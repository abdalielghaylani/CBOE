Defining an Aggregation Method

Child table fields can be added to the parent table by defining aggregation methods. Similarly, grandchild table fields can be added to its parent table (child table).
To define an aggregation method:
Drag and Drop
1. Select a field from a child table, and drag the field into the parent table on the result definition form.
2. Wait until the red line appears. The Red line indicates the position of the drop.
 

 
3. Drop the field onto the result definition form at the desired position.

 

 

 
4. Select an Aggregation method from the drop down list for the newly added child field table.

Clearing the Result Definition

To clear the Result Definition, click on the Clear form link. You will be prompted to confirm the clear result definition form. Clearing the form restores the default state of the tab when the dataview was initially selected.
