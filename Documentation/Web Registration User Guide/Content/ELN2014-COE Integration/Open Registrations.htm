Open Registrations

The Open Registrations section displays, in a table view, the list of the samples that are registered temporarily and that are waiting to be registered permanently. It also displays the samples which have already been registered outside of E-Notebook (like CBOE Manager) and are waiting to be moved to the Samples section.

You can navigate to an experiment by clicking on the Experiment ID row in the Table.

Hiding Columns

You can hide columns in the Open Registrations Table.

To hide columns in the Open Registrations Table:

  1. Right-click the column you want to hide. A right-click menu displays.
  2. Select Hide "Column Title" from the right-click menu. The column disappears.

  1. In the image below, the Sample ID column is hidden from the table.

Showing Columns

To show columns that are hidden, or are not currently displayed, in the Open Registrations Table:

  1. Right-click one of the columns in the Table of Contents. A right-click menu displays.
  2. Click Show Columns... from the right-click menu. The DBTable Columns dialog appears.
  3. In the DBTable Columns dialog, select the checkboxes corresponding to the columns that you want to show in the table. In the image below, the Sample ID column checkbox is not checked, in order to display this column in the table, simply check its corresponding checkbox.

Note that you can hide/show columns in bulk by unchecking/checking their corresponding checkboxes in the DBTable Columns dialog.

Sorting Columns

You can sort items in the Open Registrations table in either ascending or descending order.

To sort items in ascending order:

  1. Right-click the column you want to sort in the Table in an ascending order.
    A right-click menu displays.
  1. Select Sort "Column Title" Ascending.
    The items in the column you selected are sorted in ascending order.

To sort items in descending order:

  1. Right-click the column you want to sort in the Table in a descending order.
    A right-click menu displays.
  1. Select Sort "Column Title" Descending.
    The items in the column you selected are sorted in descending order.

Moving Columns in the Open Registrations Table

You can move columns in the Open Registrations table to the left or to the right by right-clicking on the column you want to move and selecting Move Left/Move Right from the right-click menu.

Deleting Registration from Dashboard

You can delete rows from the Open Registrations table by right-clicking on the row you want to remove and selecting Delete Registration from Dashboard from the right-click menu.
