Sending Data to Registration

You can register compounds in Registration Enterprise through the E-Notebook application.

To register samples in bulk from the E-Notebook application:

  1. Log in to E-Notebook.
  2. Create a Notebook and add a Discovery Chemistry or a Process Chemistry experiment.
  3. Draw a reaction in the Reaction/Process Reaction field and save your changes.
  4. Click Create Samples from the Home ribbon to create a sample.
  5. After creating samples and assigning products, click on the Sample Registration form tool and select one of the following options:
  • Quick Temporary Registration
  • Quick Permanent Registration

Note: If you have system administrator privileges, you can configure the following fields in the Properties table as read-only: Reg #, Temp ID, and Batch #. Please see the Administrator Guide for more information about setting these fields as read-only.
