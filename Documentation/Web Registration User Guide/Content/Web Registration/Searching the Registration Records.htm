Searching the Registration Records

It is possible to retrieve all the records in the permanent table or only specific records. Searches by structure, Reg Number, or by any other parameter may be run.

To search the registered compounds in the permanent table:

  1. Click the Search Registry icon, the Registration Records page displays:

  1. Click New Query button, the Search Permanent Registry page displays:

  1. Enter the search parameters and click:
  • Clear to delete the information you have entered.
  • Cancel to cancel the operation.
    Note: After you enter the search parameters and you click on Cancel button in the search form, you will be able to reload the last query you have created but you did not run by clicking on Edit Query button. If you want to create a new query from a blank form you have to click on New Query button.
  • Restore > Last Query to Form to populate in the search form the last query that you have executed.
  • Search. A list of records matching the search criteria you specified appears.

  1. Click Retrieve All to retrieve all the records in the table.
  2. Click New Query to create a new query.
  3. Click Edit Query to edit/ update the current query parameters.
  4. Click Refine Over Records to run a search over the matching results that you got from the current query.
  5. Click Manage Queries to edit, delete, restore, perform and save your queries. For more details, please refer to Managing Queries.
  6. Click Export to export the list into an SDF file.
  7. Click Print to print the results list.

Click on the right pointed arrow to display the batch(es) information:

Note: After you run a search and you get results hitlist, navigate to any form in registration other than the search form, for example, Submit a new Compound page, and then return back to the search form, you will get the last query results displayed.

You can run several types of searches, please refer to Types of Searches chapter for more details.
