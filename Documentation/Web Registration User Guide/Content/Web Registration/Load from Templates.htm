Load from Templates

Users can open saved templates using the load template option.

To load a template:

  1. Open the Submit Form.
  2. Go to Templates and select Load from Templates option

  1. The Submission Templates page displays with the list of available templates. Click on the "Load this submission template" icon to open it.

  1. The registry record is loaded. Make the required edits and click:
    • Submit to submit your record to the temporary table
    • Save as Template from the Templates dropdown list to save the template edits . An overwrite confirmation message appears. Click Yes to replace the old version with the updated one

Note: It is possible to delete a registry record template from the Submission Templates page, by clicking the delete icon corresponding to the template that you want to delete.
