Creating a registry record

Registration allows the creation of a registry record that has a single component. Pure substances consist of a single component.

Creating a Registry Record with Single Component:

To create a registry record with single component:

  1. Log in to Registration application. From the Registration page, click on Submit New Compound icon, the following page displays:


From the Submit Component page, a record may be submitted following the simple steps listed below:

  1. Entering Registry Information
  2. Entering Component Information
  3. Entering Fragment Information
  4. Entering Batch Information
  5. Clicking Submit.

Note: If ChemDraw Chemical Warning Checker is enabled and ChemDraw chemical warning is present, you will not be allowed to Submit the component until the underlying error is resolved. For more information on ChemDraw chemical warning, see ChemDraw Chemical Warning topic.
