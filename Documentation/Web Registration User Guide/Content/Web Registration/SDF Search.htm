SDF Search

Hitlists export into SDF is supported in Registration as covered in Exporting Records. Registration allows users to browse to the exported files and search for their content.

To search for an SDF content:

  1. Log in to Registration application.
  2. Click on Search Registry icon, the Registration Records page displays.
  3. Click on New Query button, the Search Permanent Registry page displays.
  4. Click on Select File corresponding to SDF Search property and browse to the SDF file that you want to search for its content or use the drag and drop feature to upload the SDF file.
