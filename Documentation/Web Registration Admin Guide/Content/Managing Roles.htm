Managing Roles

You may be able to manage roles within CBOE interface. Only the administrator user has the privilege to manage the CBOE regular user’s roles. To do that:

  1. Log in to CBOE Manager using an administrative account. Under Security section click on “Manage Roles” button as shown in the screenshot below:

  1. The following dialog displays:

Each CBOE application has its own roles, to see the roles corresponding to every application you just have to select the application from the “View Roles From” dropdown list and then the list of its roles will be displayed within the “Select a Role” field.

Adding a New Role

To add a new role; perform the below steps:

  1. Click on the “Add Role” button shown in the dialog below:

  1. The following dialog displays, enter the role name and then select the privileges that you want to assign to that role and then click on Save button.

Editing a Role

To edit a role, perform the below steps:

  1. From the “Select a Role” field, highlight the role that you want to edit, and then click on Edit button as shown in the screenshot below:

  1. The following dialog displays. Check/Uncheck the privileges that you want to go with the specific role and then click on Save button.

Editing a User

To edit the users assigned to a specific Role:

  1. In the “Select a Role” field, highlight the role that you want to edit its assigned users and then select “Edit Users” button as shown below

  1. The following dialog displays. Make your edits using the Add/Remove buttons to add or remove users who have this specific role

Editing a Role

Edit Roles option allows you to assign access for a single role to multiple CBOE applications.

For example, if you add to CSS_ADMIN role, the following two roles: ACX_BROWSER and BIOSAR_BROWSER, then if you assign to a certain user the CSS_ADMIN role, the user will have additionally the following two roles corresponding to two different CBOE applications.

To edit roles, please perform the below steps :

  1. In the “Select a Role” field, highlight the role to be edited and then select “Edit Roles” button as shown below

  1. The list of all the CBOE applications roles displays under "Available Roles" section, select the roles that you want to be associated to the selected role and then click on the arrows to add them to the "Current Roles" section. Click on Save button
