Using WebHelp Toolbar

This topic helps you going through the ChemBioOffice user's guide.

At the top of each topic in the WebHelp there is a Toolbar displayed. You may be able to specify the toolbar settings for the WebHelp and this includes determining which buttons are displayed in the toolbar.

To specify custom toolbar settings for WebHelp output:

  1. In the Project Organizer folder, Open the skin
  2. Select the WebHelp Toolbar tab
  3. Under WebHelp Toolbar Buttons, check the "Enable Custom Layout" if you want to customize the toolbar for the output. After you click this option, you can select specific features (buttons) below to include in the toolbar. You can select one or more buttons from the Available section (select multiple buttons by pressing the SHIFT or CTRL key and clicking a range or individual buttons, respectively). Then click to move the button(s) to the Selected section. (If you do not select "Enable Custom Layout," all of the available buttons will be used in the toolbar in the output).

This current topic provides you an explanation for some of the buttons that you may choose to be displayed in the WebHelp toolbar:

Add Topic to Favorites

Lets users add the active topic to the Favorites pane so that they can quickly access the topic in the future.

Toggle Navigation Pane

Lets users toggle between hiding and showing the navigation pane in the output window

Expand All

Lets users expand all elements such as togglers, drop-down effects, and expanding text effects in a topic (if they are not yet expanded)

Collapse All

Lets users collapse all elements such as togglers, drop-down effects, and expanding text effects in a topic (if they are expanded)

Print

Lets users open the Print dialog so that they can send the open topic to the printer

Quick Search

Lets users initiate a search in the active topic for the word or phrase that has been typed in the field next to the button

Remove Highlight

After a user performs a quick search in a topic, the search text found in the topic is highlighted. This button lets users turn the highlights off

Back Button

Lets users open the topic that was viewed previously

Forward Button

Lets users open the next topic in a previously viewed sequence

Stop Button

Lets users prevent a page from downloading file information

Refresh Button

Lets users update the content of the topic being viewed

Home Button

Lets users open a designated home page (such as your company's website)

Select TOC button

Lets users open the TOC feature

Select Index Button

Lets users open the Index feature

Select Search Button

Lets users open the Search feature

Select Favorites Button

Lets users open the Favorites feature
