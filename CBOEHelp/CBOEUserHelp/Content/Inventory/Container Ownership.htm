Container Ownership

Container ownership allows the owner of the container to make any modifications in the containers; any non-owner of the container can see the details but cannot modify the containers.

Creating a Container

To create a container:

  1. Login to ChemOffice Enterprise as INVADMIN or any other user.
  2. Click on Browse link to go to Inventory application.
  3. Click on New Container or Select a container Update> Edit Container.

  1. The following dialog displays with container details. Fill in the required and Optional fields (if desired) and then in the Container Admin field select an appropriate user/group based on the radio button by user/by group you select

  1. The container will be created.

Note: It is not possible to edit a container when you are not an owner. Only the following options under Other menu will be available: Lineage, Print label and View history.
