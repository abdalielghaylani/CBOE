Reporting Safety Data

There are two types of Safety Data reports:

  • At the Location Level: Print Report will print the location's containers entirely
  • At the Container level: Print Label will only print the selected container

For Safety data you can select to print the P_H sentences or the R_S sentences:

  • For P_H report (label), all safety data is reported, except the R/S sentences
  • For R_S report (label), all safety data is reported, except the P/H sentences

Print Report:

Click Print Report to print the selected location's containers entirely (at the location level).

To generate a location report:

  1. From ChemBioOffice Enterprise, Inventory Enterprise section, click Browse.
  2. Select the location you want to generate a report for.
  3. Click Print Report in the containers toolbar.

  1. Select a report layout from the drop-down list.
  2. Click Go. In the example below, Location Substance P_H Safety Report is selected.

  1. Select a report format from the drop-down list.
  2. Click Go. In the example below, Access Report Snapshot (.snp) is selected.

See the example of the following report.

About the Report:

  • Each substance occupies a separate page
  • Each substance/page displays only three symbols from Hazards and GHS Hazard tabs.

Print Label

The Print Label option prints the displayed structure only.

To print a structure:

  1. From the ChemBioOffice Enterprise, Inventory Enterprise section, click Browse.
  2. Select a location and click on the container containing the substance you want to print.
  3. From the Other menu, or from the Summary tab, click Print Label.

  1. Select the Label Type. The report generates.

Note: The Safety Data report prints in the same manner as explained in the Print Report section above.
