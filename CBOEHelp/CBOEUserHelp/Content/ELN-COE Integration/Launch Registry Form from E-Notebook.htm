Launch Registry Form from E-Notebook

It is possible for users to view the Component details in Registry system if a valid Registration Number is present. It can be accessed from Samples table under Reaction section or from Properties table under Samples section.

  1. Make sure that the Reg Number column has a valid Reg Number value under Samples or Properties table
  2. Select the CTRL key + click on the Reg Number that appears as a reference

  1. A registry form will be launched displaying the Registry Information of the selected Reg Number.
