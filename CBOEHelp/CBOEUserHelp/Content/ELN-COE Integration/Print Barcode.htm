Print Barcode

The Print Barcode Method gathers up the information on the Inventory form and attempts to create a Barcode Label. In order to print a barcode label the Container Barcode should not be null.

  1. Click “Send to Inventory” tool icon in the "Samples" section. A menu appears. Select “Open Inventory Form”.
  2. Inventory form displays showing the data associated with the corresponding sample under Samples section
  3. Select "Print Barcode" option from the "Inventory" menu

  1. An Internet browser appears

  1. Select a label template from the dropdown of label type and wait for the label to be generated

  1. The label is displayed

  1. Click the printer icon on the bottom of the toolbar, a dialog allowing the user to select a printer displays
  2. Select the printer and click on Print button

  1. Save the report somewhere on you machine
  2. Based on the printer choice you made, the barcode will be printed. In this example, we selected "Adobe PDF" printer
