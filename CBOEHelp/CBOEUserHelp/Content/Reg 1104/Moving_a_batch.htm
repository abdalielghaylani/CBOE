Moving a Batch

Users can move a batch from one registry record to another using the Move Batch option.

Note that this feature should be enabled in the system settings by the system administrator.

To move a batch:

  1. Click Move Batch. The Move/Delete Batch page appears.
  2. Select Move in the Select Action option.
  3. Specify the Batch ID and the Registry Number of the registered record to which the batch is to be moved.
  4. Click Move.
