Send Records to Inventory

If Registration Enterprise is integrated with Inventory Enterprise, users can send hits from Registration system to the Inventory system. This eliminates the need to enter the same information again in Inventory system.

To send registered records to Inventory:

  • Click Reg Number link corresponding to the record. The Registry Record page appears.
  • Click Send to Inventory.
