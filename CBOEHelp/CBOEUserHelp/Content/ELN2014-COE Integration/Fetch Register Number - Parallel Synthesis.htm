Fetch Register Number - Parallel Synthesis

Fetch Register Number lets you query the database with a registration temp ID. If a registration number is available, the registration number and batch number are returned to the associated row in the Registration Table within the PS Registry Form section.

Note: You must have Registry permissions to perform this registry action.

To fetch register numbers:

  1. Under the Selected column, check the compounds you want to fetch register numbers for.
  2. From the Home ribbon, click the Fetch Register Number form tool.

If the TempID is present in the Registration Table and the registration number is available in the Registration System, the Fetch Register Number dialog appears and displays the registration and batch numbers, as shown in the example below.

If the Temp ID is present in the Registration Table and the registration number is unavailable in the Registration System, the Fetch Register Number dialog returns: “No Compound exists with the Temp id:...” as shown below.

  1. Click OK to close the Fetch Registration Number dialog.

The registration and batch numbers are populated in the Registration Table within the PS Registry Form section.

Note: If Reg Number and Batch Number fields are already populated with data, the following message displays: “Clear the Reg Number field and Fetch again”.
