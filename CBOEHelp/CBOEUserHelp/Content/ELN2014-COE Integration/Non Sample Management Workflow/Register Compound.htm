Register Compound

With the Register Compound method, you can select a product structure from the Reaction W COE Integration section and launch a registration form that gathers the information on the registration form and attempts to submit and register a compound from E-Notebook. Upon successful registration, a Temp ID is returned to the Products table.

To submit a record:

  1. Select the Register Compound form tool from the Home ribbon.

The Select a Product Molecule dialog appears.

  1. Select a product and click OK.

The Temp ID is generated for the selected product molecule and the Submit Component form appears.

  1. Fill in the component's information and click Submit.

The Review/Register Component window appears and displays that "The temporary compound (081) was successfully submitted".

Note: The Register button is enabled only if you have the appropriate privileges assigned.

Note: You can use the Temp ID to query temporary records in the registration system.

The temporary record will be submitted in the registration system and a Temp ID is returned to the Products table of the Reaction W COE Integration section.

To register a record:

  1. Select the Register Compound form tool from the Home ribbon. The Select a Product Molecule dialog appears.
  2. Select a product and click OK. The Submit Component dialog appears.
  3. Fill in the component's information and click Register.

The Registry Record window appears and displays the registry number.

Note: You can use the registry number to query registry records in the registration system.

The compound will be registered in the registration system. A Temp ID is returned to the Products table of the Reaction W COE Integration section.
