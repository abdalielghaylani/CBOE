DataView INI settings (e.g. sample.ini)

Each ChemOffice WebServer Application can have more than one dataview. The config folder contains an ini file for the Application (cfserver.ini) and one ini file for each dataview. The dataviews that will load is determined by the list of DB_NAMES stored in the cfserver.ini. This allows enabling and disabling of data view loading. For information about the Application INI file see Application INI settings .

The Dataview files for each application are located in: <webroot>/ChemOffice/<app_name>/config
where <webroot> is your server's webroot and <app_name> in the application name.

This document discusses the sections and keys contained in a typical dataview ini file, the sample.ini file, for the Sample application. Settings for different applications may vary.  Please see an application's individual Admin guide for details.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.

When editing fields in this file, it is important to be sure what is going to change because many of these settings are necessary for the application to function properly.
