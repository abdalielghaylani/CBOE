Query History

You can retrieve a query that was executed in the current session, even if the query was not saved. The history is cleared periodically, based on the application configuration. Consult your system administrator for more information about history clearance.

To restore a query in your history:

  1. Go to History> Queries. The Queries window appears.
  2. Select the query from the drop-down list.
  3. Click Restore.
