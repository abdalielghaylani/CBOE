Adding User to a Group

To add a user to a group:

  1. Highlight the group where you want to add the user and click on the Add user icon in the right pane of the window

  1. The following dialog displays showing the available users and allowing you to add as much users as you want to the selected group

  1. Use the Ctrl+click to add multiple users and click on Save button.
  2. The following confirmation message displays. Click on Close button to complete the process.
