String Functions

  • Boolean.ToString(): Converts the Boolean type input value supplied to this function to its equivalent string representation.
  • Char.ToString(): Converts the Char type input value supplied to this function to its equivalent string representation.
  • Concat(String,String): Concatenates two instances of String type input values supplied to this function.
  • Concat(String,String,String): Concatenates three instances of String type input values supplied to this function.
  • Concat(String,String,String,String): Concatenates four instances of String type input values supplied to this function.
  • Concat(StringArray): Concatenates the values of string array supplied to this function.
  • ConvertFromUtf32(Integer): Converts the specified Unicode code point into a UTF-16 encoded string.
  • Copy(String): Creates a new instance of String using the String type input value supplied to this function.
  • DateTime.ToLongDateString(): Converts the specified DateTime type input value to its equivalent long date string representation.
  • DateTime.ToLongTimeString(): Converts the specified DateTime type input value to its equivalent long time string representation.
  • DateTime.ToShortDateString(): Converts the specified DateTime type input value to its equivalent short date string representation.
  • DateTime.ToShortTimeString(): Converts the specified DateTime type input value to its equivalent short time string representation.
  • DateTime.ToString(): Converts the specified DateTime type input value to its equivalent string representation.
  • DateTime.ToString(String): Converts the specified DateTime type input value to its equivalent string representation using the specified format.
  • Decimal.ToString(): Converts the specified Decimal type input value to its equivalent string representation.
  • Decimal.ToString(String): Converts the specified Decimal type input value to its equivalent string representation using the specified format.
  • Iif(Boolean,String,String): Evaluates the first Boolean expression; if result is True, the function returns the first string parameter else it returns the second string parameter
  • Integer.ToString(): Converts the specified Integer type input value to its equivalent string representation.
  • Integer.ToString(String): Converts the specified Integer type input value to its equivalent string representation using the specified format.
  • Intern(String): Returns the system's reference using the specified format.
  • IsInterned(String): Returns a reference to a specified string.
  • Join(String,StringArray): Concatenates a specified separator string between each element of a specified string array, resulting in a single concatenated string.
  • Join(String,StringArray,Integer,Integer): Concatenates a specified separator string between each element of a specified string array, resulting in a single concatenated string. Parameters specify the first array element and number of elements to use.
  • String.Insert(Integer,String): Inserts a specific string (the second String type value) within the first String type input value at a specified index position.
  • String.Normalize(): Returns a new string whose textual value is the same as the string supplied to this function, but whose binary representation is in Unicode normalization form C.
  • String.PadLeft(Integer): Right-aligns the characters in the specified string, padding with spaces on the left for a specified total length.
  • String.PadLeft(Integer,Char): Right-aligns the characters in the specified string, padding on the left with a specified Unicode character for a specified total length.
  • String.PadRight(Integer): Left-aligns the characters in the specified string, padding with spaces on the right for a specified total length.
  • String.PadLeft(Integer,Char): Left-aligns the characters in the specified string, padding on the right with a specified Unicode character for a specified total length.
  • String.Remove(Integer): Deletes all the characters from the specified string beginning at a specified position and continuing through the last position.
  • String.Remove(Integer,Integer): Deletes a specified number of characters from the specified beginning at a specified position. The first integer specifies the start position while the second integer specifies the number of characters to be deleted. 26 Appendix Chapter 1
  • String.Replace(Char,Char): Returns a new string in which all occurrences of a specified Unicode character in the specified string are replaced with another specified Unicode character. The first Unicode character supplied is the one that will be replaced. The second Unicode character supplied is the new character.
  • String.Replace(String,String): Involves three strings. The first string is the base string; the second string is the one whose all ocurrences in the base string will be replaced by the third string involved in this function. The function returns a new string after the replacement operation.
  • String.Substring(Integer): Retrieves a substring from the specified string. The substring starts at a position specified by the integer supplied to the function.
  • String.Substring(Integer,Integer): Retrieves a substring from the specified string. The substring starts at the character position specified by the first integer and its length is specified by the second integer.
  • String.ToLower(): Returns a copy of the specified string converted to lowercase, using the casing rules of the current culture.
  • String.ToLowerInvariant(): Returns a copy the specified string converted to lowercase using the casing rules of the invariant culture.
  • String.ToString(): Returns the specified instance of String; no actual conversion is performed.
  • String.ToUpper(): Returns a copy of the specified string converted to uppercase, using the casing rules of the current culture.
  • String.ToUpperInvariant(): Returns a copy the specified string converted to lowercase using the casing rules of the invariant culture.
  • String.Trim(): Returns a new string in which all leading and trailing occurrences of a set of specified characters from the current String type input value are removed.
  • String.Trim(CharArray): Removes all leading and trailing occurrences of a set of characters specified in a character array from the specified String type input value.
  • String.TrimEnd(CharArray): Removes all trailing occurrences of a set of characters specified in a character array from the specified String type input value.
  • String.TrimStart(CharArray): Removes all leading occurrences of a set of characters specified in a character array from the specified String type input value.
  • ToString(Boolean): Converts the specified Boolean value to its equivalent string representation (either "True" or "False").
  • ToString(Char): Converts the specified Char value to its equivalent string representation.
  • ToString(DateTime): Converts the specified DateTime value to its equivalent string representation.
  • ToString(Decimal): Converts the specified Decimal value to its equivalent string representation.
  • ToString(Integer): Converts the specified integer to its equivalent string representation.
  • ToString(Integer,Integer): Converts the first integer to its equivalent string representation in a base specified by the second integer supplied to this function.
  • ToString(String): Converts the specified string to its equivalent string representation. No actual conversion is performed.
