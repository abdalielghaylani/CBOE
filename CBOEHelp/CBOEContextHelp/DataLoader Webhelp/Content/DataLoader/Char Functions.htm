Char Functions

  • Iif(Boolean,Char,Char): Evaluates the Boolean expression; if result is True, the function returns the first Char type parameter else it returns the second Char type parameter.
  • Parse(String): Returns the character equivalent of the specified string representation of a logical value.
  • String.get_Chars(Integer): Retrieves a character from the input field of String type supplied to this function. The position of character to be retrieved is specified by the input field of Integer type supplied to this function.
  • ToChar(Boolean): Returns the character equivalent of a Boolean type input field.
  • ToChar(Char): Returns the character equivalent of a Char type input field. No actual conversion takes place.
  • ToChar(DateTime): Returns the character equivalent of a DateTime type input field.
  • ToChar(Decimal): Returns the character equivalent of a Decimal type input field.
  • ToChar(Integer): Returns the character equivalent of a Integer type input field.
  • ToChar(String): Returns the character equivalent of a String type input field.
  • ToLower(Char): Converts a character type input field to its lowercase equivalent.
  • ToLowerInvariant(Char): Converts a character type input field to its lowercase equivalent using the casing rules of the invariant culture. The invariant culture is cultureinsensitive; it is associated with the English language, but not associated with any country or region.
  • ToUpper(Char): Converts a character type input field to its uppercase equivalent.
  • ToUpperInvariant(Char): Converts a character type input field to its uppercase equivalent using the casing rules of the invariant culture.
