Retrieve Data from Saved Hit Lists - NSM

You can retrieve data from already-saved public and user hit lists within the inventory system.

To retrieve data from saved hit lists:

  1. From the Reaction W COE Integration section in the Reaction toolbar, click Add. The Select & Add Structure dialog appears.
  2. Search In: Select Inventory Search from the drop down list.
  3. Select the Show Hit Lists tab to retrieve data from already-saved public and user hit lists in the inventory system.
  4. User and Public Hit Lists: Select a hit lit from the drop down list.

  1. Click Fetch HitList Data to retrieve the hit list data you specified.

The Show Results tab appears and displays the search results.
