Manual Container Creation

To create a container manually:

  1. Create a sample and assign a product to it.
  2. Within the Reaction section, select a sample in the Samples table.
  3. From the Home ribbon, click the Inventory form tool and choose Manual Container Creation.

The Inventory Page - Manual Container Creation dialog displays. The inventory user preferences values are automatically populated under the Required tab.

  1. Fill in the remaining values in the Required tab.
  2. Fill in the values for the other tab's properties: Substance, Supplier, Contents, Optional, Owner, and Comments, if required.
  3. Click OK.

The container details are fetched from the Inventory User Preferences, unless you enter them manually. Manually-entered values override those set in the User Preferences dialog. If the mandatory fields are left blank in the User Preferences dialog, you have to enter them manually before creating the container.

The following page displays and confirms the container has been created.

  1. Click Close.

The Barcode is added to the Samples table under Reaction section and to the Sample Information SM subsection under Samples section.

Samples Table

Note: If you have system administrator privileges, you can configure the Barcode field in the Samples table as read-only. Please see the Administrator Guide for more information about setting these fields as read-only.

Sample Information SM

  1. To navigate to the container in the Inventory system, press <Ctrl> on your keyboard and click on the Barcode.

Note: You can also create containers manually under Samples section.
