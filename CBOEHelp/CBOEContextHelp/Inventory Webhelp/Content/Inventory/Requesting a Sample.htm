Requesting a Sample

From Inventory Management, you can request an entire container, or a sample of the material from that container.

To request a sample:

  1. Login to the CBOE Manager with Requestor privileges.
  2. Click Browse under the Inventory application menu.

  1. Select a location from the tree in the left frame. The list of containers in this location displays in the right frame.
  2. Select a container.

  1. From the Obtain menu, select Request.

  1. The Request Samples from an Inventory Container window appears.

  1. Request Option: Select from where the sample will be selected: Request a sample from Batch of Material or Request this specific container.
  2. Delivery Location: Click Browse to specify the delivery location from the list where the sample will be delivered to.
  3. Date Required: Use the calendar tool to enter a delivery date.
  4. Requested For: Select the requestor from the drop-down list.
  5. Amount Required: Enter the amount of the substance and select the unit from the drop-down list. The total amount of the batch content displays next to the Amount required field. The Amount Required should not exceed the maximum available amount.
  6. Proof of Approval: Click Attach the Proof of Approval to open a directory where you can select your proof of approval to attach to this order.
  7. Click OK. The following message displays.

  1. Click OK. The request displays in the Container Details Frame, as shown in the example below.

  1. If you are the requestor, click Edit in the row of the request to open the request and edit the details.
  2. If you are the requestor, click Cancel in the row of the request to cancel this request.
