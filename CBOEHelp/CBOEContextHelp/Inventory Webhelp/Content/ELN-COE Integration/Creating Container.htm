Creating Container

The Create Container Method gathers the information on the Inventory form and attempts to create a container into the Inventory system. Upon successful container creation, a Barcode Number and Compound ID will be returned to the Container Properties list of the Inventory Form.

To create a new container:

  1. Select "Open Inventory Form" from the “Send to Inventory” form tool menu under "Samples" section

  1. The “Inventory Form” appears showing the data associated with the selected product in the "Samples" section

  1. Fill in the mandatory fields and the optional fields if desired and select "Create Container" option from the Inventory form tool

  1. The container is created and the Barcode and Compound ID properties are populated automatically
