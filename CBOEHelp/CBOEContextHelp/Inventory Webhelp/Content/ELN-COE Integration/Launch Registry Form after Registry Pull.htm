Launch Registry Form after Registry Pull

It is possible for users to view the Component details in Registry system if a valid Registration Number is present .

  1. Make sure that the Reg Number column has a valid Reg Number value
  2. Select the CTRL key + click on the Reg Number that appears as a reference
  3. A registry form will be launched displaying the Registry Information of the selected Reg Number
