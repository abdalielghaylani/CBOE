FormManager

 

The COEFramework FormManager server control enables you to manage the COEFormService. Using this control, you can manage the contents of the FormTable.

 

To use the FormManager control:

 

1.       Create a Visual Studio Web Application

2.       Add the COEFramework server controls to the Toolbox.

3.       Drag the FormManager control from the Toolbox to the form.

4.       Edit the properties of the FormManager control according to your requirements.

 

Some of the commonly used properties of the FormManager control are:

  • ID: Indicates the programmatic name of the control.
  • AppName: Indicates the name of the application containing the table.
  • FieldToBeShown: Specifies the field on the basis of which you can select a record of the table.
