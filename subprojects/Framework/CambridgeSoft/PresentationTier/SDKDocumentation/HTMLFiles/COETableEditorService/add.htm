Add a new record:

 

To add a new record into a table, you need to create an object of the COETableEditorBO class, and then specify the values to the various properties, such as Name, Description, and UserName, of the COETableEditorBO object. After specifying the values to the various of the COEDataViewBO object, you need to call the Save() method of the COETableEditorBO class.

 

The following code snippet shows how to add a new record in the specified table:

 

        public void AddRecord ()

        {

            COETableEditorBO obj = null;

 

obj = COETableEditorBO.New();

 

            List<Column> colList = obj.Columns;

          

                foreach (Column col in colList)

                {

                    switch (col.FieldType)

                    {

                        case DbType.Int16:

                            col.FieldValue = Convert.ToInt16(System.Console.ReadLine());

                            break;

                        case DbType.AnsiString:

                            col.FieldValue = System.Console.ReadLine();

                            break;

                        case DbType.Boolean:

                            col.FieldValue = Convert.ToBoolean(System.Console.ReadLine());

                            break;

                        case DbType.DateTime:

                            col.FieldValue = System.DateTime.Today;

                            break;

                    }

                }

                obj.Columns = colList;

                obj.Save();

        }
