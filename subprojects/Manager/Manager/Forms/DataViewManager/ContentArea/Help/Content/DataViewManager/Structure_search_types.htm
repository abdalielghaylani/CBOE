Structure search types

If your search includes a drawn structure, you can control how ChemBioViz uses it in a query. To view the options, go to File>Preferences and click the Search tab. The basic options are:
Substructure
Returns substructure search results.
Full Structure
Returns full structure results.
Similar
Returns structures that are similar to the query structure. Use the slide bar to indicate the percent similarity.
Tautomeric
Select the tautomeric checkbox to return tautomer structures.
For more advanced queries, you can define how ChemBioViz searches for structures based on specific atom, bond, or fragment properties.
Ignore implicit H’s
Select this option to ignore implicit hydrogens. An implicit hydrogen is a hydrogen atom in which it or its bond is implied but not visible. An explicit hydrogen and its bond are visible.

Hit any charge on heteroatom
Allow uncharged noncarbon atoms in the query to match charged atoms in the target.
Charged atoms in the query must always match charged atoms in the target regardless of this setting.
Reaction query must hit reaction center
Require that any reaction center present in the query overlap with reaction centers in the target. This preference applies only to reaction searching.
Permit extraneous fragments in full structure or reaction searches.
Allow hits to contain molecular fragments in addition to the structure hit by the query.
On carbon
Allow uncharged carbon atoms in the query to match charged carbon atoms in the target.

Stereochemistry

Before you perform a stereochemical query, you must first set the search parameters:
1. Go to File>Preferences. The Preferences dialog box opens.
2. In the Preferences dialog box, click the Search tab.
3. Click Advanced.
4. In the Advanced Search Preferences dialog box, select the Stereochemistry tab.
5. In the Stereochemistry tab, select the Match stereochemistry check box.
6. Select the appropriate Tetrahedaral stereo center and Double bond options and click OK. The dialog box closes.
7. In the Preferences dialog box, click Apply.
8. Click OK to close the Preferences dialog box.
After you set the stereochemical search parameters, you can perform the query:
1. In the query form, draw the query structure.
2. In the form, click Search.
