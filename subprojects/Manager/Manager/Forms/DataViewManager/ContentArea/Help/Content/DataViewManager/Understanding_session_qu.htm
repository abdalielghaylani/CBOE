Merging Queries

You can merge two queries together to create a new query. For example, assume that you run one query for structures that have benzene ring and another query for structures that include a amino group

Viewing Results

You can view query results using either a form or table and switch between the two views as you browse the results. The Form tab and Table tab are synchronized so that the same record appears in both. The Edit tab also toggles between Edit Grid Tab and Edit Form Tab to enable you to edit the view.

Form View

The form view is useful for when you want to examine the data to be displayed using a layout of your choice for easy viewing. A form displays one record at a time. Use the record toolbar to browse through your results.

Table View

Use a table to view several records at once.
You can adjust columns and rows in a table to help you view the data more easily.

Moving columns

To move a column, click+drag the column heading to where you want it.

 

Sorting

To sort data by a particular column, right-click the column and select either Sort Ascending or Sort Descending.

Hiding columns

To hide or display a column:
1. Select Edit Grid Tab in the navigation pane. The Field Chooser window appears.
2. In the Field Chooser, select/deselect the columns you want to view.
Alternatively, you can click+drag the column heading out of the table window. To restore the column, select it in the Field Chooser window.

Resizing rows and columns

To change the column width or row height, click+drag the divider between the column or row headings.

 

Expanding record views

Some tables may have rows that can be expanded to show additional data. To expand or contract a row, click the corresponding + or - icon (tables without these icons don’t have additional data).

 

Renaming columns

To rename a column, right-click the column and select Rename header.

Magnifying Images

Details in some structural data may be difficult to see in their current size. To magnify a structure, hover your cursor over it.
