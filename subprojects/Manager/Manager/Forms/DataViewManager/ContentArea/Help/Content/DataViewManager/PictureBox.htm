PictureBox

Use a picture box to add a logo or other image to your form. To add a picturebox to your form, select the PictureBox icon in the toolbox and click+drag across the form.
Example

 

Options
To resize the picturebox, click+drag any one of its side or border handles.
To add an image, click the arrow above the box and select Choose Image.
To indicate how ChemBioViz displays the image in the box, click the arrow above the box and choose an option in the Size Mode drop-down list.
To expand the text box to the size of its container (such as a group box) select Dock in parent container.

MonthCalendar

If your data includes dates, you can display the dates in a calendar rather than a text box.
Example

 

Options
After you add the MonthCalendar field to your form, you can click+drag a corner of the field to display two or more months at a time.

SubformGrid

Although forms are designed to display data in a custom format, you may still want to display some data in a table. In this case, you use a subform grid.
Example

 

Options
To choose a data source, click the arrow above the grid and select Choose Data Source from the drop-down list.
To modify columns or to enable users to do so, click the arrow above the grid and select the appropriate option.
To expand the grid to the size of its container (such as a group box) select Dock in parent container.
