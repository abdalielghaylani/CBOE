Getting Started

When you start ChemBioViz for the first time, you will see this login window:

 

Enter your name and password as indicated. In the Server text box, either enter the name of the ChemBioViz server or select the server from the drop-down list.
If you select a name from the list, this dialog box will appear:

 

1. Enter either a service name or server address.
2. Select either 3-tier or 2-tier mode and click OK.
Each subsequent time you log in, you will need to enter only your name, password, and the server you want to use. See "Logging In".
