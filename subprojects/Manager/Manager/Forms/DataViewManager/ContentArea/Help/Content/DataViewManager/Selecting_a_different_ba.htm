Selecting a different base table

Whenever you create or edit a dataview, you can choose a different base table for the dataview.
1. In the DataView Manager, click Edit existing. The DataView Manager window opens.
2. From the list of dataviews, select a dataview to edit and click OK.
3. In the Select Tables pane, click Select Base Table. The Select Base Table pane appears.
4. In the Select Base Table pane, select a schema from the drop-down list.
5. Select a base table from the list of tables and click OK. The Select Base Table pane closes.
To save your changes:
1. In the Select Tables pane, click Submit. The Select Tables pane closes and the Validation Summary pane appears.
2. After you review the validation summary, click Submit. The Validation Summary pane closes.
3. In the confirmation window, click Home to return to the ChemBioOffice Manager.
