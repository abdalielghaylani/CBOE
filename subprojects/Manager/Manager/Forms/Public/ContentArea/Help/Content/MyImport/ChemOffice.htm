ChemBioOffice Enterprise User Guide

The ChemBioOffice Enterprise suite lets multiple users manage and share a vast resource of chemical and biological information. Users can write to and retrieve information from an extensive set of databases with an easy-to-use Web-based work area.
The ChemBioOffice Enterprise suite includes a variety of Windows and Web browser applications. In this guide, we talk about how to launch the Web applications and search databases from the ChemBioOffice Manager. For information on the Windows applications, see the ChemBioOffice Enterprise documentation library.
