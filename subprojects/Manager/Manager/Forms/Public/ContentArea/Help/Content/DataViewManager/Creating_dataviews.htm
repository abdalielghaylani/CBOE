Creating dataviews

You can create dataviews from either the master dataview or other dataviews. If there are no existing dataviews, the first one you create must be from the master.

Creating a dataview from the master

To create a dataview from the master dataview:
1. On the ChemBioOffice Enterprise Manager main page, click Create from master. The DataView Manager window opens.
2. In the Name and Description text boxes, enter a name for the new data field and a brief description.
3. In the dropdown list, select a schema from which to choose the base table.
4. Select a base table from the table list and click OK. The Select Tables window opens.
5. In the Select Tables window, select from the database tree all the tables that you want the new dataview to contain.
After you have selected that tables to include in the master dataview, you can edit the datables as needed and apply security settings for the master dataview.

Editing tables

After you select the tables for your dataview, you can edit each table as needed.
1. In the list of tables, click the name of a selected table that you want to edit. The Table Summary pane for that table appears.
2. Click Edit Table. The Edit Table and Fields pane appears.
The options are listed below.
Editing the table Alias
In the table Alias field, enter an alias name.
Creating a field alias
1. In the list of fields, select the Name for the field that you want to create an alias for.
2. Click Create Field Alias. A new field appears in the field list with the _Alias extension.
Choosing lookup field
a. In the LookUp column, click Choose for the field you want to assign as the lookup. The Edit a lookup pane appears.
b. In the left column, expand the tree and select the table to use.
c. In the right column, select the field in the Select the display field list and a Sort Direction from the dropdown list.
d. Click Submit.
Assigning a Parent Column
a. Under Parent Column, click Choose for the field you want to assign as the parent. The Define Relationships pane appears.
b. In the Current Dataview column, expand the tree and select the table to use.
c. In the Join Type dropdown list, select an option.
d. Click Submit.
Assigning an index type
In the Index Type column, select an option from the dropdown list.
Assigning a content type
In the Content Type column, select an option from the dropdown list.
Removing fields
To remove a field from the table, click Remove in the Action column.
Saving your changes
After you finished editing the table, click OK in the Edit Table and Fields pane.
Review the Validation Summary
When you finish selecting the dataview options, review the schema summary.
1. After you review your schema summary, click Submit. The Validation Summary pane opens.
Note: The validation summary displays any errors that your dataview may contain. All displayed errors must be resolved before the dataview is created. To resolve an error, click its corresponding Resolve link..
2. After you review the validation summary, click Submit. The Validation Summary pane closes.
3. Click Home to return to the ChemBioOffice Manager.
