Setting search preferences

The Preferences dialog box lets you specify the search preferences and control how results appear. These settings are global for a user throughout an application but may be different in other applications.
To set search preferences:
1. Go to File>Preferences. The Preferences dialog box appears:

 

2. In the dialog box, select from these options:
Results Display
Select either Form or List as the default for displaying search results.
Number Displayed in List View
Specify the number of records displayed per page in List View.
Specify whether charges or radicals specified on atoms in the query must match those in the target by selecting one of the following options:
Hit any Charge on Heteroatom:
Allow uncharged atoms in the query to match charged atoms in the target.
Note: Charged atoms in the query must always match charged atoms in the target regardless of this setting.
Hit any Charge on Carbon
Allow uncharged carbon atoms in the query to match charged carbon atoms in the target.
Note: Charged carbon atoms in the query must always match charged carbon atoms in the target regardless of this setting. The valence of a charged atom is taken to be the valence of the isoelectronic neutral atom.
Reaction query must hit reaction center
Specify that the reaction centers in the query must overlap the reaction centers in the target.
Permit Extraneous Fragments in Full Structure (exact) Searches
Specify exact structure searching for compounds.
Permit Extraneous Fragments in Reaction Full Structure (exact) Searches
Specify exact structure searching for reactions.
Query Fragments Can Overlap in Target
Specify whether query fragments will be used to identify targets.
Similarity search
Enter a percentage value (0-100%) to specify the degree of similarity when performing a similarity search. A lower value requires the target to be less similar.
Full structure Similarity
Search for completely similar compounds.
Reset all values, if required, by clicking the Use Defaults button.
Specify whether or not you want a structure search to consider stereochemistry by selecting on of the following check boxes:
Match Tetrahedral Stereo:
Search results will take into account tetrahedral chirality.
Note: This option greatly decreases the search speed.
Match Double Bond Stereo
Search results will take into account cis/trans double-bond geometry.
Thick Bonds Represent Relative Stereochemistry
To specify whether thick bonds will represent relative stereochemistry.
Note: Stereochemistry is perceived in the query from: hashed or wedged bonds at a tetrahedral center and unsymmetrical substitution at a C=C double bond.
