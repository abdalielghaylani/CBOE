Getting Around

To help you get started quickly, we’ll first provide a basic tour of ChemBioViz. To start ChemBioViz, go to Start>All Programs> ChemBioOffice Enterprise 2010>ChemBioViz Client.The ChemBioViz Login window appears.

Logging In

To log into ChemBioViz:
1. In the Login window, enter your user name, password, and the name of the server to which you want to login.
2. Click OK.

 

If you need a user name, password or the server name, contact your ChemBioViz administrator.

The ChemBioViz Environment

After you log in, the ChemBioViz window appears. The default layout is shown below.

 

A) navigation pane; B) record numbers box; C) tabs; D) main form.

Navigation pane

The navigation pane is on the left side of the screen and displays the main functions you use to perform a query. This is where you select the database to search, your saved queries, and the form or table in which to view your search results.

Dataviews

A dataview is a list of databases and tables that are available on a particular server. To list the database tables you can search, select Dataviews in the navigation pane.

 

An example of the dataview panel. A) a dataview; B) a parent table; C) a child table.
You can select only one dataview at a time. To modify the dataviews list, contact the ChemBioViz administrator.

Queries

The Queries tab lists all the queries you have run. You can select various queries to run them again on different databases, open hitlists to review them, or combine hitlists from multiple queries. For more information, see "Queries".

Forms

Select a form on either the Public Forms or User Forms tab to view your data. To edit a form you have chosen, select Edit Form tab. For more information, see "Forms".
