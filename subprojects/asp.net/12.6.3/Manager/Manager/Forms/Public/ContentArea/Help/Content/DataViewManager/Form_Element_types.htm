Form Element types

After you add an element, you can edit it by either changing its appearance or binding it to data. To change the appearance, select the element and go to View>Properties. The Properties window for the element opens on the right side of the form editor. You can then select the desired options in the window under the Appearance section to modify the element. Other options for the various elements are described below.

Textbox

Use a text box to display text and numbers.
Example

 

Options
To wrap text to the next line, click the arrow above the box and select the MultiLine checkbox.
To bind data to the text box, refer to the Data section in the Properties window.

Button

Use buttons to launch applications, open a Web browser, or perform other actions.
Example

 

Options
To select an action you want the button to perform, select (Advanced) in the Misc. section of the Properties window. The button properties dialog box opens.
To cause the button to perform an action, select the action from the dropdown list and enter the appropriate text in the Argument text box.
In the Button Label text box, enter the title you want to appear on the button.
(Optional) Select a variable to be included in the argument. You can enter the variable as the entire argument or include the variable in text that you type. See more in variables below.

About Variables

A variable is a place holder for data that ChemBioViz retrieves from either the database or the system. For example, if you use the variable TodaysDate, the date is displayed. To enter, select a database variable from the Data Field dropdown list or a system variable from the Variable dropdown list.
