DataView Manager

Use DataView Manager to create dataviews for ChemBioViz. A dataview is a defined set of database tables that you make available for ChemBioViz users to search. After you create a dataview and assign permissions to it, the dataview is available to ChemBioViz users. You must have administrator privileges to use DataView Manager.

The master dataview

Before you can create dataviews, you must first create the master dataview. The master dataview is the superset of all database tables from which all dataviews are made. There is only one Master dataview. Although the dataviews you create from the Master dataview are available in ChemBioViz, the master dataview is not.

Creating the master dataview

You create the master dataview by first selecting one or more schemas from your database and then selecting tables from the schemas you selected.
Selecting schemas:
1. Log into ChemBioOffice Manager as an administrative user.
2. On the ChemBioOffice Enterprise Manager main page, click Edit Master under DataView Manager. The DataView manager window opens.
3. Click Add Schema.
4. In the Database dropdown list, select a schema to add to the master dataview and enter the Password.
5. Click Publish. The schema you selected is added to the dataview.
6. (Optional) Repeat step 3. through step 6. to add more schemas.
The schemas you add to the master dataview appear on the left side of the screen.
After you add the schemas to the master dataview, you select specific tables from the schemas.

Selecting tables

To select a table:
1. In the list of schemas, click the + icon to expand the schema that includes the tables to select.
2. Click the checkbox for each table to add or click the box next to the schema name to select all of its tables.
Tip: To view a table summary, click the table name.

Submitting the master dataview

After you submit the master dataview, the master dataview is created and you can create dataviews from it.
To submit the master dataview:
1. Click either Submit button. The Validation Summary screen appears.
2. In the Validation Summary screen, review the list of tables and their relationships. If the list is acceptable, click Submit. Otherwise, click Cancel to return to the previous screen.
