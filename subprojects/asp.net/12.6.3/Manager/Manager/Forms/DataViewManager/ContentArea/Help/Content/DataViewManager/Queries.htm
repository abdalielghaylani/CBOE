Queries

To perform a basic query you simply select the database you want to search, enter your query criteria in a form, and execute the search. For more advanced queries, you can specify parameters to further control the types of results that are returned.

Selecting a database

To perform a query, you first need to select a database in the Dataviews panel. To view all the databases that are available to you, select Dataviews in the navigation pane and double-click the database. If the database you want to search is not listed, contact the ChemBioViz administrator.

Selecting a form

After you select a database, select a form from either the Public Forms or User Forms panel. The form will display your search results after you run your query. To select the form, double-click the form name.

Entering Query parameters

After you open a form, click the Query tab to enter your query parameters.

Drawing query structures

Some forms include a ChemDraw box, which lets you draw a query structure using the ChemDraw toolbar.
To draw a structure, click in the ChemDraw box. The ChemDraw main toolbar appears.
For information on how to draw structures using the ChemDraw toolbar, see the Chem & Bio Draw User Guide.
