Public Forms

Public forms are stored on a network and are available for all ChemBioViz users who have network access. This means that two or more ChemBioViz users can view data using the same form (although, they may not necessarily be viewing the same data). If you have trouble accessing a public form, contact the ChemBioViz administrator.

User Forms

User forms are stored on the server. These can be standard forms that were installed with ChemBioViz or forms that you created for your own use. These forms can be either public or private.

Private Forms

Private forms, whether stored on the network or on your local drive, are available only to you. Other ChemBioViz users cannot view or use them.
To determine whether a form is public or private, right-click the form in the navigation pane and select Properties.

Opening a form

To open a form stored on a network drive, select either the Public Forms or User Forms button in the navigation pane. Then, select the form you want.
To open a form that is on a local drive, go to File>Open local and select the form.
