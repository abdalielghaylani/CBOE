Interface Style

If you want to change the look of the ChemBioViz interface, select a new interface style. The style determines the color of the interface but not its layout.To change the style, go to File>Preferences and select the style tab.

Main Toolbar

You can choose which commands are available on the main toolbar or any of its menus.
To add a command to the main toolbar.
1. Right-click the main toolbar and select Customize. The Customize dialog box appears.
2. In the Toolbars tab, select the Main toolbar.
3. Select the Commands tab.
4. To add a command to the toolbar, click+drag an icon from the Customize pane onto the toolbar or menu where you want it.

 

For additional options, such as to display text, right-click a command icon on the toolbar.
5. Click Close, to close the Customize dialog box.
To remove a command:
1. Right-click the main toolbar and select Customize. The Customize dialog box appears.
2. In the Toolbars tab, select the Main toolbar.
3. Click+drag the command icon away from the toolbar and release the mouse button.
4. Click Close, to close the Customize dialog box.
