Entering text values

You can search for records based on all or part of the text data you want to find. For example, to find structures that include NO2 that have a molecular weight less than 150g/mole, enter NO2 in the Formula text box and <150 in the Molweight text box. If the fields you want are not in your form, edit the form as needed or choose another form.

Performing a simple query

To perform a query:
1. In the Dataview pane, select the dataview or table you want to search.
2. In either the Public forms or User forms panel, select the form you want to use in your query. The form opens.
3. Select the Queries tab.
4. Select or enter the query criteria in the form.
5. Click Search.
After your search is complete, use the Records Number box to browse through the results. Your query is stored in the Query tab.

Query Options

After you run a query, it is listed in the Queries pane. ChemBioViz provides several features you can run using this list. Right-click a query in the list and choose one of these options:
Rerun
Run the query again. ChemBioViz searches the database and returns the results that match the query parameters.
Restore Hitlist
Display the results from the previous time the query was run. A new search is not run.
Restore Query
Open the form with the search parameters entered from the previous search.
Delete
Delete the query from the Queries pane.
Rename
Enable you to change the name of the query.
Run on Opening Form
Run the query when you open a form.
