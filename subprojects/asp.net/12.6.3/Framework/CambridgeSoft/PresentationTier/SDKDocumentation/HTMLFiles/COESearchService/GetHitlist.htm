Get HitList

 

The GetHitList() method of the COESearch class of COESearchService performs a search based on the input XML files, DataView and SearchCriteria, and returns the HitListInfo object. It also saves the resulting hits into the database. 

 

The GetHitList() method contains the following parameters:

 

  • searchCriteria: Is the object of the SearchCriteria class.
  • dataView: Is the object for the COEDataView class.

 

The Search Criteria and Data View XML files are to be converted into their corresponding objects. For example, the XML file for the Search Criteria needs to be converted to the object of the SearchCriteria class. 

 

The following code snippet shows how to get the HitList information using the GetHitList() method:

 

class GetHitListTest

{

        COESearch _coeSearch = new COESearch();

 

        COEDataView dataView = new COEDataView();

        SearchCriteria searchCriteria = new SearchCriteria();

       

  

 

        // Temp Path to xml file

  string tempPath =  Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData) + @"\" +    Application.ProductName + @"\";

     

        // current file path to be displayed by the controls

        public void TestGetHitlist ()

        {

            // Build DataView

            dataView = new COEDataView();

            BuildDataViewFromXML(ref dataView);

 

            // Build SearchCriteria

            searchCriteria = new SearchCriteria();

            BuildSearchCriteriaFromXML(ref searchCriteria);

 

 

            try

            {

                this.Cursor = Cursors.WaitCursor;

                HitListInfo hitlistInfo = new HitListInfo();

    hitlistInfo =   _coeSearch.GetHitList(searchCriteria, dataView);

 

            }

            catch (Exception ex)

            {

                HandleException(ex);

            }

            finally

            {

                this.Cursor = Cursors.Default;

            }

        }

 

 

 

  private void BuildDataViewFromXML(ref COEDataView dataView)

        {

            XmlDocument doc = new XmlDocument();

            doc.Load(tempPath + “DataView.xml”);

            dataView = new COEDataView(doc);

           

        }

 

private void BuildSearchCriteriaFromXML(ref SearchCriteria searchCriteria)

        {

            XmlDocument doc = new XmlDocument();

            doc.Load(tempPath + “SearchCriteria.xml”);

            try

            {

                searchCriteria = new SearchCriteria(doc);

            }

            catch (Exception ex)

            {

                HandleException(ex);

            }

        }

 

}

 

 

The above code snippet defines GetHitList class that contains the following methods to get the HItList information:

 

  • TestGetHitlist: This method calls the GetHitList() method of the COESearch class of the COESearchService, which returns the HitListInfo object containing the HitList Information.
  • BuildDataViewFromXML: This method converts the DataView XML file into the DataVeiw object.
  • BuildSearchCriteriaFromXML: This method converts the SearchCriteria XML file into the SearchCriteria object.
