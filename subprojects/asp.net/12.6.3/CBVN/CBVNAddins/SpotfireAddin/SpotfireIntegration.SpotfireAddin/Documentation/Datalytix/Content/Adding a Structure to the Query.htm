Adding a Structure to the Query

To add a structure to the query:
1. Select the Structure Field from the drop down.
2. Select the Search Type from the drop down list. By default, Substructure is selected.
3. Double click in the Structure pane to open the Structure Editor. By default, the ChemDraw Structure Editor opens.
 

 

Note: To select a different Structure Editor, close the ChemDraw Structure Editor, and click on the Edit button in the toolbar. A list of available editors is presented (if applicable). The selection is then saved as a user preference, and will be opened by default the next time you double click on the Structure pane.
4. Draw the query structure in the Structure Editor window using appropriate tools.
 

Note: You can also copy and paste a structure into the Structure Editor window. For example, a structure can be copied from the Structure Viewer, a table, or the Details-on-Demand panel, and paste it into the Structure Pane.
5. Click OK. The structure will be displayed in the Structure pane. If Similarity is selected, use the slider to specify the degree of similarity.
