Logging Off

You are able to log off the ChemBioOffice Enterprise Server that Datalytix is connected to without having to exit Spotfire.
Note: The Log Off link is only visible in the non-default configuration when the Use Remoting preference is set to False. It is NOT available if the Use Remoting preference is set to True (default configuration). Refer to the Setting Datalytix Preferences section.
1. Click on the Log Off link in the top right hand corner of the Datalytix dialog.
