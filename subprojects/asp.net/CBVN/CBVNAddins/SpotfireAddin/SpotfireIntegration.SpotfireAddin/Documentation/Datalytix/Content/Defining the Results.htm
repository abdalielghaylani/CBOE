Defining the Results

About the Results DefinitionTab

Through the Query tab, Datalytix allows you to define what tables and data (rows of data) should be passed into Spotfire for additional analysis.
Via the Results Definition tab, you can define the Spotfire dataset in which the query results will be displayed.
 

 
Tables and fields are added and removed to/from the result definition form in the same manner as they are added and removed to/from the query form.
The tables and fields defined here are then imported and displayed in Spotfire tables for further analysis.
Via the Result Definition tab, you can:
• Define Spotfire dataset by adding/removing tables and fields to the result definition
• Define sorting of the results
• Define aggregation methods to add child tae fields to the parent table
