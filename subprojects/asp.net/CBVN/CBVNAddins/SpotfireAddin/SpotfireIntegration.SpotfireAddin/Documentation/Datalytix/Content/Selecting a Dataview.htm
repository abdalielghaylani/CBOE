Selecting a Dataview

To select a dataview:
1. From the Dataview drop down list, select a dataview.
 

Note: The logged in user can only see dataviews to which they have permissions to view. The assignment of specific security settings for a dataview is done through the Dataview Manager.
All visible Tables and Fields for the selected dataview are displayed in the left Tree panel. The tree panel displays a hierarchy of tables (i.e. base/parent table, child table, and grandchild table).
Each table/child table contains data. The tree can be expanded/collapsed as required.
 

 

Searching Tables and Fields

The Search Table and fields text box (located above the tree panel) supports ‘smart search’ and an abbreviated list of Table/Field Name matches is displayed for selection as you are typing.
