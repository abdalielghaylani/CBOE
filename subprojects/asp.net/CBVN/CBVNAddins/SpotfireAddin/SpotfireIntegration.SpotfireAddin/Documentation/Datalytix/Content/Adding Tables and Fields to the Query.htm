Adding Tables and Fields to the Query

Tables and fields are added to the query by:

• Drag and drop any table or field from the navigation tree.
• Right mouse click on any table/field from the navigation tree.
• Double click on field in the navigation tree.

When adding a table to the query, you can select to include:

• Default Fields
• All Fields
• Unique Key and Structure
These options are available for selection from the ‘When adding a table include’ drop down list located below the Available Tables and Fields pane. Alternatively, when adding a table, you can right mouse click on the table name to see these same options.
 

To add tables to the query

Drag and Drop
1. Select a table on the navigation tree.
2. From the When adding a table include drop down list, select to include Default Fields, All Fields, or Unique Key and Structure
3. Drag the table onto the query form.
4. Wait until the red line appears, The Red line indicates the position of the drop.
 

 
5. Drop the table onto the query form at the desired position.

 
Right Context Menu
1. Right mouse click on the table in the navigation tree.
2. Select Default Fields, All Fields, or Unique Key and Structure to identify what fields to include.
The table will be appended to the bottom of the query form.
 

To add fields to the query

Drag and Drop
1. Select a field on the navigation tree. Drag the field onto the query form.
2. Wait until the red line appears, The Red line indicates the position of the drop.
 

 
3. Drop the field onto the query form at the desired position.
 

 
Right Context Menu
1. Right mouse click on the field in the navigation tree.
2. Select Add.
The field will be appended to the bottom of the query form for the respective table.
Note: If the table to which the field belongs is not already part of the query form, it will be appended to the query form along with the field.
Double clicking on a field in the navigation tree
1. Double click on the field in the navigation tree.
The field will be appended to the bottom of the query form for the respective table.
Note: If the table to which the field belongs is not already part of the query form, it will be appended to the query form along with the field.

To remove tables/fields from the query

1. Drag and drop any field or table from the Query tab to the left Tree panel. The table/field will be removed with the Query form and returned to the navigation tree.
OR
2. Right mouse click on the table or field in the query form and select Remove Table or Remove Field respectively. The table/field will be returned to the navigation tree.
Note: The base table cannot be removed from the Query.
