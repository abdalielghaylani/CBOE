Performing an Advanced Search on a Dataview

You can perform an advanced search of the dataviews using Tag name and Table names.

Clicking on the Browse button located to the right of the Dataview drop down list will open the Browse Dataview window. By default, searching by Table Name is selected.

 

 

Advanced Search using Table Name

To perform an advanced search using Table Name:
1. From the Advance Search group box, select the Table Name radio button.
2. Enter the Table Name in the text box. The text box supports ‘smart search’ and an abbreviated list of Table Name matches is displayed for selection as you are typing.

 

 
3. Click on the Find button to retrieve the list of dataviews containing the specified Table Name. The
corresponding dataviews are displayed in the drop down list in the Dataview Summary group box.
 
4. Select a Dataview. Click OK.
5. The Browse Dataview window closes, and the selected Dataview is displayed in the main Datalytix window as shown in the following example.
 

 

Advanced Search using Tag Name

Dataview Manager allows user to define tagging categories on tables in the metadata of a Dataview so this information can be used when searching for tables. A particular table may have more than one tag.
Dataytix allows you to filter the list of dataview by the Table Tags assignment in the dataview. A tag from any table is the Dataview is valid. In addition, the resultant list of dataviews can be based on only tags present in the base table, or can include child tables.
 
To perform an advanced search using Tag Name:
1. From the Advance Search group box, select the Tag Name radio button.
2. Enable the Include Child Tables checkbox if you want to include child table tags in the dataview search.
3. Enter the Tag Name in the text box. The text box supports ‘smart search’ and an abbreviated list of Tag Name matches is displayed for selection as you are typing.

 

 
4. Click on the Find button to retrieve the list of dataviews containing the specified Tag Name. The corresponding Dataviews are displayed in the drop down list in the Dataview Summary group box.
5. Select a Dataview. Click OK.
6. The Browse Dataview window closes, and the selected Dataview is displayed in the main Datalytix window.
 

Retrieving all Dataviews

From the Browse Dataview window, all available dataviews can be retrieved by clicking on the Retrieve All button.

Viewing Dataview Summary

A summary of a selected Dataview can be seen in the Dataview Summary section. It provides details of the dataview including Dataview Name, Description, User and the date it was created.
