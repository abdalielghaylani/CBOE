Defining a Query

About the Query Tab

Datalytix allows you to define what tables and data (rows of data) should be passed into Spotfire for additional analysis. This is accomplished via the Query tab on the Datalytix main screen.
 

 
Via the Query tab, you can:
• Add/remove tables and fields to the query
• Define query parameters
• Define structure criteria for the query
