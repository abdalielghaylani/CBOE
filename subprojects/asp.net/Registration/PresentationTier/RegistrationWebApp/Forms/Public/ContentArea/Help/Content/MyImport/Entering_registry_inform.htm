Entering registry information

Registry information consists of these fields:
Prefix- The prefix format is defined by the administrator.
Projects- A Registry Record can be associated with one or more Projects
Identifiers- The identifiers associated with the project.
Registry Comments- Any comments about the substance.
The administrator defines the Projects, Identifiers, and the Prefix Format.
1. Fill in Registry Information by selecting values in the fields Projects, Identifiers, and Prefix. If the registry record belongs to more than one project, click Add Project and select the required values. If the registry record is assigned to more than one identifier, click Add Identifier and select the required identifier type.
The administrator defines the Projects, Identifiers, and Prefix. For more information, contact your Administrator.
2. In the Registry Comments field, enter any comments that you would like to associate with the record.
