Entering component information

Component information consists of fields:
Identifiers- The component identifiers describe the pure substance. The default component Identifiers are:
Chemical Name
CAS Number
Synonym
Draw a structure in the window. The options are:
Drawn Structure(Above) is selected by default.
Select No Structure if the component does not have a structure associated with it.
Select Unknown Structure if you don’t know the structure and you want to keep the structure field blank. You can later add the structure to the record.
Select Non Chemical Content to paste a graphic image in the structure box. To paste a graphic, right-click the structure box and select Edit>Paste from the context menu. The image appears in the structure box.
Component Comments- Comments associated with each component.
Stereochemistry Comments- Any stereochemistry remark.
