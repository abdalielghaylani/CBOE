Creating a registry record

You can create a registry record that has either a single component or multiple components. Pure substances consist of a single component whereas mixtures consist of multiple components.
To create a registry record with single component:
Click Submit new record in the Registration home page or in the ChemBioOffice Manager home page under Registration Enterprise. The Submit Record page appears.

From the Submit Record page, you can submit a record in four simple steps by:
1. Entering registry information
2. Entering component information
3. Entering fragment information
4. Entering batch information
5. Clicking Submit.

Entering registry information

Registry information consists of these fields:
Prefix- The prefix format is defined by the administrator.
Projects- A Registry Record can be associated with one or more Projects
Identifiers- The identifiers associated with the project.
Registry Comments- Any comments about the substance.
Note: The administrator defines the Projects, Identifiers, and the Prefix Format.
1. Fill in Registry Information by selecting values in the fields Projects, Identifiers, and Prefix. If the registry record belongs to more than one project, click Add Project and select the required values. If the registry record is assigned to more than one identifier, click Add Identifier and select the required identifier type.
Note: The Administrator defines the Projects, Identifiers, and Prefix. For more information, contact your Administrator.
2. In the Registry Comments field, enter any comments that you would like to associate with the record.

Entering component information

Component information consists of fields:
Identifiers- The component identifiers describe the pure substance. The default component Identifiers are:
Chemical Name
CAS Number
Synonym
Draw a structure in the window. The options are:
Drawn Structure(Above) is selected by default.
Select No Structure if the component does not have a structure associated with it.
Select Unknown Structure if you don’t know the structure and you want to keep the structure field blank. You can later add the structure to the record.
Select Non Chemical Content to paste a graphic image in the structure box. To paste a graphic, right-click the structure box and select Edit>Paste from the context menu. The image appears in the structure box.
Component Comments- Comments associated with each component.
Stereochemistry Comments- Any stereochemistry remark.

Entering fragment information

When a component is a chemical compound, only the base structure is drawn on the Structure box and the rest of the structure compound information is filled in the Fragment list. Fragment Information differs from one batch to another. Salts and Solvate are the most typical example of Fragment types
Note: The administrator defines the fragment types.
3. Fill in Fragment Information by selecting the Fragment Type, and Code. The fragment type could be a be salt, solvate or any other fragment type defined by the administrator. A component, as in the case of hydrated salts, may have both salt and solvate as fragment types. To select more than one fragment type, click New Fragment. Select the code based on the fragment type selected and enter the equivalents number for each fragment. All other fields under Fragment Information is filled automatically. For more information on defining new fragment types and adding new fragments, refer to the Registration Enterprise Administrator Guide.

Entering batch information

Batch information consists of the following fields:
Synthesis Date - Date on which the batch was created.
Chemist- Scientist who synthesized the batch.
Synthesis Date- Date on which the batch was synthesized
Notebook Reference- The reference to the chemist notebook that contains the batch synthesis information
Amount- Substance quantity obtained for this batch
Units- Units of the amount obtained.
Appearance- Visual appearance of the substance. For example, clear yellow liquid.
Purity-For example, 96.5.
Purity Comments
Sample ID
Solubility
Batch Comments
Storage Requirements and Warnings
1. Fill in the Batch Information by selecting values in the fields Chemist, synthesis Date, Units, and other fields mentioned above.
Click Submit. The record is added to the temporary table and the Submit Record window displaying the temporary record ID appears. Click Done.

Submit records with multiple components

You can create registry records with multiple components by either adding a new component or selecting a component already registered in Registration.
To create a registry record with multiple components:
1. Follow the steps of entering registry, component, fragment, and batch information for creating a record with single component.
2. You can either select a component that is already registered or add a new component.
To select an existing component, go to Manage Components>Pick an existing Component to add a registered component. The Select Component form appears. Enter the search criteria and click Search. The component information that meets the search criteria appears. Click Add Component corresponding to the component you want to select. The component is added and the window displaying the component information appears.
To add a new component, go to Manage Components> Add a new component. A new component icon appears in the New Registry tree structure on the left-hand side of the window.

 

Fill in the fields under Component Information and Fragment Information.
3. Go to Manage Components> Add a new component to add more components. To delete any component you have added, go to Manage Components> Remove Component.
4. Go to Manage Components> Complete batch information and submit when you are done adding components.
5. Specify the Batch Component Custom Properties.
6. Click Submit to add the record to the temporary table. The record is added to the temporary table and the temporary record ID appears. To view the component details, click the corresponding component in the New Registry tree structure.

Options in submit record form

Once you fill in the Submit Record form, you can not only submit the record to the temporary table but also save the form and use it later. You can also register the record directly to the permanent table if you have adequate privileges.
Options available in the Submit Record form include:
Save Template
Load Template
Submit
Register
Cancel

 

Save Template

You can save the record as a draft for later use. For example, assume you are awaiting analytical information to determine the purity of a component you want to submit and have filled in all other fields in the form. You can save the form and add the information when it is available.
To save a record as template:
1. Go to Submission Templates > Save Template. The Save Template form appears.
2. Enter a name for the record in the Template Name field and a description in Template Description field.
3. Select Public Template if you want to make the record available to other users.
4. Click Save Template.

Load Template

You can open saved templates using this option.
To load a template:
1. Go to Submission Templates > Load Template. The Load Template form displaying the list of available templates appears.
2. Double click on the template you want to open. The record information is loaded.

Submit

Use this option to submit records to the temporary table. For more information on submitting records, see "Creating a registry record".

Cancel

Use this option to exit from the Submit Record form and go to the main menu of Registration Enterprise.
