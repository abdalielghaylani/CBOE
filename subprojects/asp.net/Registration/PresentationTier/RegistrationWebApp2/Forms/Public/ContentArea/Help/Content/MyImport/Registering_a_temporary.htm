Registering a temporary record

Registration assigns a unique number to each record that is registered. when you register a batch, the batch number is added to the registration number previously assigned to the compound. In addition to registering each record individually, Registration Enterprise also lets you register all the temporary table records at once.
Records in the permanent table can be accessed only if Row Level Security is not enabled.

Registering a compound record

You can register a record by either selecting Review Register in the Registration home page or by searching for records in the temporary table using Search Temp in the Registration home page. On selecting Review Register, the Search Temporary Record form displaying all records in the temporary table appears.
You can register one record at a time or several records using bulk registration.
To register a single record:
1. Click the Review Record link corresponding to the record. The temporary record page that displays the record information appears.
2. To edit the record information, click Edit. Click Submit to save any changes made.
3. Click Register. The record is registered and the Registry Number appears.as shown below:

 

4. Click Done.
To perform bulk registration:
1. Mark the records to register.
2. Click Register Marked. The Bulk register marked page appears.
3. Choose the duplicate action to perform:
Duplicate
register as a new compound any duplicate record it encounters.
Temporary
register all records except duplicates. The duplicate records remain in the temporary table.
Batch
lets you add new batches for the duplicate compounds.
None
leave the record in temporary table.
4. Click Register Marked.
