Saving queries

If you plan to use the same search criteria over and over again, consider saving the query so that you don't need to enter the search criteria each time.
To save a query:
1. After performing a search, go to Query>Save.
2. Enter a name and description for the query.
3. Select Is Public if you want the query to be available to others.
4. Click Save.
