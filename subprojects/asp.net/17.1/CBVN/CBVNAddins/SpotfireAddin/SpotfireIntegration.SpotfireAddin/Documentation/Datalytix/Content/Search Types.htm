Search Types

The following structure query types are supported:
• Full Structure
• Substructure
• Similarity
A Full structure search finds a particular structure.
A Substructure search finds structures containing a common structural component, plus any additional attachments at the open positions.
Similarity searches find structures having features similar to the query structure. By default, a similarity search is performed based on a 90% degree of similarity. You can adjust the degree of similarity (from 0 to 100%) using the slider. The higher the value, the fewer hits found.
