Defining Structure Criteria

As part of your query, you can include a structure. Only one structure field can be used in a query. Criteria for the structure are defined through the Structure Criteria panel.
The Structure Filter is used to:
• Draw a chemical structure to include in the query.
• Query based on full structure, substructure or similarity.
The key elements of the Structure Criteria panel are:
• Structure Fields drop down list
• Structure Criteria Toolbar
• Search Type drop down list
• Structure pane
