List view

The list view lets you view multiple search results simultaneously. The list view does not display all the fields available for a record. You can specify the number of records to be displayed per page by specifying record count in the Preferences tab. If the number of hits exceeds the record count specified, browse to the next page to view the rest of the records. List View is the default view set. You can
change the default view by selecting the
required view in the Preferences tab.
