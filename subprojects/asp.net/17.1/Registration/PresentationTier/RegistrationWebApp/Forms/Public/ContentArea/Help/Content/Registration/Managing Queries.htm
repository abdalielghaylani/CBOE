Queries and Hitlist

You can use queries to search for records in temporary and permanent tables in Registration.
Registration Enterprise lets you search by any fields available in the search form.

Viewing search results

You can see your search results in either List View or Form/Details View.

List view

The list view lets you view multiple search results simultaneously. The list view does not display all the fields available for a record. You can specify the number of records to be displayed per page by specifying record count in the Preferences tab. If the number of hits exceeds the record count specified, browse to the next page to view the rest of the records. List View is the default view set. You can
change the default view by selecting the
required view in the Preferences tab.

Form view

The form view displays all the information for a particular record. To display a record in form view, click the View Record link corresponding to that record.

Marking records

Marking records lets you view the records without creating a search query for them again. However, keep in mind that, as the database is updated, your list of marked records isn't automatically revised. As a result, you may need to periodically rerun your search to ensure that your list of marked records remains accurate. However, when a marked record is updated, you can see the update when you open the marked record. You can mark the records in both list view and form view. Marked hits are user specific and can be retrieved at any time using Show Marked option under Marked menu.
To mark a record, select the check box to the left of the corresponding record in list view. After marking the record, you can view or unmark the record or export the results to other linked applications like Inventory Enterprise.
To save marked search results, go to Marked>Save. Specify a name and description for the list. Select Public, if the marked list is to be available for everyone using registration enterprise.
To mark all the records displayed in list view, click Mark All.
To unmark all marked records displayed, click Un Mark All.
