COEHitListManager

 

The Search results of the search operation using COESearchService are persisted to a database as a HitList.  The HitList gets saved as Temp HitList in the COETEMPHITLIST table for the certain number of days.  If the user explicitly saves a HitList, it is saved as a Saved HitList.

 

 

The COEFramework HitListManager server control enables you to manage the HitLists. Using this control, the following operations can be performed on the HitLists:

 

  • Edit the HitList.
  • Perform various operations, such as Intersection, Subtraction, and Union, on the HitLists.
  • Delete the HitList.
  • Save the TEMP HitList as SAVED HitList..

 

To use the COEHitListManager control:

 

1.       Create a Visual Studio Windows or Web Application.

2.       Add the COEFramework server controls to the Toolbox.

3.       Drag an HTML button onto the Web Page and enter the following code in the click event of the button:

 

debugger;

    javascript:window.open('Default.aspx?CurrentHitListID=123&CurrentHitListType=TEMP','mywindow','width=600,height=500');

 

 

 

4.       Add another page and drag the COEHitLIstManager control from the edit the properties of the DataViewManager control according to your requirements.

 

Some of the commonly used properties of the DataViewManager control are:

 

  • ID: Indicated the programmatic name of the control.
  • AppName: Indicates the name of the application containing the table.

 

 

Working with COEHitListManager Control

 

The COEHITListManager server control contains three tabs, Manage, Restore, and Save.

 

The Manage tab enables to manage any HitList or the Current HitList. Managing the HitList includes the following tasks:

 

  • Edit the HitList
  • Performing operations, such as subtraction, intersection, and union on the HitLists.
  • Delete the HitList

 

The Restore tab enables you to set a HitList as current Hitlist. The details of the HitList, which is set as Current HitList using Restore tab are displayed under Saved tab.

 

The Save tab enables you to save the Current HitList as Saved HitList.
