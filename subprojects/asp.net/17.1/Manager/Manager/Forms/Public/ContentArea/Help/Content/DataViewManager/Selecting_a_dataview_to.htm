Selecting a dataview to edit

1. Log into ChemBioOffice Manager as an administrator.
2. In the DataView Manager, click Edit existing. The DataView Manager window opens.
3. From the list of dataviews, select a dataview to edit and click OK.

Managing tables

To add or remove tables from the dataview:
1. In the left pane, expand the tree to show the list of tables in the schema.
2. Select/deselect the tables you want available to the user.
To edit a table:
1. In the left pane, select the name of the table.
Note: Select the name of the table, not its checkbox.
2. In the table summary, click Edit Table.The Edit Table and Fields pane appears.
3. In the Edit Table and Fields pane, select the appropriate options as desired and click OK. See "Editing tables" for the list of available options.
To save your changes:
1. Click Submit.The Validation Summary pane opens.
2. After you review the validation summary, click Submit. The Validation Summary pane closes.
3. In the confirmation window, click Home to return to the ChemBioOffice Manager.

Specifying security settings

You can choose which ChemBioViz users can access the dataview. You can select specific users and roles. When you select a role, all users assigned to that role have access to the dataview.
1. In the DataView Manager, click Edit existing. The DataView Manager window opens.
2. From the list of dataviews, select a dataview to edit and click OK.
3. In the Select Tables pane, click Security Settings.
4. Select either to make the dataview public or available to only specific users or roles.
5. If you choose to make the dataview public, click OK. The security is set and you are returned to the Select Tables screen.
6. If you choose to set users and roles for the dataview, the lists of available users and roles appear.
a. Select the desired user name(s) in the Available Users column and click the Add User icon in the Action column. The user(s) you selected appear under Current Users.
b. Select the desired role(s) in the Available Roles column and click the Add Roles icon in the Action column. The role(s) you selected appear under Current Roles.
c. Click OK.The security is set and you are returned to the Select Tables screen.
To save your changes:
1. Click Submit.The Validation Summary pane opens.
2. After you review the validation summary, click Submit. The Validation Summary pane closes.
3. In the confirmation window, click Home to return to the ChemBioOffice Manager.
