Managing queries

The criteria you enter in the query input form to perform a search (a drawn structure, molecular name, etc.) is collectively called a query. You can restore, save, and delete queries that you have used to search data.

Saving queries

You can save queries so that you don’t have to re-enter the criteria each time you perform a search. You just restore the query whenever you need to search for the data. See "Restoring queries".
To save a query:
1. Perform a search. For more information, see "Searching databases".
2. Go to Queries>Save. The Queries window appears:

 

3. Enter a Query Name and Description for the query.
4. To enable others to view the query, select the Make Public check box.
5. Click OK The query is saved.

Restoring queries

You can restore a saved query and re-rerun it to search for data.
To restore the last query:
1. Go to Queries> Restore>Last. The Query Input Form appears with the fields populated with the saved query criteria.
2. (Optional) Edit the criteria if needed.
3. Click Search. The search results are displayed.
To restore a saved query:
1. Do one of the following:
Go to Queries>Restore>Saved.The Queries window appears.
Go to Queries>Manage. The Queries window appears.
2. Select the query from the drop-down list in the window.
3. Click Restore.

Editing query

You can enter new identifier information for a query, such as a new name and description.
To edit a query:
1. Go to Queries>Manage. The Queries window appears.
2. Select the query from the drop-down list in the window.
3. Click Edit. The Queries window appears.
4. (Optional) Modify the Query Name and Description fields.
5. (Optional)To enable/disable others’ ability to view the query, select/deselect Public Query as needed.
6. Click OK.
7. Click Close to close the Queries window.

Deleting a query

Only saved queries can be deleted. Queries in the history remain until the history is cleared. To learn more about clearing the history, contact your system administrator.
To delete a query:
1. Go to Queries>Manage. The Queries window appears.
2. Select the query from the drop-down list in the window.
3. Click Delete. A message box appears, prompting you to confirm the deletion.
4. Click OK to delete the query.

Query history

You can retrieve a query that was executed in the current session, even if the query was not saved. The history is cleared periodically, based on the application configuration. Consult your system administrator for more information about history clearance.
To restore a query in your history:
1. Go to History> Queries. The Queries window appears.
2. Select the query from the drop-down list.
3. Click Restore.
