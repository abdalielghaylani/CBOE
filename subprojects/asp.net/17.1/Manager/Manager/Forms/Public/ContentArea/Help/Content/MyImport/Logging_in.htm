Logging in

To use the online applications, log into the ChemBioOffice Enterprise Manager.
To log in:
1. In your Web browser address bar, type http://servername where servername is the server where ChemBioOffice Enterprise is installed. The ChemBioOffice Enterprise login page appears:

 

Note: If you do not know the name of the server, contact your server administrator.
2. Enter your User Name and Password.
3. Click Log in. The ChemBioOffice Enterprise Manager page appears:
You will remain logged in for up to 25 minutes, even if you close the Web browser. To end your login session, click Log Off.
