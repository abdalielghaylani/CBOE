Record numbers box

After you select a dataview or run a query, use the Record numbers box to view the records. To browse from one record to another, click the left and right buttons. To view a specific record, enter its number in the box and press ENTER.

 

Tabs

When you start ChemBioViz for the first time, you will see three tabs, Form, Table, and Query.

Form

Choose the Form tab to view data using a form that you have chosen. Any blank forms or form copies that you create will also appear as tabs. To choose a form, select in the navigation pane either a public form or one of your own user forms.

Query

Select the Query tab to enter criteria to search the database you chose in Dataviews. To be returned as a search result, a record must meet all criteria you enter (see "Queries").

Table

Select the Table tab to view your data in a tabular format instead of a form. The Table and Form tabs are synchronized such that, as you navigate to a particular record in one tab, the record also appears in the other tab.
