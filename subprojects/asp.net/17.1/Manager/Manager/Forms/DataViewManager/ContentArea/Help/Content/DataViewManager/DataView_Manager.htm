DataView Manager

Use DataView Manager to create dataviews for ChemBioViz. A dataview is a defined set of database tables that you make available for ChemBioViz users to search. After you create a dataview and assign permissions to it, the dataview is available to ChemBioViz users. You must have administrator privileges to use DataView Manager.
