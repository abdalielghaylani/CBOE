Exporting search results

You can save a hit list as an SD or RD file. The structure, molecular weight, and formulae are saved by default but you can choose which fields to save.
When you export to a file, you use a template to define which fields are included. Always use the same template to ensure that you always select the same fields each time you export.
To export search results or hit lists:
1. Go to File>Export Hits. The Export Hits window appears.
2. Select a type of SD file:
Flat
Select this type to create multiple results for one field (a record is created for each result).
Nested
Select this type when multiple results for one field all appear in one cell.
3. Choose the template from the Export Templates drop-down list.
4. (Optional) If needed, select/deselect the fields under Export.
5. Click OK. A new window opens.
6. Click Click to Download. The File Download dialog box appears, containing a numerical name for the file, based on the date and time.
7. Click Save to display the Save As dialog box.
8. Select a location.
9. Click Save.
The exported file flattens each record so that a compound with subform fields, such as multiple batches in the Registration Enterprise is found within one record in the exported file. All subform data is concatenated with '--' between fields. The first line indicate the field names.

Creating an export template

To create a template:
1. Go to File>Export Hits. The Export Hits window appears.
2. In the drop-down list, select Create New Template. A new window appears.
3. In the new window, enter the Template Name and a Template Description.
4. To enable others to view the query, select Public.
5. Select the fields to export when using the template.
6. Click OK.

Printing

To print a hit list, go to File>Print.
