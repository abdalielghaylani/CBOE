Managing roles

The Security Manager lets you manage user roles for all ChemBioOffice Enterprise 2010 applications. To open the Security Manager, log into the ChemBioOffice Enterprise Manager and select Manage Roles in the Security section. The Security Manager window opens.

Viewing Roles

To view the roles associated with an application, select the application from the View Roles From drop down list in the Security Manager window. The roles associated with the application appear in the Select a Role list box.

Adding a new role

To add a new role to an application:
1. Select the application in the View Roles from drop down list.
2. Click Add Role. The Edit Role page appears.
3. Specify the name of the new role. You can either type the name in the Role Name text box or click Select Oracle Role to add a role from Oracle.
4. Click Save.

Eding roles

To edit a role:
1. Select the ChemBioOffice Enterprise 2010 application in the View Roles From drop down list.
2. Select the role in the Select a Role list box and click Edit Roles. A page displaying the list of available roles appears.
3. Click the left and right arrows to assign one or more Current Roles from the list of Available Roles.
4. Click Save.

Assigning users to Roles

To assign one or more users to a role:
1. Select the appropriate application from the View Roles From drop down list in the Manage Roles window.
2. Select the role in the Select a Role list box and click Edit Users. The Edit Role Users window appears.
3. In the Avaialble Users list, select the user name to assign to the role and click the right arrow. The name appears in the Current Users list.
Note: Use CTRL+click to select more than one name.
4. To remove a name from the role, select the name in the Current Users list and click the left arrow.
5. Click Save when finished.

Deleting a role

Before deleting a role, you must ensure that no user is assigned to it.
To delete a role:
1. Select an application from the View Roles from drop down list in the Manage Roles window.
2. Select the appropriate role in the Select a Role list box and click Delete. Click OK when prompted.
