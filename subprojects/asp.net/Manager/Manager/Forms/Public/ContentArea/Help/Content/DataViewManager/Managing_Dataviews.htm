Managing Dataviews

Editing the master dataview

You can delete or edit tables in the master view schema. Schemas are password protected.
1. Log into ChemBioOffice Manager as an administrator.
2. In the DataView Manager menu, click Edit Master. The DataView Manager window opens.
3. Click Add Schema.
4. In the Database dropdown list, select a schema.
5. Enter the Password for the schema and click Publish.

Selecting tables

After you select a schema, the Select Tables pane appears, which you use to select the tables you want to make available to the user.
1. In the left pane, expand the tree to show the list of tables that are in the schema.
2. Select/deselect the tables you want to be available to the user.

Viewing the table summary

The table summary shows basic information about the tables in the dataview. To view the summary for a table, click the table name in the left pane. The table summary appears in the right pane. With the summary visible, you can edit, duplicate, or remove the table from the schema.

Editing tables

1. In the table summary, click Edit Table.The Edit Table and Fields pane appears.
2. In the Edit Table and Fields pane, select the appropriate options as desired and click OK. See See "Table Options" for the list of available options.

Removing tables

1. In the DataView Manager, select a table in the left pane.
2. Click Remove Table.
3. Click Submit. The Validation Summary pane appears.
4. After you review the validation summary, click Submit. The Validation Summary pane closes.
5. Click Home to return to the ChemBioOffice Manager.

Adding a schema

The master dataview can contain more than one schema.
To add a schema:
1. Click Add Schema in the Select Tables for Master Dataview window.
2. In the Database dropdown list, select a schema.
3. Enter the Password for the schema and click Publish. The schema you selected is added to the dataview.

Removing a schema

To remove a schema:
1. Click Remove Schema in the Select Tables for Master Dataview window.
2. In the Database dropdown list, select a schema.
3. Enter the Password for the schema and click Publish. The schema you selected is removed from the dataview.
