Editing a dataview

1. Log into ChemBioOffice Manager as an administrative user.
2. In the DataView Manager menu, click Edit existing. The DataView Manager window opens.
3. In the DataView Manager, select the dataview to edit. The dataview property box appears to the right of the list.
4. Click Next.
5. (Optional) In the Name and Description text boxes, edit the name and description of the database as desired.
6. (Optional) In the dropdown list, select a schema as desired.
7. (Optional) Select base table from the table list.
8. Click OK.

Security Settings

You can control who can use the dataview by granting permission based on roles or on users’ names. See See "Security settings".

Duplicating a table

1. In the Dataview Manager, click a selected a table to duplicate (deselected tables cannot be duplicated). The table summary appears.
2. In the table summary, click Duplicate Table.
3. Click Add. The table is added to the bottom of the list in the left pane.

Deleting a dataview

1. Login as an administrative user.
2. Select Delete Existing Dataview.
3. Select a dataview from DropDown List
4. Click Delete. A confirmation window opens.
5. Click OK to confirm.
