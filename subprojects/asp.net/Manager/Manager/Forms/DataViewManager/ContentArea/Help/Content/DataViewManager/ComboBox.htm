Editing form elements

You can edit any form element to change its name, bind data to it, or modify its appearance.

Binding data to an element

For most elements, to bind data, you need to select the element in the form editor and then choose the binding options in the properties window. The steps are shown below. To bind to either a SubformGrid or a Combobox, see the sections that follow.
To bind data to an element:
a. Select the checkbox in the form editor.
b. In the Properties window, select Advanced under {DataBindings} in the Data section.
c. Click the Advanced button: . The Formatting and Advanced Binding dialog box opens.
d. In the Binding drop-down list, select the field to which you want to bind to the checkbox.

 

Binding data to a SubformGrid

1. Select the SubformGrid element in the form editor.
2. Click the arrow at the top of the form. The CBVDataGridView Tasks dialog box opens.
3. In the drop-down list, select a table to be the data source.
4. In the checkboxes, select the permissions you want to all the user to have for editing the subform.
5. To edit any of the columns in the subform, select Edit Columns.
6. To add a column to the subform, select Add Column.

Binding data to a ComboBox

1. Select the ComboBox element in the form editor.
2. Click the arrow at the top of the form. The ComboBox Tasks dialog box opens.
3. Select the Use data bound items checkbox.
4. In the Data Source drop-down list, select the data source you want to populate the combobox.
5. In the Display Member drop-down list, select the field to be displayed by default.
6. Click anywhere in the form editor to close the ComboBox Task dialog box.

Resizing an element

Most form elements have size adjustment handles. To adjust the size of an element, select the element and click+drag an adjustment handle.

Adding Labels

Some elements, such as the button and group box, have there own built-in label that you can edit. For other elements that you want to label, you can place the label next to the element.
To place a label next to an element:
1. In the form editor, click+drag the Label element into the form and position it where you want the label.
2. (Optional) Click+drag the resize handles to change the size of the label box.
3. Click the label to select it.
4. Go to View>Properties to open the properties window.
5. In the Properties window, edit the Text field in the Appearance section and press ENTER.
To edit the label on elements that have their own label:
1. In the form editor, select the element.
2. In the Properties window, edit the Text field in the Appearance section and press ENTER.

Modifying Label Properties

After you create a label, you can change its appearance, such as its color and font. To change the appearance, select the label and go to View>Properties and modify the properties as desired.

Deleting form elements

To delete a form element, select the element and press DELETE.
