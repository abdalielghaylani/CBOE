Using WebHelp Toolbar

This topic provides information about basic functionality of a Help Guide in WebHelp format as applied to ChemBioOffice administrator's guide.

At the top of each topic in the WebHelp there is a Toolbar displayed.

This current topic provides an explanation for some of the buttons that may be displayed in the WebHelp toolbar:

Add Topic to Favorites

Lets you add the active topic to the Favorites pane so that they can quickly access the topic in the future.

Toggle Navigation Pane

Lets you toggle between hiding and showing the navigation pane in the output window.

Print

Lets you open the Print dialog so that they can send the open topic to the printer.

Remove Highlight

After you perform a quick search in a topic, the search text found in the topic is highlighted. This button lets users turn the highlights off.

Back Button

Lets you open the topic that was viewed previously.

Forward Button

Lets you open the next topic in a previously viewed sequence.

Stop Button

Lets you prevent a page from downloading file information

Refresh Button

Lets you update the content of the topic being viewed.

Home Button

Lets you open a designated home page (such as your company's website).

Select TOC button

Lets you open the TOC feature.

Select Index Button

Lets you open the Index feature

Select Search Button

Lets you open the Search feature.

Select Favorites Button

Lets you open the Favorites feature.
