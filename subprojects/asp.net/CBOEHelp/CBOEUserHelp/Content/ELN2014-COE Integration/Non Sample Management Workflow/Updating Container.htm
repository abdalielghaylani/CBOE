Updating Container

The Update Container Method gathers the information on the Inventory form in E-Notebook to update a container in the Inventory system.

To update a container:

  1. In the Reaction W COE Integration section, click the Send to Inventory form tool and select Open Inventory Form. The Create Container dialog appears where you can select a molecule for container creation.
  2. Select the desired molecule and click OK.
  3. The Open Inventory Form appears and displays the data associated with the selected molecule in the Reaction W COE Integration section.

  1. Fill in the necessary attributes and modify the container attributes (for example, Container Name from Acids to Acids A or Container Size from 50 to 100).
  2. Click the Inventory form tool and select Update Container. A confirmation message appears.
  3. Click OK.

Once the container is updated from E-Notebook, the changes are reflected in the Inventory system.
