Bulk Container Creation

With Bulk Container Creation, you can create substances and inventory containers in a bulk. This works with section type: Create Containers form.

The options available are:

  • Create / Resolve substance(s)
  • Create Container(s)

To Create/Resolve Substance(s):

  1. Log in to E-Notebook.
  2. Create an Experiment, add a Create Containers form section and fill in the mandatory fields.

  1. To add a structure, double-click the Structure column. The Edit Structure dialog appears.
  2. Draw or paste the structure and click OK.

The structure appears in the Structure column, as shown below.

  1. To add more than one substance, right-click Inventory Containers table and select Insert > Insert Row Above/Insert Row Below.

A new row is inserted for another Inventory substance/container entry.

  1. Under the Selected column, check the checkbox for entries you want to send to Inventory.
  2. Click the Inventory form tool and select Create/Resolve substance(s).

The BulkContainerCreation dialog appears for confirmation.

  1. Click Yes.

The BulkConatinerCreation dialog appears showing the compound IDs are created.

  1. Click OK.

The substances with Compound ID are created and returned to the Create Containers form section.

To Create Inventory Container(s):

  1. Log in to E-Notebook.
  2. Create an Experiment, and add a Create Containers form section, and fill in the mandatory fields.
  3. Check the corresponding records (check box) for which you want to create containers.
  4. Click the Inventory form tool and select Create Containers(s). The BulkContainerCreation dialog appears.
  5. Click Yes for confirmation.

Notes:

  • If the substance associated with the container (you create) does not have a Compound ID, a container with the new Compound ID and with a Barcode Number is created and returned to the Create Containers form.
  • If the substance associated with the container (you create) does not have a Compound ID but already exists in the Inventory system, you have to resolve the substance conflicts. You can use an Existing Substance or can select Create a Duplicate. In either case, a container with a Compound ID and with a Barcode Number is created and returned to the Create Containers form.

A dialog displays with the created containers barcodes.

  1. Click OK.

The containers with barcodes are created and returned to the Create Containers form.

Once the containers are pushed from E-Notebook to the Inventory system, the containers become accessible in the Inventory system.
