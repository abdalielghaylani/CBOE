Quick Permanent Registration - Single Click

From the Samples section, you can push a sample to the Permanent table in Registration using the single click method.

Note: The values set in the Registration User Preferences form automatically populate in their corresponding fields in the Registration form.

To register a sample to the Permanent table:

  1. Draw a reaction in the Reaction field and create samples. A Samples section is automatically created.
  2. Assign a product to the created samples and navigate to the Samples section.
  3. From the Home ribbon, click Registration and select Quick Permanent Registration.

The Registration dialog displays the Reg ID and a Batch ID that display in the sample's Properties table.

Note: If you have system administrator privileges, you can configure the following fields in the Properties table as read-only: Reg #, Temp ID, and Batch #. Please see the Administrator Guide for more information about setting these fields as read-only.

  1. Click OK to close the dialog.

Notes:

  • If a sample has a temp ID, the permanent registration will fail.
  • Your system may be configured to resolve registration duplicates in various ways, as shown in the example below. Please refer to the Administrator Guide for more details.

There is one limitation for the Single Click Registration.

  • If New batch option is checked, every time a duplicate is found, a new batch is created with the same Reg number.
  • If Create Duplicate option is checked, every time you want to register a structure that is already in the Chemical Registration Database, a duplicate is created with a different Reg number.

The limitation is that if the system is re-configured to add a new batch, the registration will fail.
