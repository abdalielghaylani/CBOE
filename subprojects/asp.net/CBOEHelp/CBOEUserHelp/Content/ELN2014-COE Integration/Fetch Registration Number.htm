Fetch Registration Number

The Fetch Register Number lets you query the database with a registration Temp ID. If a Registration Number is available, the Registration Number and Batch Number are returned to the associated row in the Products Table of the Reaction section.

Note: You must have adequate Registry privileges assigned to perform this Registry action.

To Fetch the Registry Number:

  1. From the Samples section, click the Home ribbon, click the Registration form tool, and select Fetch Registration No.

If the Temp ID is present in the Properties table and the Registration Number is available in the Registration system, the Fetch Registration Number dialog appears and displays the Registration Number and Batch Number.

  1. Click OK to close the Fetch Register Number dialog.

The Registration Number and Batch Number are returned to the associated row in the Samples table of the Reaction section.
