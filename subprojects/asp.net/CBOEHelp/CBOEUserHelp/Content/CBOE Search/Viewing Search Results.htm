Viewing Search Results

You can display search results in either list view or detail view. The list view displays several records on each results page while the details view shows one record but includes more information.

The List view displays multiple records at a time.

To display the details for a record, click on its corresponding Show Details button in the list view.

A record of Benzofuran in details view.

To display the list of records, click on the Return to List button in the details view.
