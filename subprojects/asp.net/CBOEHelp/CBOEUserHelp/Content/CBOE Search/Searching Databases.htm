Searching Databases

ChemBioOffice includes the ChemACX and drug databases and a sample database. This chapter explains how to search the sample database. To search the other databases, refer to the ChemACX chapter.

To select the sample database:

  1. In the ChemBioOffice Enterprise Manager, click Search Databases under Chemical Databases. The database list opens.
  2. In the database list, select Sample. The query form opens.
  3. The Sample Database search is now possible.
