Entering Batch Information

Batch information consists of the following fields:
  • Scientist- Scientist who synthesized the batch.
  • Synthesis Date- Date on which the batch was synthesized
  • Notebook Reference- The reference to the chemist notebook that contains the batch synthesis information
  • Amount- Substance quantity obtained for this batch
  • Units- Units of the amount obtained.
  • Appearance- Visual appearance of the substance. For example, clear yellow liquid.
  • Purity- Batch purity. For example, 96.5.
  • Purity Comments- Comments associated with the Batch.
  • Sample ID
  • Solubility
  • Batch Comments
  • Storage Requirements and Warnings

A record can be associated with one or more projects. The administrator defines the projects and the user selects the project to be assigned to a given record from its corresponding dropdown list.

  • If the registry record belongs to more than one project, click Add Project and select the required values.

  • To remove a project, highlight the project's row and then click on Remove Project.
  1. Fill in the Batch Information by selecting values in the fields Chemist, Synthesis Date, Units, and other fields mentioned above
  2. Click Submit. The record is added to the temporary table and the Review/Register Component window displaying the temporary registry ID appears.
  3. The submitter is allowed to edit the submitted record if needed.
