Saving queries

If a user plans to use the same search criteria in the future, he/she can save the query avoiding the data re-entry.

To save a query:

  1. After performing a search, go to Query >> Save

  1. Enter a name and description for the query.
  2. Check Is Publiccheckbox, if the query is to be available for everyone using registration enterprise.

  1. Click Save.
