Editing Mixtures

When you edit a mixture that is created from existing components, this will affect the registered components and all the registry records that refer to them. See below for more information.

  1. Log in to CBOE and create a mixture as covered in Simple Mixture Workflow
  2. Search for that mixture in the permanent registry table and click on its corresponding Reg Number

  1. The following window displays. Highlight the component that you want to modify and click on Edit button.

  1. The following dialog displays allowing you to make edits. The Original structure is not editable, make the edits to the structure under structure field. If you want to discard the structure changes and restore the original structure, click on Restore Original Structure link.

  1. When you are done with the edits, click on Save button. A dialog displays informing that the changes made to the components will affect all registry records that refer to them

  1. Click on Continue button to accept the changes and affects
  2. Click on Create Copies to create duplicates for the original structures
  3. Click Cancel to cancel the process
