Parts of a Registry Record

A registry record represents a substance and all its associated registry data. It includes:
  • Component - a distinct pure substance within a registered record
  • Batch - a particular preparation of a registry substance
  • Batch Component - information about the components can differ for each preparation so it is tracked for each batch separately
