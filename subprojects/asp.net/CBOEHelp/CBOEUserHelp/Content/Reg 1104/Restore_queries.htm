Restore queries

 
The Restore query option lets you restore a query previously executed. You can:
  • Restore Last Hitlist
  • Perform Last query
  • Restore last Query to Form
To restore last Hitlist:
 
Click Restore Last Hitlist under Restore query option. The records in the last generated hitlist appears.
