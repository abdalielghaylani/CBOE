Batches

 
A batch is a set of quantities and properties of the same substance. In Registration Enterprise a batch is automatically created for any record registered. Users can further add batches and manage them as desired.
 
For more information please refer to the following topics:
  • Adding a Batch
  • Editing a Batch
  • Deleting a Batch
