Accepting a Request

To accept a request:

  1. Login to CBOE with Reviewer privileges.
  2. From Inventory Enterprise, click Tasks.
  3. Click Manage Sample Requests. The list of requests displays.
  4. Click Review for the sample you want to review. From here, you can review, edit, or delete the request. You can also add proof of approval.
  5. Check the Accept checkbox in the row of the request you want to accept.

The following confirmation message displays.

  1. Click OK.

  1. At the top of the window click Approved. The list of approved requests displays.

Creating Samples

For each request, a reviewer can create and add samples, and assign users and generate samples reports.

To create a sample:

  1. Click on Create Samples icon corresponding to the approved request. The Create Samples from Batch dialog displays.

  1. Enter the values for the required properties (in red font).
  2. Click Next. The following dialog displays.

  1. Check the Dispose Original checkbox.
  2. Click Next. A Preview dialog displays.
  3. Click on OK to complete the process.

The New Container has been created dialog displays.

  1. Click OK to close the dialog.
