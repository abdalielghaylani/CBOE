Substructure Search

In addition to the Advanced search fields, you can use Substructure search to search Inventory Enterprise on the basis of the Substructure, Substance Name, Molecular Formula, and MolWeight Range fields.

To perform a Substructure search in Inventory Enterprise:

  1. From the ChemBioOffice Enterprise home page, Inventory Enterprise section, click the Search link. A window containing various search tabs, such as Simple Search, Advanced Search, Substructure Search, Batch Search, Global Search, and Plate Search, displays.

Note: You can access the window containing the search tabs by clicking the Search link within the Current Location frame in the Container Management area.

  1. Click the Substructure Search tab. The Substructure Search tab window appears.

Note:The number of tabs visible in the Search window depends on your configuration settings.

  1. Draw the substructure of a substance within the drawing area and specify the appropriate substructure search type in the SearchType drop-down list. The substructure search types available in the SearchType drop-down list are:
  • Substructure
  • Full Structure
  • Exact Structure
  • Tanimoto Similarity

Note: You can draw a substructure of the substance using the ChemDraw plug-in. For information about using the ChemDraw plug-in, please refer to the ChemDraw User Guide

  1. Enter additional criteria to narrow your search results.
  2. Check the Search Sublocations check box if you want to perform the search on the location specified in the Location ID field, as well as in its child locations.
  3. Check the Exclude Special Locations check box if you want to exclude the special locations, such as, Trash Can, Disposed, and On Order, from the search. To select the locations to be excluded, click the Special link with the Exclude Special Locations check box.
  4. Check the Group results by chemical structure check box if you want the search results to be grouped on the basis of the chemical structure of the substances.
  5. Click Search. A list of substances matching the search criteria specified in the Substructure Search appears.

Viewing Substructure Search Results

If the Group results by chemical structure check box is selected, the search result is a list of substances. You can manage the search result list using the following options:

  • Zoom In: Lets you zoom in on the structure of the substance.
  • Names: Lets you gather information about the various synonyms of the substance.
  • Details: Lets you gather information about the container associated with the substance.
  • Mark: Lets you mark the substance so that you can easily locate the substance in Inventory Enterprise later, without re-creating a search query.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked menu item from the Marked Hits menu displayed in the window containing the search results.

However, if the Group results by chemical structure check box is not selected when the substructure search is performed, the search results is a list of containers. You can view the details of a container displayed in the results list similarly to the way you view the details of a container in the simple search results. For information about viewing results of the simple search, see Viewing Simple Search Results under Simple Search.
