Adding a Sample

You can add samples to approved requests.

To add a sample to an approved request:

  1. Login to CBOE.
  2. From Inventory Enterprise, click Tasks. The following dialog displays.

  1. Click Manage Sample Requests. The Manage Sample Requests window appears.

  1. Click Approved. The list of approved requests displays.
  2. Click on the corresponding Batch Amount link for the request you want to add a sample to.

The Manage Batch Details window appears.

  1. Click Edit Batch.

The Batch Details window appears.

  1. Batch Status: Click Requestable in the drop-down list to change the batch status.
  2. New Comments: Enter an optional comment for this request.
  3. Click OK.

  1. Click Add Sample for the request you want to add a sample to. The Add Container to Request window appears.

  1. Select the containers you want to add to this request.
  2. Click OK. The Fulfill Batch Request dialog appears.
  3. Click Print Receipt to print a receipt for this request.
  4. Click Print Worksheet to print a worksheet for this request.
  5. Click OK to close the window.
