Location Management

You can manage the Inventory locations using the following two frames in the Container Management area:

  • Location Tree Frame
  • Current Location Frame

Location Tree Frame

The Location Tree frame is a hierarchical structure that lets you navigate through the Inventory locations. Within this hierarchical structure, locations are nested within other locations.

The locations are represented as folders in the location tree. You access a location by navigating through the various folders in the location tree. Clicking on the +/- symbols displayed next to the folders lets you navigate through the location folders. When a folder in the Location Tree frame is selected, the content within the folder displays in the Container List frame.

The Location Tree frame contains the following links:

  • Location: Create, edit, move, or delete locations, and/or set a default location.
  • Default: Opens the Location Tree from your default location. For information about setting a location as home (default) location, see Make Default below.
  • Refresh: Populates the Location Tree frame with the latest location data and contracts the Location Tree to its original position.

Location Tree Frame Operations

Creating a New Location

To create a new location:

  1. Select New from the Location menu in the Location Tree frame.

  1. The Create a New Location window appears.

  1. Click Browse next to the Parent Location text field to select a parent location for the new location.
  2. Navigate to the location you want and select it. The Create a New Location window displays with the selected location name in the Parent Location text field.
  3. Specify the barcode ID for the location by either:
    • Typing the ID in the Location Barcode text field, or
    • Selecting Autogenerate check box (beneath the Location Barcode text field)

Note: When you select Autogenerate, the Barcode Description drop down list is displayed above the check box. You need to select Location Barcodes from the drop down list to generate a barcode ID for the location.

  1. Type the name of the location in the Location Name text field.
  2. Select appropriate location type from the Location Type drop down list.
  3. Click Add Address if you want to specify a complete, physical address for the new location. The Add an address window appears.

  1. Enter the information in the respective fields and click OK.
  2. Click OK to close the Create a New Location window.

Editing a Location

To edit a location:

  1. From the Location Tree frame, select the location you want to edit.
  2. Select Edit from the Location menu.

The Edit Location window appears.

  1. Edit the information in the Edit Location window.

Note: If the address of the currently-selected location has changed, click Add Address to modify the address.

  1. Click OK.

Moving a Location

To move a location:

  1. Highlight the location that you want to move in the Location Tree frame.
  2. Select Move from the Location menu.

The Move Inventory Location dialog appears.

  1. Next to Destination Location, click Browse to open a list for you to select the new location.
  2. Select your destination location from the list.
  3. Click OK. The location has been moved to a new location in the Location Tree.

Deleting Location

To delete a location:

  1. Highlight the location you want to delete in the Location Tree frame.
  2. Select Delete from the Location menu. The Delete Inventory Location window appears.
  3. Confirm the location you are deleting is specified in the Location to delete field.
  4. Click OK to delete the location.

Note: You can delete a location only if it is empty. Therefore, if you want to delete an non-empty location, select Recursively delete related locations and containers. When this check box is selected, all the sub-locations and containers within the selected location are deleted before the selected location is deleted.

Setting a Default Location

You can set a location as your default location. The Location Tree frame automatically opens to your default location each time the Container Management area is accessed.

To set a location as your default location:

  1. From the Location Tree frame, navigate to the location you want as your default location.

  1. Select Set Default from the Location menu. The Set your default inventory location dialog appears.

  1. Confirm the location specified in the Location Name text field is the location you want as the default location.
  2. Click OK.

Printing Location Barcodes

To print location barcodes:

  1. Select a location in the Location Tree frame.
  2. Select Print Label from the Location menu. The Print Location Labels window appears.

When the Print Location Labels window first displays, the application will determine how many location label reports are defined within the system. If only one report is defined, the application will automatically execute that report and display the resulting barcode, and the Print dialog box will appear.

If more than one report is defined, you will have the option to choose from a drop-down list of available reports.

  1. Choose a report from the list to trigger the report generation process.
  2. Click Print in the bottom-right corner to print the barcode.
  3. Click Close to close the form.

Note: To support the print location barcode function, a new report type in the Inventory database along with a new Access query and report layout were created as part of Inventory Enterprise. If you install Inventory Enterprise from scratch, you will get a default location barcode report automatically. Those who update will need to update their Cheminv_Reports Access database so that it includes the new query and new report.

Printing Barcodes on Zebra Printer

Inventory Enterprise uses the LPR service to facilitate printing of location, container, and plate barcodes on network printers, such as Zebra printers.

To print barcodes on Zebra printer:

  1. Select a location in the Location Tree frame.

Note: You can select a container/plate to print its barcode on a Zebra printer.

  1. Select Print Label from the Location menu.

Using the inv_label_printers table, the Inventory application determines whether LPR printers (network printers) are defined for the selected location. If an LPR printer is defined, the following form appears as:

Note: If no printers are defined in the database for the selected location, the Print Location Labels dialog appears. The new LPR printer definition is transparent for users who do not choose to use this feature.

  1. Select the connected Zebra printer in the Network printer reports drop-down list.
  2. Specify the number of copies to print.
  3. Click Go to print the labels using the LPR print definitions in the database.

Note: Click Go to the right of the My local printer reports to open the classic Print Label form, which displays the Access Snapshot/PDF report generator.

Note: Inventory Enterprise lets you print multiple barcodes simultaneously. To print multiple container/plate barcodes in one pass during multi-select operations, select Other > Print Labels from the Container Details frame. You can only use LPR printers (network printers) to print multiple container/plate barcodes. The print functionality during multi-select operations is identical to when a single container, or plate, is selected. If no LPR printers are defined for the selected containers/plates, an error message displays.

Current Location Frame

The Current Location frame consists of the Location text field that contains the complete path of the location, which is currently selected in the Location Tree frame. It also contains various links to perform actions on the selected location, or on the containers stored in the selected location. These links are:

  • Search: Lets you search a container, or plate, in Inventory Enterprise. When you click Search for the first time the default search form is displayed, otherwise, the search form you last-used automatically displays.
  • Manage Substances: Lets you manage chemical inventory substances.
  • My Batch Requests: Lets you view information about sample requests and reservations. You can choose the columns to display in the list of sample requests. You can cancel a request, as well.
  • Scan: Lets you scan container barcode.
  • New Container: Lets you create a new container.
  • New Samples: Lets you create new samples.
  • Receive Order: Lets you receive the ordered containers.
  • Tasks: Lets you perform administrative tasks, such as managing tables and container requests.
  • Home: Displays the login window of ChemBioOffice Enterprise.
  • Help: Opens the Inventory Enterprise user's guide.
  • Log Off: Logs off the current user.

Note: Your Current Location frame may not display all links listed above. Your privileges determine which tasks you can perform. If you do not have permission to perform a certain task, the link associated with this task will not display. To confirm if you have the appropriate privileges to access a link in the Current Location frame, see Roles and Privileges.

Current Location Frame Operations

Searching

The Search link in the Current Location frame lets you search substances, containers, and plates in Inventory Enterprise.

The various type of searches you can perform are:

  • Simple search
  • Advanced search
  • Substructure search
  • Batch search
  • Global search
  • Plate search

Manage Substances

For more information on managing substances, please refer to Substance Management.

My Batch Requests lets you view a list of sample requests. This list automatically populates when you request a sample.

Column Chooser lets you select the columns to display in the list of sample request.

Click Cancel to cancel a request.

My Reservation lets you view the samples that have been reserved following a batch search (and request for the reserved samples). For more information about reserving samples, see Reserve Sample under Viewing Contents of a Container.

Scanning

Scan lets you scan multiple containers and perform an action, such as, move or delete, on all the containers at once.

To perform scanning:

  1. Click Scan. The Scan multiple containers window appears.
  2. Type the barcode of the container (you are interested in) in the Scan Container Barcode text field.
  3. Press the <Tab> key. The container is scanned and added to the list, as shown in the example below.

Note: You can remove a container from the list by clicking Remove next to the container you want to remove.

  1. Repeat steps 1 through 3 until all containers are scanned and added to the list.
  2. Click the link associated with the action you want to perform, such as, Check Out or Move Containers. The action is performed on all scanned containers in the list.

New Container

The New Container link lets you create a new container in any location. For more information about creating containers, see Creating a New Container.

Creating a New Sample

To create a new sample:

1. Click New Samples. The Create New Samples window appears.

  1. Click Browse to open a list of locations and to select the location where you want to place the sample.
  2. You can either create a new substance by clicking New Substance, or you can add an existing substance.
  3. To search for an existing substance, click Select Substance to open the substance search form.
  4. Search for the substance and select it. For more information, see Searching for a Substance. After selecting the substance, the Compound ID and Compound Name fields automatically populate in the Create New Samples window.

Note: The search results include all the substances that match the specified criteria. This is unlike a global search, which returns a matching substance only if that substance is associated with a product.

  1. Enter the Container Name.
  2. Click Next. The Create New Samples window appears.

  1. Enter the information related to the sample you are creating.
  2. Click OK. The newly-created sample is added to the list of containers in the location wherein the sample was created.

Receiving Orders

The Receive Order feature lets you receive a shipped order. When you receive an order, the order is marked as Closed, and the ordered containers, or requested samples, are delivered to the specified location.

To receive an order:

  1. Click Receive Order in the Current Location frame. The Receive an Order window appears.

  1. Type the barcode of the container you want to receive in the Scan Container Barcode text field.
  2. Press the <Tab> key. The container is scanned and added to the selection list, as shown in the example below.

Note: You can remove a container from the list of scanned containers by clicking Remove next to the container you want to remove. You can remove all containers from the list by clicking Clear List.

  1. Repeat steps 2 and 3 until all the containers are scanned and added to the list.
  2. Confirm the check boxes next to the containers you want to receive are selected and click Update. A Success message displays.
  3. Click OK.

Tasks

When you click Tasks in the Current Location frame, the Administrative Menu window appears.

The Administrative Menu window contains following links, which let you to perform various administrative tasks, such as:

  • Manage Tables
  • Manage Grouping Fields
  • Manage Reports
  • Analyze Audit Trails
  • Customize Reports
  • Manage Approvals
  • Manage Requests
  • Manage Sample Requests
  • Manage Orders
  • Manage Container Receiving
  • Change Passwords
  • Plate Settings
  • Create Plates from Excel
  • Create Plates from Text Files
  • Help
  • Log Off

Apart from the above list, the Administrative Menu window provides links for managing grids and organizations.

To confirm if you have the required privileges to perform an administrative task, see Roles and Privileges.
