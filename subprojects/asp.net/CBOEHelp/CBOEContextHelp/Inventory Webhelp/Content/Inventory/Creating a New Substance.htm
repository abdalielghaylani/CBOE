Creating a New Substance

To confirm you have the appropriate privileges to create a new substance in Inventory Enterprise, see to Roles and Privileges.

To create a new substance:

  1. From the ChemBioOffice Enterprise home page, Inventory Enterprise section, click the Manage Substances. The Substance Management area displays.
  2. Click Add Mode. The Create or Edit a substance in Inventory Enterprise window appears.

Note:The Create or Edit a substance in Inventory Enterprise window might display more fields on your system if your system administrator has customized this window.

Note:You can access the Create or Edit a substance in Inventory Enterprise window from the Substance tab of the Create or Edit an Inventory Container window, which is displayed when you click the New Container link in the Current Location frame. This window is accessed from the Substance tab when a new substance is created and added to a new container. For more information about creating a substance through the Substance tab, see "Substance Tab" under Creating a New Container.

  1. Enter information, such as Substance Name, structure, CAS Registry #, and ACX Number for this new substance.
  2. Click OK to add the new substance to Inventory Enterprise. A window confirming the creation of this new substance in Inventory Enterprise appears.
  3. Click OK.

Note: A Conflict Resolution screen appears if a unique field, such as, Substance Name and CAS Registry #, already exists for an existing Inventory substance.
