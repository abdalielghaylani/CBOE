Using WebHelp Toolbar

The ChemBioOffice Enterprise Help file is formatted as WebHelp.

There is a Toolbar (containing action buttons) located at the top of each topic. Below is a description of the most common action buttons in the WebHelp Toolbar:

Add Topic to Favorites:

Adds the active topic, or search queries, to the Favorites pane for easy-accessibility later on.

Toggle Navigation Pane:

Toggles between hiding and showing the navigation pane in the output window.

Print:

Opens the Print dialog to send your currently-open topic to the printer.

Remove Highlight:

When you perform a quick text search in a topic, the searched text is highlighted. With this button, you turn off the highlight, and the text displays as-is.

Back:

Opens the previously-viewed (or active) topic.

Forward:

Opens the next sequential topic in a previously-viewed sequence.

Stop:

Prevents (or stops) a page from downloading file information.

Refresh:

Updates (refreshes) the content of the currently-viewed topic.

Home:

Opens a designated home page (such as your company's website).

Select TOC:

Opens the TOC.

Select Search:

Opens the Search feature to search for specific text within the topic.

Select Favorites:

Opens the Favorites feature.

Previous Topic:

Opens the previously-viewed topic (in sequential order).

Next Topic Button:

Opens the next topic in sequential order.
