Marking records

Marking records lets you view the records without creating a search query for them again. However, keep in mind that, as the database is updated, your list of marked records isn't automatically revised. As a result, you may need to periodically rerun your search to ensure that your list of marked records remains accurate. However, when a marked record is updated, you can see the update when you open the marked record. You can mark the records in both list view and form view. Marked hits are user specific and can be retrieved at any time using Show Marked option under Marked menu.

To mark a record, select the check box to the left of the corresponding record in list view. After marking the record, you can view or unmark the record or export the results to other linked applications like Inventory Enterprise.

To Mark All the records displayed in list view, check the "Marked" checkbox at the top of the list.

To Unmark all marked records displayed, uncheck the "Marked" checkbox at the top of the list.

To save marked search results:

  1. Go to Marked >> Save

  1. Specify a name and description for the list

  1. Check Is Publiccheckbox, if the marked list is to be available for everyone using registration enterprise
  2. Click Save.

Showing All Marked Hits

To show Only the Marked Hits:

  1. Go to Marked>>Show Marked

  1. All of the Marked hits will be displayed

Showing the query search results

To show the query search results:

  1. Go to Marked>>Show Search Results

  1. All the results hits will be displayed.
