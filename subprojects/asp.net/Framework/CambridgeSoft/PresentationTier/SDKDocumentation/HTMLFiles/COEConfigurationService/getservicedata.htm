Get Service Data for a Service [c1] 

 

You can get the information about a service by calling the GetServiceData(string serviceName) method of the COEConfigurationUtilities class, where serviceName represents a string containing the name of the service. This method returns the ServiceData object containing the Configuration data defining a service. You can use the following code snippet to get the service data:

 

 

ConfigurationUtilities.GetServiceData("COEConfigurationService");

 

 [c1]Is Service name represents the name of any COE service?
