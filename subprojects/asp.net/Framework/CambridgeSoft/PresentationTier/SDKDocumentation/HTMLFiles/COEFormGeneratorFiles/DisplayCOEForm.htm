Displaying COEForm

 

You need to create an .aspx page to display the COEForm on the .aspx page. The .aspx page reads the specified xml file and calls the CreateCOEForm() method of the COEForm class to convert the XML file into the COEForm object, which is displayed on the .aspx page using the COEFormGenerator class. The .aspx page can contain multiple objects of the COEFormGenerator and xml files to display the multiple COEForm in a single page. For example, you can create COEFGTest.aspx page that will display the COEForm created using the SampleXMLDate.xml xml file.
