Searching the Temporary Registry

It is possible to retrieve all the records in the temporary table or only specific records. Searches by structure, temporary ID, or by any other parameter may be run.

To run searches in the temporary table:

  1. From the Temporary Registration Records display, click on New Query button

The Search Temporary Registry page appears:

  1. Enter the search criteria under Temporary Information and then click:
  • Clear to delete the information you have entered.
  • Cancel to cancel the operation
  • Click Search. A list of records matching the search criteria you specified appears.

  1. In the Temporary Registration Records hit list, click:
  • Retrieve All, to retrieve all the records in the temporary table
  • Print, to print the list.
  • Edit Query, to edit the current query by editing /adding criteria as required
  • Manage Queries, to display the list of available queries and perform multiple actions, i.e editing, deleting, re-running, saving, etc... For more details, please refer to Managing Queries topic.
  • Export, to export the list into an SDF file.
