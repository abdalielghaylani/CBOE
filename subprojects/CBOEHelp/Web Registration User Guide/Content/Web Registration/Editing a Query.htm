Editing a Query

To edit a query name or description:

  1. In the Query Management display, click on the edit icon corresponding to the query that you want to modify, the following properties display:

  1. Edit the name and description as desired.
  2. Check the Is Public checkbox in order to make the information available to other users.
  3. Click Save.
  4. To discard the changes, click Cancel.
