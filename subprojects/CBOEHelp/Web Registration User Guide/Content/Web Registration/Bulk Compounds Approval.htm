Bulk Compounds Approval

You can approve compounds in bulk.

To approve multiple compounds simultaneously:

  1. In the Temporary Registration Records table, mark the records that you want to approve. The number of marked records and the menu options appear at the top of the table:

  1. Click Approve Marked. A confirmation message appears. Click Yes, all the marked records are approved successfully.

Save Marked

To save the marked records:

  1. Select the records to save, the menu options appear.
  2. Click Save Marked. The Save Query dialog appears.

  1. Enter a name and a description for the query and check the Is Public checkbox to share it with other users.
  2. Click Save.

Show Marked

Click Show Marked to show the marked records only.

To restore all the records, click on Show All Records button.

Delete Marked

To delete the marked records, click on Delete Marked button.
