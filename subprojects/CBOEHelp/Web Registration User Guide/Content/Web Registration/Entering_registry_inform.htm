Entering Registry Information

 
The Registry Information section consists of the following fields:
 
  • Projects: A record can be associated with one or more projects. The administrator defines the projects and the user selects the project to be assigned to a given record from its corresponding dropdown list.

  • Identifiers:The identifiers describe chemical substances in terms of layers of information to facilitate the search for such information (Ex. Alias).
  • Prefix: A mandatory field that allows you to select a prefix for the registry number.
  • Registry Comments: Any comments about the substance.
The administrator defines the Identifiers and the Registry Number Format. For more details please contact the administrator.
  1. Fill in the Registry Information by selecting, the project, the prefix and the Identifiers values from their corresponding dropdown list.
  • If the registry record is assigned to more than one identifier, click the plus sign to add a new row and select the required identifier type.
