Entering component information

Each Component is identified by a unique ID, generated automatically based on business rules defined by the Administrator. The default is: a Prefix followed by a Sequence Number.
 
The Component information consists of:
  • Identifiers- The component identifiers describe the pure substance. Each component will have their own identifiers to describe univocally the component. The default component Identifiers are:
    • Chemical Name
    • CAS Number
    • Synonym
  • Each component has a structure that may be drawn in the structure window. The options are:
    • Drawn Structure is selected by default.
    • Select No Structure if the component does not have a structure associated with it.
    • Select Unknown Structure if you donâ€™t know the structure and you want to keep the structure field blank. You can later add the structure to the record.
    • Select Non Chemical Content to paste a graphic image in the structure box.
  • Structure Comments- Comments associated with each structure.
  • Component Comments- Comments associated with each component.
  • Stereochemistry Comments- Any stereochemistry remark.

Note: If the ChemDraw Chemical Warning Checker and the required chemical warning preferences are enabled, a red wavy box appears around the problematic area if there is any error in your structure. You will not be allowed to Submit the component until the underlying error is resolved. For more information on ChemDraw chemical warning, see ChemDraw Chemical Warning topic.
