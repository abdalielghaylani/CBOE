Bulk Registration

You can register one record at a time or several records using the bulk registration feature.

To perform a bulk registration:

  1. Mark the records to register.
  2. Click Register Marked.

The "Please choose duplicate action" dialog appears providing you the duplicate resolution actions that you can perform if there is any duplicate, in the permanent registered record, for the record that you are registering.

  • Select Duplicate: to register as a new compound any duplicate record it encounters.
  • Select Batch: to add new batches for the duplicate compounds.
  • Select None or Temporary: to leave the record in the temporary table.
  1. Click Register Marked button.
