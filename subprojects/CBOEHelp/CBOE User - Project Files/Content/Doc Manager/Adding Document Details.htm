Adding Document Details

To add details to a document before submitting it to DocManager Enterprise:

  1. Click Add Document Details after locating the document in the Add Document page. The Add Document Details page, containing a preview of the selected document, appears:

  1. Enter the title, author name,comments, etc...for the document being submitted in the appropriate text boxes.
  2. Click on "Submit This Document" button to submit the document to DocManager Enterprise.
