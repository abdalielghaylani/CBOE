Doc Manager Introduction

Welcome to the Doc Manager chapter.

This chapter provides a full description of the DocManager, its features, and complete instructions on how to use them from the non-administrator user point of view.

DocManager Enterprise is used to create, store and share chemical documents. Using DocManager Enterprise, the user can:

  • Store documents at a central location and share them with other users.
  • Perform structural and text (exact phrase or word) searching.
  • Quickly retrieve large amount of data located throughout stored documents.
  • Standardize methods for delivering written information.

With DocManager's user friendly interface, any new user can begin using it instantly. You can create and maintain a secure chemical document repository, add or submit documents over the Web, keep track of relationships between documents, and quickly search and retrieve documents. The documents submitted are stored in the database and all DocManager users with proper privileges can search and view the documents.

The DocManager search feature lets you find specific information quickly and easily. The search can be based on either chemical structure or the content of the document. It supports basic and advanced document searching techniques. Users with proper privilege can remove documents that are no longer used or shared.

You can also integrate DocManager Enterprise with other Enterprise applications such as Inventory Enterprise and Registration Enterprise. As a result, you can easily access linked documents through these applications.

Note that this chapter just covers the workflow allowed for non-administrator users which is the Documents searching.
