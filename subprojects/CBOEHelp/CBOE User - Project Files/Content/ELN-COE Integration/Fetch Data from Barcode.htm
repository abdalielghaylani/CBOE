Fetch Data from Barcode

User is allowed to fetch data from Inventory by entering a valid Barcode value.

To fetch data from barcode:

  1. Enter a valid Barcode value in the Barcode column of the “Reactants’ table
  2. Click outside the cell
  3. The container properties will be returned and added to the Reactants table
