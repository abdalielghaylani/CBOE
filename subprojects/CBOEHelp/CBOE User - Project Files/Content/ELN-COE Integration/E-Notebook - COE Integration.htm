E-Notebook- COE Integration

The integration add-ins allows you to integrate E-Notebook with other ChemOffice Enterprise applications such as Inventory, ChemACX and Registry. This part of the guide defines the main functionality of the Inventory, Registration and ChemACX Push and Pull technologies, Bulk Container Creation and Bulk registration for Combi Chem Experiments.

This part of the guide deals with COE- E-Notebook 12.x integration. For more information on COE 17.1 - E-Notebook 17.1 integration, refer to E-Notebook 17.1 - COE 17.1 Integration chapter.

For more details please refer to the following topics:

  • Registry/Inventory/ACX Pull
  • Registry/Inventory Push
  • Combi Chem Bulk Registration and Bulk Container Creation

Please note that the BioAssay - ChemOfficeEnterprise integration is covered in the E-Notebook User's guide.
