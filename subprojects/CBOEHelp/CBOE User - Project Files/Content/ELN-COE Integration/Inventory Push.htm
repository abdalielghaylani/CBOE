Inventory Push

It is possible to send a compound from E-Notebook to Inventory via the Samples section and using the Open Inventory Form option.

The “Open Inventory Form” option allows selecting a structure from the Samples section and launches an Inventory form. This inventory form copies appropriate data associated with the Structure from the Samples section. Later the Inventory form allows users to ‘Create Container’ in the Inventory system or ‘Update Container’ (update information for an existing Container) and can also ‘Print Barcode'.
