Bulk Container Creation

To create containers in Bulk:

  1. Under "MPS Registry Form" section, mark the "Selected" checkbox for the entries to which you want to create containers
  2. Fill in the mandatory fields
  3. Select "Create Containers" option from the "Create Containers" form tool menu

  1. The "BulkContainerCreation" confirmation message displays, click on Yes button

  1. A dialog showing the newly created barcodes corresponding to the selected entries displays. Click on OK button

  1. The containers with Barcode will be created and returned to the Registration Table

  1. Once the containers are pushed from E-Notebook to Inventory, the containers becomes accessible in the Inventory system
