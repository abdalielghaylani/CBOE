Retrieve Data from Registry System

The user is allowed to pull data from Registry system and to use the structures in the Reaction section.

To retrieve data from Registry:

  1. Log in to E-Notebook
  2. Create an experiment and then add Reaction section
  3. In the Reaction toolbar, click on the Add button and select "Registry Compound Selector" datasource option from the "Search In" drop down list, the following window appears

  1. You can either retrieve all the records or search for specific record.
  2. The search form consists of:
  • ChemDraw control to render structures for substructure, Full Structure, Exact and similarity searching.
  • A set of text boxes are available for entering search data (Registry/Component/Batch Information).
  • You may set structure preferences by clicking “Options” tab present on structure window
  1. The search preferences provided are “General” and “Stereochemistry”. Select the desired search preferences tab and choose the search options as desired. For more details please refer to Structure Search topic.
  2. Enter specific search criteria under Registry information or Batch Information and click on the Search button
  3. A search will be executed on the basis of the criteria specified

  1. A list of the records matching the search criteria you specified will be displayed.

  1. You may choose to ‘Refine the search results’ OR conduct a ‘New Query’

Refining Search Results

You can modify your search criteria and search over the current result set. There are two options available under Refine query:

  • Edit Current Query: select this option if you want to re-run the original query and then apply the refinement query
  • Refine over Current Records: select this option if you want to apply the refinement query to the current record set.

New Query

You can conduct a new query by selecting the New Query option. A new blank Reg search form is displayed.
