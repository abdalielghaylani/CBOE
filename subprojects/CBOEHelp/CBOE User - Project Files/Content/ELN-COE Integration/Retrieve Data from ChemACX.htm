Retrieve Data from ChemACX

The user is allowed to pull data from ChemACX system and to use the structures in the Reaction section.

To retrieve data from ChemACX:

  1. Log in to E-Notebook
  2. Create an experiment and then add Reaction section
  3. In the Reaction toolbar, click on the Add button and select "ACX Reagents" datasource option from the "Search In" drop down list, the following window appears

  1. A set of text boxes are available for entering textual search data:
  • Substance Name
  • CAS Registry Number
  • ACX ID
  • Catalog Number
  • Supplier
  • Mol/Weight Range
  • Molecular Formula
  1. Enter Search criteria and click search button to execute the search
  2. A Clear button is available for clearing the form
