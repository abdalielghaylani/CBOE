Launch Inventory Form from E-Notebook

This allows the user to view an Inventory Container if a valid Barcode is present.

  1. Make sure that the Barcode column has a valid Barcode value
  2. Select the CTRL key + click on the Barcode value that appears as a reference

  1. An Inventory form will be launched displaying Container details of the selected Barcode
