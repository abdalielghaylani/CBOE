Retrieve Data from Saved Hit Lists

You can retrieve data from already saved public and user hit list in inventory system.

To retrieve data from saved Hit List:

  1. Click the Add button from the Reaction toolbar in the Reaction section.
  2. ‘Select & Add Structure’ dialog appears
  3. Select “Inventory Search” datasource from the ‘Search in’ drop down list
  4. Select “Show Hit Lists” tab to retrieve data from already saved public and user hit list in inventory system
  5. Select a hit lit from the list available

  1. Click on "Fetch HitList Data" to retrieve the hitlist data

  1. Show Results tab appears displaying the search results
