Fetch Register Number - Combi Chem

The Fetch Register Number allows users to query the database with a registration Temp ID. If a Registration Number is available then the Registration Number and Batch Number will be returned to the associated row in the Registration Table of the ‘MPS Registry Form’ section.

Note: User requires adequate Registry permissions to perform this Registry action

  1. Under “Selected” column, select the checkbox for each of the compounds you want to fetch register number
  2. Click on the “Fetch Register Number” form tool from the ‘MPS Registry Form” section
  3. If the TempID is present in the Registration Table and the Registration Number is available in the registration system then, Fetch Register Number dialog appears displaying the Registration Number and Batch Number

In case the TempID is present in the Registration Table and the Registration Number is unavailable in the registration system then the message box returns “No Compound exists with the Temp id”

  1. Click OK for the Fetch Registration Number dialog
  2. Registration Number and Batch Number will be populated in their corresponding cells in the Registration Table of the ‘MPS Registry Form” section.

If Reg Number and Batch Number fields are already populated with data then the message box returns “Clear the Reg Number field and Fetch again”.
