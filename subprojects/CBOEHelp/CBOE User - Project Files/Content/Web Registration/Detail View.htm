Detail View

The detail view displays all the information for a particular record. To display a record in detail view:

  • Click the Detail view icon corresponding to a particular record.
  • The corresponding record's data display. Scroll down to navigate through the Registry, Component and Batch information.

From the above page, you can:

  • Edit the record's information by clicking on Edit button. The editing page appears, make the required edits and click Save or click Cancel to discard any changes and to move back to the record details view.
  • Delete the record by clicking on Delete button.
  • Approve the record if you have adequate privileges.
  • Go back to the Temporary Registration Records page.

Note: The chemical structures are in CDXML format only in edit views. In list and detail views, the chemical structures appear as images, therefore it is not possible for you to copy structures from list and detail view into the chemical drawing tool. You can only copy and paste structures in edit views using the Ctrl+C shortcut or the Copy option from the ChemDraw Direct toolbar.

If approval workflow is not enabled on your system, you will see Register button instead of Approve button, you can then directly register the record skipping the approval process.

Note: Approve and Register buttons appear only when you are granted with registrar privileges.
