Save as Template

You can save the record as a draft for later use. For example, if you are waiting for analytical information to determine the purity of a component that you want to submit and you filled in all other fields in the form, you can save the form and add the information when it is available.

To save a record as template:

  1. Open the submit form and enter the registry data.
  2. Go to Templates and then select Save as Template option.

  1. Enter the template name and description

  1. Check the Public Template checkbox if desired, in order to make the record available to other users.
  2. Click Save button. The Template is saved.
