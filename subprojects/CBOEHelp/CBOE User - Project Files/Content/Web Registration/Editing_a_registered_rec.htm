Editing a Registered Record

It is possible to edit a registered record and save it:

To edit a record:

  1. Log in to Registration application.
  2. Click on Search Registry icon, the Registration Records window appears.
  3. Click on the Detail view icon corresponding to the record that you want to edit

  1. The Registry Record page displays, click Edit, make the required changes and then click Save.
