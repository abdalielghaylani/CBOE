Marking Records

Marking records lets you view the records without creating a search query for them.

To mark a record:

In the Temporary Registration Records table or in the Registration Records table, check the checkbox corresponding to the record that you want to mark

The count of marked records appears at the top, in addition to a number of menu options allowing you to perform multiple actions. For example: Delete the marked records, show the marked records only, save the marked records, approve the marked records, etc...
