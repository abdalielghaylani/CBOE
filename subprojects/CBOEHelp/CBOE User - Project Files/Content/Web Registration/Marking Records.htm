Marking Records

Marking records lets you view the records without creating a search query for them.

To mark a record:

In the Temporary Registration Records table or in the Registration Records table, check the checkbox corresponding to the record that you want to mark.

The count of marked records appears at the top, in addition to a number of buttons allowing you to perform multiple actions. For example: Delete the marked records, show the marked records only, save the marked records, approve the marked records, etc...

Marking All Records

You can mark all the records in the table with a single click. There is a maximum number of records that you are allowed to mark at once. In the default configuration the maximum marked allowed value is set to 500 records.

Note: The Mark All checkbox is enabled if the total record count in the table does not exceed the max marked allowed value, otherwise the checkbox is disabled. The max marked allowed value is set by your system administrator.

To mark all records, click on the mark all checkbox in the table header.

Clear Marked

To un-mark your marked records, click on Clear Marked button.
