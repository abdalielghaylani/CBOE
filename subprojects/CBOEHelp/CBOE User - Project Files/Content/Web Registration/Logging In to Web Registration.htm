Logging In to Web Registration

Prior to access the Web Registration, you need to have a user account created in advance in CBOE Manager.

To log in to Web Registration:

  1. Open a web browser and type in the following URL: http://Servername/Registration.Server/login
  2. The Log in page appears. Enter your credentials and click Sign in.

The Temporary Registration Records page appears:

The Temporary Registration Records page displays the total number of records that are temporarily registered.

Deleting Records

To delete a record from the list:

  1. Click on the delete icon in the second cell of the record's row.
  2. The following confirmation message appears. Click Yes to delete the record or No to cancel the operation.

Viewing Records Details

To view a record details:

  1. Click on the detail view icon in the second cell of the record's row. The following display appears:

From the above page, you can:

  • Edit the record's information by clicking on Edit button. The editing page appears, make the required edits and click Save or click Cancel to discard any changes and to move back to the record details view.
  • Delete the record by clicking on Delete button.
  • Approve the record if you have adequate privileges.

Note: The chemical structures are in CDXML format only in edit views. In list and detail views, the chemical structures appear as images, therefore it is not possible for you to copy structures from list and detail view into the chemical drawing tool. You can only copy and paste structures in edit views using the Ctrl+C shortcut or the Copy option from the ChemDraw toolbar.

If approval workflow is not enabled on your system, you will see Register button instead of Approve button, you can then directly register the record skipping the approval process.

Note: Approve and Register buttons appear only when you are granted with registrar privileges.

Logging out from Web Registration

To log out from Web Registration, click on the right-pointed arrow at the upper-right corner of the Web Registration application page.
