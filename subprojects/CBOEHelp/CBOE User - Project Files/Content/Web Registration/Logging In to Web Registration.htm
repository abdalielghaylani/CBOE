Logging In to Web Registration

Prior to access the Web Registration, you need to have a user account created in advance in CBOE Manager.

To log in to Web Registration:

  1. Enter http://servername/COEManager into your web browser address bar. <servername> is the name of the server where ChemBioOffice Enterprise is installed. The ChemBioOffice Enterprise's log in page appears.
  2. Enter your username and password in the ChemBioOffice Enterprise log in page and click Log in button.
  3. In the ChemBioOffice Enterprise Manager home page, click Main Menu under Web Registration Enterprise.

It is recommended to use Google Chrome and Firefox browsers

Note: It is not possible to automatically log in to Web Registration via CBOE Manager home page when the user name contains 'Space' and/or 'Underscore' special characters. You should provide the login credentials again in the Web Registration login page.

Deleting Records

To delete a record from the list:

  1. Click on the delete icon corresponding to the record that you want to delete.
  2. The following confirmation message appears. Click Yes to delete the record or No to cancel the operation.

Logging out from Web Registration

To log out from Web Registration, click on the right-pointed arrow at the upper-right corner of the Web Registration application page.
