Adding a batch

A batch is created after a record is registered and stored in the permanent table. More than one batch may be added to a registered record.

To add a batch:

  1. Click the Detail view icon corresponding to the record. The Registry Record page appears.
  2. Click Add Batch.

The Add Batch window appears:

  1. Fill in the Batch Information fields.
  2. Click Add. A new batch gets added to the registry record.

  1. Click Select Batch button, the list of batches displays. Select the batch that you want to show.

  1. Click Edit Batch button, to edit the batch information.
  2. Click Delete Batch button, to delete the displayed batch if it is not required anymore.
  3. Click Move Batch, to move the batch to a different registry record.

  1. Click New Container to create a container from the selected batch.
