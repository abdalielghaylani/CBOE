Date Search

Date searching allows you to search for records that are created or synthesized on a specific date or range of dates that you specify using the date pickers that are available in the search form.

To search using the date pickers:

  1. In you query form, click on the calendar icon at the end of the date field as shown in the screen shot below:

  1. An interactive calendar appears. Select the date that you want.

The date gets populated in the date field as shown in the screen shot below:

Click on the close icon beside the calendar icon if you want to clear the date that you have selected.
