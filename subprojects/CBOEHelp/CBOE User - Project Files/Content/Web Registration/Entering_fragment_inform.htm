Entering fragment information

When a component is a chemical compound, only the base structure is drawn in the Structure box and the rest of the structure compound information is filled in the Fragment list. Fragment Information differs from one batch to another. Salts and Solvates are the most typical example of fragment types.

The administrator defines the fragment types:

  1. Fill in the Fragment Information by selecting the Fragment Type, and Code.The fragment type could be a salt, solvate or any other fragment type defined by the administrator. A component, as in the case of hydrated salts, may have both salt and solvate as fragment types.

  1. To select more than one fragment type, click the plus sign to add a new row. Select the code based on the fragment type selected and enter the equivalent value for each fragment. The fragment's details get populated automatically in the Fragment Information table.

For more information on defining new fragment types and adding new fragments, refer to the Registration administrator guide.
