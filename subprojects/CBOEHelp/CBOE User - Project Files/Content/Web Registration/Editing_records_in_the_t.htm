Editing Records in the Temporary Table

It is possible to edit the records in the temporary table before they are registered in the permanent table.

To edit a record in the temporary table:

  1. In the Temporary table search for the record to edit.
  2. In the results list, click on the Detail View icon corresponding to the record. The Temporary Record page appears.
  3. Click Edit button.

  1. Make the required changes in the Temp Registry, Temp Component, or Temp Batch information and then click Save to save the changes or click Cancel to undo any unsaved changes.
