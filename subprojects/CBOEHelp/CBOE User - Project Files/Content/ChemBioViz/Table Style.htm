Table Style

You can change the color, borders, and many other table style attributes. Changing these attributes does not affect the data, it only affects how the data is displayed.

To change the table style:

  1. Select the Table tab.
  2. In the navigation pane, click Edit Grid Tab. The Properties window appears.

In the properties window, select Display Layout in the Layout section. The most common properties for modifying table styles are listed under Override in the Layout section.
