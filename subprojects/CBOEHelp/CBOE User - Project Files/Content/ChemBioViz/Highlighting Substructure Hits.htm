Highlighting Substructure Hits

While running a query in ChemBioViz, it is possible to highlight the substructure in the results hit list.

To enable the substructure's highlight feature:

  1. Log in to ChemBioViz client
  2. Go to File and select Preferences option

  1. The Preferences dialog displays, click on the Search Tab

  1. Check the "Show highlights on substructure hits" checkbox.
