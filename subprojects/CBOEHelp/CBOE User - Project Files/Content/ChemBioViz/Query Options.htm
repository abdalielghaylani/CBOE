Query Options

ChemBioViz provides several features that you can use to manage your queries. Right-click a query in the list and choose one of these options:

Restore Hitlist

Displays the results from the previous time the query was run. A new search is not run.

Rerun

Runs the query again. ChemBioViz searches the database and returns the results that match the query parameters.

Restore Query

Opens the form with the search parameters entered from the previous search.

Select for Merge

Select this query to merge with other queries. Please refer to "Merging queries".

Search Over This

Runs the new query over the results of the selected query.

Delete

Deletes the query from the Queries pane. You can also select the query and press DELETE.

Rename

Changes the name of the query.

Run on Opening Form

Runs the query when you open a form.

Keep

When you close a form with this option selected, the selected query and its corresponding hitlist are saved.

Keep All

The same as Keep but applies to all the session queries.

Properties

Displays properties of the selected query. You can change some of the properties, such as the name and description. Others are read-only.
