Building a Plot

After you create a plot, you assign data and add custom colors and labels to it as desired.

To assign data to a plot:

  1. Select the plot.
  2. Click the arrow at the upper right of the plot. The Smart Tag dialog box opens.
  3. In the Data Table drop-down list, select a data table:
  • To create a standard plot, choose the base table (the first table the list).
  • To create a subform plot, choose any table except the first one on the list.
  1. In the X Axis Field and Y Axis Field drop-down lists, choose the fields for the X- and Y-axes.To build a 1D plot, select <Record> in the X Axis Field.

Note: the X Axis Field and Y Axis Field drop-down lists display all numeric fields in the dataview under the table you select.

  1. (Optional) Enter a Plot title in the Title text box. You can also add a title outside the form editor.
  2. (Optional) Click Plot Style to customize colors and add labels to the plot. You can also make these changes outside the form editor. For a description of color and label options, see "Customizing colors and labels".
  3. Close the Smart Tag dialog. To enable your changes, exit the form editor. Note: Changes you make to the plot do not take effect until you exit the form editor.
