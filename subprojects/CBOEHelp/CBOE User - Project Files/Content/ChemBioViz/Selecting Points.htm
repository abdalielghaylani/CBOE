Selecting Points

To select a subset of the points, click+drag a rectangle around them. When you release the mouse button, the enclosed points are selected. To select additional points, press SHIFT as you drag. On release the mouse button, the enclosed points will toggle their selected states. To deselect all points, click an empty region on the plot.

Once you have selected a set of points, you can zoom to make the set fill the plot window or create a hitlist of the corresponding records.

This feature does not apply to subform plots.
