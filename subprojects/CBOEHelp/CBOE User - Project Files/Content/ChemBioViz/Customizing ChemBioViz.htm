Customizing ChemBioViz

Startup Settings

You can choose which forms and panes are displayed when you start ChemBioViz. To modify the settings, go to File>Preferences and select the Main tab. Then choose from the options described in this section.

Settings File

  • Get settings file from server

Select this option to save your settings on the ChemBioViz server. Your options are available to you on any computer you use to log into ChemBioViz.

  • Always ask for saving settings on closing

This option requires ChemBioViz to ask you whether you want to see your current settings.

Open at start

  • Mode

Choose the form you want to appear whenever ChemBioViz starts.

Choose Blank form to build a custom form from an empty template.

Choose a default form to appear. Select Default form chosen below and then choose a form from the Default Pane dropdown list.

Choose Last form viewed so that the form that is in view when you exit ChemBioViz is displayed the next time you launch ChemBioViz.

  • Form Visibility

ChemBioViz allows any two forms to have the same name as long as one of the forms is public and the other is private. When you select a form in the Default Form dropdown list, indicate under Form Visibility whether you want to select the Public or Private version of the form in case there are two forms with that name.

  • Default Pane

Choose the Default Pane that appears in the navigation pane each time you launch ChemBioViz.
