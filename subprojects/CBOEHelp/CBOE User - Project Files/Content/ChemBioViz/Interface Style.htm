Interface Style

To change the look of the ChemBioViz interface, select a new interface style. The style determines the color of the interface but not its layout.

To change the style:

  1. Go to File>Preferences and select the style tab.
  2. Select a style from the dropdown list and click Apply.
  3. Click OK.
