Drill Down Forms

You may find it easier to view child table data in a separate window, called a drill down form. You view the drill down form alongside the main form. As you navigate through records, the two forms stay synchronized, always displaying the same data.

To create a drill-down form:

  1. Open a form that includes a subform connected to a child table.
  2. Select Dataviews in the navigation pane.
  3. In the dataview tree, double-click the child form to open it in form view.
  4. Save the child table form in a database or on your local drive.
  5. Open the main form that contains the child table.
  6. Open the main form in the form editor.
  7. In the form editor, select the child table in the main form.
  8. At the bottom of the Properties window, locate the ChildFormName field.
  9. In the ChildFormName field, enter the file name or local path of the child form you saved.
  10. Click OK to exit the form editor.
  11. Save the main form.

To use a drill down form:

  1. In the main form, double-click any row in the child form table. The child form table opens in its own window.
  2. Browse through the child form records in the parent form. The data in the drill down form and the main form is the same.
