Print Setup

Users can change the page orientation or add headers/footers to the page before printing. To set up printed pages, go to File>Page Setup.

The Page Setup window includes a diagram that illustrates how the printout will look using the current settings. The large rectangle in the diagram shows the print margins. The content at the top and bottom of the diagram indicates the positions and text of the headers and footers that will appear on all printed pages. The diagram for forms includes an additional rectangle that indicates where the form will appear on the printed page.

The scrrenshot above shows the print setup diagram that illustrates the layout of the printed page.

To set up a page for printing:

  1. In the main window, select the tab for the table or form to print.
  2. Go to File>Page Setup. The Form or Grid Page Setup dialog box appears.
  3. In the dialog box, select the Settings tab.
  4. Select the paper Size and Source.
  5. Select the Orientation and Margins.

Note: The form size is determined by the tightest rectangle that fits around the form boxes and labels. To get more white space around the border, edit the form and create a group box around the space to print. To get more forms per page, create a special tab for printing, with boxes laid out horizontally.

Headers and footers

Users can add custom headers and footers text to their printed pages. ChemBioViz provides several substitution codes so that users can also add system information to the text as desired.

The substitution codes are:

Code Description Example
# Page number Page #
$F Form name Drug Lookup
$T Long time 12:45:53 PM
$D Long date Thursday June 10 2010
$U User name JSmith
$t Short time 12:45 PM
$d Short date 6/10/2010

An example of the syntax you can use is shown below.

To add a header or footer, select the appropriate tab in the Page Setup dialog box and type the text as desired. To see what your changes look like, expand the page setup dialog box to view the setup diagram. Click OK when finished.
