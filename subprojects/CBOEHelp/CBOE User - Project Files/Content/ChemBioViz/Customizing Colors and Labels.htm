Customizing Colors and Labels

You can modify the colors and labels in standard plots and subplots.

  1. In the form, select the plot to modify.
  2. Click Plot Control in the navigation pane.
  3. In the Plot Control, click Settings. The Plot Properties dialog box opens.
  4. In the Plot Properties dialog box, choose the Style tab and select from these options:
  • Plot
    Style The chart type--bar, pie, etc.
  • Background
    The chart color.
  • Selected
    The color of data points in the plot whenever the you select them.
  • Stretch on resize
    Cause the plot to resize when you resize the form (the plot edges are anchored to the form edges).

Tip: Stretch on resize works best if the lower right corner of the plot is near the lower right corner of the form before you check the box.

  • Data Point
    Select the style, color, and size of the data points that appear in the plot.
  • Highlight
    Select the style, color, and size of any data point that the user selects in the plot.
  1. Select the Labels tab.
  2. (Optional) Enter a title in the Title text box.

Note: The Title you enter in the Labels tab will override the title entered in the Smart Tag dialog box in the form editor.

  1. (Optional) Enter a plot Legend in the text box.
  2. (Optional) Enter labels for the X Axis and Y Axis. To use the names of the data columns, select Auto label axes.
  3. (Optional) For the x- and y- axes, select Grid to draw grid lines on the plot.
  4. Click OK in the Plot Properties dialog box.
