Main Toolbar

You can choose which commands are available on the main toolbar or any of its menus.

To add a command to the main toolbar:

  1. Right-click the main toolbar and select Customize. The Customize dialog box appears.
  2. In the Toolbars tab, select the Main toolbar.
  3. Select the Commands tab.
  4. Click+Drag an icon from the Customize pane onto the toolbar or menu where you want it.

Note: For additional options, such as to display text, right-click a command icon on the toolbar.

  1. Click Close, to close the Customize dialog box.

To remove a command:

  1. Right-click the main toolbar and select Customize. The Customize dialog box appears.
  2. In the Toolbars tab, select the Main toolbar.
  3. Click+drag the command icon from the toolbar and release the mouse button.
  4. Click Close, to close the Customize dialog box.
