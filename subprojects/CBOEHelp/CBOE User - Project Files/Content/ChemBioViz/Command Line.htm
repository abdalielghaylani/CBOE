ChemBioViz
User's Guide

Command Line

You can run ChemBioViz from the DOS command line so that other applications can launch ChemBioViz and pass information to it as it starts up.

The command line options are listed below. To view this list at any time, open a DOS window and type:
ChemBioViz.net.exe /?

The command line format for launching ChemBioViz is:
chembioviz.net.exe [<form>] [/option=value ...]
where <form> can be one of these values:

  • The pathname to an XML file
  • The name of a public or user form (If the name contains spaces, enclose in double quotes).
  • Blank (if no form is to be opened on startup.)

Option

/login=name,password

  • Description:
    Automatically log into ChemBioViz upon startup, bypassing the login dialog box. You pass the name/password from another ChemBioOffice Enterprise application. The /server option is required when using /login.
  • Examples:
    chembioviz.net.exe C:\myform.xml /login=me,mypass /server=abc.camsoft.com /query=Q2
    chembioviz.net.exe /server=2-tier/myoracleservice /login=me,mypass

Option

/auth=auth_ticket

  • Description Automatically log into ChemBioViz upon startup, bypassing the login dialog box by using an authentication ticket. You pass an encryption authentication ticket from another ChemBioOffice Enterprise application. The /server option is required when using /login.

Option

/server=name

  • Description
    server address. This option is ignored if there is no /login or /auth for 2-tier use server=2-Tier[/oraservice]
  • Example
    chembioviz.net.exe "CS Demo Props" /login=me,mypass /server=abc.camsoft.com
    chembioviz.net.exe /server=2-tier/myoracleservice /login=me,mypass

Option

/search=fieldname:query

  • Description
    perform text or numeric search on open
  • Example
    chembioviz.net.exe CS_Demo /public /search=molweight:>200

Option

/query=queryname

  • Description
    Run named query on open
  • Example
    chembioviz.net.exe C:\myform.xml /login=xxx /server=xxx /query=Formula:C7-8

Option

/public

  • Description
    named form is in Public collection
  • Example
    chembioviz.net.exe CS_Demo /public /search=molweight:>200

Option

/user or /private

  • Description
    named form is in User collection
  • Example
    chembioviz.net.exe CS_Demo /public /search=molweight:>200

Option

/nopanel

  • Description
    hide explorer panel on startup
  • Example
    chembioviz.net.exe /nopanel

Option

/print

  • Description
    Print a hitlist. You must specify the login options and the hitlist query.
  • Example
    chembioviz.net.exe C:\myform.xml /print /login=name,pwd /server=mothball /query=mol_id:1-100

Option

/help or /?

  • Description
    display the command line options listed in this guide
  • Example
    /help or /?
