Export to CBV Excel

To Export a hitlist to CBV Excel:

  1. In the Queries pane, select the last query you ran (The query name is bold when selected)
    Note: If you want to export an old query you need to re-run it first because the export command is applied to the current query results
  2. Go to File>Export>To CBV Excel

  1. The following warning message displays. Click Yes to continue

  1. Specify the location to save the exported excel sheet and click on Save button

  1. A message informing that the file has been exported displays. Click Ok .
