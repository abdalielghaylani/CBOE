Drawing Query Structures

Some forms include a ChemDraw box, which lets you draw a query structure for substructure, full structure, similarity, and tautomeric search types. For more information, please refer to "Structure search types".

To draw a structure, click in the ChemDraw box. The ChemDraw main toolbar appears. For information on how to draw structures using the ChemDraw toolbar, please refer to the ChemBioDraw User Guide.

Note: To use the ChemDraw toolbar on Windows Vista, you must have Vista Service Pack 1 installed.
