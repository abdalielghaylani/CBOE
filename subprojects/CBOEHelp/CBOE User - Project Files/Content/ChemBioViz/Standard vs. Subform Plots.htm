Standard vs. Subform Plots

Values to be plotted can be taken from either one or two columns of data in the database. For two-column plots, both columns are typically taken from the base table connected to the form. In a subform plot, both are from one selected child table.

A) A subform plot. To view a plot for a subform record, click its data point in the base table plot. B) a base table plot. Each point on the plot represents an record in the base table. The green dot represents the record that is displayed in the form.

A standard plot shows a point for each record in the current hitlist (roughly) , where a subform plot shows points only for the linked child records. As you browse through records in the form, a standard plot remains fixed and shows a moving highlight point, where a subform plot is recreated with a new set of data for each record.
