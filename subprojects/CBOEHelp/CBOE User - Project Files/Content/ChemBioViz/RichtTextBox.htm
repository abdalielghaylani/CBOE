RichtTextBox

Use a rich text box to display formatted text and numbers. Formatting can include bullets, text styles, and active Web addresses.

Example

Options

  • To add or edit text in the text box, click the arrow above the box and select Edit Text Lines.
  • To expand the text box to the size of its container (such as a group box) select Dock in parent container.
