Performing a Simple Query

To perform a simple query:

  1. In either the Public forms or My forms panel, select a form to use in your query. The form opens.
  2. Select the Query tab.
  3. Select or enter the query criteria in the form.

Note: You don’t need to fill in every field. However, ChemBioViz will return only those records that meet all entered criteria.

  1. Click Search.

After your search is complete, use the Records Number box to browse through the results. Your query is stored in the Queries pane.
