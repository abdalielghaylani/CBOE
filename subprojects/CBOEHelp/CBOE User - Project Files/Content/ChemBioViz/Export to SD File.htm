Export to SD File

ChemBioViz allows users to export query results to a Structure Data File (SDF). Users can then use the SDF file in other applications.

To export query results to SD File:

  1. In the Queries pane, select the query to export. (The query name is bold when selected)
    Note: If you want to export an old query you need to re-run it first because the export command is applied to the current query results
  2. Go to File>Export>SDFile. The SD Export dialog box opens

  1. In the SD Export dialog box, browse to the destination output file
  2. In the Output File text box, enter a name for the SD File and click on Save button

  1. Select an option for exporting child tables (these options are relevant only for results that contain child tables)
  • None: No child table is exported
  • Flat: Each row contains all child data, with multiple child items concatenated into multi-line data items
  • Nested: Each main row contains all child data, with each child table condensed within a single column
  1. Click on Go button to export the query results
