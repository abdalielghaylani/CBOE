Adding Elements to Forms

To add an element to a form, Click+Drag the element from the Toolbox to the form.

After you add several elements, you can move them into a logical arrangement so that data is easy to read when you use the form. To move an element, click and drag it to the desired location.

As you arrange multiple elements around the form, ChemBioViz displays colored rulers. The rulers are there to help you align the elements but don’t appear in your form.

To select more than one element at a time, SHIFT+Click each element. You can then move or resize them all at the same time.
