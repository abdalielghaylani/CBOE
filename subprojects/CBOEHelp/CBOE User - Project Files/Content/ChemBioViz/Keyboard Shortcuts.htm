Keyboard Shortcuts

You can assign keyboard shortcuts to most ChemBioViz commands.

To assign a shortcut key:

  1. Right-click the main toolbar and select Customize. The Customize dialog box appears.
  2. In the Toolbars tab, select Main Toolbar.
  3. Click on the Keyboard button. The Customize Keyboard dialog box appears.
  4. In the Commands window, select a command to which you want to assign a shortcut key.
  5. In the dropdown list, specify a shortcut key to assign to the command.
  6. Click Assign.
  7. Click Close.
