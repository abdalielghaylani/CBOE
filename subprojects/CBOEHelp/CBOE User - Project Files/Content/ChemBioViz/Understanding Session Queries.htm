Understanding Session Queries

Any new query you run is a session query by default and has an asterisk at the end of its name in the Queries pane. Session queries are not saved when you logout. To save a query (remove the session status), right-click the query in the Queries pane and either rename the query or select Keep. When you save the form, ChemBioViz will save the query as well.

Note: You must save the form when prompted prior to saving any new query you have run using that form.
