Binding Data to an Element

When you bind data to an element, you link it to a field in the dataview or enable it to perform an action. For example, you can enable a button to launch a Web site or populate a drop-down list. For most elements, to bind data, you must select the element in the form editor and choose the binding options in the properties window. The steps are shown below. To bind to a SubformGrid please refer to the section below:

To bind data to an element:

  1. Select the element in the form editor.
  2. In the Properties window, select Advanced under {DataBindings} in the Data section.
  3. Click the Advanced button: . The Formatting and Advanced Binding dialog box opens.
  4. In the Binding drop-down list, select the field to which you want to bind to the element.

Binding data to a SubformGrid

  1. Select the SubformGrid element in the form editor.
  2. Click the arrow at the top of the form. The CBVDataGridView Tasks dialog box opens.
  3. In the drop-down list, select a table to be the data source.
  4. In the checkboxes, select the permissions you want all the user to have for editing the subform.
  5. To edit any of the columns in the subform, select Edit Columns.
  6. To add a column to the subform, select Add Column.
