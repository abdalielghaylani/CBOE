Creating a Plot

You create a plot on a form in the form editor (you must have privileges to edit forms). After creating the plot, you can modify it outside the form editor using the plot control.

To create a plot:

  1. Open the form in the form editor.
  2. In the toolbox, select the CBVChartControl icon and drag it into the form.

A default plot box appears with a grid and several random points in it. A default plot that appears when first created. To assign data to the plot, click the arrow (circled) to open the plot Smart Tag dialog box.

  1. Resize and position the default plot as desired.

Tip: When creating a standard plot, position its lower right corner near the lower right corner of the form. The plot is automatically anchored so that it resizes as you size the form.
