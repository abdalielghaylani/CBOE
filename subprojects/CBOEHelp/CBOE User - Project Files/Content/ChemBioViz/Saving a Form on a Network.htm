Saving a Form on a Network

The forms that you save on the network can be either public or private.

To save a form on a network:

  1. Select the form tab.
  2. Go to File>Save as.

  1. In the Save Form As window:
  1. Enter a name for the form or keep the default name (dataview name).
  2. In the Comments text box, enter information about the form that you want to include.
  3. Select whether you want the form to be public or private.
  4. Click OK.
