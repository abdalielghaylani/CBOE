Tabs 

When you start ChemBioViz for the first time, you will see three tabs above the main window- Form, Table, and Query.

  • Form

Select the Form tab to view data using a form that you have chosen; either select a form in the navigation pane or open a local form. Any blank forms or form copies that you create will also appear as tabs.

  • Table

Select the Table tab to view your data in a tabular format instead of a form. The Table and Form tabs are synchronized such that, as you navigate to a particular record in one tab, the record also appears in the other tab.

  • Query

After you open a form, select the Query tab to enter criteria to search the database you chose in Dataviews. Only those records that meet all your search criteria are returned (see "Queries").
