Advanced Query Options

For more advanced queries, you can define how ChemBioViz searches for structures based on specific atom, bond, or fragment properties. Click Advanced in the Search tab and select from the options described below.

General Tab

Hit any charge on heteroatom

Allows uncharged non-carbon atoms in the query to match charged atoms in the target.

Note: Charged atoms in the query will return charged atoms in the database regardless of this setting.

Reaction query must hit reaction center

Requires that any reaction center present in the query overlaps with the reaction's center in the target. This option applies only to reaction searching.

Permit extraneous fragments in full structure or reaction searches.

Allows hits to contain molecular fragments in addition to the structure hit by the query.

On carbon

Allows uncharged carbon atoms in the query to match charged carbon atoms in the target.

Stereochemistry Tab

Before you perform a stereochemical query, you must first set the search parameters:

  1. Go to File>Preferences. The Preferences dialog box opens.
  2. In the Preferences dialog box, click the Search tab.
  3. Click Advanced.
  4. In the Advanced Search Preferences dialog box, select the Stereochemistry tab.
  5. In the Stereochemistry tab, select the Match stereochemistry check box.
  6. Select the appropriate Tetrahedral stereo center and Double bond options and click OK. The dialog box closes.
  7. In the Preferences dialog box, click Apply.
  8. Click OK to close the Preferences dialog box.

After you set the stereochemical search parameters, you can perform the query:

  1. In the query form, draw the query structure.
  2. In the form, click Search.
