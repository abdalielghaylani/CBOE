Smart Tags

Smart Tags provide an easy way to select and associate binding properties to a particular element in the form editor instead of selecting them from the properties script. Only administrator users with adequate privileges are allowed to do this.

To use smart tags:

  1. Log in to ChemBioViz as system administrator
  2. Select a Form and click on Edit Form Tab
  3. Select the element in the form editor and click on the small arrow at the upper right corner of the box, a dropdown list displays with the available values. Select the desired values.
