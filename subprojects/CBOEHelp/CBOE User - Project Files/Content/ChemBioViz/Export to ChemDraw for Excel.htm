Export to ChemDraw for Excel

To export to ChemDraw for Excel:

  1. In the Queries pane, select the query to export. (The query name is bold when selected)
    Note: If you want to export an old query you need to re-run it first because the export command is applied to the current query results
  2. Go to File>Export>To ChemDraw for Excel

  1. The following message displays. Click on Yes button to continue

  1. Check the fields to be exported to the ChemDraw for Excel sheet like structure, amount, etc... and select an option for exporting child tables (these options are relevant only for results that contain child tables)

  • None: No child table is exported
  • Flat: Each row contains all child data, with multiple child items concatenated into multi-line data items
  • Nested: Each main row contains all child data, with each child table condensed within a single column
  1. Click on Go button to export the query results
  2. A ChemDraw for Excel sheet opens with a confirmation message. click on Yes button to continue

  1. The excel sheet gets populated with the structures and various selected fields
