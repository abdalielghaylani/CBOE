The Queries Pane

After you run a query, it gets added to the Queries pane.

The screen shot above shows an example of a query:

  • A) The query name
  • B) A parameter used in the query
  • C) Another parameter used in the query
  • D) Total records returned.

By default, the query name begins with ‘Q’ followed by a number n to show that it is the nth query created for that form. The number is followed by a list of parameters used in the query and then total records the query returned.

You can rename the query to suit your needs.

To rename a query:

  1. Right-click the query in the Queries Pane and select Rename. A dialog box appears.
  2. In the dialog box, enter the new name and click OK.

By default, all queries are listed under the Queries folder. You can add sub-folders to the list of forms and move your forms to it.

To add sub-folders:

  1. Right-click an existing folder in the Queries pane and select New Folder.
  2. In the dialog box that appears, enter a name and click OK. A new folder appears in the list.

To move a query to the folder, click+drag the query to the corresponding folder.
