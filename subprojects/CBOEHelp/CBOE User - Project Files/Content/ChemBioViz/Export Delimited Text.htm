Export Delimited Text

Users can use delimiter in their exported files. Two delimiter types are available: Comma for the .csv extension and tab for .txt extension.

To Export a hitlist using the Delimited Text option:

  1. In the Queries pane, select the query to export. (The query name is bold when selected)
    Note: If you want to export an old query you need to re-run it first because the export command is applied to the current query results
  2. Go to File>Export>Delimited Text option

  1. The Delimited Text Export dialog displays:

  1. Specify the output file destination, check the fields to be included in the exported file and select the delimiter option:
  • Comma for .csv extension
  • Tab for .txt extension
  1. Click on Go button.
