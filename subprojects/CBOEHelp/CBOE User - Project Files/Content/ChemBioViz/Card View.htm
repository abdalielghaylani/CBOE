Card View

Card view displays data in blocks instead of rows with field headings listed vertically. The Card view is also applied when you print.

To view records in card view:

  1. Open a form and display a table tab.
  2. Right-click the tab at the top, and choose Card View from the context menu.

Note: To switch back to table view, choose the command again.

  1. To change the style, expand Card View Style in the context menu and choose one:

Standard Labels

Each card has its own set of labels

Merged Labels (default)

One set of labels at the left applies to all cards

Variable Height

Cards show only rows containing data

Compressed

Each card is shown as a button displaying one item of data, expandable to show the full card. The item displayed on each card depends on the order of the columns in the grid. To keep the tab in card view, save the form. The card view settings are saved with the form.

Structure Position

Specify whether you want the structure to be positioned at the Top, Bottom, Left or Right of the cards.
