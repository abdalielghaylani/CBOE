ChemBioViz Introduction

Welcome to the ChemBioViz chapter.

ChemBioViz is an easy-to-use environment that allows users to search chemical databases for alphanumeric and structural data. You can perform substructure, stereochemical, tautomeric, and a variety of other queries and even combine results from multiple queries to narrow or expand your results. You can then display and browse through your results in nearly any format you want using tables and custom forms.

This chapter explains how to use ChemBioViz to search chemical databases. To help you get started, we begin by describing the ChemBioViz environment. We then explain how to run queries and introduce some query techniques that you can use to help get the results you want. Finally, we explain forms, how to use them, and how you can create your own forms to run queries and view the results the way you want.
