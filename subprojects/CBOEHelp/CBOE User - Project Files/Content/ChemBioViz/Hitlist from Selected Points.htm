Hitlist from Selected Points

You can create a hitlist from selected points in a plot and examine their corresponding records. The result is the same as if you had run a query and hit only the selected points. Similarly, you can create a hitlist from selected records in a table. Please refer to "Hitlist from selected results".

To convert selected points into a list:

  1. Click+drag the points to select. SHIFT+drag points in the plot to modify the selection.
  2. Right-click the plot and choose Selection To List.

A new entry labeled “Point selection” appears in the Queries tree. This becomes the current list, and the plot is redrawn.

A point selection query cannot be restored (displayed in a query form), but it can be rerun, merged, saved, and otherwise handled like any other list-based query.

Note: Lists can be made only from points selected in standard plots.
