Selecting Data Fields

Typically, data fields assign to a plot when the plot is created in the form editor. However, if you want to view other data the plot, you can do so without the form editor.

To select data to plot:

  1. In the navigation pane, select Plot control.
  2. In the form, select the plot.
  3. In the plot control, select the data to plot using the X and Y drop-down lists.
