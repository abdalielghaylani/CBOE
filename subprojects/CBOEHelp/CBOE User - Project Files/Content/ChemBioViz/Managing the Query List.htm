Managing the Query List

By default, all the queries you run are listed in the Queries folder. To add sub-folders to the list:

  1. Right-click the Queries folder and select New Folder.
  2. In the dialog box that appears, enter a name for the new folder and click OK. The folder appears in the Queries pane.

You can create as many folders as you want. After you have created all the folders you want, you can click+drag queries and other folders into them as desired.

To save the query list, right-click any query in the list and select Keep All and save the form.
