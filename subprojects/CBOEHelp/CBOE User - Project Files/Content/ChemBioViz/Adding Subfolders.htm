Adding Subfolders

By default, all forms are listed under the My Forms or Public Forms folder. You can add subfolders to the list of forms and move your forms to it.

To add subfolders:

  1. Right-click an existing folder in the forms pane and select New Folder.

  1. In the dialog box that appears, enter a name and click OK. A new folder appears in the list.

To move a form to the folder, click+drag the form name in the tree to the folder name.
