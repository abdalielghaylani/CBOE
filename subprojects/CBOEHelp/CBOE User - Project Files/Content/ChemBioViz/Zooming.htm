Zooming

You can zoom in for a closer look at any region of a standard or subplot.

To zoom to a region:

  1. Right-click the plot and X to enlarge the plot horizontally, Y to enlarge vertically, or Both to enlarge in both directions.
  2. Click+drag the region to view:
  • If you zoom on X, the region is full height, and only the horizontal dimension is adjustable.
  • If you zoom on Y, the region is full width and only the vertical is adjustable.
  • If you zoom on both, the rectangle is fully adjustable to any dimensions.
  1. Release the mouse button. The selected region expands to fill the plot window. Scrollbars appear so you can view the unseen portions.

Zoom On Drag options apply only once. After you zoom, any subsequent drag operations will select points.

To zoom to selected points:

  1. Select a set of points.
  2. Right-click the plot and choose Zoom To Selection. The plot expands so that the selected points fit in the plot window.

To restore a zoomed view to normal size, do one of the following:

  • Right-click the plot and choose Unzoom. This restores the plot to fill the window.
  • Click a circle icon on the end of one of the scrollbars. This undoes the last zoom along the corresponding axis (You may have to repeatedly click the icons to fully unzoomed the plot).

Click a circle icon to undo the last zoom on the corresponding axis.
