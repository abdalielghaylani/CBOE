Highlighting Current Record

As you browse from one record to another, the plot point corresponding to the current record is highlighted. By default the highlight is a green circle.

Note: Record highlighting applies to standard plots only, not subform plots.

Highlighted record indicator; status bar point data

If you do not see a highlight when you move to a record, it might be because:

  • There is no point for the current record. If a record has no value on file for a field being plotted, no point appears. You can verify whether a record has a value by including a box for it on the form; when the box is blank, there is no plot point for the current record.
  • You have filtering in effect, and have filtered out the point. Please refer to "Filters".
  • The plot view is zoomed in and the point is outside the visible region. To view the highlighted point, adjust the scrollbars. Please refer to "Zooming".
  • The highlighted point is obscured beneath other points. In a dense cluster of points, the highlighted point may be underneath others;. To view the highlighted point try zooming in.
