Open Analysis From Template

Saved instances of Analysis can be opened through the Open Analysis From Template option.

To display a saved analysis:

  1. Log in to ChemBioOffice and run ChemBioViz application
  2. Open the saved analysis corresponding form, go to Spotfire menu and select Open Analysis From Template... option

Note: Spotfire supports working on a saved analysis only when its related form is opened in ChemBioViz.

  1. Browse to a previously saved instance of Analysis and open it.

  1. Spotfire launches automatically and loads the saved data in an editable page.
  2. After editing the file and while closing Spotfire a message displays asking whether to save the changes to the locally saved analysis or not. Click Yes to save the changes, or click No to discard the changes.
