User's Guide

Table View

The Table view shows several records at once.

You can adjust columns and rows in a table to help you view the data more easily.

Moving Columns

To move a column, Ctrl+click and drag the column heading to where you want it.

Sorting

To sort data by a particular column, right-click the column and select either Sort Ascending or Sort Descending.

To sort by more than one column:

  1. Select the Table tab.
  2. Go to Edit>Sort. The Sort dialog box opens.
  3. In the first drop-down list, select the first field to sort by.
  4. In the second drop-down list, select the second field to sort by.
  5. For each field you selected, indicate whether you want the data to be sorted by Ascending or Descending order.
  6. Click OK.

Hiding Columns

To hide or display a column:

  1. Select Edit Grid Tab in the navigation pane. The Field Chooser window appears.
  2. In the Field Chooser, select/deselect the columns you want to view.

Alternatively, you can click+drag the column heading out of the table window. To restore the column, select it in the Field Chooser window.

Resizing Rows and Columns

To change the column width or row height, click+drag the divider between the column or row headings.

Expanding Record Views

Some tables may have rows that can be expanded to show additional data. To expand or contract a row, click the corresponding + or - icon (tables without these icons don’t have additional data).

Renaming Columns

To rename a column, right-click the column and select Rename header.

Magnifying Images

Details in some structural data may be unclear in their current size. To magnify a structure, hover your cursor over it.

Hitlist from selected results

In the table view, you can select multiple rows and create a hitlist from these selected rows. The result is the same as if you had run a query and it hits only the selected records. Similarly, you can create a hitlist from selected points in a plot. Please refer to "Hitlist from selected points".

To convert selected records into a list:

  1. Click+drag the rows to select. SHIFT+drag rows in the table to modify the selection.
  2. Right-click the table tab and choose Selection To List.

Note: Be sure to right-click the table tab, not the selected rows.

The new hitlist appears in the queries pane and can be saved and viewed like other queries.
