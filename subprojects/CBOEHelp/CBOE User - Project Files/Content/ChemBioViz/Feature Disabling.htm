Feature Disabling

The "FeatsDisabled" value is a combination of features to be disabled. This setting is in app.config and may also be in the user's local ChemBioViz Settings file.

The default value zero, means that all features are enabled. To disable features, enter a sum of bit values:

  • Plotting = 0x01: hides plot control panel, removes chart control from toolbox
  • GrandChild = 0x02: prevents creation of grandchild subforms when creating form from dataview or when configuring subform box in form editor
  • DrillDown = 0x04: prevents double-click on subform row from opening drill-down form
  • TreeFolder = 0x08: prevents creating new folder in form or query tree
  • ActionMenus = 0x10: prevents displaying custom action menus
  • QueryTextBox = 0x20: creates standard text boxes for query forms, not query boxes
  • SubformQueryBoxes = 0x40: prevents auto-generation of subform query boxes
  • CardView = 0x80: prevents changing grid to card view (choice removed from menu)
  • Export = 0x100: removes Export menu and SD submenu
  • ComboBoxes = 0x200: removes combo box from toolbox in form editor

Example:

To disable plotting, action menus, and combo boxes, set FeatsDisabled to 0x(1 + 10 + 200) = 0x211 = 529.
