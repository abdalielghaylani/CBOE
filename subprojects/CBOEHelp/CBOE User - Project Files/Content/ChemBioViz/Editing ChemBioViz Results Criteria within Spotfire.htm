Editing ChemBioViz Results Criteria within Spotfire

The results obtained in ChemBioViz may be edited through Form Wizard by editing their Criteria in Spotfire. To do that:

  1. Log in to ChemBioOffice, run ChemBioViz application
  2. Open a form and connect to Spotfire
  3. Within Spotfire, go to Tools and select "Edit ChemBioViz.NET Results Criteria..." option

  1. The following editor dialog displays, make the desired changes, select whether to show the entire fields, default fields,etc..and click on Finish button

Aggregate function

The Aggregate function can be set for fields in Spotfire, to do that:

  1. In the editor dialog above, expand the available fields at the left side of the window
  2. Drag and drop a particular field from the Child Table to the right side of the window under the base table fields.
  3. Select the desired Aggregate function from the dropdown list and click on Finish button
