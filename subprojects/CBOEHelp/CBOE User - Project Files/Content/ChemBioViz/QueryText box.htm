QueryText box

Unlike a standard textbox, a QueryText box lets users enter a list of query values such as “2,3,5-8,15”. Targets that hit any of the listed values are returned. The querytext box also lets users enter their query values from a text file rather than type them in. Values may be listed in the file on the same line (separated by a commas) or with each value on a separate line.

You can configure a querytext box by setting a default operator that determines how the values are applied to the query, such as “less than” or “does not contain”. See the Operator dropdown list in the Properties window for all available operators.

To add a querytext box:

  1. Select a query tab and open the form editor.
  2. Click-drag a querytextbox from the toolbox onto the form.

To bind data to the query text box:

  1. In the form, select the querytext box.
  2. In the Properties window, select (DataBindings)>(Advanced) in the Data section.
  3. Click the button.The Formatting and Advanced Binding dialog box opens.
  4. In the Binding drop-down list, select the field to bind to the button. The field may be either on a subform field or the main form.
  5. Click OK. The Formatting and Advanced Binding dialog box closes.

To configure the querytext box for mutiple query values:

  1. In the Properties window, navigate to the Misc. section
  2. In the Misc. section, select True in the AllowListInput dropdown list.

When you set the AllowListInput field to True, ChemBioViz ignores all commas that the user enters in the querytext box. Otherwise, all commas are assumed to be part of the query string.

Entering query parameters from a file

Setting the AllowListInput field to True also lets users enter the name of a file that contains query text values. For users to specify a filename, they must enter "@" followed by the file path. The file may include a single query per line, or more than one per line separated by commas.

For example, suppose you have a text file C:\queryinput.txt that has three lines. To search for any of the values in the file, enter "@C:\queryinput.txt" into the form box instead of a normal query string.

Default field operators

You can set a default operator to be applied to values the user enters and enable the user to override the default if desired.

To select a default operator:

  1. Select the operator from the Operator dropdown list.
  2. To allow users to override the default operator, select True in the UseCustomOperator dropdown list.
