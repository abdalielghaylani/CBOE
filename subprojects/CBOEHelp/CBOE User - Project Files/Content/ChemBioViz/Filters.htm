Filters

Filtering lets you interactively analyze plotted data. When applied to a plot, a filter lets you hide points based on values from a selected data field.

No filtering

For example, to eliminate low-molecular-weight compounds from an analysis, you add a molweight filter and set a minimum value. Then, any points with molweights below this value disappear from the plot.

You can apply multiple filters to a plot. Each filter can be associated with any numeric field from the base table, whether or not it is represented on a plot axis or in a form box. Filtering applies only to standard plots and filters are saved with the plot data.

Filtering out low molecular weight compounds. A screenshot of the Molweight filter control is shown below the plot for illustration.

You add filters using the Plot Control in the navigation pane. The name of the table that contains the available filter fields appears at the top of the Plot Control.

To apply a filter:

  1. In the form, select the plot to filter.
  2. In the navigation pane, select Plot Control. The plot control opens.
  3. In the Plot Control, click Add Filter. A filter control appears.
  4. In the filter control drop-down list, choose the field to filter.
  5. Click+drag the filter controls to indicate the minimum and maximum values to display in the plot. Points in the plot appear and disappear to reflect the values you set.
  6. To retain the applied filter, save the form.

To remove a filter:

Choose remove this filter from the drop-down list for the filter to remove.
