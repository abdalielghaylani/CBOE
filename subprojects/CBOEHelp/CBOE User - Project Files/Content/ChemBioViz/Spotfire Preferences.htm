Spotfire Preferences

It is possible for users to set the Spotfire preferences.

To launch the Spotfire Add-in preferences dialog:

  1. Log in to ChemBioOffice Enterprise and run ChemBioViz application
  2. Go to Spotfire menu and select Preferences option

  1. The Preferences dialog displays:

  • Auto-refresh on change of ChemBioViz hitlist: the system will update automatically the changes made in the ChemBioViz hitlist.
  • Ask before modifying Spotfire data table: after modifying the ChemBioViz hitlist in ChemBioViz application, the system will ask the user before updating the Spotfire data table.
  • Ask before closing Spotfire Analysis: when the user opens a locally saved analysis, the system will prompt the user to confirm before closing the analysis whether to update the local copy or not.
  • Maximum records in data table: allows the user to specify the maximum number of hits to be loaded in Spotfire.
