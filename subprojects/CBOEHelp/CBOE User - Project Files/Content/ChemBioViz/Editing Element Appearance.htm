Editing Element Appearance

One of the ways to customize a form is to edit the appearance of the elements in it. We describe the most commonly used customization options in this guide. To view all options, select the element you want to customize, go to View>Properties, and select from the Properties window.

Changing fonts and color

Many element types, such as buttons and labels, let you change the label font or the font used to display data. You can also edit the background and foreground colors.

To change the font or color of an element:

  1. In the form, select an element to change.
  2. In the Properties window, expand the Appearance section.
  3. (Optional) To change the font properties, expand the Font section.
  4. In the left column of the Properties window Appearance section, select the property to change.
  5. In the right column, select a property from the drop-down list or enter a value in the property field.

Note: For fields that require you to enter a value, click outside the field for the value you enter to take effect.

Adding labels

Some elements, such as the button and group box, have their own built-in label that you can edit. For other elements that you want to label, you can place the label next to the element.

To place a label next to an element:

  1. In the form editor, click+drag the Label element into the form and position it where you want the label.
  2. (Optional) Click+drag the resize handles to change the size of the label box.
  3. Click the label to select it.
  4. Go to View>Properties to open the properties window.
  5. In the Properties window, edit the Text field in the Appearance section and press ENTER.

To edit the label on elements that have their own label:

  1. In the form editor, select the element.
  2. In the Properties window, edit the Text field in the Appearance section and press ENTER

Modifying label properties

After you create a label, you can change its appearance, such as its color and font. To change the appearance.

  1. Select the label and go to View>Properties. The properties window opens.
  2. Modify the properties as desired.

Resizing an element

Most form elements have size adjustment handles. To adjust the size of an element, select the element and click+drag an adjustment handle.
