1D vs. 2D plots

In a typical (“2D”) plot, values for the X- and Y- axes are taken from two different fields in the database. The alternative is a “1D” plot, which shows database values on one axis against an integral index on the other. The index is not the record number, but the zero-based position in the list of points. For example, if you have ten points in the plot, the index is a number from 0 to 9.
