Textbox

Use a text box to display plain text and numbers.

Example

Options

  • To wrap text to the next line, go to Properties pane, expand Behavior section and select True from the multiline dropdown list.
  • To bind data to the text box, refer to the Data section in the Properties window, please refer to the "Binding data to an element" for more information, or use the Smart Tags.
