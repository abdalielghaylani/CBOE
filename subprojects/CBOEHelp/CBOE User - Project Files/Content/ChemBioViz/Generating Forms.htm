Generating Forms

To generate a Form:

  1. Right-click a table within a particular dataview and then select "Generate Form".

Note: If you don't want the child table to be generated, then you have to select "Generate Without Subforms" option.

  1. Go to File and then select Save. The following dialog displays:

  1. By default the dataview name is generated, edit it if desired, enter comments and then select whether you want to make the form Public (visible to ChemBioViz users) or Private (Visible only by You).
  2. Click Ok.
  3. The form will be added either under "My Forms" or "Public Forms" as specified in step 3 above.

Note: You can edit the form before saving as well as after saving. For more details please refer to Editing Forms topic.
