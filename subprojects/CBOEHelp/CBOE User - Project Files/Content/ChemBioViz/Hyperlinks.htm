Hyperlinks

Hyperlinks behave like buttons but you use them in either a subform or grid instead of using them in a form. A hyperlink appears in each row, enabling users to perform a record-dependent action.

To create a hyperlink, you first create a button on a form and connect it to the table data:

  1. In the form editor, create a button to perform an action. See "Button".
  2. In the Properties pane, expand Data>>DataBindings and then select (Advanced)
  3. Click on the ellipsis button . The Formatting and Advanced Binding dialog box opens.
  4. In the Binding drop-down list, select the field that you want to bind to the button. The field may be either on a subform field or the main form.
  5. Click OK. If you bound the button to a subform field, then the process is completed.

If you bound the button to a field in the main form, do one of the following:

  • If no grid with the bound column exists, create a new grid: right-click the form tab and select Add Tab>Grid from Form.
  • If a grid with the bound column already exists, save, close, and reopen the form.

Executing the hyperlink

In the grid or subform, double-click the hyperlink.

Hiding the form button

To retain the hyperlink, you must keep the button on the form. However, you can hide it.

To hide the button:

  1. In the form editor, select the button.
  2. In the Properties pane, select Visible in the Behavior section.
  3. In the Behavior drop-down list, select False.
  4. Exit the form editor.
