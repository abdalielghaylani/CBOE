Form Properties

Form properties include the form name, any comments you want to enter about the form, and whether the form is public or private.

To rename a form:

  1. Right-click the form in either the Public Forms or User Forms list and click Rename form.
  2. Enter a new name for the form and click OK.

To add comments to a form:

  1. Right-click the form and select Properties.
  2. In the Form properties dialog box, enter your comment in the Comment text box and click OK.

Note: It is not possible to rename the form from the Form Properties dialog.

Note: You cannot change the location (public or private) in the Properties dialog box. You can change the location only when you save the form.
