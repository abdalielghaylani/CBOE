Logging In to ChemBioViz

ChemBioViz application is accessed through the ChemBioOffice Enterprise home page. To access the ChemBioViz application:

  1. Enter https://servername/COEManager into your web browser address bar. <servername> is the name of the server where ChemBioOffice Enterprise is installed. The User Login page of ChemBioOffice Enterprise appears.
  2. Enter a valid username and password.
  3. Click the Log in button to display the home page of ChemBioOffice Enterprise.
  4. Click on ChemBioViz Client icon under "Applications and Utilities" section

  1. The following dialog displays, click on Install button

Once installed successfully, ChemBioViz client may be started using one of the following ways:

  1. Go to Start>All Programs>ChemBioOffice Enterprise 2010>ChemBioViz Client.

  1. Simply click on ChemBioViz Client icon under "Applications and Utilities" section.

The ChemBioViz login window appears.

Logging In

When you start ChemBioViz for the first time, you will see this login window:

Enter your name and password as indicated. In the Server text box, either enter the name of the ChemBioViz server or select the server from the dropdown list.

If you select a name from the list, the following dialog box displays:

  1. Enter either a service name or server address.
  1. Select either 3-tier or 2-tier mode and to enable the secure connection check the "Use SSL" checkbox and then click on OK button.

Each subsequent time you log in, you will need to enter only your name, password, and the server you want to use.

Note: If you need a user name, password, or the server name, contact your ChemBioViz administrator.
