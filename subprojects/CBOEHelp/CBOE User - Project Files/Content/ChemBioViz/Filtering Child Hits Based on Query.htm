Filtering Child Hits Based on Query

In a ChemBioViz query, while searching a field in a child table, it is possible to "Show Only Child Data Matching Entered Criteria" instead of pulling its parent record in the base table which will in turn pull in all its child records. This option allows users to only pull the parent and child records that meet the search criteria.

To enable this feature:

  1. Log in to ChemBioViz client
  2. Go to File and select Preferences option

  1. The Preferences dialog displays, click on the Search Tab

  1. Check the "Filter child hits based on query" checkbox.
