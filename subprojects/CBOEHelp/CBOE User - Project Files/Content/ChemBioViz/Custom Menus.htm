Custom Menus

You can add submenu items to the main ChemBioViz menu to invoke commands in forms much the same way that buttons and hyperlinks do. Since submenus are part of the form design, they become part of the main menu when you open the form and are removed whenever you close the form. By default, submenu items appear under the Action menu unless you specify other location.

To create a custom submenu item:

  1. Create a button to perform an action. See "Button".
  2. In the form editor, select the button.
  3. Open the Properties window.
  4. In the Misc. section, select True in the ShowOnMenu to dropdown list.
  • (Optional) In the MenuName field, enter the name of the main menu item under which you want the submenu item to appear. By default, the submenu item will appear under the Action menu. If the name you enter doesn’t currently exist as a menu item, a new menu item will be added and the submenu item will be listed under it.

Note: Do not enter any of the standard menu item names (File, Edit, View, or Help) in the MenuName field.

  1. (Optional) To create multiple custom menus, create multiple buttons on your form and give each of them different menu names.
