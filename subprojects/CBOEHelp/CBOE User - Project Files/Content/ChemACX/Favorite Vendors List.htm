Favorite Vendors List

You can also search for substances among a list of your favorite vendors.

To create or edit the favorite vendors list:

  1. Access the ChemACX database to display the search form.
  2. Click on the "Edit Favorite Vendors List" link. The vendor list appears:

  1. Select the check box corresponding to the vendors whom you wish to include in the favorite vendors list.
  2. Click on "Save as Preferred List" button.
