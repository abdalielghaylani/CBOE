Opening ChemACX

To open a ChemACX database:

  1. Enter http://servername/COEManager into your web browser address bar. <servername> is the name of the server where ChemBioOffice Enterprise is installed. The ChemBioOffice Enterprise log in page appears.
  2. Enter a valid username and password.
  3. Click the Log in button to display the ChemBioOffice Enterprise home page.
  4. Click on Search Databases button within the Chemical Databases section in the ChemBioOffice Enterprise home page. The ChemOffice Enterprise Databases page appears showing the list of available databases.
  5. Click on ChemACX link
