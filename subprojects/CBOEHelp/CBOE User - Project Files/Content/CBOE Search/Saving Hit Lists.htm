Saving Hit Lists

You can save the list of records from a query and retrieve it later for viewing. If the database has been updated since the query was run, the retrieved hitlist will be outdated and you will need to rerun the query to get an accurate list.

To save a hit list:

  1. Perform a search. For more information, see "Searching Databases".
  2. Go to Hit lists> Save. The save hitlist window appears.
  3. Enter a List Name and Description for the hitlist.
  4. To enable others to view the hitlist, select the Make Public check box.
  5. Click OK. The hitlist is saved.
