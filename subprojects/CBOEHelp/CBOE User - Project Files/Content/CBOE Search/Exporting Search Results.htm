Exporting Search Results

You can save a hit list as a SD file.

When you export to a file, you use a template to define which fields are included. Always use the same template to ensure that you always select the same fields each time you export.

To export search results or hit lists:

  1. Go to File>Export Hits. The Export Hits window appears.
  2. Select a type of SD file:

Flat

Select this type to create multiple results for one field (a record is created for each result).

Nested

Select this type when multiple results for one field all appear in one cell.

  1. Choose the template from the Export Templates drop-down list.
  2. (Optional) If needed, select/deselect the fields under Export.
  3. Click OK. A new window opens.
  4. Click Click to Download. The File Download dialog box appears, containing a numerical name for the file, based on the date and time.
  5. Click Save to display the Save As dialog box.
  6. Select a location.
  7. Click Save.

The exported file flattens each record so that a compound with subform fields, such as multiple batches in the Registration Enterprise is found within one record in the exported file. All subform data is concatenated with '--' between fields. The first line indicate the field names.
