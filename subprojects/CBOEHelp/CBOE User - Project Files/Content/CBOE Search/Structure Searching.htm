Structure Searching

You can enter chemical structures as search parameters using the ChemDraw plug-in. Draw a structure in the plug-in window and ChemBioOffice will use the structure like any other query parameter to search the database. For information on drawing query structures, see the ChemBio Draw User Guide.

To perform a structure search:

  1. If the list or form view currently displayed, click on the New Query button to open the Query Input Form.
  2. In the Query Input Form, draw a chemical structure using the ChemDraw plug-in.
  3. In the SearchType drop-down list, select one of the structure search options (see below).
  4. Click Search.

ChemBioOffice Enterprise provides four structure search options:

  • Substructure
  • Full Structure
  • Exact Structure
  • Tanimoto Similarity
