Editing Query

You can enter new identifier information for a query, such as a new name and description.

To edit a query:

  1. Go to Queries>Manage. The Queries window appears.
  2. Select the query from the drop-down list in the window.
  3. Click Edit. The Queries window appears.
  4. (Optional) Modify the Query Name and Description fields.
  5. (Optional)To enable/disable others’ ability to view the query, select/deselect Public Query as needed.
  6. Click OK.
  7. Click Close to close the Queries window.
