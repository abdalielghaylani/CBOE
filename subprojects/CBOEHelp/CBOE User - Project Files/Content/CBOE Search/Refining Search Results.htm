Refining Search Results

If a query returns a large number of results, you can refine either the query or the hitlist by adding other search criteria.

To refine a query:

  1. While viewing the hit list, click Refine. This window appears:

  1. Select Edit current query and click OK. The Query Input Form appears.
  2. In the Query Input Form, enter the search criteria.
  3. Click Search to display the search results.

To refine a hit list:

  1. While viewing the hit list, click Refine. This window appears:

  1. Select Refine over current records found and click OK. The Query Input Form appears.
  2. In the Query Input Form, enter the search criteria.
  3. Click Apply. The current hitlist is filtered using the criteria you entered.
