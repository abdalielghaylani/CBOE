CBOE Search Introduction

Welcome to the ChemBioOffice Enterprise Search chapter.

ChemBioOffice Enterprise includes a powerful search engine that is common throughout the suite of applications.

This chapter demonstrates the types of searches that the users can perform with this feature and the results they can expect.

There are five basic types of search operations:

  • Structure search
  • Text search
  • Numeric search
  • Date search
  • Formula search

For more details please refer to the following topics:

  • Searching Databases
  • Refining Search Results
  • Setting Search Preferences
  • Managing Queries
  • Managing Hit Lists
  • Exporting Search Results
