Setting ChemBioViz Options

ChemBioViz offers several search options, as described below:

  • Maximum number of records to prompt:

Example: If this value is specified as 10 then a prompt message will appear if more than 10 hits are found. The message will prompt you to confirm whether you want to continue the search operation. If you click Yes, the search results will be displayed, depending on the specified maximum number of records to return.

  • Maximum number of records to return:

Example: If you specify this value as 30 and total hits found by the search operation is 50 then only the first 30 records will be returned.

  • Maximum hits to show structures:

Example: If you specify this value as 100 and the total hits found by the search operation is less than 100 then structures will be inserted into the sheet so that they are visible. If the number is greater than 100 structure data will be inserted but will not be visible. If a Unique Key is defined then the following structure text appears as Structure <Unique Key> in the structure cell. If not the structure text appears as Structure <Primary Key>.

  • Maximum structure height

Example: If you specify this value as 30 then only the structures whose height is 30 pixels or less will be returned.

  • Maximum structure width

Example: If you specify this value as 50 then only the structures whose width is 50 pixels or less will be returned.

To set the ChemBioViz options:

  1. Select ChemBioViz Options from the Setup menu. The ChemBioViz Options dialog box appears:

  1. Specify the maximum number of hits for displaying a prompt message.
  2. Specify the maximum number of records that will be returned after a search operation.
  3. Specify the maximum number of hits for displaying their structures in the results list.
  4. Click Structure Display Options.
  5. Specify the maximum height and width of the structures that will be returned. Structures having greater height or width will not be returned.
  6. Click OK.
