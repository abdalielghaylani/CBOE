Numeric

The Numeric, String, and Date fields contain these header options:

All (1 cell):

All the values are displayed in one cell, as shown below:

All (1 cell A-Z):

All the values are displayed in one cell, in ascending order.

All (1 cell Z-A):

All the values are displayed in one cell, in descending order.

All (Split Cells):

The values are displayed in separate cells, as shown below:

In the above figure, all the values of column D are confined to the first row. Therefore, the other rows of column D, corresponding to the rows of column C, are left blank.

All (Split Cells A-Z):

The values are displayed in separate cells, in ascending order.

All (Split Cells Z-A):

The values are displayed in separate cells, in descending order.

All (Multiple Rows):

The values are displayed in separate rows, as shown below:

In the above figure, all the values of column D are confined to the first row because All (1 Cell) is selected. However, all the rows of column D, corresponding to the rows of column C, contain the same data as displayed by the first row of column D.

All (Multiple Rows A-Z):

The values are displayed in separate rows, in ascending order. Also, the rows of other columns are filled with data of their first row.

All (Multiple Rows Z-A):

The values are displayed in separate rows, in descending order. Also, the rows of other columns are filled with data of their first row.

Average:

The average values are displayed for each row. If a cell of this column had multiple values, the average value is displayed in the cell.

Min:

The minimum values are displayed for each row. If a cell of this column had multiple values, the minimum value is displayed in the cell.

Max:

The maximum values are displayed for each row. If a cell of this column had multiple values, the maximum value is displayed in the cell.

Median:

The median of values consisting in a cell of this column is calculated and displayed in the cell.

Count:

The sum total of values consisting in a cell of this column is calculated and displayed in the cell.
