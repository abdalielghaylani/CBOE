Search Operators

Numeric

> (Greater than):

Displays results having value greater than the specified value.

< (Less than):

Displays results having value less than the specified value.

= (Equal to):

Displays results having value equal to the specified value.

>=(Greater than equal to):

Displays results having value greater than or equal to the specified value.

<= (Less than equal to):

Displays results having value less than or equal to the specified value.

!= (Not equal to):

Displays results that are not equal to the specified value. This operator is the same as <>.

- (Between):

Displays results based on specified range of values.

String

%<string>:

The % precedes one or more characters or group of characters to represent a string. Use prefix % to character(s) to represent any string of zero or more characters. It is case-insensitive. Example: %Benzene Returns: Any string that start with any character but ends with ‘Benzene’ string.

<string>%:

The % follows one or more characters or group of characters to represent a string. Use suffix % to character(s) to represent any string of zero or more characters. It is case-insensitive.

Example: Benzene%
Returns: Any string that starts with ‘Benzene’ but ends with any character.

LIKE <string>:

The <string> represents one or more characters, or group of characters in a match string. Use suffix <string> to LIKE keyword to represent any string of zero or more characters. It is case-sensitive.

Example: LIKE Benzene
Returns: Only Benzene string

NOTLIKE <string>:

The <string> represents one or more characters, or group of characters in a non-match string. Use suffix <string> to LIKE keyword to represent any string of zero or more characters. It is case-sensitive.

Example: NOTLIKE Benzene
Returns: All strings except Benzene

=<string>:

Returns all the results that match the specified string. It is case sensitive.

CONTAINS <character(s)/string>:

Takes an input of character or group of characters, and returns those strings that contain the specified character or group of characters. It is case-sensitive.

NOTCONTAINS <character(s)/string>:

Takes an input of character or group of characters, and returns those strings that do not contain the specified character or group of characters. It is case-sensitive.

STARTSWITH <character(s) >:

Takes an input of character or group of characters, and returns those strings that start with the specified character.

ENDWITH <character(s) >:

Takes an input of character or group of characters, and returns those strings that end with the specified character.

Structure

To specify a structure as search criteria:

  1. Double click the cell corresponding to Row 4 of a structure column. A confirmation message appears.
  2. Click Yes. ChemBioDraw Ultra is launched.
  3. Draw the structure using the ChemBioDraw Pro toolbar.
  4. Close ChemBioDraw Ultra. The drawn structure will appear in the selected cell of your worksheet.
