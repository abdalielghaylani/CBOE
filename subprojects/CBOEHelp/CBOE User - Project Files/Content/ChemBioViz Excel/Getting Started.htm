Getting Started

Accessing ChemBioViz Excel

You can access the ChemBioViz Excel add-in only when both ChemBioViz Excel and Microsoft Excel are installed on your system.

To access the ChemBioViz Excel add-in:

  1. Launch Microsoft Excel 2007. The following screen appears:

  1. Click on Log In button in the ChemBioViz Excel add-in toolbar. The Login dialog box appears:

  1. Enter your username and password to log into ChemBioViz Excel.
  2. Select the server where the dataview is located.

If the server you want is not listed in the Server drop-down list, add a server.

To add a server:

  1. Select Add server from the Server drop-down list. The Server dialog box appears:

  1. Select the type of architecture that your ChemBioViz application is using. If you select 3-tier, specify the middle-tier server name and choose whether to use SSL. If you select 2-tier, specify the Oracle service name.

Note: If you do not have enough information to perform the above step, contact your Administrator.

Note: The SSL feature is enabled only when you select the 3-tier mode and the server is on https protocol instead of http protocol.

  1. Click OK. The added server name appears in the Server drop-down list.

If required, you can remove a server from the Server drop-down list:

  1. Select the server in the Server drop-down list.
  2. Select "Remove selected item" from the Server drop-down list. A confirmation dialog box appears.
  3. Click Yes. The server is deleted.
  1. Click Login.

Note that ChemBioViz Excel does not support Microsoft Excel 2003.
