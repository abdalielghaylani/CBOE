Viewing Worksheet Properties

To view the properties of the current ChemBioViz worksheet, select Sheet Properties from the Sheet Options menu. The Sheet Properties dialog box appears:

The above screen displays information about the following properties:

Sheet Name:

The name of the current worksheet.

Login User:

The name of the current user.

Current Dataview:

The name of the dataview on which the current worksheet is based.

Connection Mode:

Specifies whether the current worksheet has been connected to the database in 2-tier mode or 3-tier mode.

Server/Service Name:

Specifies the name of the server or service associated with the current worksheet.

SSL:

Specifies whether SSL is being used.

Sheet Created On:

Specifies the date on which the current worksheet was created.

Sheet Created By:

The user who created the worksheet.

Sheet Modified By:

The user who last modified the worksheet.

Log Path:

The location where the log file will be saved.
