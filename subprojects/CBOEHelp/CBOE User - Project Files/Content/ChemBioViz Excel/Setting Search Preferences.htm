Setting Search Preferences

To set search preferences:

  1. Select Search Preferences from the Sheet Options drop-down menu.
  2. Select the check boxes in the General tab based on your search requirements. For more information on these check boxes, please refer to the "General" section below.
  3. Click the Stereochemistry tab.
  4. Select the options based on your search requirements. For more information on these options, please refer to the "Stereochemistry" sect
  5. Close the Search Preferences dialog box. The changes are saved.

The preference fields are described below:

General

  • Hit any charge of heteroatom

Description :Select this checkbox to allow uncharged non-carbon atoms in the query to match charged atoms in target.

Note: Charged atoms in the query must always match charged atoms in the target, regardless of this setting.

  • Reaction query must hit reaction center

Description :Select this checkbox if you require that any reaction center present in the query overlap with reaction centers in the target. This preference applies only to reaction searching.

  • Hit any charge on carbon


Description :Select this checkbox to allow uncharged carbon atoms in the query to match charged atoms in target.

Note: Charged atoms in the query must match charged atoms in the target, regardless of this setting.

  • Permit extraneous fragments in Full Structure (exact) Searches

Description :Select this checkbox to allow hits to contain molecular fragments in addition to the structure hit by the query.

  • Permit extraneous fragments in Reaction Full Structure (exact) Searches

Description:Select this checkbox to allow hits to contain molecular fragments in addition to the structure hit by the reaction searching query.

  • Query fragments can overlap in target


Description :Select this checkbox to allow fragments in the query to overlap (share one or more atoms) in the target.

  • Tautometric


Description :Select this checkbox to perform tautometric search.

  • Similarity search


Description :Select this checkbox to perform similarity search and specify the percentage of similarity desired.

Stereochemistry

  • Match Stereochemistry


Description :Select this check box to perform stereochemistry search. If you uncheck the Match Stereochemistry check box, all other options in the Stereochemistry tab are disabled.

  • Tetrahedral stereo center hits


Description :Select the tetrahedral stereochemistry option. The options are:

  • Same:
    Match the target exactly.
  • Either:
    Match same or opposite configuration at the center of the target.
  • Any:
    Match any target.
  • Double bond hits


Description :Select the double bond stereochemistry option. The options are:

  • Same:
    Match the target exactly.
  • Any:
    Match any configuration in the target.
  • Thick bonds represent relative stereochemistry

Description :Select this check box to match a relative relationship between centers.
