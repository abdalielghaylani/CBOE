ChemBioViz Excel Introduction

Welcome to the ChemBioViz Excel chapter.

ChemBioViz Excel is an add-in for Microsoft Excel that lets you run queries on dataviews and view the results in a spreadsheet.

If you have worked with ChemBioViz, many functions in ChemBioViz Excel will be familiar to you, you can select the dataview to search, enter your search parameters, and then sort and filter the results. ChemBioViz Excel is designed to display the search results in tabular format. If you would want to use forms to view your results or run queries, use ChemBioViz.

When you launch Microsoft Excel, the ChemBioViz Excel menu items are displayed in the toolbar. These menu items let you work with dataviews - you can connect to a dataview, open it in a spreadsheet, search the dataview, and display results.

To work with dataviews using ChemBioViz Excel, first log into ChemBioViz Excel then select a dataview using the New ChemBioViz Worksheet option from the toolbar.

Once you select a dataview, a new ChemBioViz worksheet is created wherein you can use the ChemBioViz Excel menu options to search data and configure the results. For example, you can sort and filter results or append the results of the current search to the results already displayed in the worksheet.
