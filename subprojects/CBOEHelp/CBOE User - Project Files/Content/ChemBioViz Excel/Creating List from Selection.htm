Creating List from Selection

At times, you may need to list a few selective items in a Word document. If you copy the selective items in Excel and paste in Word, the items will be displayed in the form of table, separated by carriage returns. However, if you would want to list the items in one line, separated by commas then create a list.

To create a list of items by selecting a few cells:

  1. Select the cell in which you want the list to be displayed.
  2. Select Create List from Selection from the Sheet Options menu. The Make List dialog box appears:

  1. Specify the range of cells to use to create the list.
  2. Click OK. The items of the cells within the specified range appear in the cell, you selected in step 1, separated by commas.
