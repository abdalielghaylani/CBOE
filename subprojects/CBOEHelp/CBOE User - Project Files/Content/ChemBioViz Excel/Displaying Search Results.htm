Displaying Search Results

The Option Header row (Row5) specifies how search results are displayed. The Option Header row contains drop-down lists that are populated based on the corresponding Table Header, Column Header, and data types of the fields.

ChemBioViz Excel classifies field data types into five categories:

  • Numeric
  • String
  • Date
  • OLE
  • Structure (Base64CDX)
