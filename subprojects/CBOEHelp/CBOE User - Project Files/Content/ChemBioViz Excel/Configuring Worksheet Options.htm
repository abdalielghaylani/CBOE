Verifying worksheet data

The header rows in a ChemBioViz worksheet display dataview information, such as the database name, table name, and primary key. For more information, please refer to "Selecting a dataview" topic.

A ChemBioViz worksheet is considered ‘valid’ if the data displayed in the header rows of the current worksheet maps correctly with the associated dataview. For example, the Category Header must display the base database of the dataview associated with the current worksheet.

To verify the worksheet data and determine whether the worksheet is valid:

  1. Log into ChemBioViz Excel and open a dataview.
  2. Select Verify Current CBV Sheet from the Sheet Options menu. If the worksheet is valid, a pop-up message informs that the current worksheet has valid tables and fields.

Note: If the Administrator has changed the dataview associated with the current worksheet, close and re-open your worksheet to incorporate the changes into the worksheet. If you do not incorporate the changes into your worksheet, it will become invalid and an error will appear when you verify your worksheet.
