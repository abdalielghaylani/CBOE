Logging Errors

The logging functionality lets you track errors that occur. For example, if you search the dataview for a value that is not there in the dataview, a search error will be logged.

To enable error logging:

  1. Select "Toggle Error Logging" from the Sheet Options menu. A success message appears confirming that logging has started. The message box also shows the path of the log file being created.
  2. Click OK.

To disable error logging, select Disable Error Logging from the Sheet Options menu.
