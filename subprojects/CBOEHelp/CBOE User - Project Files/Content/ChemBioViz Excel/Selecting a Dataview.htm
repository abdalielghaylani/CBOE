Selecting a Dataview

When you select a dataview to search, a new ChemBioViz Excel worksheet is created.

To select a dataview:

  1. Click on the New ChemBioViz Worksheet icon in the ChemBioViz Excel toolbar. The Dataview List dialog box appears:

Note: The Dataview List dialog box will appear only if you are logged into ChemBioViz Excel. If you are not logged in, the Login dialog box is displayed. The Dataview List dialog box displays all the dataviews in the currently logged-in server.

  1. Select a dataview from the drop-down list and click on OK button. A new ChemBioViz worksheet is created and data of the selected dataview is displayed, as shown below:

The first six rows of the worksheet shown above are used for data headers. The worksheet contains six default headers:

Category Header (Row 1):

Represents database name. By default, the Category Header displays the name of the ‘base database’. On changing the Category Header value, the corresponding Table Header, Column Header, Option Header, and Label Header text changes.

Note: The administrator creates a dataview from one or more databases containing one or more tables. While creating a dataview, the administrator defines a ‘base database’ and ‘base table’. The administrator also defines a column of the base table as primary key or unique key for the dataview.

Table Header (Row 2):

Represents table name. By default, the Table Header displays the name of the ‘base table’.

Column Header (Row 3):

Displays the primary or unique key, by default. However, if the base table has a structure column, two columns are displayed in the worksheet. The Column Header of the first column in the worksheet displays the name of the structure column while Column Header of the second column displays the primary or unique key.

Input Criteria (Row 4):

Lets you enter search criteria.

Option Header (Row 5):

Lets you specify how search results will be displayed.

Note: Row1, Row2, Row3, and Row5 have drop-down lists.

Label Header (Row 6):

Serves as the column name when search results are displayed.
