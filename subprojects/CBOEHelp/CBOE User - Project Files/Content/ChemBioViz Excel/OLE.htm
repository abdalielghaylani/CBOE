OLE

The OLE field contains these header options:

Image (1 cell -100%):

All images appear in a single cell in their default sizes.

Image (1 cell -75%):

The images are displayed in a single cell, and all the images are reduced to 75% of their actual size.

Image (1 cell -50%):

The images are displayed in a single cell, and all the images are reduced to 50% of their actual size.

Image (Split Cells):

The images are displayed in separate cells and corresponding cells in other columns are left blank, if they are not set to Split Cells.

Image (Multiple Cells):

The images are displayed in separate cells and corresponding cells in other columns are filled with the data of their first rows.
