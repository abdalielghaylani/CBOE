Inserting Columns

To insert a new column in the current worksheet, select Insert Data Column from the Setup menu. A new Header Category cell is created in the form of a drop-down list, as shown below:

By default, the new header category appears next to the existing column and contains category items. However, you can create new header category in any column, as shown below:

The Table Header row is populated based on the dataview selected in the Category Header row. The Column Header row is populated based on the table selected in the Table Header row.

To create new header category in any column:

  1. Click the column header so that the entire column is selected.
  2. Select Insert Data Column from the Setup menu.

Note: You can create more than one category header by selecting multiple columns. To select multiple columns, press the CTRL key and click the headers of the columns, which you want to select.
