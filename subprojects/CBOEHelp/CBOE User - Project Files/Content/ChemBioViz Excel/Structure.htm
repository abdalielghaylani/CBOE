Structure

The Structure field contains these header options:

Substructure:

Returns structures that match the specified query and have additional attachments at open positions.

Full:

Returns structures that accurately match the specified query. Matching isotopes are also returned.

Identity:

Returns structures that accurately match the specified structure and stereochemistry. Thus, duplicates are avoided.

Similarity:

Returns structures having structural features that generally correspond to those in the query.
When a table contains a structure field, the Column Header (Row3) drop-down list displays two additional items:

<Structure>.Formula:

Select this to display chemical formula instead of structure.

<Structure>.MolWeight:

Select this to display molecular weight instead of structure.
