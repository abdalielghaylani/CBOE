Searching

The Input Criteria row (Row 4) of the ChemBioViz worksheet lets you enter search criteria. Depending on the type of table/field column, the operators, which you can use in search criteria, are: numeric, string, and structure. For more information on these operators, please refer to "Search Operators" topic.

To search a dataview:

  1. Open the dataview after logging into ChemBioViz Excel. The worksheet appears as:

  1. Enter the search criteria in Row 4 corresponding to the column on whose basis you want to search. For information on how to enter a structure as search criteria, please refer to "Structure" topic.
  2. Select appropriate items from the drop-down lists in the Option header row (Row5) to define how the search results will be organized and displayed. For more information, please refer to "Displaying Search Results" topic.
  3. Select from these options in the Search menu:

Update Current Results:

Updates the existing results by taking input data from existing ChemBioViz worksheet. This feature considers the criteria row and unique key/primary key column data. If both primary key and unique key exist then unique key column data is given precedence. If none of these columns exists, or search criteria is not available then a prompt message will appear, suggesting you to use Replace All Results option.

Append Current Results:

Appends the results of the current search to the results already displayed in the worksheet.

Replace All Results:

Removes all existing results and displays the result of the current search.
