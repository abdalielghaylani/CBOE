Grid Management

The Grid Management feature in Inventory Enterprise lets you place plates and containers in grids, in the same way as they are physically kept in racks. Before placing plates and containers in grids, you first create a grid format, which specifies the number of rows and columns in the grid.

Creating Grid Format

To create a grid format:

  1. Log in to the ChemBioOffice Enterprise application as an administrator of Inventory Enterprise. The home page of ChemBioOffice Enterprise displays.
  2. Click Tasks in the Inventory Enterprise section. The Administrative Menu window appears.
  3. Click New in the Grid Management section. The Grid Format Attributes window appears.

  1. Type a name for the grid format in the Grid Format Name text field.
  2. Specify the number of rows in this grid in the Row Count text field.
  3. Specify the number of columns in this grid in the Column Count text field.
  4. Click OK. A confirmation window displays and confirms the grid is created as specified.
  5. Click OK to close the confirmation window.
  6. Click Close to close the Administrative Menu window.

Creating Rack Location

To create a rack location:

  1. From the Location Tree, select New from the Location menu.

  1. The Create a New Location window appears.

  1. Click Autogenerate to automatically generate the Location ID for this new location.
  2. Type a name in the Location Name text field.
  3. Select the grid format from the Grid Format drop-down list.
  4. From the Location Type drop-down, select whether you want to display the tree as Tree Element or as Rack. In the current example, Rack is selected. The Location Type field changes to Rack.
  5. Click OK. The new location displays in the Location Tree frame of the Inventory Enterprise home page.
  6. Click the new location to display the grid in the Container List frame, as shown in the example below.

Placing Containers on a Rack

To place containers on a rack:

  1. Create a few containers in a location other than the rack. For more information about creating containers, see Creating a New Container.
  2. Click on a container from the Container List frame to display its details in the Container Details frame.
  3. From Container Details, select Edit Container from the Update menu.

The Create or Edit an Inventory Container window appears.

  1. Click the Required tab.

  1. Click Browse.
  2. Select a rack location, or select multiple racks. The grid appears, as shown in the example below.

  1. Select a grid cell.
  2. Click OK in the Create or Edit an Inventory Container window. The Container Management area displays.
  3. Click the rack location you selected in the Location Tree frame. The selected container displays in the selected cell of the grid, in the Container List frame.

Note: By default, icons are displayed for the containers in the grid. You can change the view by clicking the down arrow in the first cell of the grid and selecting the appropriate display option.

Selecting Multiple Racks

To select multiple racks:

  1. From the Create or Edit an Inventory Container window, click Browse. The Browse tab displays.
  2. From the Browse tab, click MultiSelect Racks.

The following window appears.

  1. Select the desired racks and click the links. The following window appears.

  1. Select a cell in the grid to specify the order in which the racks will be filled.

Searching Racks

The Rack Search feature lets you search the appropriate rack location when placing containers in a rack, creating new containers, or creating new locations.

To search for the appropriate rack location:

  1. Create a few containers in a location other than the rack. For more information about creating containers, see Creating a New Container.
  2. Click on a container in the Container List frame to display its details in the Container Details frame.
  3. From Container Details, select Edit Container from the Update menu. The Create or Edit an Inventory Container window appears.
  4. Click Browse.
  5. Click Rack Search tab. The Search Criteria window appears.

  1. Enter the Number of open positions for this rack.
  2. Enter the Container Size for this rack.
  3. Click Submit to display the list of racks.
  4. Click the rack you want. The contents of the selected rack appears.
  5. Click a cell in the grid to specify the order for this rack to be filled. The selected rack appears in the Create or Edit an Inventory Container window.

Note: When Grouping Fields is configured on your system, all grouping fields are displayed under the Rack Search tab as query fields. Primary grouping fields are displayed by default along with the Number of open positions and Container Size fields.
