Enabling Application Tracing

If you are a power user or if you have appropriate privileges, you can configure CBOE applications to display trace information. Tracing provides timing information for the tasks performed in the application being traced. It helps in debugging and can be used to pinpoint specific trouble areas.

Note: You cannot enable tracing for Registration and DataView Manager applications using Application Tracing option under Security. For more information on enabling tracing in Registration Enterprise and DataView Manager, contact your system administrator.

To enable application tracing:

  1. Click Application Tracing under Security in the CBOE home page.


  2. Select an application to trace from the list of applications available (refer figure below) and click OK.


  3. Select ON in the Trace Status column. Click OK on the two pop-up windows that appear.


    Note: You can display trace information for individual pages by selecting Enable page timer for "application name".
  4. Minimize/close the window and perform the required tasks in the application.
