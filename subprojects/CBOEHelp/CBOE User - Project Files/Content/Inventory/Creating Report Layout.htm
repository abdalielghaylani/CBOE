Creating Report Layout

You can create a new report layout if the report layouts available already in Inventory Enterprise do not fulfill your requirements. To create a report layout:

  1. Click Tasks within the Inventory Enterprise section in the ChemBioOffice Enterprise home page. The Administrative Menu window appears.

  1. Click the Manage Reports link in the Inventory Management section. The Manage Reports window appears as:

  1. Click the New link. The Add Report window appears as:

  1. Perform the following tasks in the preceding windows:
  • Select the appropriate report type from the Select a report type drop down list.

Note: If you select Custom Report as report type then you will be able to select the resultant report layout of this procedure when creating a custom report.

  • Type the name of the MS Access report with which the layout is to be associated in the Enter Microsoft Access Report Name text box.

Note: The MS Access reports for Inventory Enterprise are found in a database called Cheminv_reports.mdb. You can also create a new report in Cheminv_reports.mdb if already existing reports do not meet your requirements.

  • Type the name of the layout in the Enter Report Display Name text box.
  • Type the name of the Microsoft Access Query for the report, in the Enter Microsoft Access Query Name text box.
  • Enter the SQL query for the report to be created.
  • Specify the number of parameters to be added in the Enter # of Parameters to Configure text box.
  1. Click the Go button. The following window appears as:

Note: You will have to enter value for the parameter when creating a report using this report layout.

  1. Click the Add Report button. A success message is displayed.
  2. Click OK. The newly created report layout is displayed in the Report Layouts list box, in the Manage Reports window.
