Editing Report Layout

To edit a report layout:

  1. Click Tasks within the Inventory Enterprise section in the ChemBioOffice Enterprise home page to display the Administrative Menu window.
  2. Click the Manage Reports link in the Inventory Management section to display the Manage Reports window.
  3. Select the appropriate report type in the Select a Report Type list. The report layouts corresponding to the selected report type are displayed in the Reports Layouts list.
  4. Select the report layout that is to be edited in the Report Layouts list and click the Edit link. The Edit Report Details window appears.
  5. Modify the required information and click on the Go button.
  6. Click OK when the success message is displayed.
