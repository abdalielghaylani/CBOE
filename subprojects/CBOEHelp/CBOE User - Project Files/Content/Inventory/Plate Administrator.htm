Plate Administrator

A plate's administrator is allowed to modify his/her plate; any non-admin of the plate can see the details but cannot modify the plates.

Creating a Plate

To Create Plates from Excel or Create Plates from Text File please refer to Creating a New Plate topic. The difference here is that you will have to set the plate administrator.

Case 1: Create Plates from Excel

  1. Login to ChemBioOffice Enterprise as INVADMIN or any other user.
  2. Click on Browse link to go to Inventory application.
  3. Click on Tasks link and then select "Create Plates from Excel" under Plate Management. The following dialog displays:

  1. Click the Browse button and locate the Excel spreadsheet from which the plates are to be created and then in the Plate Admin field select an appropriate user/group corresponding to the by user/by group option that you select
  2. Click OK

Case 2: Create Plates from Text File

  1. Login to the ChemBioOffice Enterprise as INVADMIN or any other user.
  2. Click on Browse link to go to Inventory application.
  3. Click on Tasks link and then select "Create Plates from Text File" under Plate Management. The following dialog displays:

  1. Perform the same steps covered in Creating a New Plate, the difference here is that you have to specify the Plate admin. To do that, select in the Plate Admin field an appropriate user/group corresponding to the by user/by group option that you select
