Making a Location Public

Public locations are special locations where group security will not work and these locations would be used to transfer the containers between the users.

To create a public location:

  1. Login to ChemBioOffice Enterprise Manager as INVADMIN or any other user
  2. Click on Browse link to go to Inventory application
  3. In the left panel, go to Location>New
  4. Pick the Access as Public checkbox and fill in the other fields
