Enabling Group Security

To enable the group security, perform the steps below:

Using the invconfig.ini

  1. On the ChemOfficeEnterprise server, go to C:\Inetpub\wwwroot\ChemOffice\ChemInv\config folder
  2. Open invconfig.ini file in edit mode
  3. Under [SECURITY] section set the following value ENABLE_OWNERSHIP= "True"

  1. Perform an IISRESET

Using the Server Configuration Tool

  1. Run the ChemBioOffice Enterprise Server Configuration Tool
  2. Click on "App Settings" tab and check the Group Security checkbox
  3. Click on Save Configuration button to save the changes

Requesting Samples

When group security feature is enabled, any user can request any sample even if he/she is a non-owner regardless of location. Users can request any sample to:

  • The location for which they have the ownership (or their group has ownership)
  • The public locations

The fulfillers can fill the request to:

  • The requested location (even if they are not part of the ownership group) OR
  • Any location for which they have ownership (or their group has ownership) OR
  • A public location

Moving Containers between Groups

While moving the container between Groups the workflow will be as follows:

Scenario:

  • User A is a member of Group A. Group A is the location admin of Location A. There is a container Container A in Location A owned by Group A.
  • User B is a member of Group B. Group B is location admin of Location B.
  • User C is a member of Group C. Group C is location admin of Location C.

Use case:

User B needs the container A from Location A.

Workflow:

  1. User A will move the container A to a public Location. Here container can be accessed and modified by anyone (UserA, UserB, UserC).
  2. User B will move the container A to Location B, still the owner is Group A, any member of Group B ( owner of Location B) can modify anything on the container A. Group A users can also edit it here. That’s means containers can be modified by Containers Admins as well as Location Admins. Here User C cannot make edits to the container A.
