Organizing Projects

You can organize tables and forms into project groups.

To update the list of project groups:

  1. Select the menu item you want to work with. The options are:
  • Table Organization: To organize tables, go to Admin > Manage > Organization.

If the Organization menu item is not enabled, select Admin > Manage > Schema, and thereafter select Manage > Organization.

  • Personal Form Organization: To organize personal forms, select Forms > Manage > Organization (my forms).

If the Organization (my forms) menu item is not enabled, select Manage > Forms from the Forms menu, and thereafter select Manage > Organization (my forms).

  • Public Form Organization: To organize public forms, select Manage > Organization (public forms) from the Forms menu.

The TreeView window appears.

  1. Highlight the project you want to update. The links to perform various actions display.
  2. Click the corresponding link with the action you want to perform. For example, click the Move link to move the form to a different location.
