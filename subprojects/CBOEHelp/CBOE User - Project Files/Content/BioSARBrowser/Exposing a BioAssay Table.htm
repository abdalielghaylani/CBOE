Exposing a BioAssay Table

To expose a BioAssay table:

  1. Open the BioAssay Enterprise application.
  2. Click the Application Settings link in the Start Page section. The BioAssay HTS Settings dialog box appears.
  3. Ensure that the user name and server name of BioSAR Browser correct.
  4. Click the OK button to close the BioAssay HTS Settings dialog box.
  5. Create a One Table Legacy Data protocol called “Integration Demo” in BioAssay Enterprise. The protocol should include the following fields:
  • Experiment
  • Date
  • Scientist
  • Compound
  • IC50
  • Comments
  1. Create a relevant file import template for the Integration Demo protocol.
  2. Import data into the protocol using a data file.
  3. Load the data from the imported file.
  4. Click the Definition tab.
  5. Click the Results table and then click the Expose Table in BioSAR Browser button. The Expose Table in BioSAR Browser dialog box appears:

The Expose Table in BioSAR Browser feature is specific to BioAssay Enterprise. The feature is not available in BioAssay Desktop.

  1. Disable the check box, Expose table showing valid data only.
  2. Type Integrate in the Table Display Name text box.
  3. Click the OK button to close the Expose Table in BioSAR Browser dialog box.
