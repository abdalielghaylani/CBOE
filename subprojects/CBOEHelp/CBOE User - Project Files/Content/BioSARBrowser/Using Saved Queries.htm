Using Saved Queries

You can save previously executed queries. Saved queries are useful when you want to repeatedly search data using the same search criteria. For example, you can save a query that searches for batch information for a particular compound. Since batches are not all made at once, one day the results may be one set, but the next day (after another batch was made), more results are retrieved.

To save a query:

  1. Perform a search operation using a form and obtain the results.
  2. Select Save from the Queries menu.

  3. Enter the name and description of the query you want to save. Optionally, you can select Make Public to make your saved query accessible to other users.
  4. Click OK.

To use a saved query:

  1. Select Queries > Restore > Last. Or select Queries > Restore > Saved, depending on the query you want to restore. If you select Queries > Restore > Last, the query is displayed in the same page that displayed the search results. If you select Queries > Restore > Saved, the Queries window appears.

  2. Select the appropriate form and the query associated with that form.
  3. Click Restore. The saved query displays.

  4. Add more information related to the search criteria, if desired.
  5. Click Search to perform the search.
