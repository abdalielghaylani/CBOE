Viewing Search Results

After a search is performed, the results display in a list form, which lets you view partial content for each record retrieved, as shown in the following example.

Note: The Jump to drop-down list lets you change forms with the same base table.  The list of forms you can access is based on a combination of security and a common base table.

To view complete information for a particular record:

  1. Click Show Details corresponding to that record.

The information displays at the different view levels, which is defined when creating a new form. You can view detailed results in different formats. The information that displays in each of the formats is identical. The screen differs in its layout appearance only.

The view formats are:

  • Frame View
  • Tab View
  • Frame/Tab View
  • Report View
  • Excel View
  • BioViz View

Sorting

You can sort results by clicking on the table header of the columns you want to sort.

Editing a Form Dynamically

Once a form is opened, you can edit the layout, or content.

To edit the currently-opened form:

  1. Select Edit this Form from the Forms menu.

  2. Make your changes to the form definition.
  3. Click Save Changes. The page refreshes with your new changes in the opened form.

Editing List View Options

List View refers to the result view, wherein the retrieved records are listed with partial information. List View provides two layout options:

  • Columns Down: Arranges fields in columns spaced horizontally.
  • Columns Across: Arranges fields in columns spaced vertically.

Displaying Search Results in Excel

When in List View, you can view search results in an MS Excel spreadsheet. (However, this option is available only if the form is integrated with MS Excel.) For more information about integrating with Excel, refer to Selecting Integration Options under Creating a Form.

To view data in an MS Excel spreadsheet, click the Display in Excel tab. The data appears in a spreadsheet in a new window.

Details View Options

After a search is performed, you can view the detailed results by clicking Show Details corresponding to a particular record. In the Details View, you can view the results in different formats. The information displayed in each of the formats is identical. The screen differs in its layout appearance only.

The view formats are:

  • Frame View
  • Tab View
  • Frame/Tab View
  • Report View
  • Excel View
  • BioViz view

BioViz View

In the BioViz view, data displays within the ChemFinder application. When you click the Display in BioViz tab, the ChemFinder application launches and displays data.

Viewing Specific Records:

You can specify the record number to display the corresponding record.

To view specific records via the record number:

  1. Click on Rec# at the top of the results list. The Go To Record dialog displays.
  2. Enter the Record number in the text field.
  3. Click OK. The record displays.

Marking Records

Marking records lets you view records without creating a search query. To mark all records at once, click Mark All Records at the top of the results list.

To mark a specific record, click on Mark Record within the corresponding record row, in the results list, as shown in the example below.

The number of Marked Records is displayed next to the Marked Hits menu. You can:

  • Display only the Marked Hits by selecting Show Marked.
  • Clear the marked Hits, by selecting Clear Marked.
