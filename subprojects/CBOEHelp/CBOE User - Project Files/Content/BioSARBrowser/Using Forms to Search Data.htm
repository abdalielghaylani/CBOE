Using Forms to Search Data

One way to retrieve data is to perform a search using a search form. For more information about defining search forms, refer to "Form Management".

To search data using a search form:

  1. From the Form Management page, select Forms > Open Form. A page containing the list of existing forms appears.
  2. Or, from the global login page of ChemBioOffice Enterprise, you can click Open Form. The following Click a link to select a query form window displays.

  3. Select a search form to use. The selected search form appears.


    Note: The form you see depends on the form group chosen.

  4. Enter the information in the form.
  5. You can select the sorting criterion from the Sort Search By drop-down menu.
  6. Click Search . The search results are displayed.

To open a form:

  1. From the ChemOffice Enterprise home page, BioSAR Enterprise section, click the Manage your forms link. The Form Management interface displays.
  2. Select the form you want to open from the drop-down list and click Open as shown in the example below.
