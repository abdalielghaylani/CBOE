Generating Reports

With BioSAR Enterprise, you can generate reports based on search results.

To generate a report:

  1. Perform a search using a public form group.
  2. Select Generate Report from the Reports menu in the menu bar.

  3. The BioSAR_Reports window appears:

  1. Select the options you want in your report. The following headings describe the report settings, which define a report:
  • Data View

Options:

  • List
  • Details

Description: Lets you to set the amount of information to display in the report.

  • Table Layout

Options:

  • Tables Across
  • Tables Down
  • Tables Down then Across

Description: Lets you to define the table layout. Tables can be placed across (arranged horizontally next to each other) or down (arranged vertically).

  • Setting Profile

Options: List of all available profiles. Profiles can be made public.

Description: Lets you define the display settings for a report. This includes font size, color, and style, as well as the inclusion, or exclusion, of logos. Use the available links to view, create, edit, or duplicate the selected profile before creating the report.

  • Page Orientation

Options:

  • Auto
  • Portrait
  • Landscape

Description: Lets you define the layout of the report. Reports can be created to fit into a portrait or landscape format. If the Auto option is selected, the appropriate size is used.

  • Page Size

Options:

  • Letter (8.5 x 11 in.)
  • Legal (8.5 x 14 in.)
  • A3 (297 x 197 mm)
  • A4 (210 x 297 mm)
  • B4 (250 x 354mm)
  • B5 (182 x 257mm)
  • Tabloid ( 11 x17in.)

Description: Lets you set the page size of the report. The page size is implemented when the report is created to fit on the page you are currently using.

  • 1 Record Each Page

Options:

  • No
  • Yes

Description: Lets you insert a page break after each record.

  • Report Format

Options:

  • SNP (Microsoft Access format)
  • PDF

Description: Lets you select the format for displaying the report.

  1. Click Run Report to generate the report.

Note: If structures do not appear in your reports, the schema may not be configured correctly.

Note: To show structures in generated reports, there must be a CLOB column that contains Base64 holding those structures. To configure the system to recognize this column as the structures column, click Lookup Column for the table with the MOL_ID. Then, select the table containing the base64 column and set that as the lookup column.

Creating a New Profile

A report profile controls the font color, style, and size of reports. In addition, report profiles control the inclusion, or exclusion, of logos and header lines.

To create a new profile:

  1. From the BioSAR Reports window, click the New link corresponding to the Settings Profile list. The BioSAR Browser Report Profile window appears.

  1. Enter the appropriate settings. The following table lists the tasks you have to perform when setting the profile:
If you would like to... Then...
Give the profile a name (*required) Enter a name for your profile in the Profile Name text field. This name is used as an identifier.
Write a brief description of the profile Enter a description in the Description text field.
Make the profile public

Select Yes in the Public drop-down list.

NOTE:When you make a profile public, it is available to other users when generating reports. Only the owner of a profile can edit or delete a profile.

Set the margin width Select the margin size in the margin drop-down lists, such as Right Margin and Left Margin.
Set the color of the header, footer, or detail in the report Select the color in the color drop-down lists, such as Header Color and Footer Color.
Adjust the font type, style, size, weight, or color of a particular part of your report Select the options from the drop-down list that corresponds to the attributes, such as font type and style.
Insert header lines to the top of the report Enter text into the Header Line text fields.
Include a logo Select the logo to include from the Logo drop-down list, or click the Add link to add a new logo.
  1. Click OK.

Adding Logos

You can add logos at the top of reports.

To add a logo to the list of options:

  1. Click Add to the right of the Logo drop-down list. The following window appears.

  1. Click Browse and browse to the logo file. There are three possible types of logo files:
  • GIF (.gif)
  • JPEG (.jpg)
  • PNG (.png)
  1. Click Add Logo.

Editing a Profile

You can edit a profile only if you have created it. Even if a profile is public, only the creator of the profile can edit this profile.

To edit an existing profile:

  1. From the BioSAR_Reports window, select the profile you want to edit in the Settings Profile drop-down list.
  2. Click the Edit link corresponding to the Settings Profile drop-down list. The BioSAR Browser Report Profile window appears.
  3. Edit the profile settings.
  4. Click OK.

Viewing a Profile

To view an existing profile:

  1. From the BioSAR_Reports window, select the profile you want to view in the Settings Profile drop-down list.
  2. Click the View link corresponding to the Settings Profile drop-down list. The selected profile appears.

Duplicating a Profile

You can create a duplicate of an existing profile, even if the profile is public.

To create a duplicate of an existing profile:

  1. From the BioSAR_Reports window, select the profile you want to duplicate in the Settings Profile drop-down list.
  2. Click the Duplicate link corresponding to the Settings Profile drop-down list. The selected profile appears.
  3. Enter a name and description for the duplicate profile.
  4. You can modify other profile settings, if required.
  5. Click OK to save the profile.

Deleting a Profile

Profiles can be deleted only by the user who created them, even if the profile is public. The default profile cannot be deleted.

To delete an existing profile:

  1. From the BioSAR_Reports window, select the profile you want to delete in the Settings Profile drop-down list.
  2. Click the Delete link corresponding to the Settings Profile drop-down list. A confirmation message displays.
  3. Click OK.

Changing Field Attributes in Reports

The attributes of fields, such as, order and size, displayed in reports are based on the field attribute values set in the corresponding forms. Therefore, to change the way in which fields display in reports, you need to change the field attributes specified in the corresponding forms.

To change the order of fields in a report:

  1. Close the BioSAR_Reports window.
  2. Select Forms > Manage > Forms from the menu bar to display the first page of the Form Management interface.
  3. Select the form that generated the report.
  4. Click the Edit link to display the list of tables related to the selected form.
  5. Click a link in the Field Order column corresponding to the appropriate table.
  6. Use the arrow buttons to move fields up and down.
  7. Click Save Changes.

To change the size of a field label displayed in a report:

  1. Close the BioSAR_Reports window.
  2. Select Forms > Manage > Forms from the menu bar to display the first page of the Form Management interface.
  3. Select the form that generated the report.
  4. Click the Edit link to display the list of tables related to the selected form.
  5. Click the Edit link under the Field Options column of the appropriate table to display a page containing the field attribute values.
  6. Scroll to the appropriate field and set the field attribute.
  7. Click Save Changes.
  8. Generate the report again. The changes you made to the field attributes are reflected in the report.
