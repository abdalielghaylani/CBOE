Duplicating a Form

Duplicating a form is useful when you need to include many properties of an existing form in a new form. For example, if you are creating a new form with the same base table and field properties of an existing form, duplicating the existing form saves time and promotes accuracy when defining the new form.

To duplicate a form:

  1. From the ChemOffice Enterprise home page, BioSAR Enterprise section, click the Manage your forms. The Form Management interface appears.
  2. Select the form you want to duplicate from the drop-down list.
  3. Click Duplicate.

The form is duplicated and appears in the forms drop-down list. The name of the new form retains the name of the duplicated form, preceded by "Copy of", as shown in the following example.
