Accessing BioSAR Enterprise

You access the BioSAR Enterprise application through the ChemOffice Enterprise home page.

To access the BioSAR Enterprise application:

  1. Enter https://servername/COEManager into your web browser. <servername> is the name of the server where ChemBioOffice Enterprise is installed. The login page of ChemBioOffice Enterprise appears.
  2. Enter a valid username and password.
  3. Click Log in. The ChemOffice Enterprise home page launches.
  4. Click Open Form within the BioSAR Enterprise section.

  1. A page that displays the existing forms appears.
  2. You can click a link to select a query form from the list below.
