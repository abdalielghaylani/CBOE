Editing a Form

After creating a form, you can edit it at anytime. To edit a form, you must be logged in as the user who created the form, or you must have the appropriate privileges assigned to be able to edit another user's forms.

To edit a form:

  1. From the ChemOffice Enterprise home page, BioSAR Enterprise section, click the Manage your forms link. The Form Management interface appears.
  2. Select a Form from the Select a Form drop-down list.

  3. A page displaying the form's definition appears:

  1. Edit the form's definition. The five properties of a form's definition are:
  • Base table
  • Child tables
  • Users and roles
  • Integration options
  • Project(s) to which the form is assigned
