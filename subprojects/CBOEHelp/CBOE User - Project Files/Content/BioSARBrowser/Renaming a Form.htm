Renaming a Form

To rename or change the description of a form:

  1. From the ChemOffice Enterprise home page, the BiSAR Enterprise section, click the Manage your forms link. The Form Management page appears.
  2. Select the form whose name you want to change from the drop-down list.
  3. Click Rename.

  4. The Rename Form: X (where "X" is the name of the form you selected from the drop-down list in step 2) appears.

  5. Edit the Name and Description in the corresponding fields.
  6. Click Save Changes.
