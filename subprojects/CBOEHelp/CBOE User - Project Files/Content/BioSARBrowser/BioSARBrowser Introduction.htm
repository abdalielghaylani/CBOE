BioSARBrowser Introduction

This chapter describes BioSAR Browser (also called BioSAR Enterprise), its features, and complete instructions on how to use them. BioSAR Enterprise is a database management tool, specific to ChemOffice Enterprise. BioSAR Enterprise combines data from multiple applications in ChemOffice Enterprise and Excel, and subsequently performs searches on the data and displays the results according to the requirements. The application is fully customizable; allowing for custom search forms and custom display data.
