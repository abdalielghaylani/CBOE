Launch Registry Form after Retrieving Data from Registry System

You can view component details in the Registry System if a valid registration number is present.

Note: Ensure the Reg # column has a valid registration number.

  1. Press the <Ctrl> key and click on the registration number that appears as a reference in the Reg # column.

A registry form is launched and displays the registry information of the selected registration number.
