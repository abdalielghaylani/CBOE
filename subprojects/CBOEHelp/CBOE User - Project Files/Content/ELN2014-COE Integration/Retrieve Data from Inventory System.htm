Retrieve Data from Inventory System

You can retrieve data from the Inventory system to use the structures in the Reaction section.

To retrieve data from Inventory:

  1. Log in to E-Notebook.
  2. Create a Discovery Chemistry (which has Reaction as default section) or a Process Chemistry (which has Process Reaction as default section) experiment.
  3. In the Reaction toolbar, click Add. The Select & Add Structure dialog appears.
  4. Search In: Select Inventory Search from the drop down list.

The search form consists of a ChemDraw control to render structures for exact and substructure searching.

A set of text fields is available for entering textual search data, such as:

  • Substance Name
  • CAS Registry Number
  • ACX ID
  • Container Barcode
  • Molecular Formula
  • MolWeight Range
  1. Enter search criteria and click Search to execute the search.

  1. Click Clear to clear the form.
