Quick Temporary Registration - Single Click

From the Samples section, you can push a sample to the temporary table using the single click method.

To register a sample to the temporary table:

  1. Draw a reaction in the Reaction field and create samples. A Samples section is automatically created.
  2. Assign a product to the created samples and navigate to the Samples section.
  3. From the Home ribbon, click Registration and select Quick Temporary Registration.

The Registration dialog displays the Temp ID and the Temp ID is automatically populated in the sample's Properties table.

Note: If you have system administrator privileges, you can configure the following fields in the Properties table as read-only: Reg #, Temp ID, and Batch #. Please see the Administrator Guide for more information about setting these fields as read-only.

  1. Click OK to close the dialog.
