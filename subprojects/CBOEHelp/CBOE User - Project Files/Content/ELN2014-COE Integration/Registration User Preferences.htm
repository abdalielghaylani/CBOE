Registration User Preferences

The registration user preferences feature lets you set default values, which are automatically used when using the single click for registration as well as for manual registration.

Note: The Registration User Preferences value can be fetched while registering from the Reaction section if no values are manually entered in the Samples table.

To display the registration user preferences:

  1. Navigate to E-Notebook application menu and select Options.

The Options dialog displays.

  1. Click Registration in the left frame. The Registration Enterprise form displays in the right frame.

  1. Fill in the default values that you want fetched when registering.
  2. Click Save.

Note: Mandatory fields are in red font, for example, Prefix, as shown in the Options dialog above. Mandatory fields must be completed when registering, but are not necessarily set as default fields in the Registration User Preferences form.
