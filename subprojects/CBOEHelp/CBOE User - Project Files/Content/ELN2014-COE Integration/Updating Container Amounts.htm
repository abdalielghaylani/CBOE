Updating Container Amounts

Update Container Amounts functionality allows to reduce the consumed amount/volume of the substance, usually these fields are Sample Mass and Volume. This is applied in the Reaction section>>Reactants & Products table where inventory reagents are used and where users can do a bulk update, to Inventory containers, of the amount consumed for multiple containers.

The Amount/Volume specified in E-Notebook is the consumed amount thus, will be reduced from the Inventory Container .

Example:

  • Container Name: Notebook-001-1, Container ID: C1042

  • Container Name: Notebook-001-2, Container ID: C1048

The Amount/Volume consumed by the container entries present in E-Notebook will be reduced from the container entries in Inventory.

To update container amounts:

  1. Go to the Reactants table in the Reaction section. Pull reactants from Inventory system. Please refer to Inventory Pull chapter for more details.

Note that the container in the reactants table must have Barcode values.

  1. Make sure that Updated (check box) is unchecked.
  2. Modify the Sample mass / volume values.

Note: You can only reduce the consumed amount.

  1. Fill in the consumed volume and select Update Container Amounts option from the Inventory form tool menu.

  1. Update Container Amounts dialog appears for confirmation. Click Yes to confirm your action.

  1. The Inventory dialog with the updated barcodes displays. Click OK.

Note that the corresponding checkboxes get automatically checked after the containers are updated.

Once the container amounts are updated in E-Notebook, the amount/volume consumed by the containers in E-Notebook will be deducted from the amount/volume of the containers in the Inventory system.
