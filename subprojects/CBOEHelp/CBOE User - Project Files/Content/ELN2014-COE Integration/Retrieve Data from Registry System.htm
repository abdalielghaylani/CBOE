Retrieve Data from Registry System

You can retrieve data from the Registry system to use structures in the Reaction section.

To retrieve data from the Registry:

  1. Log in to E-Notebook.
  2. Create a Discovery Chemistry (which has Reaction as default section) or a Process Chemistry (which has Process Reaction as default section) experiment.
  3. In the Reaction toolbar, click Add. The Select & Add Structure dialog appears.
  4. Search In: Select Registry Compound Selector from the drop down list. The following window appears:

  1. You can retrieve all records or search for specific records.

The search form consists of a:

  • ChemDraw control to render structures for substructure, full structure, exact and similarity searching. You can set structure preferences by clicking the Options tab on structure window.
  • Set of text fields for entering search data (Registry/Component/Batch Information).
  1. The search preferences are General and Stereochemistry. Select the desired search preferences and select the search options. For more information, see Structure Search.
  2. Enter specific search criteria under Registry Information or Batch Information.
  3. Click Search.

A search is executed on the basis of the specified criteria. See the example below.

A list of the records matching the search criteria you specified displays.

  1. You can refine the search results or conduct a new query.

Refining Search Results

You can modify your search criteria and search the current results set. There are two options available under Refine query:

  • Edit Current Query: Re-runs the original query and applies the refinement query.
  • Refine over Current Records: Applies the refinement query to the current record set.

New Query

You can conduct a new query by selecting New Query. A new blank registry search form displays where you can enter new search criteria.
