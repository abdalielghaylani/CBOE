Adding Compounds from ACX System to a Reaction Drawing

To add a compound from ChemACX to the reaction drawing:

  1. After executing the search, select a substance from the Show Results tab of the Select & Add Structure dialog.
  2. Click Add.
  3. Select an option from the drop down list:
  • Left of Arrow as Reactant
  • Above Arrow as Reagent
  • Right of Arrow as Product
  • To Grid as Reagent

The chemical structure is added to the Reaction field of the Reaction section. A new entry in the Reactants & Products table is added as well.
