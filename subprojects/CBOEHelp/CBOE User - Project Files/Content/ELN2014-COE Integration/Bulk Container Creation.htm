Bulk Container Creation

To create containers in bulk:

  1. From the Home ribbon, in the PS Registry Form section, click the check boxes in the Selected column in the Registration Table for each entry you want to create a container to.
  2. Fill in the mandatory fields, for example, number of copies, location, container size, container type, unit of measure, and so on.
  3. Click Inventory form tool.
  4. Select Create Container(s) from the form tool menu.

The BulkContainerCreation confirmation message displays.

  1. Click Yes.

A dialog displays the newly-created barcodes corresponding to the selected entries.

  1. Click OK.

The containers with barcodes are created and returned to the Registration Table.

Once the containers are pushed from E-Notebook to Inventory, the containers are accessible in the Inventory system, as shown in the example below.
