Structure Search

Structure searching lets you search for records based on their chemical structures. Registration Enterprise provides three types of structure searches:

  • Substructure Searching
  • Full Structure Searching
  • Exact Searching

Substructure Searching

Substructure searching is the default structure search type. It retrieves structures that meet the following criteria:
  • Matches the basic chemical structure specified
  • Contains additional attachments at open bond positions

Setting Search Preferences

Setting the search preferences lets you control the manner in which the results are displayed. These are global settings throughout the application.

To access the search preference options:
  1. Click Optionsattached to the structure window.

The Search Preferences dialog appears.

  1. Check the options you want to apply.

General tab in Search Preferences
The various options under the General tab include:
  • Hit any charge on heteroatom: Allows uncharged atoms in the query to match charged atoms in the target.
  • Reaction query must hit reaction center: Allows the reaction centers in the query to overlap with the reaction centers in the target.
  • Hit any charge on carbon: Allows uncharged carbon atoms in the query to match charged carbon atoms in the target.
  • Permit extraneous fragments in Full Structure (exact) Searches / Permit extraneous fragments in Reaction Full Structure (exact) Searches: Performs exact structure search.
  • Query fragments can overlap in target: Allow fragments to overlap in the target.
  • Tautomeric: Allows query to find tautomers.
  • Similarity search (20-100%): Retrieves chemicals that have structural features similar to the one specified in the query. It does not match specific atoms or bonds. It is useful if you have a general idea of the types of compounds you are searching for, but do not know the precise target compound. You can specify the degree of similarity. A lower value requires the target to be less similar.
  • Full structure similarity: Retrieves structures that are similar to the full structure specified in the query.
Stereochemistry Tab in Search Preferences

The various options under the Stereochemistry tab include:
  • Tetrahedral stereo center hits: To set the Tetrahedral stereo center hits, select:
    • Same: Matches the target exactly.
    • Either: Matches same or opposite configuration at the center of the target.
    • Any: Matches any target.
    • Thick bonds represent relative stereochemistry: Matches a relative relationship between centers. When activated, any thick bond drawn in the reference compound is interpreted as relative stereochemistry when searching, or comparing possible duplicates.
  • Double bonds hits: To sets the Double bonds hits, select:
    • Same: Matches the target exactly.
    • Any: Matches any configuration in the target.
