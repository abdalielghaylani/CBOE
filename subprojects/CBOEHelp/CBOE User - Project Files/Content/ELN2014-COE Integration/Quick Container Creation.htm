Quick Container Creation

To create a container using the single click feature:

  1. Create a sample and assign a product to it.
  2. Within the Reaction section, select a sample in the Samples table.
  3. From the Home ribbon, click the Inventory form tool and select Quick Container Creation.

A Barcode and a Compound ID are automatically populated in their corresponding fields in the Samples table under Reaction section, as well as, in the Sample Information SM subsection, under Samples section. This indicates that the container has been created and is accessible in the Inventory system.

The container details are fetched from the Inventory User Preferences, unless you enter values manually. Manually-entered values override those set in the User Preferences dialog. If the mandatory fields are left blank in the User Preferences dialog, you have to enter them manually before creating the container.

Note: If you have system administrator privileges, you can configure the Barcode field in the Samples table as read-only. Please see the Administrator Guide for more information about setting these fields as read-only.
