Retrieve Data from Barcode

With a valid barcode you can retrieve data from Inventory.

To retrieve data using a barcode:

  1. Enter a valid barcode value in the barcode column of the Reactants table.
  2. Click outside the cell.

The container properties associated with this barcode are returned and added to the Reactants table.
