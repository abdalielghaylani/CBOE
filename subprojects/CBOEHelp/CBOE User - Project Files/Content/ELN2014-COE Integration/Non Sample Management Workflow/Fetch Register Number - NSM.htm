Fetch Register Number - NSM

The Fetch Register Number method lets you query the database with a registration Temp ID. If a registration number is available, the registration number and batch number is returned to the associated row in the Products Table of the Reaction W COE Integration section. This method was created to support temporary registration from E-Notebook. For example, you would register a compound in the temporary table from E-Notebook and a Temp ID is returned to the Products table. Later, a Registrar would permanently register the compound into the Registration system. Subsequently, you can use the Fetch Register Number method to query the registration number and batch number, retuning them to the associated row in the Products table.

Note: You must have appropriate registry permissions assigned to perform this registry action.

To Fetch the Registry Number:

  1. Select the Fetch Register Number form tool from the Reaction W COE Integration section. If the Temp ID is present in the Products table and the registration number is available in the Registration system, the Fetch Register Number dialog appears and displays the registration number and batch number.
  2. Click OK in the Fetch Register Number dialog.

The registration number and batch number are returned to the associated row in the Products table of the Reaction W COE Integration section.

If the Temp ID is present in the Products table and the registration number is unavailable in the Registration system, the dialog returns a “No compound exists with the Temp Id:...” message. An example is shown below.

If the registration number and batch number fields are already populated with data, the dialog returns a “Clear the Reg Number field and fetch again” message.
