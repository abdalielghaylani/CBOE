Creating Container - NSM

The Create Container method gathers the information on the Inventory form and attempts to create a container into the Inventory system. Upon successful container creation, a Barcode Number and Compound ID are returned to the Container Properties list of the Inventory Form.

To create a new container:

  1. From the Home ribbon, select Open Inventory Form option from the Send to Inventoryform tool. The Select Compound for Net Container dialog appears where you select a product molecule for container creation.
  2. Select the desired structure and click OK.

  1. The Open Inventory Form appears and displays the data associated with the selected product in the Reaction W COE Integration section.

  1. Fill in the mandatory fields.
  2. Select Create Container option from the Inventory form tool.

A Barcode and a Compound ID are automatically created, confirming the container has been pushed from E-Notebook to Inventory, and that the container is accessible in the Inventory system.

Note: If the substance associated with the container (you are creating) already exists in the Inventory system, you must resolve the substance conflicts. See the example below.

You can select one of the following options:

  • Select the existing substance
  • Create duplicate from the conflicting substance

To select an existing substance:

  1. Click Select from the Existing Substance 1 section in the Resolve Substance screen.
  2. The following dialog appears displaying the selected compound ID. Click Submit.

  1. The container with the existing Compound ID, but with a different Barcode Number, is created and returned to the Container Properties list in the Inventory Form.

To create duplicate from existing substance:

  1. Click Create Duplicate from the Conflicting Substance section in the Resolve Substance screen.

A duplicate substance is created.

  1. Click Select. The following dialog displays.
  2. Click Submit.

Once the container is pushed from E-Notebook to Inventory, the container becomes accessible in the Inventory system.
