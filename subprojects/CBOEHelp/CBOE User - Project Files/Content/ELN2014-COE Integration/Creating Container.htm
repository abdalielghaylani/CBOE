Creating Container

The create container method gathers information on the Inventory form and creates a container in the Inventory system. Upon successful container creation, a barcode number and a compound ID is returned to the Container Properties list on the Inventory Form.

To create a new container:

  1. From the Home ribbon, click the Samples section and select a product.
  2. Click the Send to Inventory form tool to open a menu.
  3. Select Open Inventory Form from the form tool menu.

The Inventory form displays the data associated with the selected product in the Samples section.

  1. Fill in the mandatory fields and the optional fields, if desired, and select Create Container from the Inventory form tool.

The container is created and the barcode and compound ID populate automatically, as shown in the example below.
