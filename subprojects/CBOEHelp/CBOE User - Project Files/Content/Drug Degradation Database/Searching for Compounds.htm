Searching for Compounds

A non-administrator user is just allowed to search over the Drug Degradation Database. In order to be able to perform additional tasks like creating/editing parent compounds, degradants, conditions, salts experiments and mechanism, specific privileges should be provided. For more details on how to perform these additional tasks please refer to the ChemBioOffice administrator's guide.

Drug Degradation allows users to search for compounds using a variety of criteria: the compound name, various chemical properties, or even a drawing of all or part of the structure.

To search for compounds:

  1. On the Drug Degradation Enterprise page, click the Search link. The Drug Degradation Parent Compound Search page appears.
  2. Enter the search criteria based on which the search will be run. The criteria that may be entered are defined below:

Parent
Using the ChemDraw plugin, draw a parent structure for which to search and select the appropriate option in the SearchType dropdown list.

Formula
Enter the chemical formula.

Molecular weight
Enter the molecular weight.

Functional Group
Select a functional group from the dropdown list.

Conditions
Select a degradation condition.

Salt
Select a salt or salt type from the dropdown list.

Generic name
Enter a generic name, if any, for the compound.

Trade name
Enter a trade name, if any, for the compound.

Common/Other name
Enter another name, if any, for the compound.

Compound ID
Enter the ID, if any, for the compound.

Degradant
Draw a degradant structure for which to search. Then select the appropriate option in the Search type dropdown list.

  1. Click the Search button. The results are shown in list view.

  1. Click on "Hide Structure" button to hide the structures from the list of results.
  2. Click on "Show Details" button corresponding to each record in order to view its details.

  1. Click on View Degradants to view the Parent compound degradants.

Please refer to CBOE Search chapter for more details on the options that the Drug Degradation search form provides.
