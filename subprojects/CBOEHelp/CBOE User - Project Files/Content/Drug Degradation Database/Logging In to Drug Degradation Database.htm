Logging In to Drug Degradation Database

To log into Drug Degradation Database:

  1. Enter https://servername/COEManager into your web browser address bar. <servername> is the name of the server where ChemBioOffice Enterprise is installed. The User Login page of ChemBioOffice Enterprise appears.

Note: To find the name of your webserver, see your system administrator.

  1. Enter a valid username and password.
  2. Click the Log in button to display the home page of ChemBioOffice Enterprise.
  3. Under Drug Degradation, click the Main Page link.

  1. The Drug Degradation Enterprise page appears
