Drug Degradation Database Introduction

Welcome to the Drug Degradation Database chapter.

This chapter provides a full description of Drug Degradation Database, its features, and complete instructions on how to use them.

Drug Degradation lets users manage records for reagents that degrade over time. Users can keep track of products, reactions, and the environments in which the degradation reactions take place. Using the ChemBioOffice Webserver, users can also look up records from other Drug Degradation users.
