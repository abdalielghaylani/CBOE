Creating a registry record

Registration allows the creation of a registry record that has either a single component or multiple components. Pure substances consist of a single component whereas mixtures consist of multiple components.

Creating a Registry Record with Single Component:

To create a registry record with single component:

  1. Log in to ChemBioOffice Enterprise Manager home page, click "Main Menu" under Registration Enterprise. The Home Page of Registration appears.

  1. Click Submit New Component icon in the Registration home page or in the ChemBioOffice Manager home page under Registration Enterprise. The Submit Component page appears.



From the Submit Component page, a record may be submitted following the simple steps listed below:

  1. Entering Registry Information
  2. Entering Component Information
  3. Entering Fragment Information
  4. Entering Batch Information
  5. Clicking Submit.
    Note: If ChemDraw Chemical Warning Checker is enabled and ChemDraw chemical warning is present, you will not be allowed to Submit the component until the underlying error is resolved. For more information on ChemDraw chemical warning, See "ChemDraw Chemical Warning".
