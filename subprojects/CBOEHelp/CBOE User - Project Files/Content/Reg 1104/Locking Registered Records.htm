Locking Registered Records

Users with specific privileges can lock permanent records. Locked registry records cannot be edited, however adding batches and adding identifiers is allowed.

In order to enable the Locking feature, you need to set "EnableLocked=True"

Locking requires the following privilege: set_locked_flag

Revoking Locking requires the following privilege: toggle_locked_flag

To lock a registered record:

  1. Log in to ChemBioOffice Enterprise
  2. Click on Search Registry button under Registration section. The Search form displays. Enter specific criteria to search for specific records or click on Retrieve All button to display all the registered records. The following results window displays

  1. The locked column shows the status of the record, when the Thumb is up this means that the record is locked and when the Thumb is down this means that the record is unlocked. Note that you can filter the registered records by their status. You can then display all the unlocked records only.

  1. To lock a record, click on its corresponding Reg Number and click on the Thumb down to move it up.

  1. To lock multiple records at once, check the Marked checkbox corresponding to the records that you want to lock and click on Lock Marked link at the top of the list

To unlock a locked record, just perform the same steps described above but click this time on the Thumb up to move it down.

When a record is locked, you won't be able to delete it, the "Delete Registry Record" option is disabled as shown below:

Clicking on Edit button (Edit button is available if the user has "EDIT_COMPOUND_REG" privilege), the user will be able to add to a locked record the following:

  • Identifiers in all information sections: registry identifiers, batch identifiers, structure identifiers and component identifiers
  • Projects
  • Batches (Add Batch button is available if the user has "ADD_BATCH_PERM" privilege)

Registry Comments fields and all other fields will be disabled.

Note that when "EnableMoveBatch" is set to True in the system settings, the Move Batch button will be disabled when the record is locked.
