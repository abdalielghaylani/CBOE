Date Search

Date searching allows users to search for data based on the date field. A range may be specified using a hyphen between the values at either end of the range. Ranges may also be indicated using inequality operators(<,>) together with the AND operator.

The following table lists a few examples of date search.

Date entered

Records retrieved

Apr 11, 1971

Retrieves all entries with the exact date of April 11, 1971

Mar 31, 1971-July 1, 1971

Retrieves all entries from Apr 1-June 30 1971

>4/11/71

Retrieves any record after April 11, 1971

>4/11/71 AND <7/21/71

Retrieves any entry between April 11 and July 21, 1971
