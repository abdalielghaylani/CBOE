Export hits

Hitlists may be exported and saved using the following export formats:

  • SDF Nested
  • SDF Flat

SDF Nested and SDF Flat formats describe how child data will be presented in relation to the parent.

In SDF Nested, a record will be created for each base table record and each child table will have its data concatenated into a single sdf field as part of that base record.

In SDF Flat, the base table record will be repeated as many times as the child table with the most rows. One child row will appear for each child table. The child row data will be treated as individual fields.

To export a hitlist:

  1. Click on the Export button in the Search results hitlist window

  1. The Export Control window displays

  1. Select the Export Format desired, the export template and check the properties to be exported as shown in the screen shot below:

  1. Click on Export button, the following window displays, click Yes to export the selected records::

  1. The following dialog displays, click on Open button to open the SDF file or click on Save button to specify a location where the file is to be saved, the hitlist in the selected format is saved in the specified location.

Adding New Export Templates

It is possible for users to create their specific export templates and to use them every time they export hitlists.

To create a template:

  1. Click on Add button, beside the Current Template field

  1. The following dialog displays, enter a name and description for the new template, modify the properties names when necessary, check the default properties to be viewed while exporting and click on Save button to save the template. In order to make the template public and available to all users, check the "Is Public" checkbox

Deleting an Export Template

To delete an Export Template:

  1. Select the template that you want to delete
  2. Click on Delete button

  1. A confirmation message displays, click on Ok button to delete the selected template

Editing an Export Template

It is possible to edit an export template if changes are required. To edit an export template:

  1. Select the template that you want to edit
  2. Click on Edit button

  1. The following window displays. Make the necessary edits and click on Update button
