Submitting Mixtures

A mixture is a material made up by two or more components which are mixed but are not combined chemically. Depending on your system settings, you can either create mixtures only from registered components or create mixtures by adding new components and drawing structures as well as from existing components. When adding existing components the component information values are picked from the permanent database.

Two different settings are available:

  • Simple Mixture Settings: AllowUnregisteredComponentsInMixtures = False, EnableMixtures = True

  • Complex Mixture Settings: AllowUnregisteredComponentsInMixtures = True, EnableMixtures = True
