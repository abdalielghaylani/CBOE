Load Template

Users can open saved templates using the load template option.

To load a template:

  1. Open the Submit Form
  2. Go to "Submission Templates" and then select "Load Template" option

  1. The Load Template form displaying the list of available templates appears. Click on the record's Registry Name to open it.

  1. The Registry Record is loaded. Enter the rest of information and then Submit the Record.

Note: It is possible to delete a Registry Record Template from the Load Form, by checking the Delete checkbox corresponding to the template to be deleted and clicking on the Delete button.
