CASE A

When the system has the following settings:

  • SameBatchIdentity(SBI) = False
  • Enable Fragments = True
  • EnableMixtures = False

If the structure that the user wants to register is already in the Chemical Registration Database, a solving duplicate window will appear.The only field for checking duplicates is the structure one.

Four options are available:

  • Cancel: to cancel the registration process
  • Add Batch: to add a batch to the currently displaying duplicated registry. This will conclude the duplicates resolution process.

  • Duplicate: to create a new registry record duplicating the compound information. Then the record will be duplicated with the following properties
    • Unique Reg Number
    • Different component ID
    • Same structure
    • Different structure ID
  • Use Structure: to create a new Registry Record reusing the same structure ID. If there is a structure change both registry records will be affected.
