List view

The list view lets the user view multiple search results simultaneously. The list view does not display all the fields available for a record. An example of a list view is shown below:

It is possible to specify the number of records to be displayed per page by specifying record count in the Preferences tab. If the number of hits exceeds the record count specified, browse to the next page to view the rest of the records. List View is the default view set.

If highlight sub-structures checkbox is checked, the substructure specified in the search form will be highlighted in the structures results. If Benzene is the substructure then the Benzene structure will be highlighted in the component structure as shown in the screen shot below:
