Moving a Batch

Users can move a batch from one registry record to another using the move batch option.

Note that this feature should be enabled in the system settings by the system administrator.

To move a batch:

  1. Click on the Reg Number link corresponding to the record. The Registry Record page appears.
  2. Highlight the batch that you want to move and then click on Move Batch button.

The following page appears:

  1. Specify the Registry Number of the destination registered record where the selected batch will be moved to.
  2. Click Move Batch button.
