Web Registration - Identifier Types

Identifier information includes information about the chemical name, synonym, and CAS number of a compound. It helps to identify records easily.

Identifier information can be added in the registry or component level.

Entering a New Identifier

  1. In the Web Registration Admin window, select Identifier Types from the Custom Tables dropdown list

The Identifier Types window appears.

  1. Click on the plus sign, a new row gets added to the table

  1. Enter a name for the identifier, a description, select if you want this Identifier to be active, select the Identifier type and click the Save icon. Following are the options for the Type dropdown list:
  • Registry: Identifiers display at the Registry level
  • Compound: Identifiers display at the Component level
  • Batch: Identifiers display at the Batch level
  • Base Fragment: Identifiers display at the Structure level
  • All: Same Identifiers display at all levels.

The new identifier gets added to the Identifiers table.
