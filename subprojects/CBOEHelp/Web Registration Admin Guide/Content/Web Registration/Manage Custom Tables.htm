Manage Custom Tables

The table editor allows administrator to add, edit, or delete values from pull down menus. 

In the Web Registration Admin window:

  1. Click Custom Tables, a drop-down list displays showing all the available tables.

  1. Select a table to edit.

Note: Newly created, updated or deleted fields or tables are updated in Registration pages only after re-logging into the application.

You can edit the following tables:

  • Projects
  • Picklist Domains and Picklist Values
  • Notebooks
  • Fragments and Fragment Types
  • Sequences
  • Identifier Types
  • Sites

Editing and Deleting Tables

To edit entries in an existing table:

  1. In the Custom Table window, click on the edit icon corresponding to the row that you want to modify, the form displays:

  2. Make the required edits and click Save.

To delete an entry from the table:

  1. In the Custom Table window, click on the delete icon corresponding to the row that you want to delete.
  2. Click Yes in the confirmation window.
