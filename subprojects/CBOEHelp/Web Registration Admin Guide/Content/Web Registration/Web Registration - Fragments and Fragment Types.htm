Entering a New Fragment Type

  1. In the Web Registration Admin window, select Fragment Types from the Custom Tables dropdown list

The Fragment Types window appears.

  1. Click on the plus sign, a new row gets added to the table.

  1. Enter the fragment description and click Save icon.

The new fragment type is added.

Adding a New Fragment

  1. In the Web Registration Admin window, select Fragment from the Custom Tables dropdown list

The Fragments window appears.

  1. Click on the plus sign, the following page appears:

  1. Draw the structure using ChemDraw Direct tools, select the Fragment Type, enter a code and a description, enter MW and Formula.
  2. Click Add Fragments button.

The new fragment gets added to the Fragments table.
