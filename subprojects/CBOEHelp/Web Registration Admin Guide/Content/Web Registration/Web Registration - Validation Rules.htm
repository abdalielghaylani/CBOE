Web Registration - Validation Rules

The Validation Rule allows you to set the validation rule for every property.

To add a new validation rule to a particular property:

  1. In the Registration Admin window, click on System Behaviors button and select Manage Data Properties option from the menu.

The Manage Data Properties window appears.

  1. Click on the Validation Rules check sign corresponding to the property to which you want to add a validation rule

The Validation Rule window appears showing the list of existing validation rules

  1. Click on Add Rule button, the following window appears:

  1. Select the validation rule type from the Type dropdown list and fill in the corresponding fields
  2. Click Save Rule.

Validation Rule Types

RequiredField

Used to make the selected property required.

Error: Enter in the Error text box, the message that you want to be displayed when the validation rule is not respected

Default value "Property Type": When the property is required, enter a value to be displayed by default in the property field for the existing records. Note that when a property is required and there is no default value selected, this will create a data exception when the user edits old records because it is not possible to have Null value in a required field.

Text Length

Error: Enter in the Error text box, the message that you want to be displayed when the validation rule is not respected

Add Parameters: type the minimum and maximum text length values.

Word List Enumeration

Use this type to pre-define a specific list of words that can be used by the user in a text field while submitting a record.

Error: Enter in the Error text box, the message that you want to be displayed when the validation rule is not respected.

ValidWord: Enter the valid word and click on the plus sign to move it to the Parameters List in the right frame. You can add as much valid words as you want.

Custom

Used for custom validation rules.

Error: Enter in the Error text box, the message that you want to be displayed when the validation rule is not respected.

Client Script: Enter the validation script. For example:

source.errormessage='';arguments.IsValid = DropDown_CheckRequiredFields(source,arguments,'1','AMOUNTProperty','Amount Required');
