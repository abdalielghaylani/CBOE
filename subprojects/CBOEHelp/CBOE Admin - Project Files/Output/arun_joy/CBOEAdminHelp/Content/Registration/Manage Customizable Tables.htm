Manage Customizable Tables

The table Editor allows administrator to add, edit, or delete values from pull down menus.  These pull down menus have a limited number of options. 

In the Registration Admin window:

  • Click on "Manage Customizable Tables" link
  • The Table Editor dialog displays
  • Select a table to edit from the "Select a table/view" dropdown list

You can edit the following tables:

  • PickList Domain
  • PickList Values
  • Projects
  • People
  • Notebooks
  • Fragment Types
  • Fragments
  • Sequences
  • Identifier Types
  • Sites

Note: Newly created, updated or deleted fields or tables are getting updated in Registration pages only after doing refresh.
