Configuring Structure Settings

Settings for structures are defined in a file called MT.CDX.  This file is found in the graphics folder in an application's folder.  For example, the MT.CDX file that defines the settings for the Sample application is found in <webroot>/chemoffice/sample/graphics .

The process for editing structure settings is the same for each application.  These settings will take effect for all users of an application. The following describes how to change structure settings for the sample application.  To implement these instructions for another application, substitute the sample directory with the application directory you wish to change 

To edit structure settings for the sample application:

  1. Browse to <webroot>/chemoffice/sample/graphics.
  2. Double-click on the MT.CDX file.
    The file opens in ChemDraw.  The file appears to be empty.
  3. Under the File menu, there are 3 types of settings that can be changed:
    • Drawing Settings
    • Caption Text Settings
    • Label Text Settings

    Select the appropriate menu option.  For more information about these menu options, please see Customizing ChemDraw in the ChemDraw Manual.
    A dialog box appears.
  4. Make the changes desired.
  5. Click OK.
  6. Save and close the file (MT.CDX).
  7. Browse to <webroot>/chemoffice/cfwtemp.
  8. If there is a directory named sample (or the name of the application you are configuring settings for), delete the directory (i.e. delete <webroot>/chemoffice/cfwtemp/sample).
    The next time that application is accessed, the settings will take effect.
