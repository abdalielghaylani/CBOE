Chemoffice.ini Configuration File

The ChemOffice.ini file is necessary for any ChemOffice WebServer application to function properly. This file gives the WebServer basic information about the applications such as a start path and database names.

The file starts with the following GLOBAL settings:

  • CD_PLUGIN_THRESHOLD
  • CD_PLUGIN_DOWNLOAD_PATH
  • AppName
  • WIZARD_APPNAMES
  • CS_SECURITY
  • FORCE_ORACLE_SESSION_CURSOR_SHARING

and then continues with a section for each application. Application sections begin with a section heading (the application name in brackets. For example, the following is the default application definition for Sample:

[SAMPLE]
SECTION=CO_WebServer_STANDARD
DBNAMES=SAMPLE
SAMPLE_START_PATH="/sample/default.asp?formgroup=base_form_group&dbname=sample"
SAMPLE_DISPLAY_NAME="Sample"

Each application section can contain the following settings (any missing settings are set to the default):

  • SECTION
  • DBNAMES
  • GLOBAL_SEARCH
  • GLOBAL_SEARCH_DISPLAY_NAME
  • GLOBAL_SEARCH_START_PATH
  • <DBNAME>_START_PATH
  • <DBNAME>_DISPLAY_NAME
