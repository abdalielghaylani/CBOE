Customization and Administration

There is not much administration necessary for the ChemOffice WebServer, beyond maintaining username and passwords. As is true for many web based application, it is helpful to be familiar with IIS and Oracle.

Because all ChemOffice WebServer code regarding appearance is public, administrators have the ability to customize these applications. This includes, for example, moving fields on a page, removing fields, and adding hidden fields.

The purpose of this section of the ChemOffice WebServer Admin Guide is to offer advice in customizing ChemOffice WebServer applications.

What would you like to do?

  • Use the Administrative Tools (creating new applications and data views)
  • Customize a ChemOffice WebServer Application
  • Convert a legacy ChemOffice WebServer installation running on Access for use with Oracle or SQL Server
  • Review brief description of some common functions used in ChemOffice WebServer application code
