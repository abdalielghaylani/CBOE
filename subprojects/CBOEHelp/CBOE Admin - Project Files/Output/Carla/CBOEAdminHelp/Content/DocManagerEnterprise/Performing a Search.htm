Performing a Search

To search documents for specific information:

  1. Click Search Documents in either the ChemBioOffice Manager main page or the DocManager Main Menu. The Document Search page appears:

  1. Enter the search criteria by filling-in the various fields. The Full-Text text box allows to search exact word or phrases.For more information about drawing structures, please see “ChemBioDraw User Guide”.
  1. Click Search. The search result appears:

  1. Click the title of the appropriate document to view it in your web browser.

To delete a document, just click on the Delete button.

NOTE: You can also save a file displayed in the result by clicking on DownLoad corresponding to the file.
