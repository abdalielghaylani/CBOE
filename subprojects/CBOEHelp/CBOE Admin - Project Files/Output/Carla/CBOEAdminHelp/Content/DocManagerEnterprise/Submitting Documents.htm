Submitting Documents

To submit a document:

  1. Click Submit documents in the DocManager Enterprise section. The Add Document page appears:

  1. In the document text box, enter the directory location and file name of the document to be submitted. You can also use Browse to locate the appropriate document.
  2. (Optional) To add details about the document, click Add Document Details.
  3. In the Add Document Details page, enter the title, author name, comments, main author, report number, status of the document, writer, abstract, document date, document class and document security category for the document being submitted in the appropriate text boxes.

  1. If you added details to the document, click Submit This Document in the Add Document Details screen. Otherwise, click Submit Now on the Add Document screen.
  2. The following confirmation message displays.

  1. Clicking on Search over documents link displays a search form allowing you to search over the uploaded documents.

NOTE: If DocManager Enterprise is integrated with Registration Enterprise, and you have both DocManager and Registration access privileges, an additional field for Registry Number is displayed on this page. The Registry Number you specify in this field will be used to create a link between the submitted document and the registered compound in Registration Enterprise.
