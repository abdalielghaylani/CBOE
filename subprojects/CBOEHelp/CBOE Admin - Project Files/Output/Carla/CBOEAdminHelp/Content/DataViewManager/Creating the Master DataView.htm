Creating the Master DataView

You can create the master dataview by first selecting one or more schemas from your database and then selecting tables from the schemas you selected.

Selecting schemas

  1. Log in to ChemBioOffice Manager as an administrative user.
  2. On the ChemBioOffice Enterprise Manager main page, click on the Edit Master icon under DataView Manager.

  1. The "Select Tables for Master Dataview" window opens.

  1. In the left frame the Tables corresponding to the selected schema in the "Schema" dropdown list above are displayed and in the right frame the corresponding schema summary is displayed.
  2. To add a new schema to the Master Dataview, click Add Schema as shown below:

  1. The Add Schema window appears. In the database dropdown list, select a schema to add to the Master Dataview and in the password field enter the schema password. To publish all relationships defined in the schema, check the Publish Relationship checkbox and click Publish.

 

  1. The schema you selected is added to the Master Dataview.
  2. (Optional) Repeat step 3 through step 5 to add more schemas.

Adding Tables

To add tables to a selected schema:

  1. In the schema dropdown list, select a schema, the list of associated tables displays below as shown in the following screen shot. To view a table summary, click the table name.

  1. To delete a table simply click on its corresponding Remove button
  2. To add more tables, click Add Table, the following window appears:

Note: The tables that are already added appear in red.

  1. To add a particular table, click on its corresponding Add button, the table gets added under "Tables to Add " at the right frame. Select the tables to be added and click on Save button at the bottom. The tables are added to the schema.

Note: It is possible to add table from different schemas.

Submitting the Master Dataview

The Master Dataview should be submitted in order to make it valid and available for dataviews creation.

To submit the Master Dataview:

  1. Click Submit. The Validation Summary screen appears.

  1. In the Validation Summary screen, review the list of tables and their relationships. If the list is acceptable, click Submit. Otherwise, click Cancel to return to the previous screen.
  2. After clicking submit , the following confirmation dialog displays. Click Close.
