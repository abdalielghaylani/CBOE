Editing a Description - Password

To edit the description or password for a schema:

  1. Expand the schema whose description/password you want to edit.
  2. Click the Edit Schema Description/Password link.

  1. The following page appears:

The display name for the schema is the name that will be visible to users. The default display name is the name of the schema. It is recommended that you enter a more descriptive name for the schema so that it can be easily identified.

The password for the schema is assigned when the schema is created.

  1. Change the display name and password of the schema according to your requirements.
  2. Click the Save Changes button to save the changes.
