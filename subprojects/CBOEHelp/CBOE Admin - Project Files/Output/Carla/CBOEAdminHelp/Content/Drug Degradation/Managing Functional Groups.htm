Managing Functional Groups

Adding Functional groups

To add a functional group:

  1. In the Drug Degradation Enterprise page, click Manage functional group list.
  2. In the New functional group text box, enter the formula or name for the functional group. (examples: “CH3”, “Acetyl”)
  3. Click Add Record.

  1. A message displays, informing that the record is added. Click Ok button. The following window displays.

Deleting functional groups

You can delete only those functional groups you have added to the list.

To delete a functional group:

  1. In the Drug Degradation Enterprise page, click Manage functional group list.
  2. In the functional group list, select the Delete Functional Group link for the group you want to delete.
  3. The following confirmation window displays. Click Delete.
