Editing a Parent Compound

To edit a compound, you first search for it in the database and edit it as desired.

  1. In the Drug Degradation Enterprise main page, click Search.
  2. In the Parent Compound Search page, enter the criteria for the compound you want to edit.

  1. Click Search.
  2. In the search results, click Show Details for the compound you want to edit.

  1. In the Parent Compound Details screen, click Edit Record.

  1. Edit the parent compound data as desired.
  2. Click Update.
