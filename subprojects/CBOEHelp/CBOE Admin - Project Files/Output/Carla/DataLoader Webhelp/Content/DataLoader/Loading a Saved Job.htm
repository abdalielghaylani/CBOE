Loading a Saved Job

To load a previously saved job:

  1. Select Load a previously saved job in the Task section after logging into Data Loader.
  2. Click Next. The Load Settings window appears as:

  1. Select the job to be loaded. The selected job appears in the Load from text box.
  2. Click OK. This screen appears as:

  1. Configure the job, if required, and click Next. Data is loaded and the Summary screen appears.
  2. Click Next. The task is completed.
