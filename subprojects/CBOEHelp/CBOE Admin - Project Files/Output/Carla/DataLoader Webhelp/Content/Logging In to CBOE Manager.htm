Logging In to CBOE Manager

In order to log in to CBOE Manager, please follow the steps below:

  1. Type the following URL in the web browser: https://servername/COEManager (servername is the name of the server where CBOE is installed) the following page displays:

  1. Enter specific User Name and Password to log in to CBOE manager, and then click on Log in dialog
  2. The ChemBioOffice Enterprise Manager home page displays

The Manager displays all the main ChemBioOffice Enterprise features categorized by application. To launch a feature, click its icon.

Note: Not all users have the same features displayed on the CBOE Enterprise Manager main page. The display of sections and icons related to specific CBOE applications depends upon the roles assigned to each user.
