Moving Users

It is possible to move users between groups as shown in the steps below:

  1. Check the checkbox(es) corresponding to the user (s) to be moved and then click on Move icon in the toolbar
  2. A list of the available groups displays. Select the destination group and click on Move button.

  1. The user (s) will be moved to the target group

Note: “Duplicate users in select group” option, once checked, allows you to copy the user(s) instead of moving them, hence you will have the same user(s) in both groups.
