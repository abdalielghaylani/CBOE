Subroutine

 
          
      GetConnection(dbkey,formgroup,table)

Description

Establich an ADO connection to a table referenced in the .ini file

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

table = <name of table in ini file to look for connection information>

Example

Set DataConn= 
          
      GetConnection(dbkey,formgroup,"MolTable")
