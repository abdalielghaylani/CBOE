NAME_ADD_INFO_ITEM

Example: NAME_ADD_INFO_ITEM_1= addtoHistoryMenu(dbkey, formgroup)

Description: VB script which controls if something (dynamic or static) is added to a menu item.  For example, the Show Marked item adds the number of marked hits to the right of Marked Hits.

Default: default is dependant on the menu.

Options: VB script

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
