SHOW_RETRIEVE_ALL_BTN

Example: SHOW_RETRIEVE_ALL_BTN= 1

Description: Determines if the retrieve all button is displayed in the interface. If the retrieve all button is not displayed, a user will have to perform a search to view records. 1 indicates the retrieve all button is visible.

Default: 1

Options: 0 | 1

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
