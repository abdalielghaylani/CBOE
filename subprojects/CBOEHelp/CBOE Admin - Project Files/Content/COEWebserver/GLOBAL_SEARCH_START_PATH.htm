GLOBAL_SEARCH_START_PATH

Example: GLOBAL_SEARH_START_PATH="/chemrxn/default.asp?formgroup=global_search_form_group&dbname= chemprep"

Description: The start path (on the login page) for the link to global searching in the application.

Default: Depends on the application.

Options: Start path for global searching in the application.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
