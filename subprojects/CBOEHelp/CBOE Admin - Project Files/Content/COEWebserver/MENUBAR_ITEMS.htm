MENUBAR_ITEMS

Example: MENUBAR_ITEMS=FILE,HISTORY,QUERIES,QUERIES_RESTORE,HITLISTS,HITLISTS_RESTORE,MARKED_HITS,HELP,LOGOFF,HOME

Description: This is a list of menu names.  Submenus are listed here as <MENUNAME>_<SUBMENUNAME>.  For example, the Restore submenu found under the Queries menu is listed as QUERIES_RESTORE.  The menus are defined in the menubar.ini file under the section heading with the same name as listed in this entry. 

Default: FILE,HISTORY,QUERIES,QUERIES_RESTORE,HITLISTS,HITLISTS_RESTORE,MARKED_HITS,HELP,LOGOFF,HOME

Options: Comma delimited list of menu names.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
