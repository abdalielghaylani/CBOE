BRACKET_IN_STRUC_HANDLING

Example: BRACKET_IN_STRUC_HANDLING= "WARN"

Description: Related to brackets entered in the structure window. Brackets are not recognized as having any chemical signifigance. WARN indicates the user will be warned if brackets are used, ALLOW ignores the brackets, DISALLOW does not allow the use of brakets in the structurewindow.

Default: WARN

Options: WARN | ALLOW | DISALLOW

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
