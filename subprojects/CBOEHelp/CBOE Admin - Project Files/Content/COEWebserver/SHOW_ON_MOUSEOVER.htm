SHOW_ON_MOUSEOVER

Example: SHOW_ON_MOUSEOVER=1

Description: This controls if the menu automatically drops down from the menubar during a mouseover.  A value of 1 makes the menu drop down, a value of 0 keeps the menu from dropping down until the menubar item is clicked.

Default: 1

Options: 0 | 1 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
