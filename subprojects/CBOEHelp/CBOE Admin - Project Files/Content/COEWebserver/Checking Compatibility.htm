Checking Compatibility

CSDO.dll Version

To check to see if your CSDO.dll is up to date:

  1. Browse to  \Program Files\ChemOffice2001\Common\dlls.
  2. Right-click on the file and select Properties.
  3. Click on the Version Tab .
    If you have version 1.1.0.24 you will need to update to 1.1.0.25 or later.

CSDO_UTILS_VBS.ASP Changes

This file is located in \intepub\wwwroot\chemoffice\WebServer_source\cfserverasp\source . 

  1. Open the file in a text editor and search for "select case conn_dbms". 
  2. Make sure that the lines following include a case for SQLSERVER.  The complete select case should look like:

    select case conn_dbms
        case "ORACLE"
            dbms = 2
        case "SQLSERVER", "SQL_SERVER"
            dbms = 3
        case else
            dbms = 1
    end select

    If any of the sqlserver case is missing it should be edited to match the above.
