APP_NAME

Example: APP_NAME= sample

Description: name of the application displayed in some parts of the user interface

Default: The default value is different for each application since it indicates which application the ini file is for.

Options:

Application Name APP_NAME value
Sample Sample

Documentation Manager  

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
