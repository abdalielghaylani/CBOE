ShowResultTable

Subroutine

ShowResultTable 
      dbkey,formgroup,rsname, orientation, cell_width, table_widdth, 
      "0"

Description

Shows results from a recordset in table format

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

rsname = <name of recordset object crested at start of tile>  

orientation = table_down|table_across

cell_width = <width of individual cell in pixels>

table_width = <width of entire table in pixels or percentage>  

Example

 
         ShowResultTable 
      dbkey,formgroup,SynonymsRS,"Table_down","100","100%","0"
