ALERT_CDAX_VERSION

Example: ALERT_CDAX_VERSION=8.0.3

Description: Indicates what the minimum version of the ChemDraw plugin should be used with the software. An alert appears for users using the indicated version or lower when accessing the plugin.

Default: 8.0.3

Options: Should not be edited.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
