Deleting a User Account

To delete a user account:

  1. In the Security section in the Home page, click Manage Users. The Manage Users window appears.
  2. In the View Users From list, select the application from which you want to delete the user account.
  3. In the Select a User list, select the user account.
  4. Click Delete.
  5. Click Done.

Note: You cannot delete a user account if you are logged in using that account.
