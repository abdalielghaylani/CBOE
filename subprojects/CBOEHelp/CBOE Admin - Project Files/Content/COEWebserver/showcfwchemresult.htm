Subroutine

 
         ShowCFWChemResult 
      dbkey,formgroup,chem_type,table.field,mol_id,output_type,width,height

Description

Shows chemical result

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

chem_type = Structure|Formula|MolWeight  

table.field = <table and field name usign dot syntax>

mol_id = mol_id of chemical record to retrieve

output_type = GIF|CDX
         only valid if chem_type=Structures, 0 if not applicable

width = <width of plugin, 0 if not applicable>

height = <height of plugin, 0 if not applicable>

Example

ShowCFWChemResult 
      dbkey,formgroup,"Structure,"MolTable.Structure",BaseID,"cdx","212","156"
