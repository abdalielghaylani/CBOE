PWD_ILLEGALWORD_LIST

Example: PWD_ILLEGALWORD_LIST= password,banana

Description: A comma delimited list of words that cannot be used as a password. This applies to cs_security passwords only and is found in the cs secuirty cfserver.ini file only.

Default: password

Options: any string

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
