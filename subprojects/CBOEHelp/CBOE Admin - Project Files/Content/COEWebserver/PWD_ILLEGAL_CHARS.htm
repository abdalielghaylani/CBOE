PWD_ILLEGAL_CHARS

Example: PWD_ILLEGAL_CHARS= [\W]

Description: A regular expression defining characters that cannot be in a password. This applies to cs_security passwords only and is found in the cs secuirty cfserver.ini file only. For a description of regular expression syntax, please see: http://msdn.microsoft.com/library/default.asp?url=/library/en-us/script56/html/vspropattern.asp

Default: [\W]

Options: any valid regular expression

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
