ChemBioOffice Security

Welcome to the ChemBioOffice Security chapter.

If you are an administrator, you can create users and choose who can access each of the ChemBioOffice Enterprise applications. You can control access for each application from within the application or from the main ChemBioOffice Enterprise Home page.
