PWD_CANNOT_MATCH_USERNAME

Example: PWD_CANNOT_MATCH_USERNAME= false

Description: Indicates if a password is still valid if it matches the username. Setting to true does not allow a password to match the username. This applies to cs_security passwords only and is found in the cs secuirty cfserver.ini file only.

Default: false

Options: true | false

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
