TABLE_ALIASES

Example: TABLE_ALIASES= MolTable,Synonyms_r

Description: Names of all table definition sections in the ini file. This list must be comma delimited with no spaces. If the table name is not here, it will not be loaded by the application. Table definition sections describe all the information needed by ChemOffice Webserver for searching tables. See the MOLTABLE and SYNONYMS_R definitions for more information.

Default: MolTable,Synonyms_r

Options: Comma delimited list of tables

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
