SEARCHABLE_ADO_FIELDS

Example: SEARCHABLE_ADO_FIELDS = MolTable.MOL_ID;1,MolTable.Molname;0,Synonyms_r.SYN_ID;1,Synonyms_r.Synonym_r;0 

Description: List of fields and their datatypes that are in the input form. This list is used by JavaScript to determine what fields are populated before the submit occurs. If there is no field listed here it will not be searchable. Syntax is
tablename.fieldname;datatype
with each full item separated by a comma. These items are case sensitive relative to what is in the input and result forms.  For a list of valid datatypes, please see the ADO_Fields Datatypes Table.   

Default: N/A

Options: Comma delimited list. 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
