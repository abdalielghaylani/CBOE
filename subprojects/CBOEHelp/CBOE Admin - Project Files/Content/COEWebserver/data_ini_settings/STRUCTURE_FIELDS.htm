STRUCTURE_FIELDS

Example: STRUCTURE_FIELDS = MolTable.Structure

Description: Name of the table and field for the structure field in ChemFinder. For example, this is often MolTable.Structure. This is case sensitive relative to what is put into the input and result form. If no structure is present then enter NULL.

Default: True

Options: True | False

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
