CONNECTION_PASSWORD

Example: CONNECTION_PASSWORD= ""

Description: Contains the password for the connection.  If there is a password and you are using a DBQ or DSN then see the See Connection Passwords Tables section under "Connection Options in Dataview INI Files".  The way the password is entered is different for different datasources.  Otherwise, this entry are probably left empty.

Default: ""

Options: N/A 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
