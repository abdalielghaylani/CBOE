MW_FIELDS

Example: MW_FIELDS = MolTable.MolWeight

Description: Name of the table and field for the molecular weight field in ChemFinder. This is often MolTable.MolWeight. This is case sensitive relative to what is put into the input and result form. If no structure is present then enter NULL.  

Default: MolTable.MolWeight

Options: MolTable.MolWeight | NULL 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
