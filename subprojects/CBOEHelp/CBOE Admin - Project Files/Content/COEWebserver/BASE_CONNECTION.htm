[BASE_CONNECTION]

Description: Ado connections specified in the ado_connections section of the globals section reference connection sections else where in the ini file. An ADO_Connection group is referenced by each table alias definition. The keys specify data necessary to create the connection to the database via ado . 

The following settings are used in any ADO Connection section:

  • CONN_TYPE
  • CONNECTION_STRING
  • CONNECTION_TIMEOUT
  • COMMAND_TIMEOUT
  • CONNECTION_USERNAME
  • CONNECTION_PASSWORD

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
