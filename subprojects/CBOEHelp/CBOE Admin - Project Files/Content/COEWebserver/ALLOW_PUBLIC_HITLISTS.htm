ALLOW_PUBLIC_HITLISTS

Example: ALLOW_PUBLIC_HITLISTS= 1

Description: Determines if saved hitlists can be made public. Making a hitlist public allows all other users of the application to access that hitlist. 1 indicates hitlists can be made public.

Default: 1

Options: 0 | 1

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
