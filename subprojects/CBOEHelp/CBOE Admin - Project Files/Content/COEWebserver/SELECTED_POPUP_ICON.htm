SELECTED_POPUP_ICON

Example: SELECTED_POPUP_ICON= /cfserverasp/source/graphics/navbuttons/popup.gif

Description: The path and filename for the popup icon when an item is selected.  The selected popup icon is the icon to the right of a menu item if there is a submenu, and that item is selected.  By default, a small arrowhead.

Default: /cfserverasp/source/graphics/navbuttons/popup.gif

Options: The path and filename for the selected popup icon.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
