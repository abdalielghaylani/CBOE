EXPIRE_USER_SETTINGS_DAYS

Example: EXPIRE_USER_SETTINGS_DAYS= 365

Description: The value of this setting is in days. After the number of days specified of inactivity, user settings are reset to default settings.

Default: 365

Options: An integer indicating a number of days.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application. New versions of the ChemOffice WebServer use default configurations.
