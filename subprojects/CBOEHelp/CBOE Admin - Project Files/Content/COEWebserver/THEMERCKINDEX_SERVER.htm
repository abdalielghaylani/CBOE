THEMERCKINDEX_SERVER

Example: THEMERCKINDEX_SERVER= http://themerckindex.cambridgesoft.com

Description: Points the application to the target URL for the The Merck Index button.  If it is the local server then leave blank. If it is a different internal server, the server name or start url is required (e.g. http://servername). If it is pointing to PerkinElmer, a corporate membership is required.
Only relavant if THEMERCKINDEX_BUTTON is set to ON.
Note: No trailing slash should be included.

Default: ""

Options: URL pointing to The Merck Index server.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
