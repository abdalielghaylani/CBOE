Creating a New Application

This tool will convert a ChemFinder database to a fully searchable ChemOffice WebServer application. For information about customizing the application after it is created, please see Customization of WebServer Applications.

To create a new ChemOffice WebServer application:

  1. Click Create New Web Application from the Web Administration Page.
    The ChemOffice Web Wizard appears.

  1. Enter a name for the new application and click Next.

  1. Browse to the ChemFinder database to be converted.
    Note: The ChemFinder database must have EVERYONE:Full Control as permissions. To make sure this is the case, place the database somewhere in your ChemOffice_data folder (by default found in C:\Chemoffice_data)
  2. Indicate if Structure or Reaction data is found in the database.
  3. Click Next .

  1. Select the fields that should be shown in list view. To select multiple fields, hold the CTRL key down while highlighting fields.
  2. Click Next .

  1. Click Finish.
    The application is generated. A folder with the name of the application is added to the ChemOffice folder. This contains all of the configuration files and INI files necessary for this new application. Please see Customization of WebServer Applications for details about customizing your new application.
