DB_NAMES

Example: DB_NAMES=sample

Description: The name of the directory containing the forms necessary for proper operation of the application. Editing this value may cause unpredictable behavior.

Default: sample

Options: Usually the name of the application. If you would like to enter a list of directories, separate the list with a comma.
