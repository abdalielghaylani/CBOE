Assigning Users to Roles

To assign one or more users to a role:

  1. Select the appropriate application from the View Roles From drop down list in the Manage Roles window.
  2. Select the role in the Select a Role list box and click Edit Users. The Edit Role Users window appears.
  3. In the Available Users list, select the user name to assign to the role and click the right arrow. The name appears in the Current Users list.

Note: Use CTRL+click to select more than one name.

  1. To remove a name from the role, select the name in the Current Users list and click the left arrow.
  2. Click Save when finished.
