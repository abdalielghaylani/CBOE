ChemBioOffice Enterprise

The ChemBioOffice Enterprise 17.1 suite allows a large number of users managing and sharing an important resource of chemical and biological information. In addition to managing diverse types of data on electronic pages that are much like the pages of a paper notebook using the E-Notebook software. Users can record and retrieve information from an extensive set of databases with an easy-to-use Web-based work area. The ChemBioOffice Enterprise 17.1 suite includes a variety of applications. The current document provides information on how to:

  • Create CBOE groups and users
  • Assign Roles and Privileges to CBOE users
  • Configure CBOE 17.1 suite applications.
