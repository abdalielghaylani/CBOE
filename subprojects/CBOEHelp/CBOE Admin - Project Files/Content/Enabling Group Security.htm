Enabling Group Security

As an administrator, you can add, delete and modify user groups using the Group Security feature. In order to enable this feature, follow the steps below:

  1. Run the ChemBioOffice Enterprise Server Configuration Tool
  2. Click on "App Settings" tab and check the Group Security checkbox
  3. Click on Save Configuration button to save the changes
