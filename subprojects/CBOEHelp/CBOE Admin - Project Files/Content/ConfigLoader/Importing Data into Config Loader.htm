Importing Data into Config Loader

To import data into Config Loader:

  1. Log into Config Loader with appropriate credentials to import data.
  2. Select Configuration Import, if it is not already selected.
  3. Click Next. This screen appears as:

  1. Select the folder containing the configuration file to import. The configuration file must be in XML format.
  2. Click Next. Data is imported and the job summary appears.
  3. Click Next. The task is completed.
