Administrator's Guide

Workflow

Configuration related to registration Enterprise can be done manually using the Registration Enterprise's GUI. The configuration settings are stored in an Oracle database.

Config Loader helps the service representatives migrate the configuration settings to a brand new installation of Registration Enterprise.

The workflow of Config Loader is as follows:

  1. Using Config Loader, the service representative exports the configuration settings from Oracle database to the file system as separate files and folders.
  2. The service representative copies the exported files to the recipient system, usually a new customer installation where Config Loader has been installed and configured to interact with the database.
  3. Config Loader is run on the client system to import the configuration files from the file system into the database.

Config Loader serves two primary purposes:

  • Lets you import ChemBioOffice Enterprise's customized XML files into a database. These XML files contain data related to data views and forms used in Registration Enterprise.
  • Lets you export data to an XML file.
