Editing a Degradant

To edit a degradant:

  1. In the Degradation Experiment Details page, click Edit Degradant corresponding to the degradant you want to edit. The Edit Degradant Compound page appears.
  2. Modify the degradant text fields or drawing as desired.
  3. Click Update Record. The Degradant Compound Details page appears, displaying the updated degradant record.
  4. Click Return to Experiment to view the experiment.
