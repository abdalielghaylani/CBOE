Adding a Parent Compound

Using the ChemDraw Toolbar, you can add new parent compound structures through a ChemBioOffice Webserver form.

To add a new parent compound:

  1. In the Drug Degradation Enterprise Web page, click Add Parent Compound.

The Add Parent Compound page appears.

  1. Using the ChemDraw Toolbar, draw the parent compound.
  2. (Optional) Enter additional information in the appropriate fields:
  • Generic Name
  • Trade Name
  • Common/Other Name
  • Compound Number
  1. (Optional) If this is a salt, check the appropriate checkbox.
  2. Click Add Record.

  1. A message displays, informing that the record is added. Click Ok.
