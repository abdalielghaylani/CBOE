Managing Data Sources

In the DataView Manager, you can manage data sources.

Creating Data Sources

To create new data sources:

  1. Log in to ChemBioOffice Manager as an administrative user.
  2. On the ChemBioOffice Enterprise Manager main page, click on the Manage Data Sources icon under DataView Manager.

  1. The Manage Data Sources window opens. From here, you can add new data sources, edit existing data sources or delete existing data sources.

Adding a New Data Source

To add a new Data Source:

  1. Click on Add Data Source link. The Add Data Source dialog opens:

  1. Enter the following information:
    • Data source name: The data source alias. The Data source name can contain alphanumeric characters only and must be unique. An entry is mandatory.
    • Data source type: currently only ORACLE is supported.
    • Driver type: The way we connect to data source, currently we support ‘Oracle’ and ‘Oracle (Data Direct)’
    • Host: The host name of the data source.
    • Port: The database port of the data source.
    • SID: The service name of the data source.
  2. Click Save.

The newly added data source is appended to the Manage Data Sources list.

Editing a Data Source

To edit an existing Data Source:

  1. Select the Data Source that you want to edit and click on Edit Data Source link.

  1. The Edit Data Source dialog displays. Make the edits you want and click Save.

Note: The primary data source cannot be edited.

Deleting a Data Source

To delete a Data Source:

  1. Select the Data Source that you want to delete and click on Delete Data Source link.
  2. You will be prompted to confirm the deletion. Clicking ‘Yes’ will verify if any dataviews are published from the selected data source. If there are datataviews published from the data source, a message will appear indicating that the current data source is used by existing dataviews or Master dataview, and you will be prompted to delete the related dataviews. You can delete dataviews manually by accessing Delete Existing Dataview from the Dataview Manager.
  3. After the data source is deleted, the following message appears:

  1. Click OK to close the message dialog.

Note: The primary data source cannot be deleted.
