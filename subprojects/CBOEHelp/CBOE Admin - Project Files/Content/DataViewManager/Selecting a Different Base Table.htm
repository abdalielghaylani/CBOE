Selecting a Different Base Table

Whenever you create a dataview from Master, you can choose a different base table for the dataview.

Note: Select Base Table is disabled when you use Create from Existing option to create a dataview or when you edit an existing dataview.

  1. In the DataView Manager, click on Create from Master. The DataView Manager window opens.
  2. In the Name and Description text boxes, enter a name for the new dataview and a brief description.
  3. In the dropdown lists, select a data source and a schema and then specify a base table by selecting its corresponding radio button.
  4. Click OK.
  5. Add more than one table to the dataview that you are creating, the Select Base Table link becomes enabled. Click on it.

Note: You should add more than one table to get Select Base Table option enabled.

  1. The Select Base Table pane appears.
  2. Select a data source and schema from the corresponding dropdown lists.
  3. Select a base table from the list of tables and click OK. The Select Base Table pane closes.

To save your changes:

  1. In the Select Tables pane, click Submit. The Select Tables pane closes and the Validation Summary pane appears.
  2. After you review the validation summary, click Submit. The Validation Summary pane closes.
  3. In the confirmation window, click Home to return to the ChemBioOffice Manager.
