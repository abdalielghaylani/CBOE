Specifying Security Settings

You can choose which Datalytix users can access the dataview. You can select specific users and roles. When you select a role, all users assigned to that role have access to the dataview.

  1. In the DataView Manager, click Edit existing. The DataView Manager window opens.
  2. From the list of dataviews, select a dataview to edit and click OK.
  3. On the right pane, click on Security Settings.

  1. The following window appears:

  1. Select either to make the dataview public or available to only specific users or roles.
  2. If you choose to make the dataview public, click OK. The security is set and you are returned to the Select Tables screen.
  3. If you choose to set users and roles for the dataview, the lists of available users and roles appear.
  1. Select the desired user name(s) in the Available Users column and click the Add User icon in the Action column. The user(s) you selected appear under Current Users.
  2. Select the desired role(s) in the Available Roles column and click the Add Roles icon in the Action column. The role(s) you selected appear under Current Roles.
  3. Click OK. The security is set and you are returned to the Select Tables screen.

To save your changes:

  1. Click Submit. The Validation Summary pane opens.
  2. After you review the validation summary, click Submit. The Validation Summary pane closes.
  3. In the confirmation window, click Home to return to the ChemBioOffice Manager.
