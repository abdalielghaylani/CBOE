Managing Tables

To add or remove tables from the dataview:

  1. Log into ChemBioOffice Manager as an administrator.
  2. In the DataView Manager, click on Edit existing icon. The DataView Manager window appears.
  3. From the list of dataviews, select a dataview to edit and then click OK.
  4. The list of the dataview's table appears on the left pane.
  5. The delete a table, click on its corresponding Remove button.
  6. To add tables, click Add Table and add the desired tables.

To edit a table:

  1. In the left pane, select the name of the table.
  2. In the table summary, click Edit Table. The Edit Table and Fields pane appears.

  1. In the Edit Table and Fields pane, select the appropriate options as desired and click OK.

To save your changes:

  1. Click Submit. The Validation Summary pane opens.
  2. After you review the validation summary, click Submit. The Validation Summary pane closes.
  3. In the confirmation window, click Home to return to the ChemBioOffice Manager.
