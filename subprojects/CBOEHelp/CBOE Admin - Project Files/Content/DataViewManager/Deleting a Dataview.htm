Deleting a Dataview

To delete a Dataview:

  1. In the DataView Manager, click on Delete Existing Dataview icon.

  1. Select a data source from the dropdown List. The related dataviews table displays:

  1. Select the checkbox corresponding to the dataviews that you want to delete.
  2. Click Delete. A confirmation window opens.
  3. Click OK to confirm.
