Using an Existing Dataview

To create a dataview from an existing dataview:

  1. Log into ChemBioOffice Manager as an administrator.
  2. In the DataView Manager menu, click on Create from existing. The DataView Manager window appears.

  1. Expand the dataview list in the DataView Manager window.
  2. Select an existing dataview from which to create the new dataview. The dataview property box appears to the right of the list.
  3. Click OK.

  1. In the Name and Description text boxes, enter a name for the new data field and a brief description
  2. Click OK.

 

  1. The list of the dataview's tables opens, allowing you to add tables and proving a summary of the selected schema.

  1. To add new tables, click Add Table and perform the same steps covered in "Creating a Dataview from the Master" topic.
  2. To remove a table, click Remove corresponding to the table to be removed.
  3. You may be able to edit the dataview's name and description, and security settings. Please refer to Creating a Dataview from the Master for more information.
  4. Click Submit, the Validation Summary pane appears, displaying any errors that the dataview may contain.

  1. After reviewing the Validation Summary, click Submit.
