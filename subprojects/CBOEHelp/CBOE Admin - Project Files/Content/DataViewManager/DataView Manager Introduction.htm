DataView Manager Introduction

DataView Manager is used to create DataViews for Datalytix.

A dataview is a defined set of database tables that are made available for Datalytix users for searching purpose. After a dataview manager user creates a dataview and assigns adequate access permissions to that dataview, it becomes available to the Datalytix users.

Only users with administrative privileges are allowed to use dataview Manager.
