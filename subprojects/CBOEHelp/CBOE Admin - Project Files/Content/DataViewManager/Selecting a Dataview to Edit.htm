Selecting a Dataview to Edit

It is possible to edit an already created Dataview. To do this:

  1. Log into ChemBioOffice Manager as an administrator.
  2. In the DataView Manager, click Edit existing.

  1. The DataView Manager window opens.
  2. From the list of dataviews, select a dataview to edit and then click OK.

  1. The Dataview becomes in an editable mode. Make whatever changes you want, adding/removing tables, name and description updates, security settings changes, etc..and then click Submit.
