Web Registration - Picklist Domains and Picklist Values

Picklists are dropdown lists that provide users a list of all possible values for a field. From the Table Editor, administrator can create a list of values to make a dropdown list field. The list of values are grouped in a Domain as shown in the example below:

PickList Domain PickList Values

  • Final Product
Stereochemistry Comment
  • No Comment
  • Mixture of Isomers
  • Mixture of Diastereoisomers
Color
  • Blue

  • White

Adding a New Picklist Domain

To add a new Picklist Domain:

  1. In the Web Registration Admin window, select Picklist Domains from the Custom Tables dropdown list
  2. Click on the plus sign , a new row gets added to the table

  1. If only a list of values is needed, enter the domain description and then click on Save icon.
  2. If values come from another Table, enter the following information to build your own SQL and then click on Save icon:
  • External Table: Enter the name of the table. If this table is from another schema than REGDB, then enter the schema name before the table name (Ex. COEDB.PEOPLE)
  • External Column ID: Enter the name of the column that contains the table ID
  • External Column Display: Enter the name of the column that contains the values to display in the dropdown list
  • Additional SQL Filter: Enter a clause to filter the values to display in the dropdown list.
  • Additional SQL SortOrder:Enter a clause to specify the order of values to appear in the property's dropdown list. Note that the top most value will be selected as default value.
  • Is Locked?: Select Yes to make the row read-only, and select No to make the row editable.

Note: when a domain is used while registering a compound, it becomes automatically read-only in the Picklist Domains table.

Entering a List of Values

To enter a list of values:

  1. In the Web Registration Admin window, select Picklist Values from the Custom Tables dropdown list.
  2. Click on the plus sign, a new row gets added to the table.
  3. Select the Picklist Domain from the dropdown list and enter the value for the new Picklist value.

  1. Click Save icon. The new value gets added to the Picklist table.
