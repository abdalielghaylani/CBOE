Web Registration - Project Table

The Projects table stores the projects that can be associated with registry records or batches. This information appears in the Project dropdown list when compounds or batches are added.

Entering a New Project

To enter a new project:

  1. In the Web Registration Admin window, select Projects from the Custom Tables drop-down list. The Projects window appears.
  1. Click on the plus icon, a new form pops up as shown below.

  1. Enter the Project name in the Name text box
  2. Enter the Project description in the Description text box
  3. Select Yes from the Is Active? list box if you want the new project to appear in the Project drop-down list.
  4. Select the Type of the project, specifying at which level the Project field will be displayed, Batch level, Registry level or both.
  5. Click the Save button. The new project gets added to the Project's table.
