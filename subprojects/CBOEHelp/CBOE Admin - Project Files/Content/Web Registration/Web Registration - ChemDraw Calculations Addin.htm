Web Registration - ChemDraw Calculations Addin

The ChemDraw Calculations addin is used to apply ChemDraw control 12 transformations to a given property of the registry. This can calculate the Structure Name, Formula Weight, Molecular Formula and Percent Active. The code is extensible to any kind of transformation/operation provided by the ChemDraw control.
