Entering a New Fragment Type

  1. In the Web Registration Admin window, select Fragment Types from the Custom Tables drop-down list. The Fragment Types window appears.
  1. Click on the plus icon, a new form pops up as shown below.

  1. Enter the fragment description and click the Save button.

The new fragment type is added.

Adding a New Fragment

  1. In the Web Registration Admin window, select Fragments from the Custom Tables drop-down list. The Fragments window appears.
  1. Click on the plus icon, a new form pops up as shown below.

  1. Draw the structure using ChemDraw tools, select the Fragment Type, enter a code and a description, enter MW and Formula.
  2. Click Add Fragments button.

The new fragment gets added to the Fragments table.
