Web Registration - Notebooks

The Notebooks table stores the notebooks that can be associated with registry records or batches. This information appears in the Notebooks drop-down list when compounds or batches are added.

Adding a New Notebook

To add a new notebook:

  1. In the Web Registration Admin window, select Notebooks from the Custom Tables drop-down list. The Notebooks window appears.
  1. Click on the plus icon, a new form pops up as shown below.

  1. Enter the Notebook name in the Name text box.
  2. Enter the Notebook description in the Description text box.
  3. Select Yes from the Is Active? list box if you want the new notebook to appear in the Notebooks drop-down list.
  4. Select the notebooks owner from the User Code drop-down list.
  5. Specify the notebooks order of appearance.
  6. Click on the Save button. The new Notebook gets added to the Notebooks table.
