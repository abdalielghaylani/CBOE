Web Registration - Notebooks

The Notebooks table stores the notebooks that can be associated with registry records or batches. This information appears in the Notebooks dropdown list when compounds or batches are added.

Adding a New Notebook

To add a new notebook:

  1. In the Web Registration Admin window, select Notebooks from the Custom Tables dropdown list

The Notebooks window appears.

  1. Click on the plus sign, a new row gets added to the table

  1. Enter the Notebook name in the Name text box.
  2. Enter the Notebook description in the Description text box.
  3. Select Yes from the Is Active? list box if you want the new notebook to appear in the Notebooks dropdown list.
  4. Select the notebooks owner from the User Code dropdown list.
  5. Specify the notebooks order of appearance.
  6. Click on Save icon.
  7. The new Notebook gets added to the Notebooks table.
