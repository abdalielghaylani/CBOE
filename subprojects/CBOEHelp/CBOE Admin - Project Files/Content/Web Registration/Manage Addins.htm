Manage Addins

To manage the Web Registration addins:

  1. In the Registration Admin window, click on System Behaviors button and select Manage Addins option from the menu.

The following window displays:

From this window, you may add, edit or delete the Web Registration addins.

Adding a new Addin

To add a new addin:

  1. Log in to Web Registration with Administrative privileges
  2. Click on System Behaviors and select Manage Addins option. The Manage Addins window appears.
  3. Click on the plus sign, the Add Addins window appears

  1. Enter the Add-in name in the Friendly Name field, select an Assembly and a Class from the dropdown lists.
  2. Select a Handler and an Event from the dropdown lists.
  3. Fill in the Configuration text box by entering an xml query. Click Save Config button.
  4. Click on Save button.
