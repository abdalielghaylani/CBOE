Web Registration - Manage Data Properties

Manage Data Properties allows you to add, edit, or delete custom properties.  From this menu you can also create or delete Validation Rules.

  1. In the Registration Admin window, click on System Behaviors button and select Manage Data Properties option from the menu.

The Manage Data Properties window appears.

  1. To Add a new property, click on the plus sign

The Add New Property window appears:

  • Group Name: to select at which level you want to create the new properties. Five different levels are available:
    • Batch
    • Batch Component
    • Compound
    • Registry
    • Base Fragment
  • Name: This is the real Oracle Field Name (Spaces and other special character are not allowed)
  • Label: Display Name that will appear in all Registration Forms
  • Type: Property Type
    • TEXT: —>VARCHAR2
    • DATE —> DATE
    • BOOLEAN —> NCHAR(1)
    • NUMBER —> NUMBER(X,Y)
    • PICKLISTDOMAIN —>
    • INTEGER —> NUMBER(9)
    • FLOAT —> NUMBER(14,6)
    • URL
  1. Click Add Property button.

The new property gets added to the Properties table under the specified group.
