Manage Custom Tables

The table Editor allows administrator to add, edit, or delete values from pull down menus. 

In the Web Registration Admin window:

  1. Click Custom Tables, a dropdown list displays showing all the available tables

  1. Select a table to edit.

You can edit the following tables:

  • Projects
  • Picklist Domains and Picklist Values
  • Notebooks
  • Fragments and Fragment Types
  • Sequences
  • Identifier Types
  • Sites
