Component Information

A component is a distinct pure substance within a Registry Record.

The Component Information section consists of these fields:

  • Structure: Structure information about the component.
  • Chemical Name: A scientific name that identifies the component.
  • CAS Number: An unique numerical identifier for chemical elements, compounds, polymers, biological sequences, mixtures, and alloys.
  • Synonym: A scientific name of the substance other than its chemical name.
  • Component Comments: Comments associated with each component.
  • Stereochemistry Comments: Any stereochemistry remark.
