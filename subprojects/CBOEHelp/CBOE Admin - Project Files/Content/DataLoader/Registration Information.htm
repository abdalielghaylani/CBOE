Registration Information

In the Registration Information section, you provide top-level information about each record of your source data file. For example, you can map the output field ‘Project’ with the ‘PROJECT’ field of your source data file.

The Registration Information section consists of these fields:

  • Projects: Select the input field that contains information about the projects associated with each record.
  • Alias: The alternate name of the project.
  • Prefix: The prefix format defined by the administrator to identify substances.
  • Registry Comments: Remarks about the Registry Record.
