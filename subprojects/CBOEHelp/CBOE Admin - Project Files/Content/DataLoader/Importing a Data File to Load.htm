Importing a Data File to Load

The file formats that you can import are:

  • .cfw (ChemFinder Form)
  • .cfx (Flow-Cal file)
  • .sdf (Structure data file)
  • .xls (Excel Workbook)
  • .txt (Text file)
  • .csv (Comma Separated Values file)
  • .mdb (Access database)
  • .mst (ChemFinder Chemical Structure Index)
