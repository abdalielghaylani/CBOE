XML Mapping Template for Data Loader

XML MAPPING TEMPLATE FOR IDENTIFIERS

<mapping>
<enabled>True</enabled>
<ObjectBindingPath>BINDING_PATH</ObjectBindingPath>
<memberInformation memberType="method">
<type>instance</type>
<name>AddIdentifier</name>
<description>new identifier</description>
<args>
<arg index="0" input="constant" type="string">
<value>IDENTIFIER_NAME</value></arg>
<arg index="1" input="derived" type="string">
<value>SD_FIELD</value></arg>
</args>
</memberInformation>
</mapping>

BINDING_PATH

  • For REGISTRY property : this
  • For COMPONENT Property : this.ComponentList[0].Compound
  • For BATCH Property : this.BatchList[0]
  • For BASE_FRAGMENT Property : this.ComponentList[0].Compound.BaseFragment.Structure

IDENTIFIER_NAME

  • The Name of the Identifier in the Registration Configuration

SD_FIELD

  • The Name of the Field in the input file
