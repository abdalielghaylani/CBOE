Exporting Data to an XML

Data Loader lets you export data to an XML file so that you get the output in the form of an XML file. You can view the output XML file in any XML viewer as well as in Internet Explorer and Firefox.

Output in the form of an XML file is useful for later input into other products, such as E-Notebook.

To export data to an XML file:

  1. Select Output to an XML file in the Task section after logging into Data Loader.

NOTE: You may also select the Not logging in option to enter into Data Loader without authentication.

  1. Click Next. This screen appears as:

  1. Click Browse to select the file to import. For information on the file formats that you can import, see “Importing a data file to load”.

NOTE: If the selected input file is a .cfw or .mdb file with multiple tables, or a .xls file with multiple sheets, then you must choose the table or sheet you want to import.

  1. Click Next. This screen appears as:

  1. Edit the labels as desired and click Next. This screen appears as:

  1. Map the output fields with the input fields and click Next. The screen for selecting output file is displayed.
  2. Click the Browse button to locate the folder where the output XML file will be saved. You also need to enter a name for the XML output file.
  3. Click Next. This screen appears as:

  1. Specify the record from where processing should start.
  2. Specify the number of records to process and choose whether the processing will be done now or later.
  3. Click Next. The Summary screen appears and displays the number of records exported to the selected XML file.
