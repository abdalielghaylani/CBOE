Saving a Job

To save a job:

  1. Log into Data Loader and select one of the following tasks:
  • Load substances into registration
  • Output to a text file
  • Output to an XML file
  1. Proceed through the subsequent screens until the Configure job screen appears. For more information, see “Loading substances”.
  2. Configure the job, if required, and click Save. The Save Settings window appears as:

  1. Enter a name for the job in the Save to text box.
  2. Click OK to save the job.
