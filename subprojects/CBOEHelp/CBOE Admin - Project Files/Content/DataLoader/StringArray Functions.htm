StringArray Functions

  • DateTime.GetDateTimeFormats(): Converts the specified DateTime value to all the string representations supported by the standard DateTime format specifiers.
  • DateTime.GetDateTimeFormats(Char): Converts the specified DateTime value to all the string representations supported by the specified standard DateTime format specifier.
  • String.Split(CharArray): Returns a string array that contains the substrings in the specified sting that are delimited by elements of a specified Unicode character array.
  • String.Split(CharArray, Integer): Returns a string array that contains the substrings in the specified string that are delimited by elements of a specified Unicode character array. The Integer type value supplied to this function specifies the maximum number of substrings to return.
