Missing Indexes

On some systems the following database indexes may be missing and should be created to improve the performance of bulk loading:

Connect regdb/oracle@&&YourTNSName
create index index_compoundmol_strucid on compound_molecule(structureid);
create index index_mixcomp_mixcompound on mixture_component(mixtrureid, compoundid);

Some Addins perform database queries based on custom properties. Ensure that any custom property that may be queried against by a custom Addin is also properly indexed.
