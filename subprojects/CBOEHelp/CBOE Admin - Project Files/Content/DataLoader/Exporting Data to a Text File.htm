Exporting Data to a Text File

At times you may need to import the same raw data into multiple ChemBioOffice Enterprise applications. In such cases, you can use Data Loader to export the raw data to a text file or an XML file, and use the file for importing data into multiple ChemBioOffice Enterprise applications. Thus, exporting data to text or XML file facilitates reusability of data.

To export data to a text file:

  1. Select Output to a text file in the Task section after logging into Data Loader.

NOTE: You may also select the Not logging in option to enter into Data Loader without authentication.

  1. Click Next. This screen appears as:

  1. Select Yes, if the file to import contains a header.
  2. Select the delimiter to be used in the output file and click Next. This screen appears as:

  1. Click Browse to select the file to import. For information on the file formats that you can import, see “Importing a data file to load”.

NOTE: If the selected input file is a .cfw or .mdb file with multiple tables, or a .xls file with multiple sheets, then you must choose the table or sheet you want to import.

  1. Click Next. This screen appears as:

  1. Map the labels with their types and click Next. This screen appears as:

  1. Map the output fields with the input fields and click Next. The screen for selecting an output file is displayed.
  2. Click Browse to select the output text file.
  3. Click Next. This screen appears as:

  1. Specify the record where processing should start.
  2. Specify the number of records to process and choose whether the processing will be done now or later.
  3. Click Next. The Summary screen appears and displays the number of records exported to the selected text file.
