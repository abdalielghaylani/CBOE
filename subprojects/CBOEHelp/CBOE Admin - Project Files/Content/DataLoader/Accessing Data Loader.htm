Accessing Data Loader

To log into Data Loader:

  1. Open the ChemBioOffice Enterprise (CBOE) Manager homepage and click the DataLoader Client icon in the Applications & Utilities section. If you are accessing Data Loader for the first time, a dialog box will prompt you to install Data Loader. Else, the login screen appears as:

Note: You can also open the Data Loader login screen through the Start menu. To open the login screen, go to Start>All Programs>ChemBioOffice Enterprise>Data Loader> Data Loader

  1. Enter your username and password.

Note: If you need to perform a task that does not require connection to an Oracle database, select Not logging in. You will not require any username or password in this case. For example, if you need to load a previously saved job then you can select the Not logging in option.

  1. Click Next. This screen appears as:

This screen lists the tasks you can perform using Data Loader.
