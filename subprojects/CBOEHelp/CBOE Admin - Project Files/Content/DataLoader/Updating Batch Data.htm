Updating Batch Data

To update batch data:

  1. Log into Data Loader with appropriate credentials to load data into Registration Enterprise. The task list appears.
  2. Select Update batch data in registration and click Next. This screen appears as:

NOTE: For more information on the Pending Review and Registered options, see “Loading substances”.

  1. Click Next. The screen for selecting input file is displayed.
  2. Select an input file. The input preview screen is displayed.
  3. Modify labels of the input fields, if allowed and necessary.
  4. Click Next. The screen for mapping fields is displayed.
  5. Map the output fields with the input fields and click Next.
  6. Select the number of records to process and click Next. Batch data is updated and Summary screen appears.
  7. Click Next. The Tasks screen appears.
