Application Tracing

The command line COEDataLoader tool provides timing information for several of the steps described above. It can be used to pinpoint specific trouble areas where optimizations may be required to improve load performance. The trace files will be written to a “Performance Trace” directory directly under the working directory of the COEDataLoader application.
