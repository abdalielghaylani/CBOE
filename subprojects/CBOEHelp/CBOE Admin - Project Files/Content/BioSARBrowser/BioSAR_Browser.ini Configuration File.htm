BioSAR_Browser.ini Configuration File

The Biosar_browser.ini file, found in the following location: <webroot>/ChemOffice/biosar_browser/config defines many parameters that allow the BioSAR Enterprise application to function properly, including the definitions of formgroups (which define all forms) in the application. Most of this file should not be edited since editing this file in the wrong place could cause major problems when running the application.

Note: It is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application. New versions of the ChemOffice Enterprise use default configurations.

The BioSAR_Browser.ini file has two sections:

  • GLOBALS
  • BASE_CONNECTION
