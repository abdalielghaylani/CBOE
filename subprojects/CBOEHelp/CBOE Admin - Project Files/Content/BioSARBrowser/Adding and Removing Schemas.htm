Adding and Removing Schemas

Only administrators can add or remove schemas. The administrators can set the visibility of any schema in the Oracle database so that it is visible to BioSAR Enterprise users. If a schema is not visible, BioSAR Enterprise users will not be able to see or use the tables or views within the schema.

To add or remove schemas from the list of available schemas:

  1. Click the Add/Remove Schemas button in the first page of the Schema Management interface.

  1. A list of all the schemas in the Oracle database appears:

  1. Select the check boxes in the Exposed column corresponding to the names of schemas that you would like to display to users. To remove a schema from the list of visible schemas, disable the check box in the Exposed column corresponding to that schema.
  2. Click the Save Changes button.
