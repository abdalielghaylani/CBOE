Picklist Domain Type

To create a Picklist Domain property at the Registry level:

  1. Log in to ChemBioOffice Enterprise with administrative privilege
  2. Go to Customize Registration>>Manage Data Properties>>Registry tab
  3. Enter the property's name and label.
  4. Select PICKLISTDOMAIN from the Type dropdown list and select the desired PickList from the PickList dropdown list

  1. Click on Add Property button. The new property is added under Property List section.
  2. Click on Save button than select the newly created property and click on Validation Rules button

  1. Under New Validation Rule section, select whether to make the newly created property a required or custom field
  2. Enter in the Error text box, the message that you want to be displayed when the validation rule is not respected
  3. If the property is required, select a value to be displayed by default in the property field for the existing records. Note that when a property is required and there is no default value selected, this will create a data exception when the user edits old records because it is not possible to have Null value in a required field.

Note: the default value will not appear in the submit form when the user is submitting a new record. This is only valid for existing records.

  1. Click on Add Rule and then click on Save button
