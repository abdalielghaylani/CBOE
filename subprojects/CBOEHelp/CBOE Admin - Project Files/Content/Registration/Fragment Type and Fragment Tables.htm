Entering a New Fragment Type

  1. In the table editor page, select the "FRAGMENT TYPES" table and then click on Add button

  1. Enter the fragment description and then click on Add button

  1. The new fragment type is created.

Adding a New Fragment

  1. In the table editor page, select the "FRAGMENTS" table and then click on Add button

  1. Draw the structure, select the Fragment Type, enter a code and a description, enter MW and Formula or just click on "Get Molweight" and "Get Formula" to get these values populated.

  1. Click on Add button, the new fragment is created and added to the Fragments table.
