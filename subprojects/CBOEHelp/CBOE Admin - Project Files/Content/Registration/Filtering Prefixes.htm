Filtering Prefixes

It is possible to filter prefixes based on their assigned sites. For more clarification, please refer to the following example:

  1. Let us suppose there are two submitters, submitter A and submitter B and two sites: site A and site B
  2. Assign site A to submitter A and site B to submitter B
  3. Log in as an administrator user and create a new sequence with XY prefix
  4. Assign the site B to the sequence created in step 3
  5. Log in to ChemBioOffice Enterprise with submitter A who has site A assigned
  6. Submit a new component and notice that in the submit component window and specifically in the Prefix field, you won't be able to see the XY option in the prefix picklist
