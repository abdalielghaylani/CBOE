Customizing the Search Forms

You can customize the details that you want to be shown in the search results form for temporary and registered records.

To customize the search results forms:

  1. In the Registration Admin window, click on Edit Form XML link.

  2. The following window appears, select Search Temporary or Search Registry tabs

  1. Click on Edit XML, look for the property that you want to hide in the search results form and change the hidden value is equal to True for the corresponding column name to hide it in Search results pages.
  2. Click Save XML.
