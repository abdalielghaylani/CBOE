Create Properties

You can create new properties at five different levels:

  • Registry Record
  • Component
  • Base Fragment
  • Batch
  • Batch Component

In order to define a New Property we need to assign next values:

  • Name: This is the real Oracle Field Name (Spaces and other special character are not allowed)
  • Label: Display Name that will appear in all Registration Forms
  • Type: Property Type
  • TEXT: —>VARCHAR2
  • DATE —> DATE
  • BOOLEAN —> NCHAR(1)
  • NUMBER —> NUMBER(X,Y)
  • PICKLISTDOMAIN —>
  • INTEGER —> NUMBER(9)
  • FLOAT —> NUMBER(14,6)
  • URL
  • Length: This is only active for Text type
  • Precision: This is only active for Number Types (Number: X,Y)
  • PickList: This is only active for PickListDomain type
