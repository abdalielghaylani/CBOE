Filtering PickLists

Administrator user is allowed to create and configure properties to display only values owned by the logged in user.

The example below shows how to use this feature.

  1. Log in to ChemBioOffice Enterprise with administrator privileges
  2. Go to Registration>>Customize Registration>>Manage System Behaviors>>Manage Data Properties
  3. Create a property at the batch level:
  • Name: Notebook A
  • Label: Notebook A
  • Type: PICKLISTDOMAIN
  • PickList: Notebooks
  1. Create a second property at the batch level:
  • Name: Notebook Filtered
  • Label: Notebook Filtered
  • Type: PICKLISTDOMAIN
  • PickList: NotebooksFilteredByUser

In the Submit Component window; the filtered property, "Notebook Filtered", shows only the notebooks owned by the logged in user. The unfiltered property, "Notebook A", shows all of the notebooks available.
