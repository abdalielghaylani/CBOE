Import/Export Configuration

The Import/Export configuration allows administrator to load an initial configuration as well as export the current configuration to another system.

Import Configuration

To Import/Export the CBOE configuration perform the steps below:

  1. In the Registration Admin window, click on "Import / Export Configuration" link, the following dialog displays:

  1. Browse to the configuration file to be imported and then click on Import button.

Force Import checkbox, if checked will replace any existing configuration.

Export Configuration

To Export the Registration Configuration to another system:

  1. Enter the destination server path where the files will be exported
  2. Check the exportation options and then click on Export button.
