Registration Privileges

The table below displays the list of Registration Privileges:

Oracle Name Privileges
Search_Temp This privilege is defined as a user having the ability to query over the temporary records.
Search_Reg This privilege is defined as a user having the ability to query over the permanent registry record.
Add_Compound_Temp This privilege is defined as a user having the ability to query over the permanent registry record.
Edit_Compound_Temp This privilege is defined as a user having the ability to submit a compound and/or mixture
Delete_Temp This privilege is defined as a user having the ability to delete compounds and/or mixtures from the temporary table.
Register_Temp This privilege is defined as a user having the ability to register compounds and/or mixtures from the temporary table
Edit_Compound_Reg This privilege is defined as a user having the ability to edit a registered record
Delete_Reg This privilege is defined as a user having the ability to delete a registered record
Delete_Batch_Reg This privilege is defined as a user having the ability to delete a batch in the registered compound.
Edit_Scope_Supervisor This privilege is defined as a user having the ability to edit records belonging to users that are supervisees of the user with this privilege
Edit_Scope_All This privilege is defined as a user having the ability to edit any other user’s records, regardless of who is the supervisor
Set_Approved_Flag This privilege is defined as a user having the ability to mark a record as approved
Toggle_Approved_Flag This privilege is defined as a user having the ability to change the record once the record has been approved

Edit_Salt_Table

 This privilege is defined as a user having the ability to edit the fragments table within the table editor
Add_Salt_Table This privilege is defined as a user having the ability to add to the fragments table within the table editor
Delete_Salt_Table This privilege is defined as a user having the ability to delete from the fragments table within the table editor
Edit_Sequences_Table This privilege is defined as a user having the ability to edit the sequences table within the table editor
Add_Sequences_Table This privilege is defined as a user having the ability to make additions to the sequences table within the table editor
Delete_Sequences_Table This privilege is defined as a user having the ability to delete from the sequences table within the table editor
Edit_Notebooks_Table This privilege is defined as a user having the ability to edit the notebooks table within the table editor
Add_Notebooks_Table This privilege is defined as a user having the ability to add to the notebooks table within the table editor
Delete_Notebooks_Table This privilege is defined as a user having the ability delete from the notebooks table within the table editor
Edit_Projects_Table This privilege is defined as a user having the ability to edit the projects table within the table editor
Add_Projects_Table This privilege is defined as a user having the ability to make additions to the projects table within the table editor
Delete_Projects_Table This privilege is defined as a user having the ability to delete from the projects table within the table editor
Edit_Solvates_Table This privilege is defined as a user having the ability to edit the *fragments* table within the table editor
Add_Solvates_Table This privilege is defined as a user having the ability make additions to the *fragments* table within the table editor
Delete_Solvates_Table This privilege is defined as a user having the ability to delete from the *fragments* table within the table editor
Edit_Utilizations_Table This privilege is defined as a user having the ability to make changes to the utilizations table
Add_Utilizations_Table This privilege is defined as a user having the ability to add to the utilizations table
Delete_Utilizations_Table This privilege is defined as a user having the ability to delete from the utilizations table
Edit_Batch_Projects_Table This privilege is defined as a user having the ability to make changes to the batch projects table
Add_Batch_Projects_Table This privilege is defined as a user having the ability to add to the batch projects table
Delete_Batch_Projects_Table This privilege is defined as a user having the ability to delete from the batch projects table
Edit_Sites_Table This privilege is defined as a user having the ability to make changes to the sites table
Add_Sites_Table This privilege is defined as a user having the ability to make additions to the sites table
Delete_Sites_Table This privilege is defined as a user having the ability to delete in the sites table
Manage_People_Project This privilege is defined as a user having the ability to manage projects with row level security
Manage_System_Duplicates This privilege is defined as a user having the ability to duplicates across the application when row level security has been enabled
Register_Direct This privilege is defined as a user having the ability to directly register a record upon initial submission without the record needing to step through the temporary workflow
Config_Reg This privilege is defined as a user having the ability to manage page control settings and Registration Admin Tools
Add_Component This privilege is defined as a user having the ability to add individual components to a record
Add_Picklist_Table This privilege is defined as a user having the ability to add to the picklist values and picklist domain tables in table editor
Edit_Picklist_Table This privilege is defined as a user having the ability to edit the picklist values and picklist domain tables in table editor
Delete_Picklist_Table This privilege is defined as a user having the ability to delete the picklist values and picklist domain tables in table editor
Add_Identifier_Type_Table This privilege is defined as a user having the ability to add identifier types to the identifiers table in table editor
Edit_Identifier_Type_Table This privilege is defined as a user having the ability to edit identifier types in the identifiers table in table editor
Delete_Identifier_Type_Table This privilege is defined as a user having the ability to delete identifier types in the identifiers table in table editor
Add_Batch_Perm This privilege is defined as a user having the ability to add batches to registry records
Load_Save_Record This privilege is defined as the user having the ability to save templates and to then load those templates
Set_Locked_Flag This privilege is defined as the user having the ability to lock a particular registry record
Toggle_Locked_Flag This privilege is defined as a user having the ability to unlock the already locked record
