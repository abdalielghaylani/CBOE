Searching for Documents

DocManager Enterprise supports structure searching, basic and advanced searching for text characters and operators. To perform a simple text search, please refer to “Performing a search”. When performing a text search, you specify the search criteria by filling in values for the various fields in the query form. When performing a structure search, you draw a structure of the compound you are looking for. For more information about drawing structures, see the ChemBioDraw User guide. After entering search criteria, click on Search button to display the search results.
