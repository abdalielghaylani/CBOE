Managing Users

This topic provides information on how to create/Edit/Delete CBOE users.

Creating CBOE users

  1. Log in to CBOE Manager using an administrative account. Under Security section click on “Manage Users” button as shown in the screenshot below:

  1. The following dialog displays. Click on Add User button as shown in the screenshot below:

  1. The following dialog displays. Enter the User information. There are four mandatory fields to be filled in: User Name , Password, Password Confirmation and the Last Name marked in red as shown in the screenshot below:

  1. You may specify the user role(s) by selecting the desired option(s) from the “Available Roles” table.

Editing Users

To Edit a User, perform the following steps:

  1. In the list of available users, highlight the user that you want to edit. The Edit button becomes enabled, click on it.

  1. The User information dialog displays. Make your changes and then click on save button.

Deleting Users

To delete a user:

  1. Highlight the user in the list of the available users.
  2. Click on Delete button as shown in the screenshot below
