Manual Registration

You can register samples manually.

To register a sample manually:

  1. Draw a reaction in the Reaction field and create samples. A Samples section is automatically created.
  2. Assign a product to each created sample and navigate to the Samples section.
  3. From the Home ribbon, click Registration and select Manual Registration.

A Temp ID is automatically populated in the sample's Properties table, as shown below.

The Registration Enterprise window displays.

Note: The values set in the Registration User Preferences form are automatically populated in their corresponding fields in the Registration form in CBOE Manager, unless you enter them manually. Manually-entered values override those set in the User Preferences dialog. If the mandatory fields are left blank in the User Preferences dialog, you have to enter them manually in the Registration form .

  1. Click Submit to submit the sample to the Temporary table

  1. Click Register to push the sample to the permanent table. The following window displays showing the registry number.

Note: The Register button is enabled only if you have the appropriate privileges assigned.

  1. Click Done to end the process.
