Managing Grouping Fields

Grouping is a feature in Inventory Enterprise that lets you relate containers based on arbitrary fields, including custom fields. The fields, based on which containers are related, are called grouping fields. A set of related containers is called a group. All containers in a particular group have the same group ID.

Grouping is controlled via a GUI interface, which facilitates searching, requesting, and reserving of samples in batches. After you have classified the containers into groups, you can search for containers found in a particular batch. For example, you can create groups based on the status of the containers so that the containers with status, 'available' will be in one group while the containers with status, 'backordered item' will be put in another group. Therefore, you can search for the available containers based on their 'available' status.

To manage group fields, you specify the field(s) you want containers grouped by, and you subsequently update the container details.

Note: You can create groups of containers to which no structure is assigned.

Managing Grouping Fields

To manage grouping fields:

  1. Within the home page of ChemBioOffice Enterprise, Inventory Enterprise section, click Tasks. The Administrative Menu window appears.

Note: You can access the Administrative Menu window by clicking Tasks in the menu bar of the Inventory Enterprise home page.

  1. Click Manage Grouping Fields in the Inventory Management section. The Manage Grouping Fields window appears.

  1. Select the values you want from the drop-down list.

Note: You can select up to three columns as the grouping fields.

  1. Display Name: Type the display name of the selected grouping fields in the text field.
  2. Click OK. A window stating the batching fields are updated displays.
  3. Click OK to display the Administrative Menu window.
  4. Click Close to close the Administrative Menu window.
  5. Click Browse within the Inventory Enterprise section in the home page of ChemBioOffice Enterprise. The Inventory Enterprise home page appears.
  6. Create at least three new containers.
  7. From the Container List, View menu, select Multi-Select.

  1. Select the last two containers you created.
  2. From the Container Details, Update menu, select Update Containers. The Update Inventory Container window appears.
  3. Container Fields: Select Container Status from the drop-down list, as shown in the example below.

  1. Click OK. The Update Inventory Container window appears.

  1. Container Status: Select Backordered Item from the drop-down list.
  2. Click OK to close the Update Inventory Container window.
  3. If the Status you want is not available, select Column Chooser from the View menu of the Container List menu bar. The Choose Columns for Container list window appears.

  1. Select the status you want in the Available column and click the right arrow to move the status to the Selected column.
  2. Click Save to save your selections and close the window. The list of containers display in the Inventory Enterprise home page.
  3. Click on each container, one at a time. The first two containers with a status of 'Backordered Item' are grouped into one batch (with the same batch ID) and the remaining containers are grouped into a different batch with a status of 'Available'.
