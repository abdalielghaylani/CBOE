Creating a New Container

You can create a container in any location, such as, a freezer room, stock room, etc. To confirm that you have the appropriate privileges to create a new container, see Roles and Privileges.

To create a container:

  1. Click Browse within the Inventory Enterprise section (found in the home page of ChemBioOffice Enterprise).
  2. From the Location Tree frame, navigate to the location under which you want to create the new container.
  3. Click New Container in the Current Location frame.

  1. The Create or Edit an Inventory Container window appears.

The Create or Edit an Inventory Container window contains the following tabs:

  • Required
  • Substance
  • Supplier
  • Contents
  • Optional
  • Owner
  • Comments

Note: The Required tab appears in the example image above.. The fields marked in Red are required fields and must be completed.

  1. Enter appropriate information about this new container for each tab and click OK.

Required Tab

The Required tab lets you set the required fields of a container. Required fields must be completed. The required fields for a container are:

  • Location ID: Specifies the ID of the location for this new container.
  • Container Name: Specifies the name of this new container.
  • Container Type: Specifies the type of the container, such as, a bottle, tube, bag, etc.
  • Unit of Measure: Specifies unit of measurement for the container, such as, milligram or milliliter.
  • Container Size: Specifies the size of the container.
  • Initial Amount: Specifies the amount of the chemical substance initially available in the container.
  • Number of Copies: Specifies the number of copies created for this new container.
  • Barcode Description: Specifies the barcode type, such as, location barcode, containers barcode, or plate barcode.

For more information about other fields contained in the Required tab, see Inventory Enterprise Form Fields.

Substance Tab

The Substance tab lets you specify the substance to be stored in the new container.

To add a substance to the container using the Substance tab, you need to perform one of the two tasks:

  • Create a new substance, or
  • Select an existing substance

Creating a New Substance

To create a new substance:

  1. Click New Substance in the Substance tab window. The Create or Edit a substance in Inventory Enterprise window displays.

  1. Type the name of the substance in the Substance Name text field.
  2. Draw the chemical structure of the substance in the drawing area, using the ChemDraw plug-in.

Note: For information about using the ChemDraw plug-in, please refer to the ChemDraw User's Guide.

  1. Type the CAS registry number in the CAS Registry # field.
  2. Type the ACX number in the ACX Number field.
  3. Click OK.

Note: If you created a new substance that already exists in Inventory Enterprise, the conflict resolution screen displays when you click OK.

  1. Click Select next to the newly-created substance to add the substance to the container.

Selecting an Existing Substance

When you select an existing substance, you select it from the Inventory, Registration, or ChemACX database.

To select an existing substance:

  1. Click Select Substance in the Substance tab window. The Search for a substance to associate to a container window appears.

  1. Select the databases to search: You can search the Inventory Enterprise, Registration Enterprise, or ChemACX database. To search all three databases, check all three checkboxes.

Note: The Registration and ChemACX databases are enabled based on the privileges you have:

  • You must have the "SEARCH_REG" privilege to search compounds from the Registration database, or to select a compound from Registration Enterprise while creating a new container. Otherwise, the Registration Enterprise checkbox in the search form is disabled.

  • You must have the "BROWSE_ACX" privilege to search compounds from ChemACX database, or to select a compound from ChemACX when creating a new container. Otherwise, the ChemACX checkbox in the search form is disabled.
  1. Enter the required information and click Search. A list of substances matching the search criteria specified in the search form appears.
  2. Click the Details tab corresponding with the substance you are adding to the container.
  3. Click OK to add the selected substance to the container.

Note: The New Substance and Select Substance links are also available from the Required tab. Therefore, you can add a substance to a container from the Required tab, as well.

Supplier Tab

The Supplier tab lets you provide information about the supplier of the container, and its contents. For information about the fields found in the Supplier tab, see Inventory Enterprise Form Fields. An example of the Supplier tab is shown below.

Contents Tab

The Contents tab window lets you provide information about the contents of the container. It is important to keep track of the information regarding the contents of a container since different batches of a substance can have slightly different properties. An example of the Contents tab is shown below.

For information about the fields contained in the Contents tab, see Inventory Enterprise Form Fields.

Optional Tab

The Optional tab lets you provide other information not found in the other tabs, but is important to include.

For information about the fields found in the Optional tab, see Inventory Enterprise Form Fields.

Owner Tab

The Owner tab lets you specify the container's owner.

Comments Tab

The Comments tab lets you provide additional information about the container, such as, handling procedures and storage conditions.

For information about the fields found in the Comments tab, see Inventory Enterprise Form Fields.
