Deleting Report Layout

To delete a report layout:

  1. Click the Tasks link within the Inventory Enterprise section in the ChemBioOffice Enterprise home page to display the Administrative Menu window.
  2. Click the Manage Reports link in the Inventory Management section to display the Manage Reports window.
  3. Select the appropriate report type in the Select a Report Type list. The report layouts corresponding to the selected report type are displayed in the Report Layouts list.
  4. Select the report layout that is to be deleted in the Report Layouts list and click the Delete link.
  5. Click on the Go button to delete the report.
  6. Click on OK button when the success message is displayed.
