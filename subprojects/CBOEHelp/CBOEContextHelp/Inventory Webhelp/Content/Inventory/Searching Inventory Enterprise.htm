Searching Inventory Enterprise

With the Search feature, you can find substances, containers, or plates stored in the Inventory Enterprise database.

The type of search you can perform in Inventory Enterprise depends on your role and your set of requirements. For example, chemists can search Inventory Enterprise for chemical structures, substance names, and CAS numbers. They perform the searches using a Substructure search. Similarly, stock room, or receiving room, personnel can use Advanced search to search Inventory Enterprise for containers and location barcodes.

Refer to the following topics for more information:

  • Simple Search
  • Advanced Search
  • Substructure Search
  • Batch Search
  • Global Search
  • Plate Search
