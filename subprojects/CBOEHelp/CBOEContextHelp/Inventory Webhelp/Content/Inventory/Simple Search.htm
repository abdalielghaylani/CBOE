Simple Search

With Simple search, you can search for a container using key fields for this container, such as CAS Registry, Container ID, Location Barcode, Catalog Number, PO Number, PO Line number, Substance Name, Substance Synonym, Container Name, Container Comments, and Location ID.

To perform a simple search on Inventory Enterprise:

  1. From the Inventory Enterprise section of the ChemBioOffice Enterprise home page, click Search. The following window appears.

Note: If you performed an earlier search, the corresponding search tab is selected, by default. In this case, click the Simple Search tab to display the preceding window. You can also perform searches by clicking Search within the Current Location frame in the Container Management area.

Note:The number of tabs visible in the preceding window depends on the configuration settings.

  1. Enter your search criteria in the corresponding text fields.
  2. Check Search Sublocations if you want to perform a search on the location specified in the Location ID, field as well as on child locations of the specified location.
  3. Check Exclude Special Locations if you want to exclude the special locations, such as, Trash Can, Disposed, and On Order, from the search. You can select the locations you want to exclude by clicking the Special link.
  4. Click Search. A list of containers matching the search criteria specified in the Simple Search tab appears.

Viewing Simple Search Results

The results of the Simple search match the search criteria specified in the Simple Search tab. Click a container in the results list for more information about that container.

When you click a container in the search result list, the Container Management area displays the details of the container, for example:

The frames in Container Management with more detail about your containers are:

  • Location Tree(Left frame): Provides information about the location of the container.
  • Container List (Upper right frame): Provides information about the key attributes of the container, such as, container name and container ID.
  • Container Details (Lower right frame): Provides detailed information about the attributes of the container.
