Logging Into CBOE Manager

To launch ChemBioOffice Enterprise Manager:

  1. Type the following URL in the web browser: http://servername/COEManager. The servername is the name of the server where CBOE is installed. The following page displays:
  2. In the CBOE Manager Login dialog (in the upper-right corner of the display), enter your login credentials: User Name and Password.
  3. Click Log in.

Note: Please note that if you are logged in to CBOE manager using Internet Explorer browser then the supported drawing tool is ChemDraw ActiveX (CDAX), but if you are logged in to CBOE manager using Google Chrome browser then the supported drawing tool is ChemDraw JavaScript (CDJS).

The ChemBioOffice Enterprise Manager home page displays:

The CBOE Manager home page displays the main ChemBioOffice Enterprise applications and features in several categories. To launch an application/feature, click on its corresponding icon.

Note: Not all users will have the same applications/features displayed on the CBOE Enterprise Manager main page. Your display is specific to the CBOE applications you have access to, which depends on the roles assigned to you.

To close ChemBioOffice Enterprise Manager:

To log off from ChemBioOffice Enterprise Manager, it is recommended that you click Log Off button in the application home page (in the upper-right corner of the display).
