DateTime Functions

  • DateTime.AddDays(Decimal): Adds the specified number of days to the input field of DateTime type. The Decimal field supplied to this function specifies the number of days to be added.
  • DateTime.AddHours(Decimal): Adds the specified number of hours to the input field of DateTime type. The Decimal field supplied to this function specifies the number of hours to be added.
  • DateTime.AddMilliseconds(Decimal): Adds the specified number of milliseconds to the input field of DateTime type. The Decimal field supplied to this function specifies the number of milliseconds to be added.
  • DateTime.AddMinutes(Decimal): Adds the specified number of minutes to the input field of DateTime type. The Decimal field supplied to this function specifies the number of minutes to be added.
  • DateTime.AddMonths(Integer): Adds the specified number of months to the input field of DateTime type. The Integer field supplied to this function specifies the number of months to be added.
  • DateTime.AddSeconds(Decimal): Adds the specified number of seconds to the input field of DateTime type. The Decimal field supplied to this function specifies the number of seconds to be added
  • DateTime.AddYears(Integer): Adds the specified number of years to the input field of DateTime type. The Integer field supplied to this function specifies the number of years to be added.
  • DateTime.get_Date(): Returns the date of the DateTime type input field supplied to this function.
  • DateTime.ToLocalTime(): Converts the value of the DateTime type input field to local time.
  • DateTime.ToUniversalTime(): Converts the value of the DateTime type input field to universal time.
  • FromOADate(Decimal): Returns the Date- Time equivalent of the specified OLE Automation date.

NOTE: OLE Automation is an inter-process communication mechanism based on Component Object Model (COM). OLE Automation was intended for use by scripting languages, but is now used by languages that run on Windows. OLE Automation provides an infrastructure whereby applications called Automation Controllers can access and manipulate shared automation objects exported by other applications.

  • get_Now(): Gets the current date and time as per local time.
  • get_Today(): Gets the current local date as per local time.
  • get_UtcNow(): Returns the current date and time on this computer in terms of Coordinated Universal Time (UTC).
  • Iif(Boolean,DateTime,DateTime): Evaluates the Boolean expression; if result is True, the function returns the first Date- Time type parameter else it returns the second DateTime type parameter.
  • Parse(String): Returns the DateTime equivalent of the specified string representation of a logical value.
  • ToDateTime(Boolean): Returns the Date- Time equivalent of a Boolean type input field.
  • ToDateTime(Char): Returns the DateTime equivalent of a Char type input field.
  • ToDateTime(DateTime): Returns the Date- Time equivalent of a DateTime type input field. No actual conversion takes place.
  • ToDateTime(Decimal): Returns the Date- Time equivalent of a Decimal type input field.
  • ToDateTime(Integer): Returns the Date- Time equivalent of a Integer type input field.
  • ToDateTime(String): Returns the Date- Time equivalent of a String type input field.
