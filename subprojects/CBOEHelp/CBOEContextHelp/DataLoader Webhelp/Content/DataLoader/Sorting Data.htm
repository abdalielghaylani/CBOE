Sorting Data

To sort data:

  1. Import a data file. The preview of the selected file appears and shows input fields, data types, and values. For more information, see “Loading substances”.
  2. Click Sort. The InputFileSort window appears as:

  1. Select a field in the Fields column and click Add to move it to the Sort Fields column. Move other fields to the Sort Fields column, as desired.
  2. Select items in the Sort Fields column and click Move up or Move down to sort the list as desired. To sort the fields in the Sort Fields column in ascending or descending order, click the Sort Ascending or Sort Descending button.
  3. Click OK. The InputFileSort window closes and the input fields are sorted.
