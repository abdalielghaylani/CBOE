Output Fields

When you import data into the Registration Enterprise database using Data Loader, Registry Records are added to the temporary or primary tables of Registration Enterprise.

NOTE: A Registry Record represents a substance and all its associated registry data.

Therefore, to import data into Registration Enterprise, you need to provide all the information required for creating a Registry Record. This information includes:

  • Registration information
  • Component information
  • Fragment information
  • Batch information

You can provide the above mentioned information by correctly mapping the input fields with the output fields of the sections displayed in the Mapping page.
