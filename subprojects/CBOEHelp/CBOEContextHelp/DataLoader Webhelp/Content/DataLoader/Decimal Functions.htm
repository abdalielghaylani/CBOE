Decimal Functions

  • Abs(Decimal): Returns the absolute value of a Decimal type input value.
  • Acos(Decimal): Returns the angle whose cosine is the specified Decimal type input.
  • Asin(Decimal): Returns the angle whose sine is the specified Decimal type input.
  • Atan(Decimal): Returns the angle whose tangent is the specified Decimal type input.
  • Atan2(Decimal,Decimal): Returns the angle whose tangent is the quotient of two Decimal type input values supplied to this function.
  • Ceiling(Decimal): Returns the smallest whole number greater than or equal to the specified Decimal value.
  • Cos(Decimal): Returns the cosine of the specified Decimal value.
  • Cosh(Decimal): Returns the hyperbolic cosine of the specified Decimal value
  • DateTime.ToOADate(): Converts the specified DateTime value to the equivalent OLE Automation date.
  • Exp(Decimal): Returns the number that is obtained when ‘e’ is raised to the power specified by the Decimal type input supplied to this function.

NOTE: ‘e’ is a number whose value is approximately 2.718281828.

  • Floor(Decimal): Rounds the specified Decimal type input value to the closest integer toward negative infinity.
  • GetNumericValue(Char): Converts a specified numeric Unicode character to a decimal number.
  • GetNumericValue(String,Integer): Converts the numeric Unicode character at the specified position in a specified string to a decimal number.
  • IEEERemainder(Decimal,Decimal): Returns the remainder resulting from the division of a specified Decimal type input value by another specified Decimal type input value.
  • Iif(Boolean,Decimal,Decimal): Evaluates the Boolean expression; if result is True, the function returns the first Decimal type parameter else it returns the second Decimal type parameter
  • Log(Decimal): Returns the logarithm of a specified Decimal type input value.
  • Log(Decimal,Decimal): Returns the logarithm of a specified number based on the specified base value.
  • Log10(Decimal): Returns the base 10 logarithm of a specified Decimal input.
  • Max(Decimal,Decimal): Accepts two decimal numbers as input and returns the larger of the two numbers.
  • Min(Decimal,Decimal): Accepts two decimal numbers as input and returns the smaller of the two numbers.
  • Parse(String): Returns the Decimal equivalent of the specified string representation of a logical value.
  • Pow(Decimal,Decimal): Returns the result of raising a specified decimal number to the specified power.
  • Round(Decimal): Rounds a decimal value to the nearest integral value.
  • Round(Decimal,Integer): Rounds a decimal value to a specified number of fractional digits.
  • Sin(Decimal): Returns the sine value of the specified Decimal type input value.
  • Sinh(Decimal): Returns the hyperbolic sine value of the specified Decimal type input value.
  • Sqrt(Decimal): Returns the square root of the specified Decimal type input value.
  • Tan(Decimal): Returns the tangent value of the specified Decimal type input value.
  • Tanh(Decimal): Returns the hyperbolic tangent value of the specified Decimal type input value.
  • ToDouble(Boolean): Converts the specified Boolean type input value to the equivalent double-precision floating-point number.
  • ToDouble(Char): Converts the specified Char type input value to the equivalent double- precision floating-point number.
  • ToDouble(DateTime): Converts the specified DateTime type input value to the equivalent double-precision floating-point number.
  • ToDouble(Decimal): Converts the specified Decimal type input value to the equivalent double-precision floating-point number.
  • ToDouble(Integer): Converts the specified Integer type input value to the equivalent double-precision floating-point number.
  • ToDouble(String): Converts the specified String type input value to the equivalent double-precision floating-point number.
  • Truncate(Decimal): Calculates the integral part of a Decimal type input value supplied to this function.
