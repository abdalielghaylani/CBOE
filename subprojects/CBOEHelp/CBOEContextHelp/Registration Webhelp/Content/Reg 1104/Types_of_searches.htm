Types of Searches

 
User may search for records based on any fields in the Search Temporary Records page by specifying the required search criteria. In Registration Enterprise, the different types of searches a user can perform include:
  • Structure Search
  • Text Search
  • Numeric Search
  • Formula Search
  • Date Search
  • Boolean Search
