SDF Search

Hitlists export into SDF is supported in Registration as covered in Export Hits. Registration allows users to browse to the exported files and search for their content.

To search for the content of an exported SDF:

  1. Log in to ChemOfficeEnterprise
  2. Under Registration , click on "Search Registry" icon, the "Search Permanent Registry" window displays
  3. Click on the Browse button corresponding to the SDF Search text box and browse to the SDF file that you want to search

  1. The number of structures read from the file displays

  1. Click on Search button to get the matching records displayed.
