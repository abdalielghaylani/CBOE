Print Barcode

The Print Barcode Method gathers information from the Inventory form and creates a barcode label.

Note: To print a barcode label, the Container Barcode parameter should not be null.

To print a barcode label:

  1. In the Reaction section>>Samples table, select the sample to which you want to print the barcode.
  2. Click the Inventory form tool in the Reaction section>>Home ribbon. A menu appears. Select Print Barcode.

An Internet browser appears.

  1. Label Type: Select a label template from the drop down menu.

The label displays in the window, as shown in the example below.

  1. Click Print in the lower-right corner of the window. The Print dialog displays.
  2. Select the printer and click Print.

  1. Save the report to a location on your local machine.

Based on your printer choice, the barcode is printed. In the example below, the "Adobe PDF" printer is selected.
