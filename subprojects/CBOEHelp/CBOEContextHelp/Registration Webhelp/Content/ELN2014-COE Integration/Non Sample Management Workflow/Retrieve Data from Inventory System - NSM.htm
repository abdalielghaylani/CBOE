Retrieve Data from Inventory System - NSM

You can retrieve data from the Inventory system to use the structures in the Reaction W COE Integration section.

To retrieve data from Inventory:

  1. Log in to E-Notebook.
  2. Create an experiment and add Reaction W COE Integration section.
  3. In the Reaction toolbar, click Add. The Select & Add Structure dialog appears:

  1. Search In: Select Inventory Search, from the drop down list.

The search form consists of a ChemDraw control to render structures for exact and substructure searching.

  1. A set of text fields is available for entering textual search data:
  • Substance Name
  • CAS Registry Number
  • ACX ID
  • Container Barcode
  • Molecular Formula
  • MolWeight Range
  1. Enter search criteria and click Search to execute the search.

  1. Click Clear to clear the form.
