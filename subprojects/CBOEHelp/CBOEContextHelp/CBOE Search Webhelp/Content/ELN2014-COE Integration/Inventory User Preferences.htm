Inventory User Preferences

The Inventory User Preferences feature lets you set default values, which are used automatically when using the inventory single click as well as for manual container creation.

To display the inventory user preferences:

  1. Navigate to E-Notebook application menu and select Options from the menu.

The Options dialog displays.

  1. Click Inventory in the left frame. The Inventory form displays in the right frame.

  1. Fill in the default values that you want fetched while pushing samples to Inventory:
    • Location ID: Browse to the location where you want to create the container.
    • Container Type: Select the container type from the dropdown list.
    • Unit of Measure: Select the unit of measure from the dropdown list.
    • Container Size: Enter the container size.
    • Container Status: Select the container status from the dropdown list.
    • Supplier Name: Select the supplier name from the dropdown list.
    • Barcode Description: Select the barcode description from the dropdown list.
    • Current User: The logged-in user is automatically populated. You can select a new user from the dropdown list.
  2. Click OK. (Clear All clears all values entered in the form.)

A confirmation message displays.

  1. Click Close to close the Options dialog.
