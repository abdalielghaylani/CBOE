Adding a New User Account

You can add a new user account one of two ways. You can either type in the user’s information or retrieve the user’s information from an LDAP or Oracle server. After you retrieve the information, you can edit or add to it as needed.

To add a new user to an application:

  1. In the Security section in the ChemBioOffice Enterprise Manager home page, click Manage Users. The Security Manager window opens.
  2. In the Security Manager window, select the application in the View Users From drop down list.

Tip: Alternatively, you can select All CS Applications. This option lets you assign any role associated with any application.

  1. Click Add User. This window appears:

Note: The window will be titled either ‘Add LDAP User’ or ‘Add Oracle User’ depending on the type of your group’s user database.

  1. Do one of the following:
  • If you are not retrieving the user information from a server, click Create an Exempt and type in the user’s information as you want it to appear on screen.
  • To retrieve the user information from a server, type in the User Name in the format in which it is stored, such as ‘JSmith’ or ‘JoeS’. Then click Get from LDAP (or Oracle) User. The form is populated with the retrieved information.
  1. Continue to complete the form as described below. Fields that appear in red on your screen are required.

User code

Enter the user code.

Password

Enter a password for the user in the Password and Confirm Password fields.

Note: If you retrieve the user’s information from an LDAP or Oracle server, the stored password will be used and these fields will be grayed out.

First, Middle, and Last Names

Enter the name of the user in the fields as indicated.

Telephone, Email, and Address

Enter the user’s contact information as desired.

Supervisor

Enter the role of the user’s supervisor.

Site

Enter the site for the user

Assign Roles

Assign one or more roles to the user. The role(s) in the Current Roles window are currently assigned to the user. Roles in the Available Roles window are unassigned. To assign/unassign roles for the user, select role(s) in either window and click the arrow buttons (<< and >>) to move the roles to the desired window. To select multiple roles, CTRL+click each role.

Active

Select this checkbox to activate the account. Deselect it to make the account inactive when creating or modifying the account.

  1. Click Save.

Note: Only those roles that are available for the application you selected are listed.
