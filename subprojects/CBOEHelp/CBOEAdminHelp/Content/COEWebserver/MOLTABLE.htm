[MOLTABLE]

Description: As with other items in the globals section the TABLE_GROUP key specifies individual table groups else where in the ini file. Table Groups specify information about what table is the base table for searching, and what table holds the structures.   

The sample.ini definition is as follows:

[MOLTABLE]
TABLE_NAME=MolTable
RELATIONAL_FIELDS=ID;1,MOL_ID;1,Molname;0
PRIMARY_KEY=ID
SQL_SYNTAX=ACCESS
SELECT_KEYWORD=DISTINCT
SELECT_ADDITIONAL=NULL
SELECT_JOIN=MolTable.ID
SELECT_LINKS=MolTable.ID;1,MolTable.ID;1
INTER_TABLES=NULL
ADO_CONNECTION=base_connection
CHEM_CONNECTION=base_cfw_form
STRUC_FIELD_ID=MOL_ID

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
