Connection Options in Dataview INI Files

Connection Types

DBQ

For DBQ, CONN_TYPE must be a full text specification with the following minimum of information for each type of datasource. The ODBC control panel will help you to determine information about drivers installed on the server. In general this will use the default OLEDB driver and may not be optimized. The default driver is MSDASQL.

ACCESS
database path;DRIVER={drivername)
Example:CONNECTION_STRING = C:\ Samples\sample.MDB;
DRIVER={Microsoft Access Driver(*.mdb)}

SQLSERVER
DRIVER={drivername};DATA SOURCE=server_name;INITIAL CATALOG=database name
Example: CONNECTION_STRING=DRIVER={SQL Server};
DATA SOURCE = myServer;INITIAL CATALOB=sample

ORACLE
DRIVER={drivername}; SERVER=net8 service_name
Example:CONNECTION_STRING=DRIVER-{Microsofr ODBC for Oracle};
SERVICE = my_world

DSN

For DSN, define the DSN in the ODBC control panel.

ACCESS, SQLSERVER, ORACLE
Dsn name
Example:CONNECTION_STRING=sample

Other data source specification will depend on the driver you are using. Often DSN: the connection string should be simply the name of the dsn, such as "sample_dsn" that you have created via the ODBC control panel on the server.

OLEDB

When using OLEDB, specify the PROVIDER in addition to the other information using slightly different keywords then DBQ. In general the ODBC provider default is MSDASQL which is MicrosoftÂ’s global ODBC driver. Many datasources also have Native providers that can increase performance over the default Microsoft provider.

ACCESS:
PROVIDER=provider_name;
DATA SOURCE=database path or DSN name

Example: CONNECTION_STRING= Provider= Microsoft.Jet.OLEDB.4.0;
Data Source=C:\Samples\sample.MDB
OR with a DSN
CONNECTION_STRING= Provider=Microsoft.Jet.OLEDB.4.0;
Data Source=sample

SQLSERVER
PROVIDER=provider_name;
DATA SOURCE=server_name or dsn_namee ;
INITIAL CATALOG=database name

Example: CONNECTION_STRING= PROVIDER=SQLOLEDB;
DATA SOURCE= myServer;
INITIAL CATALOG=sample
OR With a DSN
CONNECTION_STRING= PROVIDER=SQLOLEDB;
DATA SOURCE= sample

ORACLE
PROVIDER=provider_name;
DATA SOURCE=net8_service_name or dsn_name
Example: CONNECTION_STRING= PROVIDER= MSDAORA;
DATA SOURCE= myWorld
OR with DSN
CONNECTION_STRING= PROVIDER=MSDAORA;DATA SOURCE=sample
