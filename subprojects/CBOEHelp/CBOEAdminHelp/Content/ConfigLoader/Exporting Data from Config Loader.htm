Exporting Data from Config Loader

To export data:

  1. Log into Config Loader with appropriate credentials to export data.
  2. Select Configuration Export, if it is not already selected.
  3. Click Next. This screen appears as:

  1. Select the destination folder where files will be exported.
  2. Click Next. The files are exported and the job summary appears as:

  1. Click Next. The task is completed.

Note: If you export files multiple times to the same output folder, the existing files in the output folder will be overwritten.
