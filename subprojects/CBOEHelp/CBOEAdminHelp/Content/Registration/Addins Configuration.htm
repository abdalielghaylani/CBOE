Add-ins Configuration

To manage the Registration AddIns:

  1. On the Registration Admin section click on "Manage AddIns" link

  2. The following dialog displays:

  3. In this dialog, you may add, delete or edit the Registration AddIns. Refer to the following topics for more details about the Registration AddIns:
  • Normalized Structure Add-in
  • Structure Aggregation Add-in
  • ChemDraw Calculations Add-in
  • Find Custom Field Duplicates Add-in
  • Salt Stripping Add-in
  • ChemDraw Chemical Warning Checker Add-in
  • Python Property Calculation Add-in

Adding a new Add-in

To add a new add-in:

  1. Log in to CBOE with Administrative privileges
  2. In the CBOE home page, click on Customize Registration under Registration section
  3. The Registration Admin window displays, click on "Manage AddIns" link, the following window displays:

  1. Enter the Add-in name in the Friendly Name field, select an Assembly, a Class, a Handler and an Event from the dropdown lists.
  2. Fill in the AddIn Configuration text box by entering an xml query.
  3. Click on Add Addin.
