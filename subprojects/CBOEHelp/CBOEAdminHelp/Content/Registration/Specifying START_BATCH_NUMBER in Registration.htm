Specifying START_BATCH_NUMBER in Registration

By default, the system Batch number always starts with '1'. However, if you want to starts the Batch number with any other number other than '1', you make it possible by updating the web.config file. For example, if you update the web.config file with '0' as described below, the Batch number starts with zero.

To update the web.config file:

  1. Browse to <Install Directory>\Program Files (x86)\PerkinElmer\ChemOfficeEnterprise\Registration\WebApp\.
  2. Open the Web.config file for edit.
  3. Update the tag mentioned below with the desired value. The default value is '1'. If you update it as '0', then the Batch number starts with zero.

    <add key="ConfigBatchNumber" value="1"/>

For DataLoader

  1. Browse to <Install Directory>\Program Files (x86)\PerkinElmer\ChemOfficeEnterprise\DataLoader\.
  2. Open the DataLoader.exe.config file for edit.
  3. Update the tag mentioned below with the desired value. The default value is '1'. If you update it as '0', then the Batch number starts with zero.

    <add key="ConfigBatchNumber" value="1"/>

For DataLoader 2

  1. Browse to <Install Directory>\Program Files (x86)\PerkinElmer\ChemOfficeEnterprise\DataLoader2\.
  2. Open the DataLoaderGUI.exe.config file for edit.
  3. Update the tag mentioned below with the desired value. The default value is '1'. If you update it as '0', then the Batch number starts with zero.

    <add key="ConfigBatchNumber" value="1"/>
