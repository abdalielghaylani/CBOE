Web Registration - Sort Properties Order

The sort order feature allows the administrator to specify the order of appearance of a property. Note that the top most value will appear the first.

To sort the properties order:

  1. In the Web Registration Admin page, click on System Behaviors and select Manage Data Properties option.

The Manage Data Properties window appears.

  1. Use the Level up and Level down arrows to change the properties order in the table.
