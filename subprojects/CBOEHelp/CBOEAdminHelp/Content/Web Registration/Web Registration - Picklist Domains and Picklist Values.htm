Web Registration - Picklist Domains and Picklist Values

Picklists are drop-down lists that provide users a list of all possible values for a field. From the Table Editor, administrator can create a list of values to make a drop-down list field. The list of values are grouped in a Domain as shown in the example below:

Adding a New Picklist Domain

To add a new Picklist Domain:

  1. In the Web Registration Admin window, select Picklist Domains from the Custom Tables drop-down list
  2. Click on the plus icon, a new form pops up as shown below.

  1. If only a list of values is needed, enter the domain description and then click on Save button.
  2. If values come from another Table, enter the following information to build your own SQL and then click on Save button:
  • External Table: Enter the name of the table. If this table is from another schema than REGDB, then enter the schema name before the table name (Ex. COEDB.PEOPLE)
  • External Column ID: Enter the name of the column that contains the table ID
  • External Column Display: Enter the name of the column that contains the values to display in the dropdown list
  • Additional SQL Filter: Enter a clause to filter the values to display in the dropdown list.
  • Additional SQL SortOrder:Enter a clause to specify the order of values to appear in the property's dropdown list. Note that the top most value will be selected as default value.
  • Is Locked?: Select Yes to make the row read-only, and select No to make the row editable.

Note: When a domain is used while registering a compound, it becomes automatically read-only in the Picklist Domains table.

Entering a List of Values

To enter a list of values:

  1. In the Web Registration Admin window, select Picklist Values from the Custom Tables drop-down list.
  2. Click on the plus icon, a new form pops up as shown below.

  1. Enter the following details.
  • Picklist Domain: Select a picklist domain from the drop-down list.
  • Picklist Value: Enter a value for the new picklist.
  • Sort Order: Enter an integer value to specify the order of values to appear in the property's dropdown list.
  • Is Active?: Select Yes to make the new picklist active. Otherwise, select No.
  1. Click the Save button. The new value gets added to the Picklist table.
