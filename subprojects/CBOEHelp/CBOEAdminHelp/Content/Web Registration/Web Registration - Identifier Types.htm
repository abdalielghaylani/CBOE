Web Registration - Identifier Types

Identifier information includes information about the chemical name, synonym, and CAS number of a compound. It helps to identify records easily.

Identifier information can be added in the registry or component level.

Entering a New Identifier

  1. In the Web Registration Admin window, select Identifier Types from the Custom Tables drop-down list. The Identifier Types window appears.
  1. Click on the plus icon, a new form pops up as shown below.

  1. Enter the following details:
  • Name: A name for the identifier
  • Description: A description for the identifier
  • Is Active?: Select Yes, if you want this Identifier to be active. otherwise, select No.
  • Type: Select the Identifier type. The following are the options for the Type drop-down list:
  • Registry: Identifiers display at the Registry level
  • Compound: Identifiers display at the Component level
  • Batch: Identifiers display at the Batch level
  • Base Fragment: Identifiers display at the Structure level
  • All: Same Identifiers display at all levels
  1. Click the Save button. The new identifier gets added to the Identifiers table.
