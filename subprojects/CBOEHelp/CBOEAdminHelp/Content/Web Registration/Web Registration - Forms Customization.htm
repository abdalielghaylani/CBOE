Web Registration - Forms Customization

To customize forms, please perform the steps below:

  1. In the Web Registration Admin page, click on System Behaviors and select Customize Forms option.

The following page displays:

  1. Click on the Edit icon.
  2. In the forms customization form you may be able to:
    • Change the Label Name
    • Select the Control Type
    • Define Width/Height
    • Change the field visibility
