Administrator's Guide

Editing the Master Dataview

After you create the master dataview, you can edit it at any time. You can add or remove schemas from it and then choose which tables from those schemas will be available to create dataviews.

To begin editing the master dataview, click Edit Master under DataView Manager on the ChemBioOffice Enterprise Manager main page. The DataView manager window opens.

Adding schemas

To add schemas to the existing Master Dataview:

  1. Click Add Schema.
  2. In the Database dropdown list, select the schema and enter the schema Password.
  3. Click Publish. The schema is added to the Master Dataview.

Removing schemas

Note: When you have a clean installation and no schema is published, both Remove Schema and Refresh Schema buttons will be disabled. When only one schema is published, Refresh Schema will be enabled and Remove Schema will remain disabled. Remove Schema will be enabled only when more than one schema is published.

To remove a schema from the Master Dataview:

  1. Select the schema name in the schema list on the left side of the screen.
  2. Click "Remove Schema".

  1. A confirmation message displays. Click OK, the schema is removed from the list.

Selecting tables

To include a table in the Master Dataview:

  1. Select the schema containing the tables to be added
  2. Click Add Table, to add tables to the Master Dataview

  1. To add a particular table, click on its corresponding Add button, the table gets added under "Tables to Add " at the right frame. Select the tables to be added and click on Save button at the bottom. The tables are added to the schema.

Note: The names of tables that are already added appear in red.

To exclude tables from the Master Dataview, simply click on Remove button corresponding to the tables to be removed as shown in the screen shot under step 2 above.

Saving your changes

  1. After you have selected the schema and tables to add to the Master Dataview, click Submit. The Validation Summary screen appears.
  2. In the Validation Summary screen, review the list of tables and their relationships. If the list is acceptable, click Submit. Otherwise, click Cancel to return to the previous screen.
