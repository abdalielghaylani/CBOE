Searching the Database

Searching the ChemACX database is similar to searching in other ChemBioOffice Enterprise applications. You can search the ChemACX database by chemical name, substructure, ACX number, molecular formula, molecular weight, or catalog number.

Unless you search by substructure, you can simplify the search by using only the chemical name or Chemical Abstracts Service (CAS) number. For information, see "Non-plugin searches" topic.

To search the ChemACX database:

  1. Log in to ChemACX and click on the Search button to display the search form.
  2. Enter information about the chemical that you are searching for.
  3. Select one of the following options:
  • Search all vendors: Searches chemicals from all the vendors available in ChemACX.
  • Search your favorite vendors: Searches chemicals only from the vendors available in the favorite vendors list. For information about viewing or editing favorite vendors list, please refer to "Favorite Vendors List" topic.
  • Display all substances even if no vendors are found: Searches for all chemicals that match your search criteria regardless of whether they have an associated vendor or not.
  1. Click Search. A page containing the search results appears. For information about viewing the search results, see "Viewing Search Results" topic.
