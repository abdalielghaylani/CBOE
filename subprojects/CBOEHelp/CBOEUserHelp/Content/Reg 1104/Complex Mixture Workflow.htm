Complex Mixture Workflow

If your system settings is set to the complex mixture workflow, then you can either add new components and draw structures or add from existing components.

To submit a mixture:

  1. Log in to CBOE with submitter privileges
  2. In the CBOE home page, select Submit New Mixture icon under Registration

Note: The user is allowed to submit mixtures only if he/she has "ADD_COMPONENT" privilege.

  1. The Submit Mixture window displays. Draw the desired structure and fill in the Registry and component information

  1. To add further components:
  • Select "Add a new component" option from the "Manage Mixture Components" menu, to add new components and draw new structures
  • Select "Pick an existing component" option from the "Manage Mixture Components" menu, to search and add components from the permanent registry

  1. When you are done with adding components, go to Manage Components and select Complete batch information and submit.Specify the percentage of each component and click on Submit button.
    Note: If ChemDraw Chemical Warning Checker is enabled and ChemDraw chemical warning is present, you will not be allowed to Submit the mixture until the underlying error is resolved. For more information on ChemDraw chemical warning, See ChemDraw Chemical Warning.
  2. The record is added to the temporary table and the temporary record ID appears. To view each component details, click the corresponding component in the New Registry tree structure

After submitting a mixture, the same process is followed as for a single component. The registrar reviews and registers the mixture.
