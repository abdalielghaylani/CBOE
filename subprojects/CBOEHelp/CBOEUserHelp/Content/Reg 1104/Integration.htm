Integration

Registration can be integrated with other applications such as Inventory Enterprise, ChemACX database, E-Notebook, etc. When integrated, Registration provides a solution to the problems associated with organizing, storing, and sharing chemical information. Each product shares information through common databases. Each product also provides interfaces to operations performed in different applications. For example, it is possible to register a compound in Registration Enterprise through E-Notebook. Sharing a common database makes all information consistent, up to date and accessible to all users with specific privileges. This process saves time and reduces the number of errors in re-interpreting and re-entering data in multiple locations.
 
Registration Enterprise may be linked to any of these ChemBioOffice Enterprise applications, if they are installed:
Inventory Enterprise
It is possible to search for compounds in Registration Enterprise using Inventory Enterprise. This allows users to include Registration Enterprise compounds in inventory containers without re-entering data. In addition, it is also possible to send Registration Enterprise search results to Inventory Enterprise. Any changes made to the record in the Registration Enterprise is immediately reflected in Inventory Enterprise.
ChemACX
It is possible to register compounds from a catalog listing. This populates the registration record with all the data in the ChemACX record.
Structured Data
It is possible to register a compound in Registration Enterprise through Structured Data.
E-Notebook
It is possible to register a compound in Registration Enterprise through E-Notebook.
