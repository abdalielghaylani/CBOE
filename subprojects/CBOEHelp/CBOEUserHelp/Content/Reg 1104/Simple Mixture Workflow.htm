Simple Mixture Workflow

If your system settings is set to the simple mixture workflow, then you are only allowed to create mixtures form registered components.

To submit a mixture:

  1. Log in to CBOE with submitter privileges
  2. In the CBOE home page, select Submit New Mixture icon under Registration

Note: The user is allowed to submit mixtures only if he/she has "ADD_COMPONENT" privilege.

  1. The following dialog displays. Fill in the Mixture registry and the Batch information fields

  2. Click on Pick Component to select a component from the permanent registry. The search form window displays, enter specific criteria, based on which the query will be run and click on search button. The component information that meets the search criteria appears. Click Add Component link corresponding to the component that you want to add.
    To display all the registered record, click on Retrieve All button.

  3. Select the structure and click on Add Component link

  4. The selected component gets added to the mixture registry.
  5. To add more components, go to Manage Mixture Components and select Add a New Component.

  6. A new component icon appears in the New Registry tree structure on the left-hand side of the window
  7. The Search form displays allowing users to run specific query and search for components in the permanent registry
  8. To delete any component that is already added, go to Manage Components> and select Remove Component
  9. When you are done with adding components, go to Manage Components and select Complete batch information and submit.The following dialog displays:

  1. Enter the percentage for each component and click on Submit button to move the record to the temporary table.
    Note: If ChemDraw Chemical Warning Checker is enabled and ChemDraw Chemical warning is present, you will not be allowed to Submit the mixture until the underlying error is resolved. For more information on ChemDraw chemical warning, Make Default.
  2. The record is added to the temporary table and the temporary record ID appears. To view each component details, click the corresponding component in the New Registry tree structure

After submitting a mixture, the same process is followed as for a single component. The registrar reviews and registers the mixture.
