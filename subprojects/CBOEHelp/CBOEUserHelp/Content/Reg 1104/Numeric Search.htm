Numeric Search

Data such as boling points and molecular weights can be retrieved using numeric search. When performing a numeric search, users can specify a range using a hyphen between the values at either end of the range. Ranges may also be indicated using inequality operators(<,>) together with the AND operator.

The query is a decimal value or range. If a single value is given, the number of significant digits determines the precision of the search. A hit is any value that rounds off to the query.

The following table lists a few examples of numeric search.

Entry in a Numerical Field Results Found
78.1 Values rounded off to 78.1, for example, 78.15. 78.32, 78.09, etc and NOT 78.0
78 Values rounded off to 78, for example, 77.6, 77.9, 78.2, etc..
89.5-90.5 Values from 89.5 to 90.5
90.05-91.05 Values from 90.05 to 91.05
90-100 Values from 90 through 100, inclusive
>90 and <100 Values from 90 through 100, exclusive
