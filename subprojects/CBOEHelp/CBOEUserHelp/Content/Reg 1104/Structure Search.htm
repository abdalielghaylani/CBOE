Structure Search

Structure searching allows the user to search for records based on their chemical structures. Registration Enterprise 11.0.4 provides three types of structure searches:

  • Substructure Searching
  • Full Structure Searching
  • Exact Searching

Substructure searching

Substructure searching is the default structure search type. It retrieves structures that meet the following criteria:
  • Matches the basic chemical structure specified
  • Contains additional attachments at open bond positions
To narrow down or broaden search, attach different features such as atom lists and variable bond types to the structure using ChemDraw drawing tools. Below is an example of Substructure Searching.

Full Structure Searching

Full structure searching retrieves structures that match the structure specified in the query, without considering additional attachments. Isotopes are recognized as hits. Below is an example of Full structure searching.

Exact Searching

Exact searching retrieves structures that are identical to the structure specified in the query. Isotopes are not recognized as hits. Below is an example of Exact Searching.

Setting search preferences

Setting the search preferences allows the user to control the manner in which the results are displayed. These settings are global for a user throughout the application.

To access the search preference options:
  1. Click "Options"attached to the structure window.

  1. The Search Preferences dialog box appears.
General tab
The various options under the general tab include:
  • Hit any charge on heteroatom: allow uncharged atoms in the query to match charged atoms in the target.
  • Reaction query must hit reaction center: allow the reaction centers in the query to overlap with the reaction centers in the target.
  • Hit any Charge on Carbon: allow uncharged carbon atoms in the query to match charged carbon atoms in the target.
  • Permit extraneous fragments in full structure(exact) searches / Permit extraneous fragments in reaction full structure(exact) searches: perform exact structure search.
  • Query Fragments can overlap in target selected: fragments can be overlapped.
  • Tautomeric: allow to find tautomers in the query.
  • Similarity search: retrieves chemicals that have structural features similar to the one specified in the query. It does not match specific atoms or bonds. It is useful if you have a general idea of the types of compounds you are looking for, but do not know the precise target compound. You can specify the degree of similarity. A lower value requires the target to be less similar.
Stereochemistry tab

The various options under the stereochemistry tab include:
  • Tetrahedral stereo center hits: To set the Tetrahedral stereo center hits:
    • Select Same to match the target exactly.
    • Select Either to match same or opposite configuration at the center of the target.
    • Select Any to match any target.
    • Select Thick bonds represent relative stereochemistry to match a relative relationship between centers. When activated, any thick bond drawn in the reference compound is interpreted as relative stereochemistry when searching or comparing possible duplicates
  • Double bonds hits: To sets the Double bonds hits:
    • Select Same to match the target exactly.
    • Select Any to match any configuration in the target.
