Batch Highlighting

Batch Highlighting feature allows batches to be highlighted when some other field is in a particular state. In order to enable the batch highlighting feature, the administrator should configure and provide specific values to the following fields in the system settings dialog:

  • BatchHighlightingField
  • BatchHighlightingValue
  • BatchHighlightingTooltip

Let us suppose that the corresponding values are the following:

  • Amount
  • 25
  • QA wants To be verify

When the submitter submits a component that has 25 value as amount, then after approving and registering the component, the batch will be highlighted as shows the screen shot below:

Moving the mouse over the highlighted batch, a message displays explaining why the batch is highlighted.
