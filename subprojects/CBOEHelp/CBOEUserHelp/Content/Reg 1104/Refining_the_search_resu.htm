Refining the search results

Users can modify the search criteria and search over the current result set.

To refine a query, go to Refine menu and then select one of the available options:

  • Edit Current Query: to re-run the original query and then apply the refinement query.
  • Refine Over Current Records: to apply the refinement query to the current record set
