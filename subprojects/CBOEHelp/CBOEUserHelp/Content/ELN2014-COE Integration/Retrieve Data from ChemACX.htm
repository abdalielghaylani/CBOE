Retrieve Data from ChemACX

You can retrieve data from ChemACX system to use the structures in the Reaction section.

To retrieve data from ChemACX:

  1. Log in to E-Notebook.
  2. Create a Discovery Chemistry (which has Reaction as default section) or a Process Chemistry (which has Process Reaction as default section) experiment.
  3. In the Reaction toolbar, click Add. The Select & Add Structure dialog appears.
  4. Search In: Select ACX Reagents from the drop down list. The following window appears:

A set of text fields is available for entering textual search data:

  • Substance Name
  • CAS Registry Number
  • ACX ID
  • Catalog Number
  • Supplier
  • Mol/Weight Range
  • Molecular Formula
  1. Enter the search criteria and click Search to execute the search.
  2. Click Clear to clear the form.
