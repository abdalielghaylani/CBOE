Fetch Register Number

The Fetch Register Number allows users to query the database with a registration Temp ID. If a Registration Number is available then the Registration Number and Batch Number will returned to the associated row in the Products Table of the Reaction section. This method was created to support temporary registration from E-Notebook. For example, users would register a compound in the temporary table from E-Notebook and a TempID would be returned to the Products table. Later a Registrar would then permanently register the compound into the Registration system. Subsequently, the E-Notebook user can use the Fetch Register Number method to query the Registration number and Batch number, retuning them to the associated row in the Products table.

Note: user requires adequate Registry permissions to perform this Registry action.

To Fetch the Registry Number:

  1. Under Home ribbon, select the Fetch Register Number form tool within the Samples section.
  2. If the TempID is present in the Products table and the Registration Number is available in the Registration system then, Fetch Register Number dialog appears displaying the Registration Number and Batch Number.
  3. Click OK for the Fetch Register Number dialog.
  4. The Registration Number and Batch Number will be returned to the associated row in the Samples table of the Reaction section.

In case the TempID is present in the Samples table and the Registration Number is unavailable in the Registration system then the message box returns “No Compound exists with the Temp id:...”

If Reg Number and Batch Number fields are already populated with data then the message box returns: Clear the Reg Number field and Fetch again.
