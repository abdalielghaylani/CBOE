Global Search

Global search lets you to search for a substance in Inventory Enterprise, as well as in Registration Enterprise, and ChemACX. With Global Search, you can search all these applications simultaneously.

Note: Global search can be performed on the Registration Enterprise and ChemACX applications only if the system running Inventory Enterprise is installed and configured with these applications.

Global Search lets you search a substance on the basis of the Substructure, Substance Name, CAS Registry, ACX Number, Catalog Number, Molecular Formula, MolWeight Range, Registry Number, Registry Sequence, Reg Alternate IDs, Salt Name, and Salt Equivalents fields.

To perform global search on Inventory Enterprise:

  1. From the ChemBioOffice Enterprise home page, Inventory Enterprise section, click the Search link. A window containing various search tabs, such as Simple Search, Advanced Search, Substructure Search, Batch Search, Global Search, and Plate Search is displayed.

Note:You can access the Search tabs window by clicking the Search link within the Current Location frame in the Container Management area.

  1. Click the Global Search tab. The Global Search tab window appears.

Note:The number of tabs visible in the Search tabs window depends on your configuration settings.

Note:You can access the Global Search tab window by clicking the Global Search link within the Inventory Enterprise section in the ChemBioOffice Enterprise home page.

  1. Select the check boxes next to the applications over which the search will be run, such as Inventory Enterprise, Registration, and ChemACX.

Note: The Registration and ChemACX databases checkboxes are enabled or disabled based on the privileges assigned to you:

  • You must have SEARCH_REG privilege to search compounds from the Registration database, or to select a compound from Registration while creating a new container. Otherwise, the registration checkbox in the search form will be disabled.

  • You must have BROWSE_ACX privilege to search compounds from ChemACX database, or to select a compound from ChemACX while creating a new container. Otherwise the ChemACX checkbox in the search form will be disabled.
  1. Enter your search criteria.
  2. Click Search. A list of substances matching the search criteria specified in the Global Search tab window appears.

Note: Global Search does not return substances that match the specified criteria, but are not associated with any product.

Viewing Global Search Results

In the Global Search, the results from various applications are stored in different lists. For example, Registration Enterprise, Inventory Enterprise, and ChemACX substances that match the search criteria specified in the Global Search tab are stored in separate lists.

You can view the results of a particular application by clicking the radio button next to the application, as shown in the example below.

You can manage the search result list using the following options:

  • Zoom In: Zoom in on the structure of the substance.
  • Names: Gather information about the various synonyms of the substance.
  • Details: Gather information about the container associated with the substance.
  • Mark: Mark the substance so that you can easily locate the substance in Inventory Enterprise later, without re-creating a search query for it.
  • MSDS: View the Material Safety Data for a chemical.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked from the Marked Hits menu displayed in the window containing the search results.
