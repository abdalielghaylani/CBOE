Searching for a Container

You can easily locate a container in the Location Tree frame if you know its location.

To search for a container in the Location Tree frame and view its details:

  1. Click on Browse within the Inventory Enterprise section in the ChemBioOffice Enterprise home page. The Container Management area appears.
  2. From the Location Tree, navigate to the location that contains the container.
  3. Select the appropriate location. The location's contents displays in the Container List.
  4. Click the container in the Container List to view its details in the Container Details.

For more information about the Location Tree and how to navigate through it, see Location Management.

When you do not know the location of the container that you are looking for, you can search for the container using the search forms. The search forms lets you search for a container based various container attributes, such as container name and type. Inventory Enterprise provides search forms letting you search for containers based of various search criteria. For more information about the different search forms, see Simple Search, Advanced Search, Substructure Search, and Global Search.
