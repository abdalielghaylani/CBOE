Location Administrator

To create a new location and specify the administrator:

  1. Login to ChemBioOffice Enterprise Manager as INVADMIN or any other user
  2. Click on Browse link to go to Inventory application

  1. In the left pane go to Location>>New

  1. Fill in the required and Optional fields (if desired) and then in the Location admin field select an appropriate user/group corresponding to the by user/by group option that you select

  1. When completed, click on Ok button the new location is created

Note: It is not possible to edit a location or to add a sub-location to a location when you are not an owner. The Field will appear in read only mode as shown in the screen shot below.

Note: when you select Propagate the Location Admin to child locations/containers/plates checkbox, the location admin gets updated in the corresponding sub-locations, containers and plates.
