Ordering and Receiving Containers

The Ordering and Receiving feature lets you place an order for a container through ChemACX and get it delivered to the desired Inventory location, which is set as default. Therefore, it is important when ordering a container, you have set a default location. To set a location as the default location, see Make Default in Location Management.

Placing an Order

Placing an order involves searching compounds through ChemACX and filling a shopping cart.

To place an order:

  1. Login to the ChemACX application. The following window appears.

  1. Click Search to search for the chemical you want to place in the shopping cart. The following window appears.

  1. You can click Details for any compound for more information. The following window appears, for example:

  1. Select an item from the All Vendors list. The following window appears.

  1. Check the checkbox next to add to cart to add a substance.
  2. Click the VIEW SHOPPING CART icon in the ChemACX interface. The Shopping Cart Frameset window displays.
  3. Click Send to Inventory Manager.

A confirmation screen appears as:

  1. Click Browse to open a list of destination locations.
  2. Destination Location: Select a destination location from the list.

Note: Regardless of the destination location you specify, by default, the ordered container displays within the “On Order” section in Inventory Enterprise. You can change this behavior so the ordered container displays under the specified destination location. You have to configure a parameter within the invconfig.ini file. For more information, see your system administrator.

  1. Project: Select the project associated with this order from the drop-down list.
  2. Job: Select the job associated with this order from the drop-down list.
  3. Due Date: Use the calendar tool to enter a due date for this order.
  4. Order Reason: Select a reason for this order from the drop-down list.

Note: When the invconfig,ini file is configured to change the default behavior of the ordering process, ChemACX does not prompt you to enter the order details mentioned in this step. The only fields that appear on the confirmation screen of ChemACX form are the destination location and the owner.

  1. Click OK. The following window appears.

  1. Click Commit. The following window appears.

  1. Click the View link. The ordered container is displayed in the Container Management area. The container is in On Order location.

Receiving an Order

To receive a placed order:

  1. Open the Administrative Menu window.
  2. From the Container Management section, click Manage Container Receiving. The Receive Ordered Compounds window appears.

  1. Specify a filter criteria, such as CAS Number.
  2. Click Filter. The following window appears.

  1. Specify a scan barcode for the container to be received.
  2. Click OK at the bottom of the window. The status of the container changes to Received.
  3. Click Close to close the window.
