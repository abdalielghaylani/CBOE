Submitting a New Compound

You can submit a record to the temporary table in the database or, if you have adequate privileges, approve and register the record directly to the permanent table.

To submit a new compound:

  1. Log in to Registration.
  2. Click on Submit New Compound icon. The following page displays:

  1. Enter the compound's information in the corresponding fields and click Submit.

Note: Use the ChemDraw Direct tools to draw and edit structures. For ChemDraw Direct there is no right-click menu available to edit the structures. You can use some shortcuts for structure editing as shown in the table below:

In addition, the following options are not available in ChemDraw Direct:

  • Structure Perspective
  • Fragmentation Tools
  • Table Tool
  • Chromatography Tools
  • Query Tools
  • Drawing Elements
  • Orbitals
  • Pen Tools

After clicking Submit, the compound gets added to the temporary table.

Note: You can see the Register button only if you have adequate privileges assigned to you by the system administrator and the approval workflow is disabled. You can skip then the temporary registration and send the record directly to the permanent table if desired.
