Printing Records

You can print your temporary registered, permanent registered or search results lists.

Note: the maximum number of records that you are allowed to print is 1000. If the records table contains more than 1000 records, the Print button will not be available.

To print records:

  1. Navigate to the list that you want to print and click Print button.

  1. The Print table page appears. Specify the print settings, for example, choose the printer, the pages that you want to print, the layout, the color, the paper size, the margins, etc... When done with your selections, click Print.

Note: the print function depends on the browser print feature, prepare the print sheet to fit your preferences using the tools available in each browser (for example: print landscape, zoom out, etc...).

Note: Pop-ups blockers may prevent pop-up windows from displaying in certain web browsers and this may stop the printing process until you turn off the pop-up blocker.
