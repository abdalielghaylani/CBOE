Restoring a Query

Many options for restoring a query or hitlist are available:

To Restore a query hitlist:

In the Query Management display, click on the Restore Hitlist icon corresponding to the query that you want to display its hitlist.

The corresponding hitlist gets displayed.

To Re-run a query:

In the Query Management display, click on the Perform Query icon corresponding to the query that you want to display its hitlist.

To restore a query to form:

In the Query Management display, click on the Restore Query To Form icon corresponding to the query that you want to restore. The query input gets restored into the search form.

Advanced

The Advanced option provides you three options for restoring a query.

To restore a query:

In the Query Management display, click on the Advanced icon corresponding to the query that you want to restore. The following options display:

  • Intersect with current list: displays only those records that appear in both the current hit list and in the saved hit list. This is a Boolean AND operation.
  • Union with current list: displays all records in either the current list or the saved hit list. This is a Boolean OR operation.
  • Subtract from current list: displays only those records that are in the saved hit list but are not in the current hit list.
