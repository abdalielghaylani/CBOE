Creating inventory containers

To create containers in Inventory Enterprise from ChemACX chemicals:

1. Open the shopping cart.
2. Click Send to Inventory Manager. The Create Inventory Containers from ChemACX page appears:

Please consult your system administrator, if you do not see the Send to Inventory Manager button.
3. Provide the required information and click OK. The following page appears:

4. Click Commit. The following page appears:

5. Click View to view the newly created containers.
