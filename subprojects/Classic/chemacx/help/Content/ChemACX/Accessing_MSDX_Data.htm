Accessing MSDX data

You can view the Material Safety Data (MSDX) for a chemical by clicking the MSDX link available in a chemical record. The MSDX link is available in both, List view and Details view, as shown in the following figures:

List View

Details View

Note: If MSDX data is not available for a chemical record, the MSDX link for the record is disabled.

To view MSDX data for a chemical:

1. Click MSDX in the chemical record in either List or Details view. A page listing all the datasheets available for the chemical appears:

If only one datasheet is available for the chemical, the datasheet appears directly and the list of the datasheet is not displayed.
2. Click View MSDS corresponding to the required datasheet. The datasheet appears:
