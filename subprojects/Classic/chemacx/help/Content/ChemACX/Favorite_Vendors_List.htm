Favorite vendors list

You can also search for substances under a shorter list of preferred vendors. This shorted list helps to limit search results to include products from only your favorite vendors.

To create or edit the favorite vendors list:

1. Access the ChemACX database to display the search form.
2. Click EDIT FAVORITE VENDORS LIST. The following page appears:

3. Select the check box next to the vendors whom you need to include in the favorite vendors list.
4. Click Save as Preferred List.
