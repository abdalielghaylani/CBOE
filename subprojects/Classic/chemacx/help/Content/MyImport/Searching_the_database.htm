Searching the database

Searching the ChemACX database is similar to searching in other ChemBioOffice Enterprise applications. You can search the ChemACX database by chemical name, substructure, ACX number, molecular formula, molecular weight, or catalog number.
Unless you search by substructure, you can simplify the search by using only the chemical name or Chemical Abstracts Service (CAS) number. For information, see "Non-plugin searches".
Note: For more information on searching techniques, see “Performing Searches” in the ChemBioOffice Webserver Guide.
To search the ChemACX database:
1. Access the ChemACX database to display the search form.
2. Enter information about the chemical that you need to search.
3. Select from these options:
Search all vendors: Searches chemicals from all the vendors available in ChemACX.
Search your favorite vendors: Searches chemicals only from the vendors available in the favorite vendors list. For information about viewing or editing favorite vendors list, please see "Favorite vendors list".
Display all substances even if no vendors are found: Searches for all chemicals that match your search criteria regardless of whether they have an associated vendor.
4. Click Search. A page containing the search result appears. For information about viewing the search results, see "Viewing search results".

Non-plugin searches

A non-plugin search limits the search fields to only chemical name and CAS number. This search method is useful when you do not need to search ChemACX by substructure, and therefore do not need the Chem & Bio Draw plugin.
To perform a non-plugin search in ChemACX:
1. Access the ChemACX database to display the search form.
2. Click CLICK FOR NON-PLUGIN SEARCH. This page appears:

:

3. Enter the name or CAS number of the chemical that is to be searched.
4. Click Search.

Favorite vendors list

You can also search for substances among a list of your favorite vendors..
To create or edit the favorite vendors list:
1. Access the ChemACX database to display the search form.
2. Click EDIT FAVORITE VENDORS LIST. The vendor list appears:

:

3. Select the check box next to the vendors whom you wish to include in the favorite vendors list.
4. Click Save as Preferred List.
