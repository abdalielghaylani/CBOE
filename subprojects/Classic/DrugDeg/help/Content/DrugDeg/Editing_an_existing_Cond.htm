Editing an existing Condition

To edit a condition already in the conditions list:
1. In the CambridgeSoft Drug Degradation Enterprise page, click Manage Experimental Conditions List. The Manage Degradation Condition List appears.
2. Click the corresponding Change Text button for the condition you want to edit. The Change Degradation Condition Text page appears.
3. Enter a new name for the condition.
4. Click Update.
