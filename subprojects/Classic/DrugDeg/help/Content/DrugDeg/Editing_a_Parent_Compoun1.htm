Editing a Parent Compound

To edit a compound, you first search for it in the database and edit it as desired.
1. In the CambridgeSoft Drug Degradation Enterprise main page, click Search.
2. In the Parent Compound Search page, enter the criteria for the compound you want to edit.
3. Click Search.
4. In the search results, click Show Details for the compound you want to edit.
5. In the Parent Compound Details screen, click Edit Record.
6. Edit the parent compound data as desired.
7. Click Update.
