Managing Salts Lists

The salt lists appear in parent record submission forms. These forms allow users to select salts to be associated with parent compounds.
To add to or edit the salt lists, do one of the following:
Click Manage salt list in the CambridgeSoft Drug Degradation Enterprise page.
Click Add/Edit Salts in the ChemBioOffice Webserver global login page.
The Manage Parent Salt List page appears.
