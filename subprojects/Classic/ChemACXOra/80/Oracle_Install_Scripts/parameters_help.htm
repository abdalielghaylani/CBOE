Oracle Script Parameters - ChemACX

The following descriptions are intended to lead you through the Oracle Parameters file. The default values are adequate for standard, smaller installations, but should be adjusted by a knowledgeable DBA to suit the environment of midsize-large production installations.  Changing these parameters has a direct effect on how the application schema is created, so it is important to fully understand the parameters before making changes.

The parameters file is found here.

General

The parameters listed here under General should not be changed to ensure proper operation of the application.

schemaName

The name of the schema.

schemaPass

The password for the schema.

schemaVersion

The version of the schema.

appVersion

The version of the application.

securitySchemaName

The name of the security schema (default = cs_security)

securitySchemaPass

The password for the security schema.

privTableName

The privilege table name.

min_priv

The minimum privileges granted to users of the application.

Table Space Names

tableSpaceName

The name of the schema table space.  This table space is created during script execution.

indexTableSpaceName

The name of the index table space.  This table space is created during script execution.

tempTableSpaceName

The name of the temporary table space.  By default, the TEMP tablespace is used (this is the temporary tablespace added in a default Oracle installation).  This table space is not created as part of these scripts.  Be sure the name of this table space refers to an existing table space before running the script.

lobsTableSpaceName

The name of the lob table space.  This table space is created during script execution.

msdxTableSpaceName

The name of the MSDX table space.  This table space is created during script execution.

securityTableSpaceName

The name of the security table space.  This must match the name given to the cs_security table space during creation.  This table space is not created as part of these scripts.

Path to Table Space

defaultTableTSPath

The path to the table space file.  The path must end with a slash (\) and the path must be valid.

defaultIndexTSPath

The path to the index table space file.  The path must end with a slash (\) and the path must be valid.

defaultLobTSPath

The path to the lob table space file.  The path must end with a slash (\) and the path must be valid.

defaultMsdxTSPath

The path to the MSDX table space file.  The path must end with a slash (\) and the path must be valid.

Table Space File Names

This file name must include the full path.  By default the following convention is used:
 '&&defaultTableTSPath&&tableSpaceName..DBF'
Where &defaultTableTSPath refers to the value of the defaultTableTSPath parameter and &tableSpaceName refers to the value of the tableSpaceName parameter.

tableSpaceFile

The name of the schema table space file.

indexTableSpaceFile

The name of the index table space file.

lobsTableSpaceFile

The name of the lob table space file.  

msdxTableSpaceFile

The name of the MSDX table space file.  

Block Sizes

Custom block sizes can be defined for Oracle 9i installations.  In an Oracle 8i installation, these parameters are ignored.
The values for your block size rely heavily on your installation's constraints.

dataBlockSize

The data block size.

indexBlockSize

The index block size.

lobBlockSize

The lob block size.

msdxBlockSize

The MSDX block size.

dataBlockSizeClause

The data block size clause.  By default, BLOCKSIZE followed by the value of the dataBlockSize parameter.

indexBlockSizeClause

The index block size clause.  By default, BLOCKSIZE followed by the value of the indexBlockSize parameter.

lobBlockSizeClause

The lob block size clause.  By default, BLOCKSIZE followed by the value of the lobBlockSize parameter.

msdxBlockSizeClause

The MSDX block size clause.  By default, BLOCKSIZE followed by the value of the msdxBlockSize parameter.

Table Space Size

The default values are adequate for standard, smaller installations, but should be adjusted by a knowledgeable DBA to suit the environment of midsize-large production installations. See the parameters for tips in deciding how large to make your table spaces.

tablespaceSize

The schema table space size.

idxTablespaceSize

The index table space size.

lobTablespaceSize

The lob table space size.

msdxTablespaceSize

The MSDX table space size.

tablespaceExtent

The extension size for the schema table space.

idxTablespaceExtent

The extension size for the index table space.

lobTablespaceExtent

The extension size for the lob table space.

msdxTablespaceExtent

The extension size for the MSDX table space.

Lob Space Allotted

lobB64cdx

The amount of space allotted to lobs containing Base64cdx data. 

msdxLob

The amount of space allotted to lobs containing MSDX data.
