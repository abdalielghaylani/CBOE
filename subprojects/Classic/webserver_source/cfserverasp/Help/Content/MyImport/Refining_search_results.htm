Refining search results

If a query returns a large number of results, you can refine either the query or the hitlist by adding other search criteria.
To refine a query:
1. While viewing the hit list, click Refine. This window appears:

 

2. Select Edit current query and click OK. The Query Input Form appears.
3. In the Query Input Form, enter the search criteria.
4. Click Search to display the search results.
To refine a hit list:
1. While viewing the hit list, click Refine. This window appears:

 

2. Select Refine over current records found and click OK. The Query Input Form appears.
3. In the Query Input Form, enter the search criteria.
4. Click Apply. The current hitlist is filtered using the criteria you entered.

Undoing a refined search

To return to results obtained before refining a search, click Undo Refine.
