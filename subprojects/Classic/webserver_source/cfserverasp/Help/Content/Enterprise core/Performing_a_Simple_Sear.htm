Performing a simple search

To perform a simple search:
1. Open a query input form.
2. Draw the chemical structure of benzene in the ChemDraw Plug-in area using the toolbar available in the Query Input Form page.
3. Click the Search button. The results of the search appear in the Results List View page.
Note: Search results are displayed according to the Preferences dialog box.
