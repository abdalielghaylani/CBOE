Using the history

The application history can be used to retrieve queries and hit lists, which had been executed earlier, but were not saved. The history is cleared periodically, based on the application configuration. Consult your system administrator for more information about history clearance.
To restore a query in your history:
1. Select Queries from the History menu. The Queries window appears.
2. Select the appropriate query or hit list from the Select One drop down list.
3. Click the Restore button.
