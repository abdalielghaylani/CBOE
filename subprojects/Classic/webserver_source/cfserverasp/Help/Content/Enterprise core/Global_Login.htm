Global login

ChemBioOffice Enterprise uses a global login interface to manage user related information. Once you login with your username and password, you can access all the ChemBioOffice Enterprise applications to which you have the necessary permissions.
When you log into ChemBioOffice Enterprise, a cookie is written to your machine. This cookie holds login information. Therefore, login information is recognized even after a browser is closed.
Note: The cookie expires if you do not perform any action for 25 minutes.
