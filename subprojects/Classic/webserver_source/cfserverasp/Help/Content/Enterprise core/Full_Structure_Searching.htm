Full structure searching

Full structure searching retrieves structures that match the structure drawn in the query input form, without considering additional attachments. Isotopes are recognized as hits.
To perform a Full Structure search:
1. Click the New Query button to display a blank search form.
2. Draw a structure.
3. Select Full Structure in the Search Type drop-down list.
4. Click the Search button. The records including isotopes are retrieved.
