Structure searching

Structure searching allows you to search for chemicals based on their chemical structures, which can be drawn using the ChemDraw plug-in. For more information on using ChemDraw, see the ChemDraw documentation.
ChemBioOffice Enterprise provides four options for searching chemicals by structure. These options are:
Substructure
Full Structure
Exact Structure
Tanimoto Similarity.
