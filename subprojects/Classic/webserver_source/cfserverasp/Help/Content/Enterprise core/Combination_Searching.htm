Combination searching

You can perform a search that is a combination of the search types mentioned earlier and obtain a boolean AND search result.
For a complete list of types of searches, see "Types of searches".
Note: OR and NOT searches are not supported in the ChemBioOffice Enterprise.
To perform a sample combination search:
1. Open a query input form.
2. Type *ene in the Molecule Name text box and 100-200 in the Molecular Weight text box.
3. Draw a bond with a chloride using the ChemDraw plug-in.
4. Select Substructure in the Search Type drop-down list.
5. Click the Search button. The result of combination search that matches the entered criteria is displayed.
