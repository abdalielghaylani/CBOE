Defining Views

Views expose Registration data to Inventory Enterprise. The three views that needs to be created for integrating Registration Enterprise with Inventory Enterprise are:

  • INV_VW_REG_BATCHES
  • INV_VW_REG_ALTIDS
  • INV_VW_REG_STRUCTURES

All three views must be created within the Inventory Enterprise schema (cheminvdb2).
