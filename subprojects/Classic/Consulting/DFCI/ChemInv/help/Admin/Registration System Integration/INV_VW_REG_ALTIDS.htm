INV_VW_REG_ALTIDS

This view defines alternate identifiers for the Registration substances. The following table lists the alternate identifier for a Registration substance:

Field Name Type Description
RegID Number Specifies a unique internal identifier for a substance.
Identifier Varchar Specifies synonyms associated with a  substance.
