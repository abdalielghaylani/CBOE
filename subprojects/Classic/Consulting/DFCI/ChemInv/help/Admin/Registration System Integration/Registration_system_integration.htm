Integration with Registration Enterprise

The integration of Registration Enterprise with Inventory Enterprise allows you to access the Registration data directly from Inventory Enterprise. However, if you want to view additional Registration data in addition to the standard data, you can customize the integration. In order to integrate Inventory Enterprise with Registration Enterprise and customize the integration, you need to:

  1. Define views
  2. Define comments for each Registration field to be accessed from Inventory Enterprise
  3. Add new fields to the view and define comments for the new fields to customize the integration
