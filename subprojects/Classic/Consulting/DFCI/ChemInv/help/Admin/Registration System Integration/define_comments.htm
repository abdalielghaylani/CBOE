Displaying Registration Fields and Labels

After creating views, you need to define comment for each Registration field that is to made accessible from Inventory Enterprise. This exposes the Registration fields in the Inventory Enterprise interface. Comments are defined in the following format:

 [Interface Display Name]::[state]

In this format, Interface Display Name refers to the user friendly name for the field in the Inventory Enterprise interface and state specifies whether or not the field is to be exposed. The state parameter can take two values, off and on, and it exposes the field in the interface when it is set to on.

Example

The following comments and display settings expose the standard Registration fields to Inventory Enterprise:  

COMMENT ON COLUMN inv_vw_reg_batches.regnumber IS 'Reg Number::on'
COMMENT ON COLUMN inv_vw_reg_batches.regid IS 'Reg ID::on'
COMMENT ON COLUMN inv_vw_reg_batches.regcas IS 'Reg CAS::on'
COMMENT ON COLUMN inv_vw_reg_batches.regname IS 'Reg Name::on'
COMMENT ON COLUMN inv_vw_reg_batches.batchnumber IS 'RegBatch Number::on'
COMMENT ON COLUMN inv_vw_reg_batches.regbatchid IS 'Reg Batch ID::on'
COMMENT ON COLUMN inv_vw_reg_batches.regpage IS 'Reg Page::on'
COMMENT ON COLUMN inv_vw_reg_batches.regpurity IS 'Reg Purity::on'
COMMENT ON COLUMN inv_vw_reg_batches.regscientist IS 'Reg Scientist::on'
COMMENT ON COLUMN inv_vw_reg_batches.regamount IS 'Reg Amount::on'
COMMENT ON COLUMN inv_vw_reg_batches.cpdid IS 'Reg Compound ID::on'
COMMENT ON COLUMN inv_vw_reg_batches.regnotebook IS 'Reg Notebook::on'
COMMENT ON COLUMN inv_vw_reg_structures.regbase64 IS 'Reg Structure::on'
COMMENT ON COLUMN inv_vw_reg_structures.regid IS 'Reg Structure::on'
