INV_VW_REG_STRUCTURES

INV_VW_REG_STRUCTURES is a materialized view, which defines structure data for the Registration substances. This view is created within the CHEMINVDB2 schema, and is updated on the basis of the LastUpdated field on a defined schedule.

Structure data is stored in the MOL format and is maintained by ABC. However, when the INV_VW_REG_STRUCTURES view is created and updated, the structure data is converted from the MOL to CDX format  After the structure data is converted from MOL to CDX format, you can reference it the main view, INV_VW_REG_BATCHES, using the unique identifier for the substance.

Field Name Type Description
MolData CLob Specifies structure data, which is stored in the MOL format.
RegID Varchar Specifies unique identifier for the structure of the substance.
AdditionalField1 Varchar Specifies value of AdditionalField1.
AdditionalField2 Varchar Specifies value of AdditionalField2.
AdditionalField3 Varchar Specifies value of AdditionalField3.
AdditionalFieldN Varchar Specifies value of AdditionalFieldN.
LastUpdated DateTime Specifies the timestamp for the last updation or creation of INV_VW_REG_STRUCTURES.

In addition to the preceding fields, an unlimited number of other fields can be added to the INV_VW_REG_STRUCTURES view and can be viewed and/or searched directly from the Inventory Enterprise interface.
