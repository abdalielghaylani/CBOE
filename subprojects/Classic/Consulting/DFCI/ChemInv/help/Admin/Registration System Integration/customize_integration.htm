Customizing Registration Enterprise Integration

By customizing the integration of Registration Enterprise with Inventory Enterprise, you can access additional Registration fields also in addition to the standard Registration fields, from Inventory Enterprise. In order to customize the integration, you need to add new fields to a view and define comments for the new fields.

Example

The following example illustrates the creation of the INV_VW_REG_BATCHES view with 3 additional custom fields, CustomField1, CustomField2, and CustomField3:

CREATE OR REPLACE VIEW INV_VW_REG_BATCHES AS
SELECT 
rt.field1 AS RegNumber
, rt.field2 AS RegID
, rt.field3 AS RegBatchID
, rt.field4 AS RegScientist
, ot.field1 AS CustomField1
, ot.field2 AS CustomField2
, ot.field3 AS CustomField3
FROM reg_table rt, other_table ot
WHERE rt.key = ot.key

COMMENT ON COLUMN inv_vw_reg_batches.RegNumber IS 'Reg Number::on'
COMMENT ON COLUMN inv_vw_reg_batches.RegID IS 'Batch Number::on'
COMMENT ON COLUMN inv_vw_reg_batches.RegBatchID IS 'Reg BatchID::on'
COMMENT ON COLUMN inv_vw_reg_batches.RegScientist IS 'Reg Scientist::off'
COMMENT ON COLUMN inv_vw_reg_batches.CustomField1 IS 'Custom Field A::on'
COMMENT ON COLUMN inv_vw_reg_batches.CustomField2 IS 'Custom Field B::on'
COMMENT ON COLUMN inv_vw_reg_batches.CustomField3 IS 'Custom Field C::on'
