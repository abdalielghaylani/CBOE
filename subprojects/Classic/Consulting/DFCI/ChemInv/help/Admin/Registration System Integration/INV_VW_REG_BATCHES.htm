INV_VW_REG_BATCHES

This view specifies the Registration data that can be viewed and/or searched directly from the Inventory Enterprise interface. The following table lists the standard Registration data that can be accessed directly from Inventory Enterprise on integrating Registration Enterprise with Inventory Enterprise:

Field Name Type Description
RegID Number Specifies a unique internal identifier for a substance.
RegNumber Varchar Specifies registration number of a substance.
BatchNumber Number Specifies batch number of a substance.
RegBatchID Varchar Specifies a combination of Registration and Batch ID.
RegPage Varchar Specifies the page reference of a substance.
RegPurity Varchar Specifies purity of a substance.
RegNoteBook Varchar Specifies the notebook reference of a substance.
RegScientist Varchar Specifies the scientist associated with a substance.
RegName Varchar Specifies the name of a substance.
RegCas Varchar Specifies CAS Number of a substance.
RegAmount Varchar Specifies the amount of a substance. It also includes unit of measurement for the amount.
cpdID Number Specifies FK to structures view.

Example

The following code allows you to create the INV_VW_REG_BATCHES view:

CREATE OR REPLACE VIEW INV_VW_REG_BATCHES AS
SELECT 
       field1 AS RegNumber
       , field2 AS RegID
       , field3 as RegBatchID
       , field4 AS RegScientist
FROM reg_table
