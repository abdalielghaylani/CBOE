CHEM_CONNECTION_NAMES

Example: CHEM_CONNECTION_NAMES = cheminv_cfw_form

Description: Names of all ChemFinder connection definition sections in the ini file. This list must be comma delimited with no spaces. If the connection name is not here, it will not be loaded by the application. A ChemFinder connection definition section describes variables for connecting to ChemFinder for searching of chemical information. The default definition "base_cfw_form" is normally present (unless there is no chemical information). In dataviews that deal with reaction databases, there may be three (or more) definitions, one for the reaction, one for the solvent and one for the catalyst. (e.g.config/chemprep.ini file in the chemrxn application).

Default: cheminv_cfw_form

Options: Comma delimited list of ChemFinder connection definitions found in the ini file.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
