Testing the Installation

To ensure that Inventory Enterprise is installed successfully, perform the following steps:

  1. Login in to ChemOffice Enterprise using the following username and password:
  • Username: invadmin
  • Password: invadmin
  1. Add a compound to Inventory Enterprise.

Note: For information about adding compounds to Inventory Enterprise, please see Substance ManagementàCreating a New Substance in the Inventory Enterprise User's Guide.

  1. Add a container to Inventory Enterprise.

Note: For information about adding containers to Inventory Enterprise, please see Container ManagementàCreating a New Container in the Inventory Enterprise User's Guide.

  1. Add a ChemACX compound to an Inventory container.

Note: For information about adding a ChemACX compound to an Inventory container, please see Container ManagementàCreating a New ContaineràSubstance Tab in the Inventory Enterprise User's Guide.

  1. Add a Registration compound to an Inventory container.

Note: For information about adding a Registration compound to an Inventory container, please see Container ManagementàCreating a New ContaineràSubstance Tab in the Inventory Enterprise User's Guide.

  1. Generate a barcode for a container.

Note: For information about generating barcode, please see Container ManagementàViewing the Contents of a ContaineràContainer Details Frame FunctionsàPrint Label in the Inventory Enterprise User's Guide.

Note:  The Inventory Enterprise User's Guide can be accessed from the following location:
http:// <IIS_SERVER_NAME>/ChemOffice/cheminv/help/, where IIS_SERVER_NAME is the name of the server on which IIS is installed.
