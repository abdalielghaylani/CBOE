Installing Inventory Enterprise

To install Inventory Enterprise:

  1. On the installer CD, double-click ChemInv_9->setup.exe to run the installation wizard.
  2. Read and accept the License terms and conditions.
  3. Step through the rest of the wizard to complete the installation of Inventory Enterprise.
