Creating Inventory Enterprise Database

Note: If you have already set up the Inventory Enterprise database or schema, leave this step and move on to Step 4, Making Required Configurations.

The files necessary to set up the Inventory Enterprise database can be installed directly from the Inventory Enterprise installation wizard. However, if you choose not to set up the Inventory Enterprise database at that time, you can do so by performing the following steps:

  1. Browse to <webroot> /cheminv/config/oracle_install_scripts/.
  2. Double-click the OracleSetup.exe file. This file allows you to set up the Inventory Enterprise database. If you already have Inventory Enterprise on your machine, then you only need to upgrade the database instead of creating the new database.
