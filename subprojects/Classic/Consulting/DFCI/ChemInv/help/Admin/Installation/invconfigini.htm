Invconfig.ini Configuration File

The Invconfig.ini file, found in the following location: <webroot>/ChemOffice/cheminv/config
defines many parameters that allow Inventory Enterprise to function properly, including the format of custom fields. Most of this file should not be edited since editing this file in the wrong place could cause major problems when running the Inventory Manager application.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
