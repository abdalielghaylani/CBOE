Mapping invconfig.ini Files

In the SR3 version of Inventory Enterprise, several parameters of the invconfig.ini file have been moved to different sections. In addition, some of the parameters have been renamed. Therefore, the section of a particular parameter in the invconfig.ini file of Inventory Enterprise SR3 might be different from the section of that parameter in the invconfig.ini file of Inventory Enterprise SR2. The following tables list the various parameters in the invconfig.ini file of Inventory Enterprise SR2 and provide information regarding their sections and new names (if applicable):

Note: In the invconfig.ini file of Inventory Enterprise SR2, there are various custom fields in the end. In the Inventory Enterprise SR3 version, the custom fields are listed under the section to which they belong. For example, the following custom field appears in the CONTAINERS section:

////////////////////////////////////////////////////////////////

//Custom field format: Field Name:Field Label;Required(0|1)

// Valid custom fields are FIELD_x where x=1-10 and DATE_x where x=1-5

// Example:

// CUSTOM_FIELDS =FIELD_1:Custom Field 1;0,FIELD_2:Custom Field 2;0,FIELD_3:Custom Field 3;0,FIELD_4:Custom Field 4;0,FIELD_5:Custom Field 5;0,FIELD_6:Custom Field 6;0,FIELD_7:Custom Field 7;0,FIELD_8:Custom Field 8;0,FIELD_9:Custom Field 9;0,FIELD_10:Custom Field 10;0,DATE_1:Date Field 1;0,DATE_2:Date Field 2;0,DATE_3:Date Field 3;0,DATE_4:Date Field 4;0,DATE_5:Date Field 5;0

CUSTOM_FIELDS

 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
