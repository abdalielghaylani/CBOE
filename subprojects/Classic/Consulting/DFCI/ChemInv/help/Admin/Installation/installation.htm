Installation of Inventory Enterprise

You need to perform the following steps for installing Inventory Enterprise:

  1. Identify minimum software requirements for Inventory Enterprise.
  2. Install Inventory Enterprise.
  3. Create the Inventory Enterprise database.
  4. Make the required configurations.
  5. Integrate Inventory Enterprise with other ChemOffice applications, such as Registration Enterprise, ChemACX, and DocManager Enterprise.
  6. Enable barcode and PDF support.
  7. Test the installation.
  8. Install Inventory Loader.

Note: Installation of InvLoader is an optional step in installation of Inventory Enterprise.
