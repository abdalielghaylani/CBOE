Enabling PDF and Barcode Support

PDF Support

The RPT Report Writer installed with Inventory Enterprise can generate reports in four different formats, which are: MS Access Report Viewer (SNP), Rich Text Format (RTF), Excel Workbook (XLS), and Portable Document Format (PDF).

Support for the PDF format is optional and requires a PDF printer driver to be installed on the server. To enable the PDF support:

  1. Browse to <webroot>\ChemOffice\ ChemInv\Installation\win2pdf.
  2. Install the win2pdf driver by double-clicking w2pdf151.exe.
  3. Select StartàSettingsàControl PanelàPrinters.
  4. Check that the win2pdf printer is the default Windows printer. If not, perform the following steps to set the win2pdf printer as the default Windows printer:
    1.  Right-click the win2pdf printer.
    2. Select Set As Default Printer.
    3. Purchase a license and registration code from: http://www.daneprairie.com/
    4. Right-click the win2pdf printer and select Properties.
    5. Enter the regcode you purchased.
  5. Browse to <webroot>/ChemOffice/WebServer_source/cfserverasp/RPT.
  6. Stop the Reports Service (RPTService).
  7. Open the ChemInv_reports.mdb file, which is available in the ChemOffice_data/Cheminv folder.
  8. Open the ReportFormats table and check the Available checkbox for the PDF format.
  9. Close Microsoft Access.
  10. Restart the Reports Service.

Barcode Support

To enable the barcode support:

  1. Browse to <webroot>\ChemOffice\ ChemInv\Installation\Tbarcode.
  2. Run the Tbarcode.ocx installer.
  3. Perform an IISRESET after finishing the Inventory Enterprise installation.
