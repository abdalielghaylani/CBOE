Installing the Chemical Inventory Application

  1. Run setup.exe from the ChemInv installer
    • Read and accept the License Terms and Conditions
    • Select the installation path for the application and database file.  (Note: it is recommended to keep the default paths when possible) 
  2. chemInv.ini settings
    • Verify that the STRUC_DB_PATH entry agrees with the database path chosen in step 1 above.
    • Verify that the CONNECTION_STRING entry agrees with the application path chosen in step 1.
    • The INV_SERVER_NAME should point to Server hosting ChemInvs databse
  3. cfserver.ini settings
    • Set the ORA_SERVICENAME entry to the Oracle service name of your target server 
  4. invconfig.ini settings
    • Verify that the RPT_PATH entry agrees with the ChemOffice WebServer installation path
    • Verify that the REPORT_DB_PATH entry agrees with the database path chosen in step 1 above.
  5. cheminv.udl
    • the file cheminv/config/ChemInv.udl should point to the Oracle service hosting the CHEMINVDB2 schema 

To enable ChemACX Integration:

  • cheminv/config/ChemACX.udl should point to the location of the ChemACX.mdb file
  • The ACX_SERVER_NAME entry in cheminv/config/invconfig.ini should point to the Server hosting ChemACX
    Note: If set to NULL ChemACX integration will be disabled. 
  • In chemacx/config/acxconfig.ini set EXPORT_TO_CHEMINV="True"

To enable ChemReg Integration:

  • cheminv/config/ChemReg.udl should point to the Oracle service hosting the REGDB schema 
  • The REG_SERVER_NAME entry in cheminv/config/invconfig.ini should point to the Server hosting ChemReg
    Note: If set to NULL ChemReg integration will be disabled. 
  • In cheminv/config/cfserver.ini add CHEMREG_PRIVILEGES to the PRIVILEGE_TABLE_LIST entry.
    1. Open chem_reg/config/Cfserver.ini
    2. Change POST_MARKED_SEND_TO_PAGE=NULL to:
      POST_MARKED_SEND_TO_PAGE= http://YOUR_SERVER/cheminv/gui/ImportFromChemReg.asp
      (where YOUR_SERVER is the server name that is hosting Chemical Inventory2)
      You will now see an additional button appear when you mark hitsfor registered compounds.< /LI>
