EmptyTrash

Deletes the contents of the Trash Location.

Syntax

/cheminv/api/EmptyTrash.asp?LocationID=adInteger&TrashType=adVarChar

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adInteger Specifies the ID of the Trash location.   Y INV_LOCATIONS LOCATION_ID
TrashType adVarChar Specifies the type of trash location. Containers | Plates Y N/A N/A

Output

None.

Example

The following code deletes all the plates stored in the Trash location:

/cheminv/api/EmptyTrash.asp?LocationID=3&TrashType=Plates
