UpdatePlateType

Updates a plate type.

Syntax

/cheminv/api/UpdatePlateType.asp?PlateTypeID=adNumeric&PlateTypeName=adVarChar&MaxFreezeThaw=adNumeric

Input 

Parameter Name Data Type Description Options Required? Table Name Database Field Name
PlateTypeID adNumeric Specifies the ID of the plate type to be updated.   Y INV_PLATE_TYPES PLATE_TYPE_ID
PlateTypeName adVarChar Specifies the name of the plate type.   Y INV_PLATE_TYPES PLATE_TYPE_NAME
MaxFreezeThaw adNumeric Specifies the maximum number of freeze thaw cycles allowed for the plate type.   Y INV_PLATE_TYPES MAX_FREEZE_THAW

Output

The ID of the updated plate type, which is stored in the PlatetypeID parameter.

Example

The following code updates a plate type with ID 4:

/cheminv/api/UpdatePlateType.asp?PlateTypeID=4&PlateTypeName=NewPlateType&MaxFreezeThaw=5
