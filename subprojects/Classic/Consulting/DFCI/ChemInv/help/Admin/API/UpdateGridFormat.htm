UpdateGridFormat

Updates a grid or storage format.

Syntax

/cheminv/api/UpdateGridFormat.asp?grid_format_id=adNumeric&name=adVarChar&row_count=adNumeric&col_count=adNumeric&zero_padding_count=adNumeric[&row_prefix=adVarChar&col_prefix=adVarChar&row_use_letters=adNumeric&col_use_letters=adNumeric&name_separator=adVarChar&number_start_corner=adNumeric&number_direction=adNumeric&description=adVarChar&grid_format_type=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
Grid_format_id adNumeric Specifies the ID of the grid to be updated.   Y INV_GRID_FORMAT GRID_FORMAT_ID
grid_format_type adNumeric Specifies the type of the grid format.   N INV_GRID_FORMAT GRID_FORMAT_TYPE_FK
row_count adNumeric Specifies the number of rows that the grid should contain.   Y INV_GRID_FORMAT ROW_COUNT
col_count adNumeric Specifies the number of columns that the grid should contain.   Y INV_GRID_FORMAT COL_COUNT
row_prefix adVarChar Specifies the prefix to be used for the grid rows.   N INV_GRID_FORMAT ROW_PREFIX
col_prefix adVarChar Specifies the prefix to be used for the grid columns.   N INV_GRID_FORMAT COL_PREFIX
row_use_letters adNumeric Specifies whether to label rows with letters or with numbers.   N INV_GRID_FORMAT ROW_USE_LETTERS
col_use_letters adNumeric Specifies whether to label columns with letters or with numbers.   N INV_GRID_FORMAT COL_USE_LETTERS
name_separator adVarChar Specifies the character that separates the row and column IDs. For example, '-' is the name separator for the following column and row IDs: A-1 and B-1.   N INV_GRID_FORMAT NAME_SEPARATOR
number_start_corner adNumeric Specifies the value with which the plate numbering is to be started.   N INV_GRID_FORMAT NUMBER_START_CORNER
number_direction adNumeric Specifies whether to fill the rows first or to fill the columns first.   N INV_GRID_FORMAT NUMBER_DIRECTION
name adVarChar Specifies the name of the grid format.   Y INV_GRID_FORMAT NAME
description adVarChar Specifies the description of the grid format.   N INV_GRID_FORMAT DESCRIPTION
zero_padding_count adNumeric Specifies the number of digits with which the column and row numbers are to be padded. Usually, this parameter is set to 2.   Y INV_GRID_FORMAT ZERO_PADDING_COUNT

Output

The ID of the updated grid format, which is stored in the new_grid_format_id parameter. The data type of the new_grid_format_id parameter is adNumeric.

Example

The following code updates a grid format with ID 14: 

/cheminv/api/UpdateGridFormat.asp?grid_format_id=14&name=TestGrid&row_count=12&col_count=8&zero_padding_count=1[&col_prefix=C]
