SolvatePlate

Solvates a plate(s) with the specified type and amount of solvent.

Syntax

/cheminv/api/SolvatePlate.asp?PlateIDList=adVarChar&SolventIDList=adVarChar&SolventVolumeList=adVarChar&SolventVolumeUnitIDList=adVarChar&ConcentrationList=adVarChar&ConcentrationUnitIDList=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
PlateIDList adVarChar Specifies a comma delimited list of the IDs of the plates to be solvated.   Y INV_PLATES PLATE_ID
SolventIDList adVarChar Specifies a comma delimited list of IDs of the solvents to be used for solvating plates.   Y INV_PLATES SOLVENT_ID_FK
SolventVolumeList adVarChar Specifies a comma delimited list of the volumes of the solvents.   Y INV_PLATES SOLVENT_VOLUME
SolventVolumeUnitIDList adVarChar Specifies a comma delimited list of the IDs of the units in which the solvent volumes are to be measured.   Y INV_PLATES SOLVENT_VOLUME_UNIT_ID_FK
ConcentrationList adVarChar Specifies a comma delimited list of the concentrations of the plates.   Y INV_PLATES CONCENTRATION
ConcentrationUnitIDList adVarChar Specifies a comma delimited list of the IDs of the units in which the concentrations are to be measured.   Y INV_PLATES CONC_UNIT_FK

Output

None.

Example

The following code solvates the P1234 plate with a solvent with ID 120:

/cheminv/api/SolvatePlate.asp?PlateIDList=P1234&SolventIDList=120&SolventVolumeList=10&SolventVolumeUnitIDList=4&ConcentrationList=.1&ConcentrationUnitIDList=4
