MergePlates

Merges the contents of two plates into one. 

Note: Plates can be merged only if there are childern of the same parent plate.

Syntax

 /cheminv/api/MergePlates.asp?MergePlateIDs=adVarChar&PlateTypeID= Int&NumPlates= Int

Inputs

MergePlateIDs

PlateIDs for the plates to be merged.  A list of IDs should be separated with commas.

PlateTypeID

PlateTypeID for the destination plate.

NumPlates

The number of plates to be merged.           

Outputs

None

Example

A user wants wto merge two plates with IDs 12345 and 12346. 

 /cheminv/api/MergePlates.asp?MergePlateIDs=12345,12346&PlateTypeID= 4&NumPlates= 2
