RemoveOrderContainers

Removes containers from an order but does not remove the order itself. Each container's status is reverted to its previous value prior to being added to the order, and the container's location is left unchanged.

Syntax

/cheminv/api/RemoveOrderContainers.asp?OrderID=adNumeric&RemovedContainerIDs=adVarChar

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
OrderID adNumeric Specifies the ID of the order to be received.   Y INV_ORDERS ORDER_ID
RemovedContainerIDs adVarChar Specifies a comma delimited list of containers that are to be removed from the order.   Y INV_ORDER_CONTAINERS CONTAINER_ID_FK

Output

Comma-delimited list of containers removed from the order.

Example

The following code removes containers 1006 and 1007 from order with ID 21, and returns those containers to their original status: :

/cheminv/api/RemoveOrderContainers.asp?OrderID=21&RemovedContainerIDs=1006,1007
