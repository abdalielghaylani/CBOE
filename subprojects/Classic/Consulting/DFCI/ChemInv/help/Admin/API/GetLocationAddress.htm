GetLocationAddress

Retrieves the address attributes for a given location.

Syntax

/cheminv/api/GetLocationAddress.asp?LocationID=adNumeric

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the location, whose address attributes are to be retrieved.   Y INV_LOCATIONS LOCATION_ID

Output

Parameter Name Data Type Description
ADDRESS_ID adNumeric Specifies the ID of the address of the location.
CONTACT_NAME adVarChar Specifies the contact name.
ADDRESS1 adVarChar Specifies the first line of the address.
ADDRESS2 adVarChar Specifies the second line of the address.
ADDRESS3 adVarChar Specifies the third line of the address.
ADDRESS4 adVarChar Specifies the fourth line of the address.
CITY adVarChar Specifies the name of the city.
STATE_ID_FK adNumeric Specifies the ID of the state.
COUNTRY_ID_FK adNumeric Specifies the ID for the country.
ZIP adVarChar Specifies the zip code.
FAX adVarChar Specifies the fax number.
PHONE adVarChar Specifies the phone number.
EMAIL adVarChar Specifies the email address.
State_Name adVarChar Specifies the name of the state.
State_Abbreviation adVarChar Specifies the abbreviation for the state.
Country_Name adVarChar Specifies the name of the country.

Example

The following code retrieves attributes of the location with ID 1013:

/cheminv/api/GetLocationAddress.asp?LocationID=1013
