CreateDataMap

Creates a data map and returns the ID of the newly created data map.

Syntax

/cheminv/api/CreateDataMap.asp?DataMapTypeID=adNumeric&DataMapName=adVarChar&DataMapComments=adVarChar&NumColumns=adNumeric&ColumnDelimiter=adVarChar[&NumHeaderRows=adNumeric&DataMapFieldList=adVarChar&DataMapColumnList=adVarChar&UseWellCoordinates=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DataMapTypeID adNumeric Specifies the ID of the type of the data map.   Y INV_DATA_MAPS DATA_MAP_TYPE_ID_FK
DataMapName adVarChar Specifies the name of the data map.   Y INV_DATA_MAPS DATA_MAP_NAME
DataMapComments adVarChar Specifies the comments associated with the data map.   Y INV_DATA_MAPS DATA_MAP_COMMENTS
NumHeaderRows adNumeric Specifies the number of header rows the data map should contain.   N INV_DATA_MAPS NUM_HEADER_ROWS
NumColumns adNumeric Specifies the number of columns the data map should contain.   Y INV_DATA_MAPS NUM_COLUMNS
ColumnDelimiter adVarChar Specifies the column delimiter to be used for the data map.   Y INV_DATA_MAPS COLUMN_DELIMITER
DataMapFieldList adVarChar Specifies the field list for the data map.   N N/A N/A
DataMapColumnList adVarChar Specifies the column list for the data map.   N N/A N/A
UseWellCoordinates adNumeric     N INV_DATA_MAPS USE_WELL_COORDINATES

Output

The ID of the newly created data map, which is stored in the DataMapID parameter. The data type of the DataMapID parameter is adNumeric.

Example

The following code creates a data map, with type ID 19, name TestTemplate, comments creating plates, number of columns 4, and column delimiter tab:

/cheminv/api/CreateDataMap.asp?DataMapTypeID=19&DataMapName=TestTemplate&DataMapComments=Creating Plates&NumColumns=4&ColumnDelimiter=tab
