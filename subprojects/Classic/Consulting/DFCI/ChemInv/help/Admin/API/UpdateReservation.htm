UpdateReservation

Updates a container reservation.

Syntax

/cheminv/api/UpdateReservation.asp?ReservationID=adNumeric&ContainerID=adNumeric&ReservationUserID=adVarChar&QtyReserved=Int[&ReservationTypeID=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ReservationID adNumeric Specifies the ID of the reservation to be updated.   Y INV_RESERVATIONS RESERVATION_ID
ContainerID adNumeric Specifies the ID of the container the contents of which are to be reserved.   Y INV_RESERVATIONS CONTAINER_ID_FK
ReservationUserID adVarChar Specifies the ID of the user who wants to reserve the container contents.   Y INV_RESERVATIONS USER_ID_FK
QtyReserved adNumeric Specifies the amount of the quantity to be reserved.   Y INV_RESERVATIONS QTY_RESERVED
ReservationTypeID adNumeric Specifies the ID of the type of reservation.   N INV_RESERVATIONS RESERVATION_TYPE_ID_FK

Output

The ID of the updated reservation, which is stored in the ReservationID parameter.

Example

The following code updates a reservation with ID 42:

/cheminv/api/UpdateReservation.asp?ReservationID=42&ContainerID=1742&ReservationUserID=INVADMIN&QtyReserved=.5
