UpdateContainer2

Updates the specified attributes of the container(s).

Syntax

/cheminv/api/UpdateContainer2.asp?ContainerIDs=adVarChar&ValuePairs=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerIDs adVarChar Specifies a “::” list of the IDs of the containers to be updated.   Y INV_CONTAINERS CONTAINER_ID
ValuePairs adVarChar Specifies a comma delimited list of the pairs of the name and value of the parameters to be updated.

For example: colnam1%3Dval1,colnam2%3Dval2 

   Y N/A N/A

Output

None.

Example

The following code updates the qty_remaining and container_description fields of the containers with IDs 212 and 213:

Updatecontainer2.asp?ContainerIDs=212,213&ValuePairs=qty_remaining%3D5,container_description%3Dupdated
