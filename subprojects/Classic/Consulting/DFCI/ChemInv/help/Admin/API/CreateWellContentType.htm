CreateWellContentType

Creates a new well content type or well format. 

Note: Only those wells that are assigned the Compound well format can contain a compound. 

Syntax

/cheminv/api/CreateWellContentType.asp?WellFormatName=adVarChar

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
WellFormatName adVarChar Specifies the name of the well format.   Y INV_ENUMERATION ENUM_VALUE

Output

The ID of the new well format. This ID is stored in the newWellFormatID parameter, the data type of which is adVarChar.

Example

The following code creates a new well format with name NewFormat:

/cheminv/api/CreateWellContentType.asp?WellFormatName=NewFormat
