DeleteLocations

Deletes multiple locations. You can delete a location only if it is empty. The location can be emptied by deleting the containers, plates, and sublocations from it using the DeleteContainer, DeletePlate, or DeleteLocation APIs, respectively.

Syntax

/cheminv/api/DeleteLocations.asp?LocationList=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationList List of adNumeric Specifies a comma delimited list or the range of the IDs of the locations that are to be deleted. You need to use dash(-) to define the range of the location IDs.   Y INV_LOCATIONS LOCATION_ID

Output

None.

Example

The following code deletes two locations with IDs 111 and 112:

/cheminv/api/DeleteLocations.asp?LocationList=111,112
