AllotContainer

Samples or splits an existing container into new containers. It also allows you to calculate the amount of substance that is to be left in the container after it is sampled, and specified amount of its contents are transferred to the new containers. The value of the Action parameter specifies whether to sample or split the container.

Syntax

/cheminv/api/AllotContainer.asp?ContainerID=adNumeric&NumContainers=adInteger&LocationID=adNumeric&ContainerTypeID=adNumeric&Action=adVarChar&ContainerSize=adNumeric[&QtyRemaining=adNumeric&Split#=adNumeric&SampleQtyUnit=adNumeric&BarcodeDescID=adVarChar&DateCertified=adDBTimeStamp&ContainerStatusID=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerID adNumeric Specifies the ID of the container to be splitted or sampled.   Y INV_CONTAINERS CONTAINER_ID
NumContainers adInteger Specifies the number of the new containers to be created after sampling or splitting the container.   Y N/A N/A
LocationID adNumeric Specifies the location where the newly created containers are to be stored.   Y INV_CONTAINERS LOCATION_ID_FK
ContainerTypeID adNumeric Specifies the type of the new containers.   Y INV_CONTAINERS CONTAINER_TYPE_ID_FK
Action adVarChar Specifies the action to be taken on the container. The Action parameter can take two values, split and sample. If you set the Action parameter to split, all the contents of the original container are transferred to the new containers and the original container is discarded. However, if you set the Action parameter to sample, some of the contents remain there in the container and it is not discarded. split | sample Y N/A N/A
ContainerSize adVarChar Specifies the size of the containers to be created after sampling or splitting the container.   Y INV_CONTAINERS QTY_MAX
QtyRemaining adNumeric Specifies the quantity of the substance to be left in the original container.   if Action = sample INV_CONTAINERS QTY_REMAINING
Split# adNumeric Specifies the quantity of the substance to be allotted to the new containers. A separate Split parameter is associated with each new container. For example, if a container is to be splitted into three containers, the call to AllotContainer should contain three Split parameters, Split1, Split2, and Split3   if Action = split INV_CONTAINERS QTY_INITIAL
SampleQtyUnit adNumeric Specifies the unit in which the quantity of the contents of the new containers is to be measured.   N INV_CONTAINERS UNIT_OF_MEAS_ID_FK
BarcodeDescID adVarChar Specified the barcode description associated with the new containers.   N INV_BARCODE_DESC BARCODE_DESC_ID
DateCertified adDBTimeStamp Specifies the certification date of the new containers.   N INV_CONTAINERS DATE_CERTIFIED
ContainerStatusID adNumeric Specifies the status for the new containers.   N INV_CONTAINERS CONTAINER_STATUS_ID_FK

Output

A Pipe (|) delimited list of the IDs of the new containers, which are stored in the ContainerID parameter. For example: 100|101.

Example

The following code splits a container with ID 111  into 4 containers of size 100 and quantity 50:

/cheminv/api/AllotContainer.asp?ContainerID=111&NumContainers=4&LocationID=1014&ContainerTypeID=2&Action=split&ContainerSize=100[&split1=50&split2=50&split3=50&split4=50]
