ApproveSampleRequest

Approves or declines a sample request, as specified. If the request is approved, the status of the sample request is set to Approved, otherwise Declined.

Syntax

/cheminv/api/ApproveSampleRequest.asp?ApprovedRequestIDList=adVarChar&DeclinedRequestIDList=adVarChar[&DeclineReasonList=adVarChar]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ApprovedRequestIDList adVarChar Specifies a comma delimited list of IDs of the requests to be approved.  

Y, if DeclinedRequestIDList is empty

 INV_REQUESTS REQUEST_ID
DeclinedRequestIDList adVarChar Specifies a comma delimited list of the IDs of the requests to be declined.  

Y, if ApprovedRequestIDList is empty

 INV_REQUESTS REQUEST_ID
DeclineReasonList adVarChar Specifies a comma delimited list of the reasons for declining requests.   N INV_REQUESTS DECLINE_REASON

Output

None.

Example

The following code approves the sample request with IDs 69, 70, and 43:

/cheminv/api/ApproveSampleRequest.asp?ApprovedRequestIDList=69,70,43
