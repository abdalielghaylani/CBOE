DeliverRequest

Delivers the requested container(s) to the specified location(s). When the container is delivered, its status and location are modified.

Syntax

/cheminv/api/DeliverRequest.asp?DeliveredRequestIDList=adVarChar

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DeliveredRequestIDList adVarChar Specifies a comma delimited list of the IDs of the requests to be delivered.   Y INV_REQUESTS REQUEST_ID

Output

The number of requests delivered, which is stored in the rowCount parameter. The data type of the rowCount parameter is adInteger.

Example

The following code delivers the requests with IDs 23, 26, and 27:

/cheminv/api/DeliverRequest.asp?DeliveredRequestIDList=23,26,27
