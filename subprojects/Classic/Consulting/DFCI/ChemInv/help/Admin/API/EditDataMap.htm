EditDataMap

Updates a data map, as specified.

Syntax

/cheminv/api/EditDataMap.asp?DataMapTypeID=adNumeric&DataMapID=adNumeric&DataMapName=adVarChar&DataMapComments=adVarChar&NumColumns=adInteger&ColumnDelimiter=adVarChar[&NumHeaderRows=adInteger&DataMapFieldList=adVarChar&DataMapColumnList=adVarChar&UseWellCoordinates=adInteger]

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DataMapTypeID adNumeric Specifies the ID of the type of the data map.   Y INV_DATA_MAPS DATA_MAP_TYPE_ID_FK
DataMapID adNumeric Specifies the ID of the data map.   Y INV_DATA_MAPS DATA_MAP_ID
DataMapName adVarChar Specifies the name of the data map.   Y INV_DATA_MAPS DATA_MAP_NAME
DataMapComments adVarChar Specifies the comments associated with the data map.   Y INV_DATA_MAPS DATA_MAP_COMMENTS
NumHeaderRows adInteger Specifies the number of header rows contained in the data map.   N INV_DATA_MAPS NUM_HEADER_ROWS
NumColumns adInteger Specifies the number of columns contained in the data map.   Y INV_DATA_MAPS NUM_COLUMNS
ColumnDelimiter adVarChar Specifies the column delimiter used for the data map.   Y INV_DATA_MAPS COLUMN_DELIMITER
DataMapFieldList adVarChar Specifies the field list for the data map.   N N/A N/A
DataMapColumnList adVarChar Specifies the column list for the data map.   N N/A N/A
UseWellCoordinates adInteger     N INV_DATA_MAPS USE_WELL_COORDINATES

Output

The ID of the edited data map, which is stored in the DataMapID parameter.

Example

The following code edits data map with ID 20:

/cheminv/api/EditDataMap.asp?DataMapTypeID=19&DataMapID=20&DataMapName=TestTemplate&DataMapComments=Creating Plates&NumColumns=8&ColumnDelimiter=tab
