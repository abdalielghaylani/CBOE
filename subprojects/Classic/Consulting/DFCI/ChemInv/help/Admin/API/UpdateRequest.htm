UpdateRequest

Updates a request.

Syntax

/cheminv/api/UpdateRequest.asp?RequestID=adNumeric&UserID=adVarChar&QtyRequired=adInteger&LocationID=adNumeric&ContainerTypeID=adNumeric&NumContainers=adNumeric&QtyList=adVarChar[&DateRequired=Date&RequestComments=adVarChar&RequestTypeID=adVarChar&ShipToName=adVarChar&ExpenseCenter=adVarChar]

Inputs 

Parameter Name Data Type Description Options Required? Table Name Database Field Name
RequestID adNumeric Specifies the ID of the request to be updated.   Y INV_REQUESTS REQUEST_ID
UserID adVarChar Specifies the ID of the user who has requested the container.   Y INV_REQUESTS USER_ID_FK
QtyRequired adInteger Specifies the quantity of the substance the requested container should contain.   Y INV_REQUESTS QTY_REQUIRED
LocationID adNumeric Specifies the ID for the location where the request is to be delivered.   Y INV_REQUESTS DELIVERY_LOCATION_ID_FK
DateRequired Date Specifies the date on which the requested container is to be delivered. The date is specified in the date-time literal format. For example, 01-27-2006.   N INV_REQUESTS DATE_REQUIRED
RequestComments adVarChar Specifies the comments associated with the request.   N INV_REQUESTS REQUEST_COMMENTS
RequestTypeID adNumeric Specifies the type of the request.   N INV_REQUESTS REQUEST_TYPE_ID_FK
ContainerTypeID adNumeric Specifies the type of the sample containers.   Y INV_REQUESTS CONTAINER_TYPE_ID_FK
NumContainers adNumeric Specifies the number of samples requested.   Y INV_REQUESTS NUMBER_CONTAINERS
QtyList adVarChar Specifies the list of quantities that each sample container should contain.   Y INV_REQUESTS QUANTITY_LIST
ShipToName adVarChar Specifies the user to whom the request is to be shipped.   N INV_REQUESTS SHIP_TO_NAME
ExpenseCenter adVarChar Specifies the Expense Center for the sample request.   N INV_REQUESTS EXPENSE_CENTER

Output

The ID of the updated request, which is stored in the RequestID parameter.

Example

The following code updates a request with ID 121:

/cheminv/api/UpdateRequest.asp?RequestID=121&UserID=mjs&QtyRequired=10&LocationID=1123&ContainerTypeID=1&NumContainers=2&QtyList=10,10
