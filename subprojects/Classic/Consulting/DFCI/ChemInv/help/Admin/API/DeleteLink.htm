DeleteLink

Deletes the specified link.   

Syntax

/cheminv/api/DeleteLink.asp?URLID=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
URLID adNumeric Specifies the ID of the link to be deleted.   Y INV_URL URL_ID

Output

The ID of the deleted link, which is stored in the URLID parameter.

Example

The following code deletes a link with ID 1: 

/cheminv/api/DeleteLink.asp?URLID=1
