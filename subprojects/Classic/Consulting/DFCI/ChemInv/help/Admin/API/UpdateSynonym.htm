UpdateSynonym

Updates a synonym of a substance.

Syntax

/cheminv/api/UpdateSynonym.asp?SynonymID=adNumeric&SynonymName=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
SynonymID adNumeric Specifies the ID of the synonym to be updated.   Y INV_SYNONYMS SYNONYM_ID
SynonymName adVarChar Specifies the updated synonym of the substance.   Y INV_SYNONYMS SUBSTANCE_NAME

Output

The ID of the updated synonym, which is stored in the SynonymID parameter.

Example

The following code updates a synonym with ID 121:

/cheminv/api/UpdateSynonym.asp?SynonymID=121&SynonymName="Benzenol"
