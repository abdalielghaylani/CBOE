CreateCustomReport

Causes the RPT report writer to generate a custom report. Custom reports are created by the users, as per their requirements. 

Syntax

/cheminv/api/CreateCustomReport.asp?ReportName=adVarChar&QueryName=adVarChar&ReportID=adNumeric[&ReportFormat=adVarChar]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ReportName adVarChar  Specifies the name of the custom report.   Y INV_REPORTS REPORTDISPLAYNAME
QueryName adVarChar Specifies the name of the query on the basis of which the custom report is to be created.   Y INV_REPORTS QUERYNAME
ReportID adNumeric Specifies the ID of the report template to be used to generate the report. The template name is specified in the similar way as it is defined in the MS Access reports database.   Y INV_REPORTS ID
ReportFormat adVarChar Specifies the format of the report. A default value for the ReportFormat parameter is defined in the invconfig.ini file. Access Report SnapShot (SNP) | Adobe Acrobat Portable Document Format (PDF) | Rich Text Format (RTF) | Microsoft Excel (XLS) N INV_REPORTFORMATS FORMATDISPLAYNAME

Output

The file name of the custom report that is saved in the server's reports directory. The location of the server's reports directory is /cfserverasp/RPT/Reports/.

Note: To determine the location of the server's reports directory on your server, please check the invconfig.ini file.

Example

The following code creates a custom report with name CustomTestReport:

/cheminv/api/CreateCustomReport.asp?ReportName=CustomTestReport&QueryName=qryCustomTest&ReportID=10
