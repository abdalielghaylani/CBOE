CreateOrder

Creates a new order.

Syntax

/cheminv/api/CreateOrder.asp?DeliveryLocationID=adNumeric&ShipToName=adVarChar[&ShippingConditions=adVarChar&SampleContainerIDs=adVarChar&StatusID=adNumeric]

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DeliveryLocationID adNumeric Specifies the ID of the location where the order is to be delivered.   Y INV_ORDERS DELIVERY_LOCATION_ID_FK
ShipToName adVarChar Specifies the user to whom the order is to be shipped.   Y INV_ORDERS SHIP_TO_NAME
ShippingConditions adVarChar Specifies the shipping conditions for the order.   N INV_ORDERS SHIPPING_CONDITIONS
SampleContainerIDs adVarChar Specifies a comma delimited list of IDs of the containers included in the order.   N INV_ORDER_CONTAINERS CONTAINER_ID_FK
StatusID adNumeric Specifies the ID of the status of the containers included in the order.   N INV_CONTAINERS CONTAINER_STATUS_ID_FK

Output

None.

Example

The following code creates an order that is to be delivered to the location with ID 1032 and to be assigned to INVADMIN:

/cheminv/api/CreateOrder.asp?DeliveryLocationID=1032&ShipToName=INVADMIN
