CreateGridFormat

Creates a grid format and returns the ID of the newly created grid format.

Syntax

/cheminv/api/CreateGridFormat.asp?row_count=adNumeric&col_count=adNumeric[&grid_format_type=adNumeric&row_prefix=adVarChar&col_prefix=adVarChar&row_use_letters=adNumeric&col_use_letters=adNumeric&name_separator=adVarChar&number_start_corner=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
grid_format_type adNumeric Specifies the type of the grid format.   N INV_GRID_FORMAT GRID_FORMAT_TYPE_FK
row_count adNumeric Specifies the number of rows in the grid.   Y INV_GRID_FORMAT ROW_COUNT
col_count adNumeric Specifies the number of columns in the grid.   Y INV_GRID_FORMAT COL_COUNT
row_prefix adVarChar Specifies the prefix to be used for grid rows.   N INV_GRID_FORMAT ROW_PREFIX
col_prefix adVarChar Specifies the prefix to be used for grid columns.   N INV_GRID_FORMAT COL_PREFIX
row_use_letters adNumeric Specifies whether to label rows with letters or with numbers.   N INV_GRID_FORMAT ROW_USE_LETTERS
col_use_letters adNumeric Specifies whether to label columns with letters or with numbers.   N INV_GRID_FORMAT COL_USE_LETTERS
name_separator adVarChar Specifies name separator to be used in the grid format.   N INV_GRID_FORMAT NAME_SEPARATOR
number_start_corner adNumeric Specifies start corner for the grid format.   N INV_GRID_FORMAT NUMBER_START_CORNER
number_direction adNumeric Specifies whether to fill the rows first or to fill the columns first.   N INV_GRID_FORMAT NUMBER_DIRECTION
Name adVarChar Specifies the name of the grid format.   Y INV_GRID_FORMAT NAME
Description adVarChar Specifies the description of the grid.   N INV_GRID_FORMAT DESCRIPTION
zero_padding_count adNumeric Specifies zero padding count for the grid format.   Y INV_GRID_FORMAT ZERO_PADDING_COUNT

Output

The ID of the new grid format, which is stored in the new_grid_format_id parameter. The data type of the new_grid_format_id parameter is adNumeric.

Example

The following code creates a grid format with row count 8 and column count 12:

/cheminv/api/CreateGridFormat.asp?row_count=8&col_count=12
