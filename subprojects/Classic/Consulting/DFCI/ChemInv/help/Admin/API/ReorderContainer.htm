ReorderContainer

Places order for an already ordered or existing container.

Syntax

/cheminv/api/ReorderContainer.asp?DeliveryLocationID=adNumeric&ContainerID=adNumeric&ContainerName=adVarChar&Comments=adVarChar&OwnerID=adVarChar&CurrentUserID=adVarChar&NumCopies=adNumeric&Project=adVarChar&Job=adVarChar&RushOrder=adNumeric&ReorderReson=adVarChar&ReorderReasonOther=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DeliveryLocationID adNumeric Specifies the ID of the location where the container is to be delivered.   Y INV_CONTAINER_ORDER DELIVERY_LOCATION_ID_FK
ContainerID adNumeric Specifies the ID of the container to be reordered.   Y INV_CONTAINERS CONTAINER_ID
ContainerName adVarChar Specifies the name of the container to be reordered.   Y INV_CONTAINERS CONTAINER_NAME
Comments adVarChar Specifies the comments associated with the container.   Y INV_CONTAINERS CONTAINER_COMMENTS
OwnerID adVarChar Specifies the owner of the container.   Y INV_CONTAINERS OWNER_ID_FK
CurrentUserID adVarChar Specifies the current user of the container.   Y INV_CONTAINERS CURRENT_USER_ID_FK
NumCopies adNumeric Specifies the number of container copies to be ordered.   Y N/A N/A
Project adVarChar Specifies the project under which the container is to be ordered.   Y INV_CONTAINER_ORDER PROJECT_NO
Job adVarChar Specifies the job associated with the project under which the container is ordered.   Y INV_CONTAINER_ORDER JOB_NO
RushOrder adNumeric Specifies whether or not it is a rush order. 1 indicates that the order is rush order, 0 indicates that the order is not a rush order. 1|0 Y INV_CONTAINER_ORDER ISRUSHORDER
DueDate Date Specifies the date on which the order should be delivered. The date is specified in the date-time literal format. For example 01-27-2001.   Y INV_CONTAINER_ORDER DUE_DATE
ReorderReason adVarChar Specifies the reason for reordering the container.   Y N/A N/A
ReorderReasonOther adVarChar Specifies the reason for reordering container, if the reason is other than those defined already in Inventory Enterprise.   Y, If Other is specified in ReorderReason. N/A N/A

Output

The ID of the newly created container, which is stored in the NewContainerID parameter. The data type of the NewContainerID parameter is adNumeric.

Example

The following code reorders a container with ContainerID 1764:

/cheminv/api/ReorderContainer.asp?DeliveryLocationID=1&ContainerID=1764&ContainerName=nitrobenzene&Comments=adVarChar&OwnerID=adVarChar&CurrentUserID=System&NumCopies=1&Project=2&Job=1&RushOrder=1&ReorderReson=Insufficient Quantity
