DeleteTableRow

Deletes row(s) of a table in the cheminvdb schema.   

Syntax

/cheminv/api/DeleteTableRow.asp?TableName=adVarChar&pkColumnName=adVarChar&pkIDs=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
pkColumnName adVarChar Specifies the name of the primary key column of the table. This column uniquely identifies the row that is to be deleted.   Y N/A N/A
TableName adVarChar Specifies the name of the table.   Y N/A N/A
pkIDs List of adNumeric Specifies a comma delimited list of the primary key IDs corresponding to the rows to be deleted.   Y N/A N/A

Output

None.

Example

The following code deletes rows from the INV_SUPPLIER table: 

/cheminv/api/DeleteTableRow.asp?TableName=INV_SUPPLIERS&pkColumnName=Supplier_Name&pkIDs=11,12,14
