CopyPlate

Creates duplicate copy of a plate.

Syntax

/cheminv/api/CopyPlate.asp?PlateID=adNumeric&NumCopies=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
PlateID adNumeric Specifies the ID of the plate to be copied.   Y INV_PLATES PLATE_ID
NumCopies adNumeric Specifies the number of copies to be created.   Y N/A N/A

Output

The ID of the new plate, which is stored in the NewPlateID parameter. The data type of the NewPlateID parameter is adNumeric.

Example

The following code creates 2 copy of the plate with ID 1004:

/cheminv/api/CopyPlate.asp?PlateID=1004&NumCopies=2
