CheckDuplicateBarcode

Checks if the barcode that is to be assigned to a container, plate, or location exists already in Inventory Enterprise.

Syntax

/cheminv/api/CheckDuplicateBarcode.asp?Barcodes=adVarChar&BarcodeType=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
Barcodes adVarChar Specifies the barcode.   Y INV_CONTAINERS BARCODE
INV_LOCATIONS LOCATION_BARCODE
INV_PLATES PLATE_BARCODE
BarcodeType adVarChar Specifies the type of the barcode. Plate|Container|Location Y N/A N/A

Output

None.

Example

The following code checks whether the container barcode, 295 exists already in Inventory Enterprise:

/cheminv/api/CheckDuplicateBarcode.asp?Barcodes=295&BarcodeType=container
