Delete Location

To delete a location:

  1. Select the location you need to delete, in the Location Tree frame.
  2. Select Delete from the Location menu in the Location Tree frame. The Delete Inventory Location window appears:

  1. Specify the location that is to be deleted in the Location to delete field.
  2. Click the OK button to delete the specified location.

Note: You can delete a location only if it is empty. Therefore, if you want to delete an non-empty location, you should select the check box, Recursively delete related locations and containers. When this checkbox is selected, all the sub locations and containers within the selected location are deleted before the location.
