Placing Containers on a Rack

To place containers on a rack:

  1. Create a few containers in a location other than the rack. For more information about creating containers, see Creating Containers.
  2. Click on a container in the Container List frame to display its details in the Container Details frame.
  3. Select Edit Container from the Update menu of Container Details frame. The Create or Edit an Inventory Container window appears:

  1. Click the Browse link. The following window appears:

  1. Select a rack location or multiple racks. The grid appears, as shown in the following figure:

  1. Select a grid cell.
  2. Click the rack location you selected in the Location Tree frame. The selected container will be displayed in the selected cell of the grid, in the Container List frame.

Note: By default, icons are displayed for the containers in the grid. You can change the view by clicking the down arrow in the first cell of the grid and then selecting the appropriate display option.

Multi-select Racks

To select multiple racks:
