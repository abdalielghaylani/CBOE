Make Default

You can set a location as the default location so that it opens in the Location Tree frame automatically, each time the Container Management area is accessed.

To set a location as the default location:

  1. Navigate to the location that is to be set as the default location, in the Location Tree frame.
  2. Select Set Default from the Location menu in the Location Tree frame. The Set your default inventory location window appears:

  1. Make sure that the location specified in the Location Name text box is the location that is to be set as the default location.
  2. Click the OK button.
