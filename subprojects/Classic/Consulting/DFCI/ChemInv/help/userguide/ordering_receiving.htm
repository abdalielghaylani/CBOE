Ordering and Receiving

The ordering and receiving feature allows you to place and order for a container through ChemACX and get it delivered to the desired Inventory location, which is set as default. Therefore, it is important that the user trying to order a container has set a default location. To set a location as the default location, please see, make default.

Receiving an Order

To receive the placed order:

  1. Create an order for the container that has been sent from ChemACX to On Order location in Inventory Enterprise. For details, please see Creating New Order.
  2. Ship the order. For details, please see Shipping an Order.
  3. Receive the shipped order. For details, please see Receive Order.
