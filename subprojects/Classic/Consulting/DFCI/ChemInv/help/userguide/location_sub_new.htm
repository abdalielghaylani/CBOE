Creating a New Sub Location

To create a new sub location:

  1. Navigate to the location within which you want to create the new sub location, in the Location Tree frame.
  2. Click the New Location link in the Current Location frame. The Create a New Inventory Location dialog box appears:

  1. Specify barcode ID for the location either by typing the ID in the Location ID text box or by selecting the Autogenerate barcode ID check box.

Note: When you select the Autogenerate barcode ID check box, the Barcode Description list box is displayed above the checkbox. You need to select Location Barcodes from the list box to generate barcode ID for the location.    

  1. Type the name of the sub location in the Location Name text box.
  2. Select appropriate location type from the Location Type list box.
  3. Click the OK button. You can observe that the new sub location has been created within the desired location.
