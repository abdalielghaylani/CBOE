Location ID

Type of field: Integer

This field is meant to hold the location ID. Location ID is a unique number that identifies a location in the Inventory Enterprise database. It is automatically generated when the location is entered into Inventory Enterprise. The location ID is linked to the location name and other information in the Inventory Enterprise database.
