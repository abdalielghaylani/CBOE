Location Type

Type of field: Text (picklist)

This field allows users to specify the type of the location. A location can be of any of the following type: Bench, Bin, Box, Building, Cabinet, Company, Cylinder Storage, Disposal/Neutralization Room, Dry box, Freezer, GMP, Hood, Instrument Room, Laboratory, Pan, Receiving, Refrigerator, Safe, Shelf, Site, Solvent Cabinet, Solvent Room, Stock Room, Ultra-Freezer, Utility Room, unknown, Disposed, On Order, Trash Can, Stock Room Shelf, Stock Room Cabinet, Stock Room Bin, Missing, Stock Room Refrigerator, Stock Room Top of Cabinet, Stock Room Freezer, Stock Room Returns Room, and Plate Map.
