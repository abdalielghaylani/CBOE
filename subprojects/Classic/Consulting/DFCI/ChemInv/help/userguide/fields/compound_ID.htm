Compound ID

Type of field: Number

This fields is meant to hold the compound ID, which identifies a compound in the Inventory Enterprise database. Compound ID is generated automatically when the compound is entered into Inventory Enterprise. This number is linked to the compound name and other information in the Inventory Enterprise database.
