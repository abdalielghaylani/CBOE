Plate ID (Internal)

Type of field: Number

This field is meant to hold plate ID for a plate. Plate ID is a unique number that identifies a plate in the Inventory Enterprise database. It is generated automatically when the plate is created in Inventory Enterprise. Plate ID is linked to the plate name and other information in the Inventory Enterprise database.

The parent plate ID is the ID of a plate's parent. A plate has a parent if it is derived from another plate.
