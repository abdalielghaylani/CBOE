Conflict Resolution - Selecting an Existing Substance

The option of selecting an existing substance is available only when you are creating a substance for adding it to a container. This option is not available when you create a new substance using the Substance Management area.

To select the existing substance, click the Select link next to the existing substance. The selected substance is added to the container and you are returned to the Create or Edit an Inventory Enterprise dialog box, using which you can continue the process of creating a new container in Inventory Enterprise.
