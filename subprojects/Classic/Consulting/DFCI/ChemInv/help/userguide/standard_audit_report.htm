Standard Audit Report

Standard audit report allows you to gather detailed information about all the actions performed on a substance in Inventory Enterprise. To audit an Inventory Enterprise substance using a Standard audit report:

  1. Perform a search in the Substance Management area of Inventory Enterprise to search the substance that is to be audited. For information about searching substances in Inventory Enterprise, please see Searching for a Substance.
  2. Click the Details tab corresponding to the appropriate substance. A page containing the details, such as molecular weight and molecular formula of the substance appears.
  3. Click the View History link. The Audit Report page, displaying the Standard audit report, appears:

By default, the Standard audit report displayed in the Audit Report page contains the details of all the actions performed on the substance. You can filter the actions using the Filter Criteria area of the Audit Report page. The Filter Criteria area allows you to filter the actions on the basis of the following fields:

  • Table: Specifies the name of the table.
  • User: Specifies the user who has performed the action.
  • RID: Specifies the ID of the action.
  • LocationID: Specify the ID of the location at which the action is performed.
  • ContainerID: Specifies the ID of the container on which the action is performed.
  • From Date and To Date: Specify the range of date, the actions performed in which are to be tracked.
  • Action: Specifies the type of the SQL action performed on the substance, such as Delete, Update, and Insert.
  • CompoundID: Specifies the ID of the substance on which the action is performed.
  1. Provide the information on the basis of which you want to filter the actions, in the Filter Criteria area.
  2. Click the Filter button. The results are displayed in the lower frame of the Audit Report page.
