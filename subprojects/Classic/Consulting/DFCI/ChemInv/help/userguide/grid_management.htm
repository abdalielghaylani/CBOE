Grid Management

The Grid Management feature in Inventory Enterprise allows you to place plates and containers in grids, in the same way as they are physically kept in racks. Before placing place plates and containers in grids, you need to create a grid format, which specifies the number of rows and columns of the grid.

Note: In order to enable the Grid Management feature, you need add a new line, ENABLE_GRID_MANAGEMENT= "True", in the invconfig.ini file.
