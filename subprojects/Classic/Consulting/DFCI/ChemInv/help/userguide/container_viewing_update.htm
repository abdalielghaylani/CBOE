Update Contents

There are situations when the locations of the plates or containers are changed, but Inventory Enterprise is not updated with the changes. This results in discrepancies in Inventory Enterprise. The Update Contents function of Inventory Enterprise allows you to check for these discrepancies and verify whether a plate is at the appropriate location.

However, unlike the content rectification feature, the update contents feature does not allow you to correct these discrepancies. It allows you to import the plates that are not present at a location, but should have been present. However, it does not allow you to move irrelevant plates that are present at a location, to some other location.

Note: You can update location contents only if you have appropriate privileges to do so. To see if you have appropriate privileges to update the contents of a location, please see Roles and Privileges.

To update the contents of a location:

  1. Navigate to the location contents of which are to be updated, using the Location Tree frame.
  2. Click the Update Contents link, which appears in the Container List frame when the location contains plates. The Reconcile Location window appears:


The lower frame of the Reconcile Location window contains a list of containers that are expected to be there at the selected location.

  1. Type the barcode of the container, whose availability is to be verified, in the Scan Barcode text box.
  2. Press the Tab key. The following window appears:

However, if the scanned plate is not available at the location, the plate is added to the list of the misplaced plates.

Note: Clear the check box next to the misplaced plate, if you do not want to move it to the location, contents of which are being updated.

  1. Repeat steps 3 and 4 for all the plates, whose availability is to be verified.
  2. Click the OK button.
