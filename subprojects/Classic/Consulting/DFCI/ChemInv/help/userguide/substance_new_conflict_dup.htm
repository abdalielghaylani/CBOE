Conflict Resolution - Creating Duplicate Substance

You can add a conflicting substance to Inventory Enterprise, regardless of the conflict, by creating a duplicate substance. To create a duplicate substance:

  1. Click the Create Duplicate link next to the conflicting substance. A screen, informing you that the duplicated substance is created, appears.
  2. Click the OK button.

Creating a duplicate substance is not recommended, but is an option. You can search all the duplicate substances in Inventory Enterprise by selecting the Duplicate Compounds option from the Compound Type list box while performing Advanced Search on Inventory Enterprise.
