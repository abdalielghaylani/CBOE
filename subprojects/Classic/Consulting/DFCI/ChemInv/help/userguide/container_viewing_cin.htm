Check In

Checking in allows you to restore the checked out container to its original location and assign it to its original user again, even if you do not remember the original location and original user of the container. To see if you have the appropriate privileges to check in a container, please see Roles and Privileges.

To check in a checked out container:

  1. Navigate to the location that contains the container, which is to be checked in by using the Location Tree frame.
  2. Select the appropriate container in the Container List frame. The details of the container are displayed in the Container Details frame.
  3. Select Check In from the Obtain menu in the Container Details frame. The Check out/in an Inventory Container window appears:

Note: For information about the fields available in the Check out/in an Inventory Container window, please see Form Fields.

  1. Observe that the Destination Location text box and Current User drop down list are already populated with the original location and original user of the container.
  2. Modify the status of the container using the Container Status drop down list.
  3. Click the OK button. Observe that the container is moved to its original location.
