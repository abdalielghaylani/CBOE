Tasks

When you click the Tasks link in the Current Location frame, the Administrative Menu window appears:

The Administrative Menu window contains various links, which are as follows:

Search Inventory Change Password
Manage Tables Manage Users
Manage Batching Fields Manage Roles
Manage Reports Plate Settings
Analyze Audit Trails Create Plates from Excel
Custom Reports Create Plates from Text File
Manage Approvals Manage Sample Requests
Manage Requests Manage Orders
Help Log Off
Close  

Apart from the above links, the Administrative Menu window provides links for managing grids and organizations.

Note: To see if you have the privileges to perform an administrative task, please see Roles and Privileges.
