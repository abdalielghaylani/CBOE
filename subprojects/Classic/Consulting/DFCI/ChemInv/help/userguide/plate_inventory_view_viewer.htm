Plate Viewer Tab

The Plate Viewer tab displays the plate in the grid format. Each cell of the grid represents a well. The following figure displays the Plate Viewer tab page:

You can click a cell of the grid to gather information about the well represented by it. When you click a cell, the View a Plate Well window appears:

The View Plate Well page contains the Edit Well link that allows you to edit contents of an individual well. For more information about editing contents of an individual well, please see Edit Well.
