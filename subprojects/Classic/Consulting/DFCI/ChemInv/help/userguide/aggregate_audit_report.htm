Aggregate Audit Report

Aggregate audit report allows you to filter the actions on the basis of the table name, user, action, and date. Unlike the Standard audit report, Aggregate audit report does not provide detailed information about the actions.

In the Aggregate audit report, you can group the filtered results on the basis of either, Table Name, Modified by, or Action field. For example, you can group the filtered results by Table Name to obtain a list of tables on which a particular user has performed some action. You can also group the filtered results on the basis of hour, day, month, or year.

To audit an Inventory Enterprise substance using an Aggregate audit report:

  1. Perform a search in the Substance Management area of Inventory Enterprise to search the substance that is to be audited. For information about searching substances in Inventory Enterprise, please see Searching for a Substance.
  2. Click the Details tab corresponding to the appropriate substance. A page containing the details, such as molecular weight and molecular formula of the substance appears.
  3. Click the View History link. The Audit Report page appears.
  4. Click the Aggregate link. The following page appears:

In the preceding page, you can specify the criteria according to which you want to filter the actions by setting the following fields:

  • Table: Specifies the name of the table.
  • User: Specifies the user who has performed the action.
  • From Date and To Date: Specify the range of date, the actions performed in which are to be tracked.
  • Action: Specifies the type of the SQL action performed on the substance, such as Insert, Update, and Delete.
  1. Specify the filter criteria in the Filter Criteria area.
  2. In the Group by Fields area, select the radio button corresponding to the field according to which you want to group the filtered results.
  3. Select the appropriate radio button in the Group by Date area.
  4. Click the Filter button. The results are displayed in the lower frame of the Audit Report page:
