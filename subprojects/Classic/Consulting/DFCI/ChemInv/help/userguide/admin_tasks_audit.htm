Analyze Audit Trail

The Analyze Audit Trail link in the Administrative Menu window allows you to keep track of the actions performed on the containers, locations, compounds in Inventory Enterprise. For example, if the Qty Remaining field of a container is modified, you can audit Inventory Enterprise to determine the old and new value of the field, date and time at which the field is edited, and the user who has edited the field.

You can audit Inventory Enterprise using two different types of audit reports, which are as follows:

  • Standard
  • Aggregate

Note: In addition to auditing Inventory Enterprise, you can also audit DocManager Enterprise using the auditing feature. However, you can audit DocManager Enterprise only if it is installed and configured properly on the system from which you are accessing Inventory Enterprise.
