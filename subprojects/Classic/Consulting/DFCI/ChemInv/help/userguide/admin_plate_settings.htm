Plate Settings

The Plate Settings link in the Administrative Menu window allows you to create, edit, delete, and copy a plate setting. The various plate settings that can be created, edited, deleted, and copied using the Plate Settings link, are as follows:

  • Grid Formats
  • Physical Plate Types
  • Plate Formats
  • Plate Types
  • Well Formats
  • Reformatting Maps
  • Plate Import Templates

When you click the Plate Settings link, the following window appears:

Creating a New Plate Setting

To create a new plate setting:

  1. Click the New link next to the plate setting that is to be created.
  2. Provide the required information.

Note: Fields highlighted in red are the required fields.

  1. Click the OK button.

Editing a Plate Setting

To edit a plate setting:

  1. Select the plate setting that is to be edited, from the appropriate plate setting drop down list.
  2. Click the Edit link next to the plate setting.
  3. Make the required changes.

Note: Fields highlighted in Red color are the required fields.

  1. Click the OK button.

Note: Editing cannot be done for the reformatted maps. Reformatted maps can only be created and deleted.

Deleting a Plate Setting

To delete a plate setting:

  1. Select the plate setting that is to be deleted from the appropriate plate setting drop down list.
  2. Click the Delete link next to the plate setting.

Note: Fields highlighted in Red color are required.

  1. Click the OK button.

Copying a Plate Setting

Copies can be created only for the Plate Import templates. To copy a Plate Import template:

  1. Select the Plate Import template that is to be copied from the Plate Import Templates drop down list.
  2. Click the Copy link.
  3. Click the OK button.
