EH&S Tab

You can view the Environmental Health and Safety (EH&S) tab in the Container Details frame only if you the privileges to edit the EH&S data for a container. To see if you have the appropriate privileges to edit the EH&S data, please see Roles and Privileges.

The EH&S tab provides (EH&S) data for the container. The following figure displays the EH&S tab page:

The EH&S tab is customizable by the system administrator, therefore, the fields shown in the preceding figure may or may not exist in your EH&S tab. For more information about the fields in the EH&S tabs, please see your system administrator.

You can record the EH&S information at two levels, which are as follows:

  • Substance level
  • Container level
