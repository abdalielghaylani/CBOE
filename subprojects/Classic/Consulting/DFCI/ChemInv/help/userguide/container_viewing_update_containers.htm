Update Containers

The Update Containers link is available in the Update menu of Container Details frame only when the Container List frame is in the multi select mode and a container is selected. To view the Container List frame in the multi select mode, you need to click the Multi-Select link from the View menu in the Container List frame.

The Update Containers link allows you to update multiple containers, at once. To update multiple containers using the Update Containers link:

  1. Navigate through the Location Tree frame to reach to the location, which contains the desired containers.
  2. Click the Multi-Select link from the View menu in the Container List frame.
  3. Select the check box next to the desired container(s).
  4. Select Update Containers from the Update menu in the Container Details frame. The Update Inventory Container window appears:

  1. Select the field that is to be updated in the Container Fields list box.

Note: You can also select multiple fields in the Container Fields list box. To select multiple fields, you need to hold down the CTRL key.

  1. Click the OK button. The following page, asking you to enter values for the fields selected in the Container Fields list box, appears:

Note: The preceding page displays only those fields, which you have selected for updating.

  1. Enter the appropriate values in the respective fields and click the OK button.
