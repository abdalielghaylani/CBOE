Global Search - ChemACX Substances

The ChemACX substances displayed in the global search result are linked directly to ChemACX. This allows you to retrieve the ChemACX data from Inventory Enterprise. You can retrieve the ChemACX record of a substance from Inventory Enterprise by clicking the ACX number of the substance.

The ChemACX substances in the results of the global search are displayed in Red.

 

Various tabs are associated with a ChemACX substance to allow you to gather information about the substance. These tabs are:

  • Zoom In, : Allows you to zoom in on the structure of the substance.
  • ACX, : Allows you to access the ChemACX record of the substance from Inventory Enterprise.
  • Inv, : Allows you to view the details of the container associated with the substance
  • Mark, : Allows you to mark the substance so that you can easily locate the substance later, without creating a search query for it again. For more information about marking substances records, please see Marking Records.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked menu item from the Marked Hits menu displayed in the window containing the search results.
