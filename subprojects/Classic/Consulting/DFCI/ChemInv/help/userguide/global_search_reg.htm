Global Search - Registration Enterprise Substances

The Registration Enterprise substances displayed in the global search result are linked to Registration Enterprise. This allows you to retrieve the Registration Enterprise data directly from Inventory Enterprise. You can retrieve the Registration Enterprise record of a substance from Inventory Enterprise by clicking the Registration Number of the substance.

The Registration Enterprise substances in the results of the global search are displayed in Blue.  

Various tabs are associated with a Registration Enterprise substance to allow you to gather information about the substance. These tabs are:

  • Zoom In, : Allows you to zoom in on the structure of the substance.
  • Names, : Allows you to gather information about the various synonyms of the substance.
  • Details, : Allows you gather information about the container that contains the substance.
  • Mark, : Allows you to mark the substance so that you can easily locate the substance later, without creating a search query for it again. For more information about marking substances, please see Marking Records.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked menu item from the Marked Hits menu displayed in the window containing the search results.
