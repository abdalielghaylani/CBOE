Manage Users

The Manage Users interface of Inventory Enterprise allows you to add, update, and delete user related information of various ChemOffice Enterprise applications, such as BioAssay Enterprise and BioSAR Enterprise. The following screenshot shows the Manage Users interface of Inventory Enterprise:
 

Add User

To add a new user to a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Users from drop down list.
  2. Click the Refresh link. The list of users of the selected application is displayed in the Select a User list box.
  3. Click the Add User button to display the Add new User page.
  4. Enter relevant information in the respective fields.
  5. Click the OK button. The new user is added to the list of users of the selected application.

Update User

To update the information about an existing user of a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Users from drop down list.
  2. Click the Refresh link. The list of users of the selected application is displayed in the Select a User list box.
  3. Select the user whose information you need to update, in the Select a User list box.
  4. Click the Update User button to display the Update User page.
  5. Modify the information in the respective fields according to your requirements.
  6. Click the OK button. The information about the selected user is updated.

Delete User

To delete an existing user of a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Users from drop down list.
  2. Click the Refresh link. The list of users of the selected application is displayed in the Select a User list box.
  3. Select the user that you need to delete, in the Select a User list box.
  4. Click the Delete User button to display a page, which confirms whether you really want to delete the selected user.
  5. Click the OK button. The selected user is deleted.
