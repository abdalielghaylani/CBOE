Creating Grid Format

To create a grid format:

  1. Log on to the ChemOffice Enterprise application as an administrator of Inventory Enterprise. The home page of ChemOffice Enterprise appears:

  1. Click the Tasks link in the Inventory Enterprise section. The Administrative Menu window appears:

  1. Click the New link in the Grid Management section. The CreateGrid Format window appears:

  1. Type a name for the grid format in the Grid Format Name text box.
  2. Specify the number of rows and columns to be contained in the grid, in the Row Count and Column Count text boxes.
  3. Click the OK button to display a window confirming that the grid has been created.
  4. Click the OK button to close the confirmation window.
  5. Click the Close button to close the Administrative Menu window.
