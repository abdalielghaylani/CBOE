Substructure Search

In addition to the Advanced Search fields, you can use the substructure search to search Inventory Enterprise on the basis of the Substructure, Substance Name, Molecular Formula, and MolWeight Range fields.

To perform substructure search on Inventory Enterprise:

  1. Click the Search link within the Inventory Enterprise section in the home page of ChemOffice Enterprise. A page containing various search tabs, such as Simple Search, Advanced Search, Substructure Search, Batch Search, Global Search, and Plate Search, is displayed.

Note: The page containing the search tabs can also be accessed by clicking the Search link within the Current Location frame in the Container Management area.

  1. Click the Substructure Search tab. The Substructure Search tab page appears:

Note: The number of tabs visible in the preceding page depends on the configuration settings.
  1. Draw the substructure of a substance in the available drawing area in the Substructure Search tab page and specify the appropriate substructure search type in the SearchType drop down list, if you want to search Inventory Enterprise on the basis of the substructure of the substance stored in a container. The SearchType drop down list can take the following values:
  • Substructure
  • Full Structure
  • Exact Structure
  • Tanimoto Similarity

Note: You can draw substructure of the substance using the ChemDraw plug-in. For information about using the ChemDraw plug-in, please see ChemDraw Quick Reference.

  1. Enter the additional information on the basis of which you want to search the container.
  2. Select the Search Sublocations check box if you want to perform search on the location specified in the Location ID field as well as on the child locations of the location.
  3. Select the Exclude Special Locations check box if you want to exclude the special locations, such as Trash Can, Disposed, and On Order, from the search. For information about specifying which special location is to be excluded from the search, please see Excluding Special Locations.
  4. Select the Group results by chemical structure check box if you want the search results to be grouped on the basis of the chemical structure of the substances.
  5. Click the Search button. A list of substances matching the search criteria specified in the Substructure Search tab page appears. For information about viewing the results of the substructure search, please see Viewing Substructure Search Results.

Viewing Substructure Search Results

If the substructure search is performed with the Group results by chemical structure check box selected, the result of the search is a list of substances. You can view details of a particular substance in the search result list using various tabs, which are as follows:

  • Zoom In: Allows you to zoom in on the structure of the substance. The Zoom In tab is represented by a magnifying glass icon.
  • Names: Allows you to gather information about the various synonyms of the substance.
  • Details: Allows you gather information about the container associated with the substance.
  • Mark: Allows you to mark the substance so that you can easily locate the substance in Inventory Enterprise later, without creating a search query for it again. For more information about marking substances records, please see Marking Records.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked menu item from the Marked Hits menu displayed in the window containing the search results.

However, if the Group results by chemical structure check box is not selected when the substructure search is performed, the result of the search is a list of containers. You can view details of a container displayed in the result list of substructure search in the similar manner as the details of a container in the simple search results are viewed. For information about viewing the details of a container in the simple search results, please see Viewing Simple Search Results.
