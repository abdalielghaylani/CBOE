Manage Orders

The Manage Orders link allows you to gather information about all the sample request and container orders placed in Inventory Enterprise. It also allows you to create, edit, cancel, and ship orders. The following figure displays the Manage Orders page:

The Manage Orders page contains various links to allow you to manage the orders with different statuses. These links are as follows:

  • New
  • Shipped
  • Closed
  • Cancelled
