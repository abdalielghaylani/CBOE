Privileges

The following table lists the privileges available in Inventory Enterprise along with the roles associated with the privileges:

Privilege Oracle Privilege Name Role(s) Associated with Privilege
Change the quantity in a container INV_CHANGEQTY_CONTAINER CHEMIST, FINANCE, ADMIN
Change a container status INV_CHANGE_STATUS_CONTAINER CHEMIST, RECEIVING
Check out/in a container INV_CHECKOUT_CONTAINER CHEMIST, ADMIN
Check out/in a container INV_CHECKIN_CONTAINER CHEMIST, ADMIN
Reserve a container INV_RESERVE_CONTAINER CHEMIST, ADMIN
Retire a container INV_RETIRE_CONTAINER CHEMIST, FINANCE, ADMIN
Print a container's label INV_PRINT_LABEL_CONTAINER CHEMIST, FINANCE, RECEIVING, REGISTRAR, ADMIN
Print a report INV_PRINT_REPORT CHEMIST, FINANCE, RECEIVING, REGISTRAR, ADMIN
Create a new location INV_CREATE_LOCATION FINANCE, ADMIN
Edit a location INV_EDIT_LOCATION FINANCE, ADMIN
Move a location INV_MOVE_LOCATION FINANCE, ADMIN
Delete a location INV_DELETE_LOCATION FINANCE, ADMIN
Create a new container INV_CREATE_CONTAINER CHEMIST, FINANCE, RECEIVING, ADMIN
Edit a container INV_EDIT_CONTAINER CHEMIST, FINANCE, RECEIVING, ADMIN
Move a container INV_MOVE_CONTAINER CHEMIST, FINANCE, RECEIVING, ADMIN
Delete a container INV_DELETE_CONTAINER FINANCE, RECEIVING, ADMIN
Add/edit/delete links INV_MANAGE_LINKS REGISTRAR, ADMIN
Add/edit/delete substances INV_MANAGE_SUBSTANCES REGISTRAR, ADMIN
View audit trails INV_VIEW_AUDIT_TRAIL FINANCE, REGISTRAR, ADMIN
Update a location INV_UPDATE_LOCATION_CONTENTS Associated with a role by the ADMIN role, when required.
Rectify a location INV_RECTIFY_LOCATION_CONTENTS ADMIN
Order a container INV_ORDER_CONTAINER Associated with a role by the ADMIN role, when required.
Reorder a container INV_REORDER_CONTAINER Associated with a role by the ADMIN role, when required.
Certify container INV_CERTIFY_CONTAINER Associated with a role by the ADMIN role, when required.
Approve containers INV_APPROVE_CONTAINER Associated with a role by the ADMIN role, when required.
Edit EH&S data INV_EDIT_EHS_DATA Associated with a role by the ADMIN role, when required.
Create plate INV_CREATE_PLATE ADMIN, CHEMIST, FINANCE, RECEIVING
