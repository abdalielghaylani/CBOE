Receive Order

The Receive Order feature allows you to receive a shipped order. When you receive an order, the order is marked as Closed and the ordered containers or requested samples are delivered to the specified location.

To receive an order:

  1. Click the Receive Order link in the Current Location frame. The Receive an Order window appears:

  1. Type the barcode of the container that is to be received in the Scan Container Barcode text box.
  2. Press the Tab key. Observe that the container is scanned and added to the selection list, as shown in the following figure:

Note: You can remove a container from the list of the scanned containers by clicking the Remove link next to the container. You can remove all the containers from the list by clicking the Clear List link.

  1. Repeat steps 2 and 3 until all the required containers are scanned and added to the list.
  2. Ensure that the check boxes next to the containers that are to be received are selected and click the Receive Containers link.
  3. Click the OK button when the success message appears.
