Edit Container

To edit a container:

  1. Navigate through the Location Tree frame, to reach to the location that contains the container, which is to be edited.
  2. Select the appropriate container in the Container List frame. The details of the container are displayed in the Container Details frame.
  3. Select Edit Container from the Update menu in the Container Details frame. The Create or Edit an Inventory Container window, displaying the container information in the edit mode, appears:

Note: To see if you have the appropriate privileges to edit a container, see Roles and Privileges.

  1. Click the appropriate tab and make the required changes.
  2. Click the OK button.

Note: For more information about the tabs in the Create or Edit an Inventory Container window, please see Creating a New Container.
