Creating a Source Plate

To create a source plate from a text file:

  1. Click the Tasks link within the Inventory Enterprise section in the home page of ChemOffice Enterprise. The Administrative Menu window appears.
  2. Click the Create Plates from Text File link. The Create Plates From Text File window appears:

  1. Select the Create Source Plate(s) radio button.
  2. Set the fields in the Create Plates From Text File window in the similar manner as they are set for the target plates.
  3. Click the OK button.
