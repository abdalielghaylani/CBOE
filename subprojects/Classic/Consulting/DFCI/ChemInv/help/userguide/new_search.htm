Search

The Search link in the Current Location frame allows you to search substances, containers, and plates in Inventory Enterprise. The following figure displays the search form that appears on clicking the Search link:

You can perform various type of searches in this search form:

  • Simple search
  • Advanced search
  • Substructure search
  • Batch Search
  • Global search
  • Plate search
