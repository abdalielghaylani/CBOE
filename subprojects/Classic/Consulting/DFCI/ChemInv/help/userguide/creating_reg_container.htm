Creating a Container Using Registration Enterprise

To create a new container using a substance in Registration Enterprise:

  1. Open the Inventory Enterprise application.
  2. Click the New Container link in the Current Location frame. The Create or Edit an Inventory Container window appears:

  1. Click the Select Substance link. The following window appears:

  1. Click the Registration Enterprise check box.
  2. Search for a substance and select it. The batch id of the selected compound is automatically displayed in the Registry Batch ID text box, in the Create or Edit an Inventory Container window.
  3. Enter values in the required fields.
  4. Click the OK button to create the container. The Container Details frame displays the Reg Substance tab as shown in the following figure:
