Global Search

Global search enables you to search a substance in Inventory Enterprise as well as in Registration Enterprise and ChemACX. You can search all these applications simultaneously using the global search.

Note: Global search can be performed on the Registration Enterprise and ChemACX applications only if the system running Inventory Enterprise is installed and configured with these application.

Global search allows you to search a substance on the basis of the Substructure, Substance Name, CAS Registry, ACX Number, Catalog Number, Molecular Formula, MolWeight Range, Registry Number, Registry Sequence, Reg Alternate IDs, Salt Name, and Salt Equivalents fields.

To perform global search on Inventory Enterprise:

  1. Click the Search link within the Inventory Enterprise section in the home page of ChemOffice Enterprise. A page containing various search tabs, such as Simple Search, Advanced Search, Substructure Search, Batch Search, Global Search, and Plate Search is displayed.

Note: You can also access the page containing the search tabs by clicking the Search link within the Current Location frame in the Container Management area.

  1. Click the Global Search tab. The Global Search tab page appears:

 

Note: The number of tabs visible in the preceding page depends on the configuration settings.

Note: You can also access the Global Search tab page by clicking the Global Search link within the Inventory Enterprise section in the home page of ChemOffice Enterprise.

  1. Select the check boxes next to the applications that are to be searched, such as Inventory Enterprise, ChemReg, and ChemACX.
  2. Enter the information on the basis of which you want to search the selected applications.
  3. Click the Search button. A list of substances matching the search criteria specified in the Global Search tab page appears. For information about viewing the global search results, please see Viewing Global Search Results.

Note: For more information about searching Inventory Enterprise, please see Performing Searches, Search Result, and Types of Searches.  

Viewing Global Search Results

In the global search, the results from different applications are stored in different lists. For example, Registration Enterprise, Inventory Enterprise, and ChemACX substances that match the search criteria specified in the Global Search tab page are stored in separate lists. You can view the result list of a particular application by clicking the radio button next to the application, as shown in the following figure:

You can view details of the substances in the search result list of Inventory Enterprise using various tabs, which are as follows:

  • Zoom In: Allows you to zoom in on the structure of the substance.
  • Names: Allows you to gather information about the various synonyms of the substance.
  • Details: Allows you gather information about the container associated with the substance.
  • Mark: Allows you to mark the substance so that you can easily locate the substance in Inventory Enterprise later, without creating a search query for it again. For more information about marking substances records, please see Marking Records.

Note: You can view or unmark the marked records by selecting Show Marked or Clear Marked menu item from the Marked Hits menu displayed in the window containing the search results.

For  information about viewing details of the Registration Enterprise and ChemACX substances displayed in the global search result list, please see Global Search - Registration Enterprise Substances and Global Search - ChemACX Substances.
