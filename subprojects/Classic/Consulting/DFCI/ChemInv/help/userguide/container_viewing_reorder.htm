Reorder Container

The Reorder Container feature allows you to order an already ordered or already existing container again. If you want to order a container that does not exist in Inventory Enterprise, please see Order Container. To see if you have appropriate privileges to reorder a container, please see Roles and Privileges.

To reorder a container:

  1. Navigate through the Location Tree frame, to reach to the location that contains the required container.
  2. Select the appropriate container in the Container List frame. The details of the container are displayed in the Container Details frame.
  3. Select Reorder Container from the Create menu in the Container Details frame. The Reorder a Container window appears:

Note: For information about the fields available in the Reorder a Container window, please see Form Fields.

  1. Enter appropriate information about the reordered container in the respective fields.
  2. Click the OK button.
