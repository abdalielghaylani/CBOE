New Orders

New orders are the orders that have been created, but not yet shipped. New orders can be viewed and managed under the New link in the Manage Orders page. The following figure displays the Manage Orders page with the New link selected:

You can take the following actions on the new orders under the New link:

  • Create new orders
  • Edit an order
  • Cancel an order
  • Ship an order

You can also  for sample requests and containers under the New link.

Creating New Order

 To create order for a sample request or container:

  1. Click the Create Order link in the lower frame of the Manage Orders window. The Create or Edit an Order window appears:

  1. Type the ID of the container for which the order is to be created in the Scan Container Barcode text box.
  2. Press the Tab key. The container gets added to the selection list.
  3. Repeat steps 2 and 3 for all the required containers.
  4. Specify the location at which the order is to be delivered in the Delivery Location text box.
  5. Type the name of the user responsible for receiving the order in the Ship To text box.
  6. Click the Create Order link. The following window appears:

Editing an Order

To edit an order:

  1. Click the Edit link to the left of the appropriate order. The Create or Edit an Order window, containing the order information in the edit mode, appears:

  1. Make the required changes in the order and click the Update Order link.

Canceling an Order

To cancel an order:

  1. Click the Cancel link to the left of the appropriate order. The Cancel an Order window appears:

  1. Enter reason for canceling the order, if required, in the Reason for Cancel text area.
  2. Click the OK button. The order will now appear under the Cancelled link.

Shipping an Order

To ship an order:

  1. Select the Ship check box to the right of the appropriate order.
  2. Click the OK button.
  3. Click the OK button when the success message appears. The order will now appear under the Shipped link.
