Lineage 

The Lineage link allows you to determine the complete hierarchy of a container. When you select a container in the Container List frame and click the Lineage link, a tree type structure appears:

Note: The Lineage link is available in the Other menu of the Container Details frame.

The tree type structure contains the parent and child containers of the selected container, if any. It also contains all other containers having same parent container as the selected container. In addition, the tree structure allows you to determine if the parent container of the selected container further has any parent or the child containers of the selected container further have any children.
