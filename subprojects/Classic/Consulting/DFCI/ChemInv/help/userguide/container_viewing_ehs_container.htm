EH&S Tab - Container

You can store the EH&S data at the container level only if the substance stored in the container possesses the EH&S data, CAS number, supplier name, and catalog number. To store the EH&S data at the container level:

  1. Click the Edit EH&S Data link in the EH&S tab page. The Create or Edit EH&S Information dialog box appears.

Note: The Edit EH&S Data link does not appear in the EH&S tab page, if the substance stored in the container does not possesses EH&S data, or a valid CAS number, supplier name, and catalog number does not exist for the container.

  1. Make the required changes and click the OK button.
