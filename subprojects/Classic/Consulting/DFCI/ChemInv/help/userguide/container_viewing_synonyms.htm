Synonyms

The Synonyms link is available in the Substance tab of the Container Details frame. When you click the Synonyms link, a list of all the synonyms associated with the substance stored in the container is displayed:

 

The Synonyms link does not allows you to add new synonyms or edit or delete existing synonyms. For information about adding new synonyms or editing or deleting existing synonyms, please see Managing Synonyms.
