New Location

To create a new location:

  1. Select New from the Location menu in the Location Tree frame. The Create a New Location window appears:

  1. Click the Browse link next to the Parent Location text box to select a parent location for the new location. The following window appears:

  1. Navigate to the desired location within which you need to create the new location, and select it. The preceding window closes and the Create a New Location window is displayed with the selected location name in the Parent Location text box.
  2. Specify barcode ID for the location either by typing the barcode ID in the Location ID text box or by selecting the Autogenerate check box.

Note: When you select the Autogenerate check box, the Barcode Description list box is displayed above the check box. You need to select the location barcodes from the list box to generate barcode ID for the location.    

  1. Type the name of the location in the Location Name text box.
  2. Select appropriate location type from the Location Type drop down list.
  3. Click the Add Address link if you want to specify complete physical address of the new location. The following window appears:

  1. Enter relevant information in the respective fields and click the OK button.
  2. Click the OK button to close the Create a New Location window.
