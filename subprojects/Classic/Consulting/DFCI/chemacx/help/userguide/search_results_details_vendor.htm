Browsing Vendor Information

To browse through the vendor information in the Details view:

  1. Click one of the following tabs:
  • ALL VENDORS: Lists all the vendors available in ChemACX.
  • YOUR FAVORITE: Lists only favorite vendors.
  • ONLINE VENDORS: Lists only online vendors.
  1. Select the required vendor from the list. The catalogs associated with the selected vendor appears at the right of the vendors list, as shown in the following figure:

 

  1. Hold down the Ctrl key and select multiple vendors, if you want to compare the catalogs of the different vendors,
  2. Click the CATALOG DETAILS link corresponding to a catalog. The complete details about the catalog appears, as shown in the following figure:

  1. Click the Close button to close the preceding window.
  2. Click the COMPACT VIEW tab, if you want to view the catalogs in the Compact view, as shown in the following figure:

By default, the catalogs are displayed in the Catalog view. If you want to switch back to the Catalog view from the compact view, click the CATALOG VIEW tab.

In the Details view, you can also add products to the shopping cart and view the shopping cart. For information about this, please see Using the Shopping Cart.
