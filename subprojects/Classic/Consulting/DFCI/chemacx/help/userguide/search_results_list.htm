List View

By default, the search results are displayed in the List view, as shown in the following figure:

In the List view, you can perform various tasks, which are as follows:

  • Browse through the records using the navigation buttons.
  • Zoom-in on the structure of a chemical by clicking the following icon:
  • View the synonyms of a chemical by clicking the following icon:
  • View detailed information about a chemical, such as molecular weight and vendors by clicking the following icon:
  • Mark a chemical record for viewing later by clicking the following icon:
  • Register a chemical in Registration Enterprise.
  • Access the MSDX data for a chemical.
