Performing Non-Plugin Search

Non-plugin search limits the search fields only to chemical name and ACX number. This type of search method is useful when you do not need to search ChemACX on the basis of substructure, and therefore do not need to use the ChemDraw plugin.

To perform a non-plugin search in ChemACX:

  1. Access the ChemACX database to display the search form.
  2. Click the CLICK FOR NON-PLUGIN SEARCH link. The following page appears:

  1. Enter the name or CAS number of the chemical that is to be searched.
  2. Click the Search button to display the result.
