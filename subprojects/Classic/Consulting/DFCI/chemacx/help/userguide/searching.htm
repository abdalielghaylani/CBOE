Searching the ChemACX Database

Searching the ChemACX database is similar to searching all other ChemOffice Enterprise applications. You can search a chemical in the ChemACX database on the basis of various fields, such as name, substructure, ACX number, molecular formula, molecular weight, and catalog number of the chemical.

If searching by substructure is not necessary, you can simplify the search by querying the ChemACX database on the basis of only chemical name or Chemical Abstracts Service (CAS) number. For information about this, please see Performing Non-Plugin search.

Note: For more information about the searching techniques, please see Performing Searches.

Note: CAS is a division of the American Chemical Society that registers chemicals and provides unique id to the registered chemicals.

To search the ChemACX database:

  1. Access the ChemACX database to display the search form.
  2. Enter information about the chemical that you need to search.
  3. Select one of the following options:
  • Search all vendors: Searches chemicals from all the vendors available in ChemACX
  • Search your favorite vendors: Searches chemicals only from the vendors available in the favorite vendors list. For information about viewing or editing favorite vendors list, please see Editing Favorite Vendors List.
  1. Select the check box, Display all substances even if no vendors are found if you want to search those chemicals also, which do not have a vendor associated with them.
  2. Click the Search button. A page containing the search result is displayed. For information about viewing the search results, please see Viewing Search Results.
