Minimum Server Requirements

Server

Operating System

  • Windows 2000 Server OR 2003 Server

Oracle

  • Oracle 9.2.0.4

CambridgeSoft Software

  • CambridgeSoft ChemDraw Pro Plugin/ActiveX Control 8.0.7 or 9.0

Client

  • IIS and IE security patches
  • MSXML3 (installed with Windows 2000 but must be added for WinNT and Win 98 clients)
  • CambridgeSoft ChemDraw Pro Plugin/ActiveX Control 8.0.7 or 9.0
  • Adobe Acrobat Reader (to view MSDS data)

  • A resolution of 1024 x 768 is recommended

Note: All ChemOffice Enterprise applications must be installed by an administrator (i.e. the user must be logged into the server as the administrator of the machine).
