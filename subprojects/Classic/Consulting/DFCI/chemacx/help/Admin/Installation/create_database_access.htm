Create a Microsoft Access Database

Please follow the following steps:

  1. Run the ChemACX installer found on the CD at \ChemACX_9.0\ChemACX_9.0 Application
  2. Create a ChemACX Database.
    The database is contained in a self extracting ZIP file.
    1. Browse to your ChemOffice_data folder. Usually, the ChemOffice_data folder is located at:
      C:\Chemoffice_data
      and create a new directory called ChemACX in that folder.
    2. Double-click on the exe file found in this directory.
    3. Browse to the ChemACX folder you created in step (1).
    4. Click OK .
  3. Make sure the database files are NOT read only
    1. Right-click on the files (.mdb, .msi, .mst) and Select Properties.
    2. Make sure the Read-only checkbox is not selected.
  4. Make sure that everyone has full access to the folder where the database is located.
    1. Browse to C:\ and right-click on the ChemOffice_Data folder.
    2. Select Properties .
    3. Click the Security Tab.
    4. Scroll Down and Highlight Everyone in the Name box.
    5. Select the checkbox to the right of Full Control in the Allow column.
    6. Click the Advanced Button.
    7. Select the checkbox on the bottom which says "Reset permissions on all child objects and enable propagation of inheritable permissions".
    8. Click OK .
    9. Click Yes.
  5. Browse to <webroot>/ChemOffice/ChemACX/Config
  6. Double click on the ChemACX.udl file.
  7. Make sure the Database the UDL is pointing to is the ChemACX database you just moved (by default C:/ChemOffice_Data/ChemACX.mdb).
    The path to the mdb file is found in the Database name textbox.
    Click Browse and browse to the location of your mdb file if the default location is incorrect.

Step 4: Test the Installation
