Set up the Server to allow Export to Excel function

Please follow the following steps to allow exporting data to Excel from ChemACX:

  1. From the Start menu on the server, go to: Start>Run.
  2. Enter 'dcomcnfg' without the quotation marks into the Run box and click OK.
  3. Highlight the Microsoft Excel Application under the Applications Tab.
  4. Click Properties.
  5. Under the Security Tab select the radio button labeled "Use custom access permissions".
    1. Click Edit.
    2. Click Add.
    3. Highlight "Everyone" in the list and click Add.
    4. Click OK.
    5. You will be returned to the previous window and "Everyone" should be listed. If this is true, click OK.
  6. Under the Security Tab select the radio button labeled "Use custom launch permissions".
    1. Click Edit.
    2. Click Add.
    3. Highlight "Everyone" in the list and click Add.
    4. Click OK.
    5. You will be returned to the previous window and "Everyone" should be listed. If this is true, click OK.
