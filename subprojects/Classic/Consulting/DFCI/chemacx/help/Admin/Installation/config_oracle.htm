Additional Configuration

Please follow the following steps:

  1. Browse to <webroot>/ChemOffice/chemacx/config
  2. Open cfserver.ini
  3. Adjust the following:
    • PRIVILEGE_TABLE_LIST = "CS_SECURITY_PRIVILEGES"
    • MINIMUM_REQUIRED_PRIVILEGE = "CSS_LOGIN"
    • USE_CS_SECURITY_APP = 1
  4. Save the file.

Notes about users for this configuration:

  • CS_SECURITY must be altered to include CHEMACX roles and privileges
  • Users must have either ACX_BUYER or ACX_BROWSER to enter ChemACX
  • Additional roles will be required to Register and Create Inventory containers and compounds, if ChemACX is integrated with Inventory Enterprise and/or Registration Enterprise. See those application's Admin guides for more details.
