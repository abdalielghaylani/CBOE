Testing the Installation

The ChemACX user's guide can be accessed at:
http:// <IIS_SERVER_NAME>/Chemoffice/chemacx/help/
where IIS_SERVER_NAME is the name of the server where IIS is installed.

Perform the following steps to ensure ChemACX was successfully installed:

  1. Click on the ChemACX link on the ChemOffice Enterprise homepage.
  2. Search for a compound in the ChemACX database. See ChemACX User's Guide>Searching ChemACX in the User's Guide.
  3. Click the ChemACX link.
  4. Search for a compound in the ChemACX database.
    See ChemACX User's Guide>Searching ChemACX in the User's Guide.
    1. Choose Exact Search in the Search Type drop down menu.
    2. Type "benz" without the quotation marks in the text box next to Substance Name.
    3. Click Search.
      The results will be displayed in list view.
    4. Click the details button for one of the structures displayed.
      Tabs with vendor names will be displayed.
    5. Click on a vendor name.
      Package data will be displayed in the right frame. 

  5. Add the compound to your Shopping Cart.
    See ChemACX User's Guide>Viewing a Record and Using the Shopping Cart in the User's Guide.
  6. Export your shopping cart to Excel.
    See ChemACX User's Guide>Ordering the Chemicals in your ChemACX Shopping Cart in the User's Guide.
