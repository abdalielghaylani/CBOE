Upgrading to Windows Scripting version 5.6

Windows scripting version 5.6 is required to run the Oracle Setup Help File, which steps you through the Oracle configuration steps for ChemOffice Webserver.  The following steps will help you upgrade to version 5.6:

  1. Go to the download area on the Microsoft website to download the upgrade.
  2. Download the file and follow the instructions.
    Note: A reboot is necessary after installing.
  3. Go to START>Programs>ChemOffice Webserver>APPNAME>Oracle Configuration
    Note: This START menu shortcut points here -  <webroot>\ChemOffice\chem_reg\config\oracle_install_scripts

This executes the Oracle Configuration file.  After upgrading your scription version it is still necessary to run the required Oracle scripts.
