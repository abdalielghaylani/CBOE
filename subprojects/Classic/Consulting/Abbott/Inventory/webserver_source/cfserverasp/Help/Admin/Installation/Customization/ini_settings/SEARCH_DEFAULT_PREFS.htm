SEARCH_DEFAULT_PREFS

Example: SEARCH_DEFAULT_PREFS = S1:true,S2:true,S3:true,S4:true,S5:true,S6:90,S7:true,S8:true,S9:true,S10:true,S11:false,S12:false,S13:false

Description: The default settings for search preferences.  This is a comma separated list with each element having the following format:
<S#>:<true|false>
 where S# is the ID for a particular search preference.

ID Search Preference
S1 MATCH_TET_STEREO
S2               MATCH_TET_DB

S3 HIT_ANY_CHARGE_CARBON

S4 RXN_HIT_RXN_CENTER

S5 HIT_ANY_CHARGE_HETERO

S6 SIM_SEARCH_THRESHOLD

S7 FULL_STRUCTURE_SIMILARITY

S8 EXTRA_FRAGS_OK

S9 EXTRA_FRAGS_OK_IF_RXN

S10 FRAGS_CAN_OVERLAP

S11 IDENTITY

S12 RELATIVE_TET_STEREO

S13 ABSOLUTE_HITS_REL

Default: S1:true,S2:true,S3:true,S4:true,S5:true,S6:90,S7:true,S8:true,S9:true,S10:true,S11:false,S12:false,S13:false

Options: A comma separated list in the above format. 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
