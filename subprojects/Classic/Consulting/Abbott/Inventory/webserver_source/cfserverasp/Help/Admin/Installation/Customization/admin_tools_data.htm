Adding a DataView

This tool will add a new datview to an existing ChemOffice WebServer application.

To create a new dataview:

  1. Select an application name from the listbox and click  Add Data View.
  2. Browse to the ChemFinder database you would like to convert into a dataview in the chosen application. Note: The ChemFinder database must have EVERYONE:Full Control as permissions.  To make sure this is the case, place the database somewhere in your ChemOffice_data folder (by default found in C:\Chemoffice_data)
  3. Indicate if Structure or Reaction data is found in the database.
  4. Click Next.
  5. Select the fields that should be shown in list view.  To select multiple fields, hold the CTRL key down while highlighting fields.
  6. Click  Next.
  7. Click Finish.
    The data view is added to the chosen application.
