Errors due to Cached Information in the CFWTemp Directory

Certain information is saved in a temporary directory when applications are launched to make starting the application and taking certain actions more efficient.  Sometimes information in this temporary directory may cause problems.  If you are experiencing unexplained errors, try clearing out the temporary directory:

  1. Browse to \InetPub\wwwroot\ChemOffice\cfwtemp\
  2. Delete all files and folders in this directory.
  3. Perform an IISreset.
