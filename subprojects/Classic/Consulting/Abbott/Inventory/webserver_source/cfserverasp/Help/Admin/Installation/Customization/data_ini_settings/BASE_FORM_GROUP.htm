[BASE_FORM_GROUP]

Description: The form groups specified in the key, references individual form group sections elsewhere in the ini file. Form groups are a way for ChemOffice Webserver to specify different input and result forms and functionality via Form_group_flags.

The Base_form_group definition is the default definition for standard ChemOffice searching using the ChemDraw plugin.   

The sample.ini definition is as follows:

[BASE_FORM_GROUP]
INPUT_FORM_PATH=sample_input_form.asp
INPUT_FORM_MODE=search
RESULT_FORM_PATH=sample_result_list.asp
RESULT_FORM_MODE=list 
PLUGIN_VALUE=True
STRUCTURE_FIELDS=MolTable.Structure
FORM_GROUP_FLAG=SINGLE_SEARCH
MW_FIELDS=MolTable.MolWeight
FORMULA_FIELDS=MolTable.Formula
SEARCHABLE_ADO_FIELDS=MolTable.MOL_ID;1,MolTable.Molname;0,Synonyms_r.SYN_ID;1,Synonyms_r.Synonym_r;0 REQUIRED_FIELDS=NULL
SDFILE_FIELDS=TABLES:MOLTABLE,SYNONYMS_R
TABLE_GROUP=base_table_group
NUM_LIST_VIEW=5

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
