Testing the Installation

The ChemOffice Enterprise user’s guide can be accessed at: http:// IIS_SERVER_NAME/Chemoffice/WebServer_source/cfserverasp/help/ where IIS_SERVER_NAME is the name of the server where IIS is installed. Perform the following steps to ensure the ChemOffice WebServer was successfully installed:

  1. Access the ChemOffice Enterprise homepage (http:// IIS_SERVER_NAME where IIS_SERVER_NAME is the name of the server where IIS is installed).
  2. Click the Sample link.
  3. A page containing a ChemDraw plugin window appears.
  4. Draw benzene in the plugin window.
  5. Click Search.
    See ChemOffice Enterprise Help>Performing Searches>The Query Input Form User Interface in the User’s Guide.

The first five structures should appear on the page and the hit list totals 123 hits.
