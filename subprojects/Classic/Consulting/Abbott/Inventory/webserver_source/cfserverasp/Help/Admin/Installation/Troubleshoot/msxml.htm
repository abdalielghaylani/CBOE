MSXML Errors

Occasionally, users of ChemOffice WebServer applications are given a MSXML3 Access Denied error when trying to access pages containing MSXML. This will not happen on all installations.

To correct this issue if you experience it:

  1. Go to START>Programs>Administrative Tools>Internet Services Manager
  2. Expand your machine on the IIS tree and right-click on Default Web Site. 
  3. Select Properties from the menu. 
  4. Click on the Directory Security Tab. 
  5. Click on the Edit button in the Anonymous access and authentication control box. 
  6. Click on the Edit button in the checked Anonymous access section of the Authentication Methods window. 
  7. Click Browse in the Anonymous Users window and select Administrator. 
  8. Enter the local machine's administrator password.
    Note: You may have to uncheck the "Allow IIS to control password" box.
  9. Allow the action to cascade to all of the other relevant web sites, (ie ChemReg, ChemInv, etc.)
