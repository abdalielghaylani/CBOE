Preferences

The Preferences dialog box allows you to specify the searching preferences and how the search results are displayed. These settings are global for a user throughout an application. The settings can be different in different applications.

To access the Preferences dialog box:

  1. Click Preferences.
    The Preferences dialog box appears.
  2. What would you like to do?
    Specify whether results are displayed in a form or a list
    Specify the number of records displayed in List View
    Specify whether charges or radicals specified on atoms in the query must match those in the target
    Require that the reaction centers in the query must overlap the reaction centers in the target.
    Specify Exact Structure Searching Options
    Specify how similar structures need to be when performing a similarity search
    Reset all values to selected and the similarity search value to 90
    Specify whether or not you want a structure search to consider stereochemistry

Specify whether results are displayed in a form or a list
Click one of the Display Preferences buttons. Choose:

  • Form to view one record at a time.
  • List to display several records. The number displayed in list view is controlled by the ChemOffice Enterprise administrator. The default is 5 records.

Specify the number of records displayed in List View
Do one of the following:

  • Enter the maximum number of records to be displayed in the Number box.
  • Select All Records to display all the records in the database in List view.

Specify whether charges or radicals specified on atoms in the query must match those in the target
Select the desired options:

  • Hit any Charge on Heteroatom - allow uncharged atoms in the query to match charged atoms in the target (Note: charged atoms in the query must always match charged atoms in the target regardless of this setting).
  • Hit any Charge on Carbon - allow uncharged carbon atoms in the query to match charged carbon atoms in the target (Note: charged carbon atoms in the query must always match charged carbon atoms in the target regardless of this setting).
The valence of a charged atom is taken to be the valence of the isoelectronic neutral atom

Require that the reaction centers in the query must overlap the reaction centers in the target
Select Reaction Query Hit Reaction Center.

Specify Exact Structure Searching Options
Select the appropriate checkbox next to Permit Extraneous Fragments in Full Structure or Reaction searches.
If Permit Extraneous Fragments (for structure or reaction searching) is selected, the search will allow hits to contain molecular fragments in addition to that which was hit by the query. 

Specify how similar structures need to be when performing a similarity search
In the Similarity search box, type a value between 0 and 100. A lower value requires the target to be less similar.

Reset all values to selected and the similarity search value to 90
Click Use Defaults.

Specify whether or not you want a structure search to consider stereochemistry
Click one of the following:

  • Match Tetrahedral Stereo to recognize tetrahedral chirality.
    Note: Keeping this flag on greatly decreases the speed of searching the registry. If this type of duplicate checking is not important to you we recommend turning this option off.
  • Match Double Bond Stereo option to recognize cis/trans double-bond geometry

When either of these options are selected, the search includes any stereo-chemistry indicated.

NOTE: Stereochemistry is perceived in the query from (a) hashed or wedged bonds at a tetrahedral 
center, and (b) unsymmetrical substitution at a C=C double bond.
