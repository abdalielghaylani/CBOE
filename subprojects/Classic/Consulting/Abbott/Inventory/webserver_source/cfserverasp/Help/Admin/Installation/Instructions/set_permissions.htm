Set Admin Permissions

  1. Go to Start > Programs > Administrative Tools > Internet Services Manager .
  2. Expand the node with the name of your machine.
  3. Expand the Default Web Site node.
  4. In the Default Web Site node, right click on CFServerAdmin and select Propertiesfrom the drop down menu.
  5. Select the Directory Security tab.
  6. Under Anonymous Access and Authentication Control, click the Edit button.
  7. Make sure the Allow Anonymous Access check box is selected and click the Edit button.
  8. Change the Username to ADMINISTRATOR (or the name of a local administrator) and set the Password to the for the local machine.
    Note: It is necessary to deselect the Allow IIS to control password check box, in order to change the password.

Note: All virtual directories default to using the Administrator user for the user name and password. If you would like to change this username, or if the Administrator does not have administrator privileges, please be sure to change the username and password to a user with the appropriate privileges. Exit any open windows.
