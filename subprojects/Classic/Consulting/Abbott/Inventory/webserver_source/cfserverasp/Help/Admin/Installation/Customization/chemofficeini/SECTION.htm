SECTION

Example: SECTION= CO_WEBSERVER_STANDARD

Description: The section that the application should be listed under onthe ChemOffice WebServer start page.

Default: Depends on the application

Options: 

Section Value Location on Startpage
CO_WEBSERVER_STANDARD far left column
CO_WEBSERVER_ULTRA center column
ENTERPRISE center column
GLOBAL_LOGIN center column
WIZARD far right column

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
