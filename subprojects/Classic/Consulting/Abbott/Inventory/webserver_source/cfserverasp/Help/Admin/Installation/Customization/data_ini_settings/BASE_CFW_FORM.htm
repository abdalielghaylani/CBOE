Description: ChemFinder connection names specified in the chem_connections section of the globals section reference connection sections elsewhere in the ini file. If a table contains chemical information stored by ChemFinder then the CHEM_CONNECTION name is referenced by the table definition. The keys specify data necessary to create the ChemFinder connection. 

The following settings are used in any ChemFinder Connection section:

  • STRUC_ENGINE
  • STRUC_FORM_NAME
  • STRUC_DB_PATH
  • STRUC_TABLE_NAME

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
