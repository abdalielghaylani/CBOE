Saving and Restoring Queries

You can save your search criteria and recall it for a search at a later time.

To Save a Query:

  1. On the Results form, click Save Query.
    The following dialog box appears:
  2. Type the name of your query and click OK.
    Your query is saved and the Saved Queries button appears.

To Restore a saved query:

  1. Click Saved Queries.
    The list of saved queries appears.
  2. Select the query to restore and click .
    The query is restored to the Query Input Form.
