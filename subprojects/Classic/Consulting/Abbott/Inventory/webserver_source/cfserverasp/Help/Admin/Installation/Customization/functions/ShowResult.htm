Subroutine

 
      ShowResult 
      dbkey,formgroup,rsname,table.field,style,width,height

Description

Shows results from a recordset in table format

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

rsname = <name of recordset object crested at start of tile>  

table.field = <table and field name using dot syntax>

style = RAW|PLAIN|TEXT|HIDDEN|TEXTAREA|PICTURE
         raw : raw HTML output and a non-breaking space
         plain : raw HTML output
         text : output in a text box
         hidden : output in a hidden field
         textarea : output in textarea
         picture : output in a structure window (indicates that the field is a LOB containing an image) 

The PICTURE style should be in the following format:
PICTURE:MIME_TYPE:LOB_DISPLAY

where:
  • MIME_TYPE – the mimetype of the image e.g. image/gif; image/jpeg, etc…
  • LOB_DISPLAY – is either IN_LINE (the default) or NEW_WINDOW. IN_LINE displays the image in the calling window, NEW_WINDOW displays a link that when clicked displays the image in a new window
For Example: 
PICTURE:image/gif:IN_LINE

width = <width of text box or rows of textarea, 0 if not applicable>

height = <height of textarea, 0 if not applicable>

Example

  • Display MolTable.Molname in raw HTML output and a non-breaking space
    ShowResult dbkey,formgroup,BaseRS,"MolTable.Molname","raw",0,0

     
  • Display Graphics.Graphic as a gif in the calling window
    ShowResult dbkey,formgroup,GraphicsRS,"Graphics.Graphic","PICTURE:image/gif:IN_LINE", "80", "80"
