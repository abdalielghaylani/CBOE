Upgrading ChemOffice WebServer from 7.1 to 7.2

Please follow the following steps, if you are upgrading from 7.1 to 7.2:

  1. Uninstall Webserver Core 7.1 and Registration System 7.1 (if app.) in the following order:
    1. Registration System 7.1:
      1. Backup Regdb.mst, Regdb.msi files and cfserver.ini configuration file, custom databases and customized forms.
      2. Backup the chemoffice.asp file.
      3. Run the uninstall from add/remove programs.
    2. Webserver Core 7.1:
      1. Run the uninstall from add/remove programs.
  2. Upgrade Webserver Core to 7.2:
    1. Run installer.
    2. Go to Components Services>Com+ Objects and delete CSDO and CSDO7 COM objects.
    3. Open an execute the following file:
      ChemOffice\cs_security\config\Oracle_Install_Scripts\create_blank_cs_security_DB\alter_cs_security_for_global_security.cmd
    4. Update the cs_security.udl with the current Oracle Service name.

Step 3: Register CSDO 7.1 Objects with COM+ Component Services
