Exact Searching

Exact searching retrieves only structures exactly as drawn (taking stereochemistry into account). Isotopes are not recognized as hits.

To perform an exact structure search:

  1. On the Results Form, click New Query.
    A blank Search form appears.
  2. Draw a structure.
  3. From the Search Type list, select Exact Structure.
     
  4. Click Search.
    One record retrieved.
