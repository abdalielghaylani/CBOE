ChemDraw Shortcuts

ChemDraw has a number of built in hot keys which allow users to perform common functions quickly. 

Drawing Techniques

If you want to ... Here's How
Draw a first bond Drag to draw the bond
Add a bond to a structure Click an atom or drag from an atom
Label and atom using a bond or ring tool Double-click an atom (triple-check to repeat last label)
Label and atom using a text tool Click an atom (double-click to repesat last label)
Label and atom using a HotKey Point at an atom and press a HotKey (e.g., "c")
Draw a double bond (solid, dashed, bold only) Select bond type from Shortcut menu
Draw a triple bond (solid only) Select bond type from Shortcut menu
Reduce bond order using Eraser tool Select bond type from Shortcut menu
Change double bond alignment Click center of double bond using any bond tool
Change double bond type Click center of double bond using different bond tool
Move an atom SHIFT+drag an atom using a bond or selection tool
Change bond type Click center of double bond using different bond tool
Change bond layering using bond tool Double-click center of bond you want forward
Draw a ring Click or drag using a Ring tool
Create a spiro-linklage Click or drag from an atom usign a Ring tool
Crate a fused ring Click a bond using a Ring tool
Change size of non-bond object SHIFT+drag starting point of object (e.g., arrowhead)

Hard Coded HotKeys

Atom and Bond

HotKey Function
Enter or Return Opens the atom label text box.
space Removes an atom label.
/ (slash) Displays the Atom Properties Dialog Box when pointing to an atom or the Bond Properties Dialog Box when potining to a bond.
= (equal) Displays the list nickname dialog box.
. (period) Adds an attachment point.
' (apostrophe) Adds an atome number.
Delete Deletes selected objects.
1 When pointing to a bond, changes the bond to single.
2 When pointing to a bond, changes the bond to double.
3 When pointing to a bond, changes the bond to triple.
f Send bond to which you are pointing to the front of the drawing.
b Send bond to which you are pointing to the back of the drawing.
c Centers the double-bond to which you are pointing.
l Positions the double bond to which you are pointing to the left.
r Positions the double bond to which you are pointing to the right.

Working with Selections

If you want to ... Here's How
Add items to a selection SHIFT+select items not currently selected
Remove items from a selection SHIFT+select items currently selected
Duplicate a selection ALT+drag Selection Rectangle
Constrain dragging to X or Y direction SHIFT+drag Selection Rectangle
Duplicate and constrain dragging SHIFT+ALT+drag Selection Rectangle
Switch between selection and another tool CTRL+ALT+TAB
Resize a selection excluding atom labels Drag Resize handle of Selection Rectangle
Resize a selection including atom labels ALT+drag Resize handleof Selection Rectangle
Rotate selection excluding atom labels Drag Rotation handle of Selection Rectangle
Rotate a selection including atome labels ALT+drag Rotation handle of Selection Rectangle
Distort aspect ratio of selection SHIFT+drag Resize handle of Selection Rectangle
Open rotate or Resize dialog box Double-click Resize handle or Rotation handle
