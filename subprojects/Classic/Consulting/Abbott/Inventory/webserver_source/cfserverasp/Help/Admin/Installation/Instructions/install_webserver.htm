Install ChemOffice Enterprise

To install the ChemOffice Enterprise:

  1. Run autorun.exe (found in /WebServer_Core_2005 ) on the installation CD.
    Note: If you are not using the Autorun, you can run the installer separately.  Check the minimum requirements and double-click on the file called setup.exe (found in the /Installers/Webserver) on the installation CD.  See the ChemOffice Enterprise readme for more details about running the ChemDraw Ultra and Oracle Cartridge installers separately from the Autorun.
  2. Enter the administrator username and password you would like used for setting up the ChemOffice Enterprise components. 
    Note: If you would like to create a new account, the installer will allow you to do so.
  3. Click the Install button.
  4. Click Finish .
  5. Restart your computer.
