Substructure Searching

A substructure search is the default structure search type. Substructure searching finds structures that contain the query and any additional attachments at the open positions. You can attach different features, such as atom lists and variable bond types, to a structure using ChemDraw plug-in, to perform a narrower or broader search.

For more information about what query features you can use and how they affect a search, do the following:

  • See Appendix A of the ChemDraw manual.
  • See the ChemDraw Quick Reference Guide.

To perform a substructure search:

  1. Click New Query.
  2. Draw pyrrole.
  3. From the Search Type menu, choose Substructure.
  4. Click Search.
    Seven records are found because the pyrrole moiety is found in seven structures in the database.
