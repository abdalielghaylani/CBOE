Viewing Results in Details View

Details View displays all of the information available in the database about one record. The result details view display varies from application to application. See your application’s user guide for more information.

To display a record in Details View, from the Results List Form:

  • Click Show Details.
    The record you chose appears in Results View.

In Results View, you can scroll through the record set using the browse buttons in order to view additional hits.

    To display the record set in List View:

    • Click  Return to List.
