Tips for Installing on Windows 2003 Server

Windows 2003 Server requires a few additional steps to be taken before concluding.  If you are installing on Windows 2003 Server, please do the following:

  1. Add the "Application Server" role which installs IIS 6.0 (No longer installed by default).
    1. Go to START>Settings>Control Panel.
    2. Double-click Add/Remove Programs.
    3. Click Add/Remove Windows Components (make sure the checkmark is in a white box, indicating all components will be installed).
    4. Click Next.
       
  2. Enable ASP, Server Side Includes. 
    1. Go to START>Programs>Administrative Tools>IIS
    2. Expand  the node with the name of your computer.
    3. Click on the Web Service Extentions node.
    4. Highlight Active Server Pages.
    5. Click the Allow Button.
       
  3. "Enable Parent Paths" for asp application virtual folders.
    Note: This used to be the default in IIS 5.0 but it changed in IIS 6.0.  To change it: . 
    1. Go to START>Programs>Administrative Tools>IIS Management Console
    2. Expand the node with the name of your computer.
    3. Exapnd the Websites node.
    4. Expand the Default Website node.
    5. Left click Sample.
    6. Select Properties.
    7. Select the Virtual Directory Tab.
    8. Click the Configuration button.
    9. Select the Options Tab.
    10. Select the Enable Parents Folder checkbox.
    11. Click OK.
