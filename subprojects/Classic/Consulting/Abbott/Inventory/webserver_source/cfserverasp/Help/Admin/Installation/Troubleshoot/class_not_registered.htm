Class Not Registered (on login)

The Problem: In some Oracle 9i installations users are unable to log into any Oracle based applications using an OLEDB connection. The symptom is a "Class Not Registered" error message at the log in screen.

The Solution: Change the security on the Oracle Home directory (the default is C:/Oracle/Ora92). Specifically, you must give the user named "Authenticated User" read and execute permissions.
