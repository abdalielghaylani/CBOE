Types of Searches

ChemOffice Enterprise forms contain several different types of data. Since records can be searched for by almost any field, ChemOffice Enterprise allows the following types of searches:

  • Text
  • Structure
  • Formula
  • Numeric
  • Combination

You can set further preferences to allow or disallow valence modifications during the search in the Preferences dialog box. For more information on setting preferences, see Setting User Preferences.
