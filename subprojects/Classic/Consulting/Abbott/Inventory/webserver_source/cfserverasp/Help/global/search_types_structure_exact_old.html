Exact Structure Searching

Exact structure searching retrieves structures as drawn, with no additional attachments. Isotopes are recognized as hits.   

To perform an exact structure search:

  1. On the Results Form, click New Query.
    A blank Search form appears.
  2. Draw a structure, for example, Chlorobenzene.
  3. From the Search Type list, select Exact Structure.

  4. Click Search.
    One record is retrieved.
