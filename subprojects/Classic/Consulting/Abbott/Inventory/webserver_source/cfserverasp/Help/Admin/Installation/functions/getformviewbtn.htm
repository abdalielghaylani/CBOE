Subroutine/Function

 
getFormViewBtn file_path,form_name,index_value

Description

Link from one item in list view to the corresponding function in form view

Parameters

file_path = <path to GIF>
         relative to cfserverasp/source/graphics

form_name = <name of form to link to>
         relative to <application>/<dataview>/<dataview>

index_value = <index value of item to display>
