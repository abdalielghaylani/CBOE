Saving Queries

Users can save a query to be recalled at a later time. This is often useful when you expect to run the same search more than once.

To Save a Query:

  1. After performing a search, select Save from the Queries Menu.
  2. Enter a name and description for the query.
  3. Select the Make Public checkbox if you would like other users to be able to access the query.
  4. Click OK.
    Your query is saved. For information about retrieving this query, please see Restoring Queries.
