ChemOffice Webserver
ChemOffice WebServer for use with Win 2000 

Register CSDO7.1 Objects with COM+ Microsoft Transaction Services

Please follow the following steps:

  1. Go to Start > Programs > Administrative Tools > Component Services.
  2. Expand Component Services>Computers>My Computer node.
  3. Expand COM+ Applications node.
  4. Click on COM+ Applications folder and right click.
  5. Select New > Application from the menu.
  6. Click Next
  7. Click the Create an Empty Application button.
  8. Enter the name: CSDO71 in the text field.
  9. Select Server application option.
  10. Click the Next button.
  11. Select this user radio button.
  12. Under User: enter “Administrator”
  13. Under Password: enter the password for the local Administrator account
  14. Confirm the password.
  15. Click Next .
  16. Click the Finish button.
  17. The CSDO71 Package should appear in the COM+ Applications folder.
  18. Expand the CSDO71 Package and left click on the Components folder.
  19. Right click on the selected Components folder and choose New > Component.
  20. Click Next
  21. Click the Import Component(s) that are already registered button.
  22. From the Components on My Computer list, select the following components:
    1. CSDO71.Connection
    2. CSDO71.Request
  23. Click Next.
  24. Click the Finish button. The CSDO71 components should appear in the Components folder.
  25. Exit program
