Text Searching

Information can be entered using wildcard ("*") characters to broaden the search results or "=" to narrow to exact wording.

NOTE: You cannot enter "AND", "OR", or "NOT" into the text searches.

The following table shows examples of text queries.

Text Entered Results Found
ben Finds records that start with "ben", but may have any characters after. Finds benzene and benzofuran, but not bromobenzene.
*ben Finds records that have any character before "ben" and any character after "ben" in the name field. Finds benzene, bromobenzene, and benzofuran.
=Benzene Finds the exact text typed. Finds Benzene.
Ben*chl Finds compounds with "ben" at the beginning, any character after "ben" and before "chl" and any character after "chl". Finds Benzyl chloride.
