STRUC_DB_PATH

Example: STRUC_DB_PATH = C:\Inetpub\wwwroot\ChemOffice\sample\database\sample.mst 

Description: The path to the mdb file that ChemFinder uses. Even if you are storing all data in Oracle or SQLServer, ChemFinder still uses an MDB file to link tables.

Default: C:\Inetpub\wwwroot\ChemOffice\sample\database\sample.mst

Options: Relative or absolute path to the ChemFinder mdb file. 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
