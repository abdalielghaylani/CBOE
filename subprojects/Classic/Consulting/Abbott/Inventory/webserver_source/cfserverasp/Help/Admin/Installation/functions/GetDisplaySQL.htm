Subroutine

 
GetDisplySQL(dbkey,formgroup,sql_get_fields,sql_base_table,sql_info,uniqueid,sql_options)

Description

Create SQL string to get a recordset

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

sql_get_fields = <the fields to return>

sql_base_table = <the base table to search>

sql_info = <additional sql for statement>

uniqueid = <used to create recordset>

sql_options = SHOWSQL|SQL_WHERE|SQL_FULL|SQL_NO_DISTINCT
         description of type of sql in sql_info
         showsql : response.write resulting in SQL; use for debugging
         sql_where : WHERE clause
         sql_full : full SELECT statement
         sql_no_distinct : removes distinct/distinct row from SQL statement

Example

 
sql=GetDisplaySQL(dbkey,formgroup,"MolTable.*","MolTable","",BaseID,"")

Note

 Once you have a data connection and SQL statement, the following creates the recordset: 
Set BaseRS = DataConn.Execute(sql)
