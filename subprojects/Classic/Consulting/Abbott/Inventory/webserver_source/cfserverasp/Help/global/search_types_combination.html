Combination Searching

You can perform a search that has any combination of the previously mentioned types of searches and (for a complete list of types of searches, see Types of Searches) obtain a Boolean AND search for all the entered data.

NOTE: OR and NOT searches are not supported in the ChemOffice Enterprise.

To perform an example combination search:

  1. Type *ene in the Molecule name field, 100-200 in the MW field.
  2. Draw a bond with a chloride.
  3. From the Search Type list, select Substructure.
  4. Click Search.

    This combination search returns 1 result that matches all the entered criteria.
