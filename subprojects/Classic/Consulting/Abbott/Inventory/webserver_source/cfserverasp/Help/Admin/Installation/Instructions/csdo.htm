Register CSDO 7.1 Objects

To register the CSDO Objects:

  1. Go to Start > Programs > Administrative Tools > Component Services .
  2. Expand the Component Services > Computers > My Computer node.
  3. Expand the COM+ Applications node.
  4. Left-click on the COM+ Applications folder.
  5. Right-click on the COM+ Applications folder.
  6. Select New > Application from the menu.
  7. Click Next .
  8. Click the Create an Empty Application button.
  9. Enter the name: CSDO71 in the text field.
  10. Select the Server application option.
  11. Click the Next button.
  12. Select the This user radio button.
  13. Under User: enter Administrator (or the name of a local administrator)
  14. Under Password: enter the password for the local Administrator account
  15. Confirm the password.
  16. Click Next .
  17. Click the Finish button.
    The CSDO71 Package should appear in the COM+ Applications folder.
  18. Expand the CSDO71 Package and left-click on the Components folder.
  19. Right click on the selected Components folder and choose New > Component .
  20. Click Next
  21. Click the Import Component(s) that are already registered button.
  22. From the Components on My Computer list, select the following components:
    • CSDO71.Connection
    • CSDO71.Request
  23. Click Next .
  24. Click the Finish button.
    To select more then one component at once, click on the first component, then hold down the CTRL key while clicking on the second component.
    The CSDO71 components should appear in the Components folder.
  25. Exit the Component Services window.
