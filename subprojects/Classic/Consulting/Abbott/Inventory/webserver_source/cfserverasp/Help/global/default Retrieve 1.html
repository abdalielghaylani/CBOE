ChemOffice Enterprise Help

ChemOffice Enterprise is a World Wide Web (www) chemical information server. Through a web browser, you can use the ChemOffice Enterprise web applications to search databases for chemical information.

This help is designed to give the user general information and tips dealing with searching in ChemOffice Enterprise applications.
