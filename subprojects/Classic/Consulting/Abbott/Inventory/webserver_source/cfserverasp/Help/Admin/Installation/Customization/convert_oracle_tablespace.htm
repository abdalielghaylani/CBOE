TableSpaces in Oracle

  1. Browse to:
    START>Programs>Oracle>Enterprise Manager>Console
  2. Enter a user name and password (i.e. system, manager ).
  3. Expand Tablespaces .
  4. Look and see if there is a tablespace with the same name as the mdb file you are going to convert. If there is, you can do two things:
    • Rename the mdb file.
    • Remove the tablespace:
      If the tablespace was generated from an earlier conversion of the same mdb file:
      1. Delete the table space.
      2. Go to the oracle home directory/databases directory (i.e. if your setup uses all the defaults this is orant/databases) and remove the physical table space file.
      3. Open the Security manager and remove the user that has the same name.
