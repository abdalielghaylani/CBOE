ChemOffice Webserver
ChemOffice WebServer for use with Win 2000 

Set Admin Permissions

  1. Go to Start > Programs > Administrative Tools > Internet Services Manager. <
  2. Expand the Internet Information Server folder > yourservername > Default Web Site.
  3. In the Default Web Site folder, right click on CFServerAdmin and select Properties from the drop down menu.
  4. Select the Directory Security tab.
  5. Under Anonymous Access and Authentication Control, click the Edit button.
  6. Make sure the Allow Anonymous Access check box is selected and click the Edit button.
  7. Change the Username to ADMINISTRATOR and set the Password to the Administrator password for the local machine.

    NOTE: It is necessary to deselect the Allow IIS to control password check box, in order to change the password.

Note: All virtual directories default to using the Administrator user for the user name and password.  If you would like to change this username, or if the Administrator does not have administrator privileges, please be sure to change the username and password to a user with the appropriate privileges.
