Global Login

ChemOffice Enterprise uses a global login interface to manage user information. After logging in once with your user name and password, users have access to all ChemOffice Enterprise applications that they have permissions to, without having to login to each application separately.

After logging in, a cookie is written to the client machine, holding login information. The cookie expires after 25 minutes of inactivity. Therefore, login information is recognized even after a browser is closed.  

What would you like to do?

Login to the ChemOffice WebServer
Manage Users and Roles for the applications
