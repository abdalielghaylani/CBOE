Subroutine

ShowStrucInputField 
      dbkey,formgroup,table.field,fieldtype,width,height,search_options,display_options

Description

Shows the ChemDraw plugin box for structure input and displays options for search

Parameters

dbkey = <dataview name>  

formgroup = <formgroup name>

table.field = <table and field name using dot syntax>  

fieldtype = <indicates chemical information>  

width = <width of plugin box >  

height = <height of plugin box>

search_options = All_Options|SubStructure|Exact|Tanimoto 
         All_Options: show drop down list with options
         Substructure: always substructure search
         Exact: always exact search
         Tanimoto: always similarity search

display_options = None|Select_List|Select_List_Edit
         None: no display type needed
         Select_list: shows chocies - substructure, exact, similarity
         Select_List_Edit: structure button

Example

ShowStrucInputField 
      dbkey,formgroup,"MolTable.Structure","5","340","AllOptions","Select_List"
