Saving Hit Lists

Save Hit List allows the user to save a hit list and recall it at a later time.

To Save a Hit List:

  1. After performing a search, select Save from the Hit Lists Menu.
     
  2. Enter a name and description for the hit list.
    Note: User can make save hit list as a public, by selecting Make Public check box.
  3. Click OK.
    Your hit list is saved. For information about retrieving this hit list, please see Restoring Hit Lists.
