DeleteReservation

Deletes a single reservation.

Syntax

/cheminv/api/DeleteReservation.asp?ReservationID=adNumeric&ContainerID=adNumeric

Inputs

ReservationID

The reservation_ID associated with the reservation made previously and now needs updating.  

ContainerID

The Container_ID value of the container being checked out .

Outputs

None

Example

A user wants to delete the reservation (ID = 5) he made for the compound with ID=23745

/cheminv/api/DeleteReservation.asp?ReservationID=5&ContainerID=23745
