Inventory Enterprise User's Guide

Minimum Requirements

Server

Operating System

  • Windows 2000 Server, OR Windows 2000 Advanced Server SP2

Oracle and Microsoft

  • For Oracle 8i Installations
    • Oracle OLEDB Provider 8.17.00
    • Oracle ODBC Driver 8.01.07
        
  • For Oracle 9.2.0.4 or 9.2.0.5 Installations
    • Oracle OLEDB Provider 9.2.0.4
    • Oracle ODBC Driver 9.2.0.4
       
  • Microsoft Access 2000 or XP (required for report writer)
     
  • MSXML 4 (The MSXML 4 installer is found at /installers/msxml4/msxml.msi on the installation CD, or find it on the Microsoft website)  

CambridgeSoft Software

  • CambridgeSoft ChemDraw Pro Plugin/ActiveX Control 8.0.7, 9.0 
  • CS_Security schema (see the ChemOffice Enterprise Admin Guide for more details about setting up Global Login)
  • CS Oracle Cartridge version 9.0

Client

  • IIS and IE security patches
  • MSXML3 (installed with Windows 2000 but must be added for WinNT and Win 98 clients)
  • CambridgeSoft ChemDraw Pro Plugin/ActiveX Control 8.0.7, 9.0 
  • CambridgeSoft ChemFinder 8.0.7, 8.0 (for use with InvLoader)

  • A resolution of 1024 x 768 is recommended

Note: All ChemOffice Enterprise applications must be installed by an administrator (i.e. the user must be logged into the server as the administrator of the machine).
