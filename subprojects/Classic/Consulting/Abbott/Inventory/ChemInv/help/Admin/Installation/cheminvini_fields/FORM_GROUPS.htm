FORM_GROUPS

Example: FORM_GROUPS= base_form_group,containers_form_group,containers_np_form_group,substances_form_group,add_record_form_group,gs_form_group,global_substanceselect_form_group

Description: Names of form group definition sections in the ini file. This must be comma delimited, with no spaces. If the form group names are not here, they will not be loaded by the application. Form_groups specify a sub set of information including the input and result forms to use, the fields that can be searched, the table group to use for searching as well as additional information that will be described in later sections. Form_groups are referenced by hyperlinks from the main.asp page and carried throughout all functions and subroutines within ChemOfficeWebserver .

Default: base_form_group,containers_form_group,containers_np_form_group,substances_form_group,add_record_form_group,gs_form_group,global_substanceselect_form_group

Options: Comma delimited list of form group definitions found in the ini file.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
