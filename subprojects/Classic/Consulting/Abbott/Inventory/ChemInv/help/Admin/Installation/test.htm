Inventory Enterprise User's Guide

Test the Installation

The Inventory Manger user's guide can be accessed at:
http:// <IIS_SERVER_NAME>/Chemoffice/cheminv/help/
where IIS_SERVER_NAME is the name of the server where IIS is installed.

Perform the following steps to ensure Inventory Enterprise was successfully installed:

  1. Login in to the ChemOffice WebServer using the following username and password:
    • Username: invadmin
    • Password: invadmin
  2. Add a compound.
    See Inventory Enterprise User's Guide>Substance Management>Creating a New Substance in the User's Guide.
  3. Add a container.
    See Inventory Enterprise User's Guide>Container Management>Creating a New Container in the User's Guide.
  4. Add a compound from ChemACX to a container.
    See Inventory Enterprise User's Guide>Container Management>Creating a New Container>Substance Tab in the User's Guide.
  5. Add a compound from Registration System to a container.
    See Inventory Enterprise User's Guide>Container Management>Creating a New Container>Substance Tab in the User's Guide.
  6. Create a Barcode Report.
    See Inventory Manager User's Guide>Container Management>Viewing the Contents of a Container>Available Functions>Print Label/Report in the User's Guide.
