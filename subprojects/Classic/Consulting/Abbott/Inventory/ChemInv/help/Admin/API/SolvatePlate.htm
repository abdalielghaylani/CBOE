SolvatePlate

Adds the specified amount of solvent to a plate, or plates.

Syntax

/cheminv/api/SolvatePlate.asp?PlateIDList=adVarChar&SolventIDList=adVarChar&SolventVolumeList=adVarChar&SolventVolumeUnitIDList=adVarChar&ConcentrationList=adVarChar&ConcentrationUnitIDList= adVarChar

Inputs

Plate_IDList

List of plateIDs (IDs are separated by a comma). 

SolventIDList

List of IDs associated with solvents.  There is one solvent for each plate.  Separate solvent IDs by a comma in the order they appear in Plate_IDList.

SolventVolumeList

The amount of solvent added.  There is one volume for each plate.  Separate solvent volumes by a comma in the order they appear in Plate_IDList. 

SolventVolumeUnitIDList

The units for the amount of solvent added.  There is one volume Unit for each plate.  Separate volume units by a comma in the order they appear in Plate_IDList.     

ConcentrationList

The concentration of solvent added.  There is one concentration for each plate.  Separate concentrations by a comma in the order they appear in Plate_IDList. 

ConcentrationUnitIDList

The units for the concentration of solvent added.  There is one concentration Unit for each plate.  Separate concentration units by a comma in the order they appear in Plate_IDList.        

Outputs

None

Example

A user wants to add a solvent with ID 120 to the plate with ID P1234.  10mL of solvent is added.  mL has a ID of 4. 

/cheminv/api/SolvatePlate.asp?PlateIDList=P1234&SolventIDList=120&SolventVolumeList=10&SolventVolumeUnitIDList=4
