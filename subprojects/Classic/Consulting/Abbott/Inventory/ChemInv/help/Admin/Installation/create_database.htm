Create the Inventory Enterprise Database

The files neccessary to set up the Inventory Enterprise database can be installed directly from the Inventory Enterprise installer.  If you choose not to set up the Inventory Enterprise schema at that time, follow the follinw instructions.  If you have already set up the Inventory Enterprise schema, move on to Step 4: Required Configuration.

If you have not yet set up the Inventory Enterprise schema:

  1. Browse to <webroot> /cheminv/config/oracle_intall_scripts/
  2. Double click the file OracleSetup.exe
    This file will lead you through setting up the Invnetory Manager schema.  If you have a previous installation of Inventory Enterprise, be sure to Upgrade instead of Creating a new schema.
