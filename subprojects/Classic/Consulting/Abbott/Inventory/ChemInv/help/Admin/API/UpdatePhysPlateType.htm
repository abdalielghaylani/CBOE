UpdatePhysPlateType

Updates a physical plate type according to the inputs.

Syntax

/cheminv/api/UpdatePhysPlateType.asp?PhysPlateID=adNUmeric&GridFormatID=adNumeric&Name=adVarChar&RowCount=adNumeric&ColCOunt=adNumeric&WellCapacity=Int&CapacityUnitID=adNUmeric[&GridFormatID=adNumeric&RowPrefix=adVarChar&ColPrefix=adVarChar&RowUseLetters=adNumeric&ColUseLetters=adNumeric&NameSeparator=adVarChar&NumberStartCorner=adNumeric&NumberDirection=adNumeric&PhysPlateName=adVarChar&IsPreBarcoded=adVarChar&SupplierID= adNumeric]

Inputs 

PhysPlateID

The ID for the physical plate type being updated.

GridFormatID

The ID for the grid format.

Name

The name of the physical plate type.

RowCount

The number of rows in the physical plate type.

ColCount

The number of columns in the physical plate type.

WellCapacity

The capactity of the wells in the physical plate type being defined.

CapacityUnitID

The ID for the units used to measure the well capacity.

[RowPrefix]

A prefix added to the start od the row ID.

[ColPrefix]

A prefix added to the start od the column ID.

[RowUseLetters]

Use letters instead of .

Outputs

None
