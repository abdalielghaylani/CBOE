DEFAULT_CONTAINER_STATUS

Example: DEFAULT_CONTAINER_STATUS= 1

Description: Indicates what container status should appear as the default in the New Container dialog.  The value must be a valid container status ID in the INV_CONTAINER_STATUS table. 

Default: 

Options: A valid container status ID 

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
