CreateContainer

Creates a new container and returns its Container_ID.

Syntax

/cheminv/api/CreateContainer.asp?LocationID=adNumeric&UOMID=adNumeric&QtyMax=adDouble&QtyInitial=adDouble&ContainerTypeID=adNumeric&ContainerStatusID=adNumeric[&Barcode=adVarChar&CompoundID=adNumeric&RegBatchID=adVarChar&ExpDate=adDBTimeStamp&MinStockQty=adDouble&MaxStockQty=adDouble&ContainerName=adVarChar&ContainerDesc=adVarChar&TareWeight=adDouble&UOWID=adNumeric&Purity=AdDouble&UOPID=adNumeric&Concentration=adDouble&UOCID=adNumeric&Grade=adVarChar&Solvent=adVarChar&Comments=adVarChar&SupplierID=adNumeric&SupplierCatNum=adVarChar&LotNum=adVarChar&DateProduced=adDBTimeStamp&DateOrdered=adDBTimeStamp&DateReceived=adDBTimeStamp&ContainerCost=adNumeric&OwnerID=adVarChar&CurrentUserID=adVarChar&PONumber=adVarChar&ReqNumber=adVarChar&NumCopies=adVarChar]

Inputs

LocationID

The Location_ID value of the location where the container is stored.

UOMID

The unit of measure to be used for this container as a UNIT_ID value. Use ShowContainerUnits to see valid UNIT_ID values.

QtyMax

Maximum capacity of the container in the units defined by UOMID.

QtyInitial

Quantity of material present in the container at creation time in the units defined by UOMID.

ContainerTypeID

Type of container as a Type_ID value. Use ShowContainerTypes to see valid Type_ID values.

ContainerStatusID

Status of container as a Status_ID value. Use ShowContainerStatus to see valid Status_ID values. 

[Barcode]

A unique alphanumerical identifier used to print a container barcode.

[CompoundID] 

Compound_ID value associating the substance in the container with a preexisting inventory compund.

[RegID] 

A registry number from the Registration System.

[BatchNumber] 

A batch number (associated with the given RegID) from the Registration System.

[ExpDate]

Content's expiration date as date-time literal (ie. 01-27-2001) 

[MinStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue a re-stock warning.

[MaxStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue an over-stock warning.

[ContainerName]

The name of the container.  If empty, defaults to the system's internal ContainerID.

[ContainerDesc]

A description of the container.

[TareWeight]

The tare weight of the physical container in the units of UOWID.

[NetWeight]

The net weight of the physical container in the units of UOWID.

[FinalWeight]

The final weight of the physical container in the units of UOWID.

[UOWID]

Unit of weight value as a Unit_ID value from the Units table.  ShowContainerUnits to see valid UNIT_ID values.

[Purity]

The purity of the substance designated by CompoundID in units of UOPID.

[UOPID]

Unit of purity value as a Unit_ID value from the Units table.  ShowContainerUnits to see valid UNIT_ID values.

[Concenctration]

The concentration of the substance designated by CompoundID in units of UOCID.

[Density]

The density of the substance.

[UOCID]

Unit of concentration value as a Unit_ID value from the Units table.  ShowContainerUnits to see valid UNIT_ID values.

[UODID]

Unit of density value as a Unit_ID value from the Units table.  ShowDensityUnits to see valid UNIT_ID values.

[Grade]

The quality or purity designation associated with the substance designated by CompoundID.

[Solvent]

The name of the substance's solvent.

[Comments]

Comments or instructions associated with this container. Maximum length is 4000 characters. It may be passed as form value to avoid truncation by querystring size limit.

[SupplierID]

Supplier_ID value associating the Supplier with a preexisting Supplier Name.  Use ShowSuppliers to see valid Supplier_ID values.

[SupplierCatNum]

The catalog number for the product assigned by the supplier.

[LotNum]

Supplier's Lot Number

[DateProduced]

The date the compound was produced as date-time literal (ie. 01-27-2001).

[DateOrdered]

The date the compound was ordered as date-time literal (ie. 01-27-2001).

[DateReceived]

The date the compound was received as date-time literal (ie. 01-27-2001).

[ContainerCost]

The unit cost of the container.  Currency is assumed to be USD.

[OwnerID]

User_ID value of the user who owns responsibility of the container.

[CurrentUserID]

User_ID value of the user currently responsible for the container.

[PONumber]

The PONumer under which the container was ordered.

[POLineNumber]

The POLine number for the container.

[ReqNumber]

The internal requisition number under which the container was ordered.

[NumCopies]

Number of container copies to create.

[Field_1]

Value of the customizable Field_1.

[Field_2]

Value of the customizable Field_2.

[Field_3]

Value of the customizable Field_3.

[Field_4]

Value of the customizable Field_4.

[Field_5]

Value of the customizable Field_5.

[Field_6]

Value of the customizable Field_6.

[Field_7]

Value of the customizable Field_7.

[Field_8]

Value of the customizable Field_8.

[Field_9]

Value of the customizable Field_9.

[Field_10]

Value of the customizable Field_10.

[Date_1]

Value of the customizable Date_1.

[Date_2]

Value of the customizable Date_2.

[Date_3]

Value of the customizable Date_3.

[Date_4]

Value of the customizable Date_4.

[Date_5]

Value of the customizable Date_5.

Outputs

Pipe (|) delimited list of new container IDs. For example: 100|101

Example

A user wants to create a container in Location 101 which is a flask (ID=13) with a max quantity of 200 ml (ID=3).  The container will initially have 100 ml in it and the status is In Transit (ID=6).    

/cheminv/api/CreateContainer.asp?LocationID=101&UOMID=3&QtyMax=200&QtyInitial=100&ContainerTypeID=13&ContainerStatusID=6
