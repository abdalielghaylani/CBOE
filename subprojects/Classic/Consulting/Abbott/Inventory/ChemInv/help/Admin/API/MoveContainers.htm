MoveContainers

Moves one or more containers to a new location.

Syntax

/cheminv/api/MoveContainers.asp?LocationID=adNumeric&ContainerList=List[&LocationList=List]
or
/cheminv/api/MoveContainers.asp?LocationID=adNumeric&LocationList=List[&ContainerList=List]

Inputs

LocationID

The Location_ID value of the destination location where the containers should be moved to.

ConatinerList

A list of containers to be moved. Use a comma to separate multiple Conatiner_ID values. Use a dash to express a range of Container_ID values.

[LocationList]

A list of Location_ID values. Use a comma to separate multiple Location_ID values. Use a dash to express a range of Location_Id values. All containers in the given locations will be moved.

Outputs

NumContainers

The number of containers moved.

Example

A user wants to moves the containers with ID 12345 and 12222 to the location with ID 111.

/cheminv/api/MoveContainers.asp?LocationID=111&ContainerList=12345,12222


A user wants to moves all of the containers in the location with ID 110 to the location with ID 111.

/cheminv/api/MoveContainers.asp?LocationID=111&LocationList=110
