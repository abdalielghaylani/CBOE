CreatePlateType

Creates a new plate type according to the inputs.  

Syntax

/cheminv/api/CreatePlateType.asp?PlateTypeName=adVarChar&MaxFreezeThaw= adNumeric

Inputs

PlateTypeName

The name of the new plate type. 

MaxFreezeThaw

The maximum numer of freeze thaw cycles for this plate type.   

Outputs

None

Example

A user creates a new plate format called NewPlateType which allows 5 freeze thaw cycles. 

/cheminv/api/CreatePlateType.asp?PlateTypeName=NewPlateType&MaxFreezeThaw= 5
