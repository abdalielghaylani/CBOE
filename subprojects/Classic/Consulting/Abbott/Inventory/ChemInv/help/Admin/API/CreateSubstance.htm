CreateSubstance

Creates a new substance and returns its Compound_ID. This function also checks for a duplicate substance in the database.

Syntax

/cheminv/api/CreateSubstance.asp?SubstanceName=adVarChar[&Structure=base64cdx&SubstanceName=adVarChar(255)&CAS=Text(15)&ACX_ID=Text(15)&Alt_ID_1=adVarChar&Alt_ID_2=adVarChar&Alt_ID_3=adVarChar&Alt_ID_4=adVarChar&Alt_ID_5=adVarChar]

Inputs

SubstanceName

The name of the substance.

[Structure]

The chemical structure of the substance as base64 encoded cdx data. Due to the length of cdx data, it is recommended that this parameter be posted as form data rather than passed via the query string.

[CAS]

Chemical Abstracts Service registry number.

[ACX_ID]

ChemACX ID number.

[Alt_ID_1]

The value of Alt_ID_1, an optional extra ID field for the substance.

[Alt_ID_2]

The value of Alt_ID_2, an optional extra ID field for the substance.

[Alt_ID_3]

The value of Alt_ID_3, an optional extra ID field for the substance.

[Alt_ID_4]

The value of Alt_ID_4, an optional extra ID field for the substance.

[Alt_ID_5]

The value of Alt_ID_5, an optional extra ID field for the substance.

Outputs

SubstanceID

Example

A user creates a substance with the name NewSubstance.  The Substance has an ACX_ID of X1001304-8. 

/cheminv/api/CreateSubstance.asp?SubstanceName=NewSubstance[&ACX_ID=X1001304-8]
