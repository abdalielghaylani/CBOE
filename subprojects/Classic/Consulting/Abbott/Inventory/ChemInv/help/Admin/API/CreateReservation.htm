CreateReservation

Allows a user to reserve a specified quantity of a substance in a specified container.

Syntax

/cheminv/api/CreateReservation.asp?ContainerID=adNumeric&ReservationUserID=adVarChar&QtyReserved=adDouble[&ReservationTypeID=adNumeric]

Inputs

ContainerID

The Container_ID value of the container being checked out .

ReservationUserID

The user_ID associated with the user making the reservation.  

QtyReserved

The amount of the compound reserved. 

[ReservationTypeID]

Type of reservation as a Type_ID value. Use ShowReservationTypes to see valid Type_ID values.

Outputs

None

Example

A user (ID=mjs) creates a reservation for 10 microliters of the substance in the container with ID 12312.

/cheminv/api/CreateReservation.asp?ContainerID=12312&ReservationUserID=mjs&QtyReserved=10
