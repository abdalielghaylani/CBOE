Tips for Installing XML XDK

XML XDK 9.2.0.5 is a required component for the propoer function of Inventory Enterprise.  This component is not installed by default in Oracle 8i installations.  After installing Oracle 8i, users should install the XML XDK. For more information about Oracle and XML, please see http://otn.oracle.com/tech/xml/index.html

Installing on a Windows server

The following is some guidelines for installing XML XDK 9.2.0.5 on a Windows server.  For more information, please see your Oracle DBA: 

  1. Download XDK 9.2.0.5 and copy the 3 folders (bin, xdk and lib) to the C: drive of the server.
  2. Open SQL Plus Worksheet.
  3. Execute the following:
    grant javasyspriv to cheminvdb2; 
  grant javauserpriv to cheminvdb2;
  4. Close SQL Plus Worksheet
  5. Go to START>Run.
  6. Enter cmd into the Open textbox.
  7. Change Directories to C:\bin by entering the following:
    cd C:\bin
  8. Enter the following into the prompt:
    • For Oracle 8.16 installations:
      xdkload –u cheminvdb2/oracle –s –dbver 816
    • For Oracle 8.17 installations:
      xdkload –u cheminvdb2/oracle –s –dbver 817
    • For Oracle 9i installations:
      xdkload –u cheminvdb2/oracle –s
  9. Hit any key to run.
    NOTE: This installs the data in the cheminvdb2 schema.  You can install in any appropriate tablespace. 
    NOTE: If the schema containing the XDK is dropped, XML XDK must be reinstalled.
