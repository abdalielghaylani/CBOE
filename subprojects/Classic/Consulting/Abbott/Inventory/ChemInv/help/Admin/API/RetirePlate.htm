RetirePlate

Moves the identified plate to the Retired location.

Syntax

/cheminv/api/RetirePlate.asp?Plate_ID=adVarChar&Location_ID_FK=adDouble&Status_ID_FK=adNUmeric&Qty_Remaining= adDouble

Inputs

Plate_ID

The ID for the plate that is being retired.

Location_ID_FK

The ID of the location where the plate is retired to.

Status_ID_FK

The ID for the plate status (most likely retired).

Qty_Remaining

The quantity remaining in the plate.

Outputs

1 if successful

Example

A user retires a plate of ID P888 to the retired location (ID=1212) changing the status of the plate to retired (ID=5) with a remaining quantity of .001 micro liters. 

/cheminv/api/RetirePlate.asp?Plate_ID=P888&Location_ID_FK=1212&Status_ID_FK=5&Qty_Remaining=0.001
