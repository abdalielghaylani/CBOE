CreateLocation

Creates a new location and returns its Location_ID

Syntax

/cheminv/api/CreateLocation.asp?ParentID=adNumeric&LocationName=adVarChar(128)[&Barcode=adVarChar(4000)&LocationDesc=adVarChar(255)&LocationTypeID=adNumericGridFormatID=adNumeric&PlateTypeList=adVarChar&LocationOwnerID=adVarChar]

Inputs

ParentID

The Location_ID value of the parent location.

LocationName

The name of the location.

[Barcode]

A unique alphanumerical identifier used to produce a location barcode.

[LocationDesc]

A description of the location.

[LocationTypeID]

A Type_ID value describing the type of location.  Use ShowLocationTypes to see valid Type_ID values. 

[GridFormatID]

The ID associated with the Grid Format for the location.  Grid Formats allow the user to create sublocations for the location during creation. 

[PlateTypeList]

A list of the plate types allowed to reside in the new location.  Plate types are separated by a comma. 

[LocationOwnerID]

The ID, associated with a particular user or owner, for the location being created. 

Outputs

 

Example

A user wants to create a location named "Fridge" which is in Lab5 (ID=121). 

/cheminv/api/CreateLocation.asp?ParentID=121&LocationName=Fridge
