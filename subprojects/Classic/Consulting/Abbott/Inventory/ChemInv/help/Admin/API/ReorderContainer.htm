ReorderContainer

Create anew order for a container that already exists.

Syntax

 /cheminv/api/ReorderContainer.asp?

Inputs

DeliveryLocationID

The Location_ID value of the location where the container is to be delivered.

ContainerID

The container ID for the container that is being reordered..

ContainerName

The name of the new container.

Comments

Any additional comments concerning the order or the container.

OwnerID

The ID for the owner of the new container. 

CurrentUserID

The ID of the user creating the reorder request (the user currently logged in).

NumCopies 

The number of containers to be ordered.

Project

The project the new container is being ordered for and will therefore be grouped under.

Job 

The job in the project the new container belongs to.

RushOrder

If this is a rush order, the value is 1. 

DueDate

The date by which the new container must be recieved as date-time literal (ie. 01-27-2001).

ReorderReason = <A value from the INV_CONTAINER_ORDER_REASON.CONTAINER_ORDER_REASON_ID field>

The reason for reordering the container.

ReorderReasonOther

If other is chosen for the ReorderReason, the reason in free text.          

Outputs

ContainerID

The new container ID for the cotnainer being ordered
