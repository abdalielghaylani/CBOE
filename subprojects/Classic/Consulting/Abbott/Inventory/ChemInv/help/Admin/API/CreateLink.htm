CreateLink

Creates a new link for a container record.  

Syntax

 /cheminv/api/CreateLink.asp?FK_Value=adVarChar&FK_Name=adVarChar&Table_Name=adVarChar[&URLHref=adVarChar&ImageSource=adVarChar&LinkText= adVarChar]

Inputs

FK_Value

FK_Name

Table_name

[URLHref]

The URL text. 

[URLType]

The type of link (e.g. file, www, etc.). 

[ImageSource]

. 

[LinkText]

The text to appear.  If no text is specified, the link text defaults to the link itself. 

Outputs

None

Example

A user wants to create a link .

 /cheminv/api/CreateLink.asp?
