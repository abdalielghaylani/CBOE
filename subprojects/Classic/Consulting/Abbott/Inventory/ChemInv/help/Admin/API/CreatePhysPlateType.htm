CreatePhysPlateType

Creates a new physical plate type according to the inputs.  

Syntax

/cheminv/api/CreatePhysPlateType.asp?PhysPlateName=adVarChar&RowCount=adNumeric&ColCount=adNumeric&WellCapacity=Int&CapacityUnitID=adNumeric[&RowPrefix=adVarChar&ColPrefix=adVarChar&RowUseLetters=adNumeric&ColUseLetters=adNumeric&NameSeparator=adVarChar&NumberStartCorner=AdNumeric&NumberDirection=adNumeric&IsPreBarcoded=adVarChar&SupplierID= adNUmeric]

Inputs

PhysPlateName

The name of the new physical plate type. 

RowCount

The number of rows in the plate using this physical plate type. 

ColCount

The number of columns in the plate using this physical plate type.

WellCapacity

The capacity of each well in the plate using this physical plate type.

CapacityUnitID

The ID corresponding to the appropriate units used for the Well Capacity measurement.

[RowPrefix]

If the rows are labelled by a prefix, the prefix for the row. 

[ColPrefix]

If the columns are labelled by a prefix, the prefix for the column.

[RowUseLetters]

A value of 1 if the rows are identified by consecutive letters (instead of numbers). 

[ColUseLetters]

A value of 1 if the columns are identified by consecutive letters (instead of numbers).  

[NameSeparator]

. 

[NumberStartCorner]

[NumberDirection]

. 

[IsPreBarcoded]

If Y, the plates of this physical plate type are already barcoded. 

[SupplierID]

The supplier ID for the plates of this physical plate type. 

Outputs

None

Example

A new physical plate type (Name is NewPlateType) is created with 8 rows and 12 columns.  The wells have a capacity of1 mL (unit ID= 9).  The supplier for this physical plate type has ID 1123. 

/cheminv/api/CreatePhysPlateType.asp?PhysPlateName=NewPlateType&RowCount=8&ColCount=12&WellCapacity=1&CapacityUnitID=9[&SupplierID= 1123]
