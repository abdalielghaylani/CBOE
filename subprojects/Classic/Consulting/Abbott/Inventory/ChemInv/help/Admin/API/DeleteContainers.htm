DeleteContainers

Deletes one or more containers. Returns the number of containers deleted.  Either a list of containers or locations must be provided.

Syntax

/cheminv/api/DeleteContainers.asp?ContainerList=List[&LocationList=List]
or
/cheminv/api/DeleteContainers.asp?LocationList=List[&ContainerList=List]

Inputs

ConatinerList

A list of containers to be deleted. Use a comma to separate multiple Conatiner_ID values. Use a dash to express a range of Container_ID values.

[LocationList]

A list of Location_ID values. Use a comma to separate multiple Location_ID values. Use a dash to express a range of Location_Id values. All containers in the given locations will be deleted.

Outputs

None

Example

A user wants to delete all of the containers in the location with ID 111.

/cheminv/api/DeleteContainers.asp?LocationList=111
