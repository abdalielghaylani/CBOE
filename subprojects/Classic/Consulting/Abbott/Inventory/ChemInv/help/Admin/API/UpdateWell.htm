UpdateWell

Updates the contents of a well.

Syntax

/cheminv/api/UpdateWell.asp?PlateIDs=adVarChar&WellIDs=adVarChar&ValuePairs= adVarChar

Inputs 

PlateIDs

List of PlateIDs associated with plates whose wells will be updated.  IDs should be separated by a comma.

WellIDs

List of Well IDs that will be updated.  IDs should be separated by a comma.

ValuePairs

The value pairs defining how to update the wells.

Outputs

0 if successful

Example

A user wants to update wells A-1 and A-2 in plate P111 with a new QtyRemaining of 10 ml (ID= 2).    

/cheminv/api/UpdateWell.asp?PlateIDs=P111&WellIDs=A-1,A-2&ValuePairs=qty_Remaining=10,quantity_unit_fk=2
