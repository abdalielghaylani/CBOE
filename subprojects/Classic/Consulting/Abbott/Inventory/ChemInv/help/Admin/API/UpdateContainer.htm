UpdateContainer 

[Deprecated function. Do Not Use!]

Updates one or more container attributes. Ommiting a given optional parameter from the function call leaves the ommited attribute unchanged.

Syntax

/cheminv/api/UpdateContainer.asp?ContainerID=adNumeric[&UOMID=adNumeric&QtyMax=adDouble&QtyRemaining=adDouble&ContainerTypeID=adNumeric&CompoundID=adNumeric&MinStockQty=adDouble&MaxStockQty=adDouble&ExpDate=adDBTimeStamp&ContainerName=adVarChar&ContainerDesc=adVarChar&UOWID=adNumeric&UOCID=adNumeric&UOPID=adNumeric&TareWeight=adDouble&Concentration=adDouble&Purity=adDouble&Grade=adVarChar&Solvent=adVarChar&Comments=adVarChar]

Inputs

ConatinerID

ContainerID of container to be moved.

[UOMID]

The unit of measure to be used for this container as a UNIT_OF_MEAS_ID value. Use ShowContainerUnits to see valid UNIT_OF_MEAS_ID values

[QtyMax]

Maximum capacity of the container in the units defined by UOMID.

[ContainerTypeID]

Type of container as a Type_ID value. Use ShowContainerTypes to see valid Type_ID values.

[CompoundID]

Compound_ID value associating the substance in the container with a preexisting inventory compund.

[ExpDate]

Content's expiration date as date-time literal (ie. 01-27-2001) 

[MinStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue a re-stock warning.

[MaxStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue an over-stock warning.

[ContainerName]

The name of the container.

[ContainerDesc]

A description of the container.

[TareWeight]

The weight of the physical container in the units of UOWID.

[UOWID]

Unit of weight value as a Unit_ID value from the Units table.

[Purity]

The purity of the substance designated by CompoundID in units of UOPID.

[UOPID]

Unit of purity value as a Unit_ID value from the Units table.

[Concenctration]

The concentration of the substance designated by CompoundID in units of UOCID.

[UOCID]

Unit of concentration value as a Unit_ID value from the Units table.

[Grade]

The quality or purity designation associated with the substance designated by CompoundID.

[Solvent]

The name of the solvent.

[Comments]

Comments or instructions associated with this container. Maximum length is 4000 characters. It may be passed as form value to avoid truncation by querystring size limit.
