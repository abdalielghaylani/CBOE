DeliverRequest

Changes status and location of a requested container when it is delivered.

Syntax

 /cheminv/api/DeliverRequest.asp?

Inputs

DeliveredRequestIDList

A comma delimited list of Request IDs.      

Outputs

None

Example

 /cheminv/api/DeliverRequest.asp?DeliveredRequestIDList=23,26,27
