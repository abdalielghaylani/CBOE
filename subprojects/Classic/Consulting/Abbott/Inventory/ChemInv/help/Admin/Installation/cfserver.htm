CFServer.ini Configuration File

The CFServer.ini file, found in the following location:
<webroot>/ChemOffice/cheminv/config
defines parameters that allow Inventory Enterprise to find necessary information and how to react in certain situations.  When editing fields in this file, it is important to be sure what is going to change because many of these settings are necessary for the application to function properly.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
