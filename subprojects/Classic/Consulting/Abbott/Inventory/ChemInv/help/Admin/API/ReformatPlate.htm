ReformatPlate

Creates new plate(s) from the contents of a plate which already exists.

Syntax

/cheminv/api/ReformatPlate.asp?SourcePLateIDList=adVarChar&ReformatMapID=adNumeric&PlateTypeIDList=adVarChar&AmtList=adVarChar&AmtUnitIDList=adVarChar&AmtType=adVarChar&SolventIDList=adVarChar&SolventVolumeList=adVarChar&SolventVolumeUnitIDList=adVarChar&LocationIDList=adVarChar&NumTargetPlates= adNumeric

Inputs

SourcePlateIDList

A list of plate IDs for the source plates to be reformatted.  When there is more than one plate, the IDs are listed separated by commas. 

ReformatMapID

The ID of the reformat map stored in the inv_xmldocs table .       

AmtType=SourceVolumeTaken|SourceAmountTaken|TargetConcentration

This determines how the amtlist and amtunitlist are interpreted. 

AmtList

Comma delimited list of amounts   

AmtUnitIDList

Comma delimited list of amount unit ids.      

SolventIDList

Comma delimited list of solvent ids from the inv_solvents table.         

SolventVolumeList

Comma delimited list of target solvent volumes.    

SolventVolumeUnitIDList

Comma delimited list of target solvent volume units.    

LocationIDList

Comma delimited list of target plate location ids.    

NumTargetPlates

The number of plates (in the reformatted format) created from the original plate.    

[PlateTypeIDList]

Comma delimited list of target plate types .    

Outputs

Pipe (|) delimited list of new plate IDs. For example: 100|101

Example

A user wants to make 2 new plates by reformatting plate ID P101 to use the platemap with ID 3, taking 10 ml (ID=2) from the source and combining it with 1ml of Water (ID=4) and put the plates in the location with ID101. 

/cheminv/api/ReformatPlate.asp?SourcePlateIDList=P101&ReformatMapID=3&AmtList=10&AmtUnitIDList=2&AmtType=SourceVolumeTaken&SolventIDList=4&SolventVolumeList=1&SolventVolumeUnitIDList=2&LocationIDList=101&NumTargetPlates= 2
