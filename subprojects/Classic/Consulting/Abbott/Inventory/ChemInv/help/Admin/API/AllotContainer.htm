AllotContainer

Used to calculate the amount of substance that should be alloted to new (or existing) containers which are being sampled or split.  Which action is specified by a parameter.   

Syntax

/cheminv/api/AllotContainer.asp?ContainerID=adNumeric&NumContainers=Int&Action=adVarChar&ContainerSize=adNumeric[&QtyRemaining= adNumeric]

Inputs

ContainerID

The ID of the conatiner (the parent container) to be split or sampled. 

NumContainers

The number of  conatiners the parent container is split or sampled into.

Action = split|sample

The only two options are split and sample. When a container is split, the container contents if dispersed among new containers, the original conatiner being removed.  On the other hand, when a container is sampled some quantity of the entire quantity of the container is moved into new containers, the original container left in tact, with less contents.

ConatinerSize

 The size of the newly created containers.

[QtyRemaining]

The quantity left in the original container - this field is required if Action = sample. 

[Split#]

The quantity in the new container - this field is required if Action = split. 
There is a split parameter required for each new container.  For example, if there are 3 container being made, a split1, split2, and split3 are required.

Outputs

Pipe (|) delimited list of new container IDs. For example: 100|101

Example

A container (ID=111) is split into4 container (each having a size of 100 and quantity of 50)

/cheminv/api/AllotContainer.asp?ContainerID=111&NumContainers=4&Action=split&ContainerSize=100&split1=50&split2=50&split3=50andsplit4= 50
