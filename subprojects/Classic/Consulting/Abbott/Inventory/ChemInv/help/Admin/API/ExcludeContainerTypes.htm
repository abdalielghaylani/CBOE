ExcludeContainerTypes

Makes it impossible to store specified container types in a particular location.

Syntax

 /cheminv/api/ExcludeContainerTypes.asp?LOcationID=adNumeric&ContainerTypeIDList= adVarChar&AllowContainers= adVarChar

Inputs

LocationID

The location ID where particular container types will be excluded.      

ContainerTypeIDList

A list of container type IDs to exclude (separated by commas).      

[AllowContainers]

.      

 

Outputs

None

Example

 

 /cheminv/api/ExcludeContainerTypes.asp?
