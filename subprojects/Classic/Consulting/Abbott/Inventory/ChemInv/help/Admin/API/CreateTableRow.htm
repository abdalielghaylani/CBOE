CreateTableRow

Creates a new row in any table in the schema.  

Syntax

/cheminv/api/CreateTableRow.asp?TableName=adVarChar&ValuePairs= adVarChar

Inputs

TableName

The name of the table in the schema. 

ValuePairs

A comma delimited list of value pairs.  In this case, the value pair is the column name and column value. 
For example: colnam1=val1,colnam2=val2       

Outputs

0 if successful

Example

A user creates a new row in the table Inv_Suppliers. 

/cheminv/api/CreateTableRow.asp?TableName=Requests&ValuePairs=Supplier_Name=Wyeth,Supplier_Short_Name=WYT,Supplier_Code= WH1432
