UpdateReservation

Updates the given reservation accordingly.

Syntax

/cheminv/api/CreateReservation.asp?ReservationID=adNumeric&ContainerID=adNumeric&ReservationUserID=adVarChar&QtyReserved=adDouble[&ReservationTypeID=adNumeric]

Inputs

ReservationID

The reservation_ID associated with the reservation made previously and now needs updating.  

ContainerID

The Container_ID value of the container being checked out .

ReservationUserID

The user_ID associated with the user making the reservation.  

QtyReserved

The amount of the compound reserved. 

[ReservationTypeID]

Type of reservation as a Type_ID value.
