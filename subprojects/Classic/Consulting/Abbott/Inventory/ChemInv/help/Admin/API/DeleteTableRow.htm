DeleteTableRow

Deletes the specified row in the specified table.   

Syntax

/cheminv/api/DeleteTableRow.asp?TableName=adVarChar&pkColumnName=adVarChar&pkIDs= adNumeric

Inputs

TableName

The name of the table.

pkColumnName

The name of the column in the table.    

pkIDs

A comma delimited list of primary key IDs.

Outputs

0 if successful

Example

/cheminv/api/DeleteTableRow.asp?TableName=INV_SUPPLIERS&pkColumnName=Supplier_Name&pkIDs= 11,12,14
