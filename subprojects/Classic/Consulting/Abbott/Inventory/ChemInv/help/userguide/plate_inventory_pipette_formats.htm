Pipette Log Template Options

Pipette logs imported into Inventory Enterprise through the Create Plates from Pipette Logs interface must be in one of the following supported formats:

Template Name # of Header Rows File Delimiter # of Columns
CS Default 0 comma 23
2 Column 1 tab 2
Hitlist Format 0 tab 5
Unified Transfer Format 7 comma 15
Plate 1 1 tab 7
Plate 2 1 tab 7
BTK File Format 0 comma 8

CS Default

The CS Default format is delimited by commas and contains 0 header lines. The following columns make up this type of file:

Column Contains ...
A Source Plate Barcode
B Source Well Number
C Target Plate Barcode
D Target Well Number
E Initial Quantity
F Units of Measure ID
G Concentration
H Units of Concentration ID
I Solvent ID
J Field 1
K Field 2
L Field 3
M Field 4
N Field 5
O Field 6
P Field 7
Q Well Field 1
R Well Field 2
S Well Field 3
T Well Field 4
U Well Field 5
V Well Date 1
W Well Date 2

 

2 Column

The 2 column format is delimited by tabs and contains 1 header line. The following columns make up this type of file:

Column Contains ...
A Target Plate Barcode
B Source Plate Barcode

'
 

Hitlist Format

The Hitlist Format is delimited by tabs and contains 0 header lines. The following columns make up this type of file:

Column Contains ...
A Target Plate Barcode
B Target Well Coordinate
C Source Plate Barcode
D Source Well Coordinate

Unified Transfer Format

The Unified Transfer Format is delimited by commas and contains 7 header lines. The following columns make up this type of file:

Column Contains ...
A N/A
B N/A
C N/A
D N/A
E Source Plate Barcode
F Source Well Number
G N/A
H N/A
I N/A
J Target Plate Barcode
K Target Well Number
L N/A
M N/A
N N/A

Plate 1

The Plate 1 format is delimited by tabs and contains 1 header line. The following columns make up this type of file:

Column Contains ...
A Well Coordinate
B N/A
C N/A
D Initial Quantity
E Solvent
F Concentration
G Lot comment

 

Plate 2

The Plate 2 format is delimited by tabs and contains 1 header line. The following columns make up this type of file:

Column Contains ...
A Well Coordinate
B N/A
C N/A
D Initial Quantity
E Concentration
F Solvent
G Lot comment

 

BTK file format

The BTK File format is delimited by commas and contains 0 header lines. When imported, this file uses a z pattern interleaving process to layout the new plate. The following columns make up this type of file:

Column Contains ...
A N/A
B N/A
C Target Plate Barcode
D Target Quadrant
E Source Plate Barcode
F N/A
G N/A
