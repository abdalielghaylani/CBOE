Edit Container

To see if you have the appropriate privileges to edit a container, see roles and privileges.

The Edit Container function allows editing of all container attributes. You may switch between the various tabs to access all container attributes. Changes to the container are only committed to the database once the OK button is clicked.

After the Edit Container window opens, proceed by editing any fields necessary and click OK. For more information about the fields, see Creating a New Container.
