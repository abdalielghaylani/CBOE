Link Management

The manage link area of Inventory Enterprise is available as part of substance management. To add a substance link, please take the following steps:

  1. From the Substance Management area, access the search form.

  2. Perform a search for the appropriate substance.

  3.  While in list view, click the Details button for the substance.

  4. Click on Manage Link.

The Manage Links window opens.

This area allows adding a link to the substance.

  1. Click New to add link.

The Add a Link window opens.

 

  1. Enter the appropriate information.

  • Enter a URL in the URL list box.

          NOTE: You must enter http:// for the URL if applicable.

  • Enter Text that you would like the link to be displayed as in the Link Text textbox. 

  1. Click OK.

To edit a link:

  1. Click Edit next to the link to be edited.

  2. Edit the appropriate information.

  3. Click OK.

To delete a link:

  1. Click Delete next to the appropriate link.

  2. If you do want to delete this link from the list, click OK.
