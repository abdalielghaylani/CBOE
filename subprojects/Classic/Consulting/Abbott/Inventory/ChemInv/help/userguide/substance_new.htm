Creating a New Substance

To see if you have the appropriate privileges to create a new substance, see roles and privileges.

The "New Substance" form is available in the Substance Management area of Inventory Enterprise. To reach this form, do the following:

  1. From the homepage, click Manage Substances.
  2. Click Add Mode.

This form is also available from the New Container Substance Tab.

Click on a field in the screenshot below for more information about a particular field.

How to create a new substance:

  1. Enter information about the new substance. For more information about individual fields, click on the field pictured above.
    Note: More fields may appear in your form. These forms are customized for your site. Please see your system administrator for more details.
  2. Click Add Record in the toolbar.
    If a conflict is recognized in the system (i.e. the same structure or Substance Name), the Conflict Resolution page appears. Otherwise, the substance is added to the system.
  3. Click  OK.

Creating Inventory Substances from Registration Enterprise and ChemACX

Inventory substances can be crested from both the Registration Enterprise and ChemACX. If this option is not available from your instance of the Registration Enterprise or ChemACX, please see your system administrator.

Please see the User's Guide for the appropriate application for instructions in using these functions.
