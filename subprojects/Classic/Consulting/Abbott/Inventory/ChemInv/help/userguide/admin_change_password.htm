Change Password

Users always have the option to change their password.

To change your password:

  1. From the Home page, click Change Password.

    The Change Password window opens.


     

  2. Enter the new password in the New Password textbox.
  3. Enter the new password again in the Confirm New Password textbox.
  4. Click OK.
