Reservations Tab

To see if you have the appropriate privileges to create, edit, or delete a reservation, see roles and privileges.

The Reservations Tab of the View Container frame provides information about any reservations made for the container's contents. This tab also provides the ability to enter a new reservation as well as edit or delete an existing reservation. For more information regarding the fields found under the Reservations Tab, click on the appropriate field in the image below.
