Creating a New Location

To see if you have the appropriate privileges to create a new location, see roles and privileges.

Containers are located in different places throughout a facility. The types of locations necessary in your inventory system is highly dependent upon your setup.

In order to create a new location:

  1. From the homepage, click Manage Containers.
  2. Use the location tree to select where the new location should be created. 
  3. Click New Location in the Current Location Frame.

The New Location window opens.

  1. Enter the desired information in the window that opens. For more information about a particular field, click on the field in the image above.
  2. Click Add Address link.

The Add Address window opens.

  1. Enter the desired information about address in the window.
  2. Click OK.
