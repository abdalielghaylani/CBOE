EH&S Tab - Container

Container level safety data requires the following things to be true:

  • The substance in the container must have substance level safety data associated with it
  • A valid Supplier name must be part of the container record
  • A valid catalog number must be part of the container record

If all of the above requirements are met, the EH&S Tab in the container viewing frame contains a link: Edit EH&S Data. 

To add substance level safety data:

  1. Open a substance record through a search.
  2. Click Add EH&S Data.
    A page allowing the addition of safety data appears. The fields in this form are customized by your system administrator, so your screen may look different from the screenshot below.
  3. Enter the appropriate information.
  4. Click OK.
    Substance Level Environmental Health and Safety Data has been added. This data will appear in the EH&S Tab for any container containing this substance.
