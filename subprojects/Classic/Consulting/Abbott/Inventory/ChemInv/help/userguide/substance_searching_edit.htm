Editing a Substance's Details

To edit a substance:

  1. Perform a search and click on the  Details button for the substance.
    A page appears displaying all retrieved details for the appropriate record.
  2. Click  Edit Mode.
    The record appears in Edit Mode.
  3. Enter or edit the appropriate information.
  4. Click OK.

Note: If a duplicate for any required field is found in the system, the conflict resolution page appears. Please see Substance Conflict Resolution for more information.
