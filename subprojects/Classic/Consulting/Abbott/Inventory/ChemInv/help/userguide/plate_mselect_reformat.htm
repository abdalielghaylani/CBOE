Reformat Plates

Depending upon the plates which were selected, user can reformat them in a single plate in different ways. There are limited numbers of reformats maps available in the default configuration. All the selected plates must have some quantity in it, they also can be of different Plate Types, and But Format Type should be the same.

To reformat plates:

  1. Select plates from multi select option.
  2. Select Reformat Plates from Plate Details Frame.
    The Reformat Plates dialog appears.

     
  3. Click Next.
    The following window appears.

    Note: User can rearrange barcode position of plates.

  4. Click Next. 

    Note: User can select Yes or No option for plates.

  5. Click Next.

  6. Enter the appropriate information.
  7. Click OK.
