Rectify Contents

At times, plates are moved and this is not updated in the inventory system. Rectify Plates help users to easily update the system confirming that a plate is actually where the system thinks it is.

The Rectify Contents link opens a window which aids users in keeping the list of inventory plates in a particular location up to date. This tool is a formal way to correct inventory discrepancies. That is, if a particular plate is not confirmed, that plate is moved to the Missing Location in Inventory Enterprise. Rectify Contents is meant to be used on a quarterly or biannual basis during official inventory reconciliation efforts. For a less formal tool which only verifies the existence of a plate in a location, please use the Update Contents tool.

Rectify contents accomplishes the following:

  • confirm that plates are still where the system thought them to be.
  • move plates unexpectedly found at that location to specific location.
  • allow for unknown plates to be created at specific location.
  • move any plates that are not explicitly found to a missing location.

To rectify the contents of a location:

  1. Open the appropriate location and click Rectify Contents in the Plates List Frame.

    The Rectify Contents window opens.


  2. Using a barcode scanner, or by manually selecting the checkboxes, select the plate ID for each plate that is being verified.

    Note: Any plate IDs entered that are not currently in this inventory location, are listed in the upper frame.
    Select the checkbox next to any plates you would like moved to this location.

  3. Click  OK.

  5. Click  OK if you would like plates which have not been verified moved to the Missing Location.
