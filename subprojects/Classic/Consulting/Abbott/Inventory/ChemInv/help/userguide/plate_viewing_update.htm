Update Contents

At times, plates are moved and this is not updated in the inventory system. Update Plates helps users to easily update the system confirming that a plate is actually where the system thinks it is.

The Update Contents link opens a window which aids users in keeping the list of inventory plates in a particular location up to date. This tool is a casual way for users to confirm the existence of a particular plate. If a particular plate is not confirmed, no action is taken on that plate. For a tool to correct any discrepancies in a location's contents, please use the Rectify Contents tool.

Update contents accomplishes the following:

  • confirm that plates are still where the system thought them to be.
  • move plates unexpectedly found at that location to specific location.
  • allow for unknown plates to be created at specific location.

To update the contents of a location:

  1. Open the appropriate location and click Update Contents in the Plate List Frame.

    The Update Contents window opens.


  2. Using a barcode scanner, or by manual selecting the checkboxes, select the plate ID for each plate that is being verified.
  3. Click OK.
