Submitting documents

To submit a document:
1. Click Submit documents in the DocManager Enterprise section. The Add Document page appears:

 

2. In the document text box, enter the directory location and file name of the document to be submitted. You can also use Browse to locate the appropriate document.
3. (Optional) To add details about the document, click Add Document Details.
4. In the Add Document Details page, enter the title, author name, comments, main author, report number, status of the document, writer, abstract, document date, document class and document security category for the document being submitted in the appropriate text boxes.
5. If you added details to the document, click Submit This Document in the Add Document Details screen. Otherwise, click Submit Now on the Add Document screen.
Note: If DocManager Enterprise is integrated with Registration Enterprise, and you have both DocManager and Registration access privileges, an additional field for Registry Number is displayed on this page. The Registry Number you specify in this field will be used to create a link between the submitted document and the registered compound in Registration Enterprise.

Viewing recent activities

The Recent Activities lists the documents that users have submitted.
To view recent activities performed by the user:
1. Click Recent Activities in the main page of DocManager Enterprise. The View Recent Activities page appears:

 

Note: You can also view preceding page by clicking the Recent Activities link in the DocManager Enterprise section of the Chem & Bio Office Enterprise 2010 home page.
2. Select an appropriate option to sort the documents.
3. Click Show Activities to view the sorted result.
