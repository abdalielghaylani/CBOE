Logging into ChemBioOffice

To use DocManager, you must first log into ChemBioOffice Enterprise 2010.
To log in:
1. Enter http://hostname into the address bar of your web browser where hostname is the name of the server on which ChemBioOffice Entereprise 2010 is installed. The login page appears.
If you do not know the name of the server, contact your system administrator.

 

2. Enter your ChemBioOffice Enterprise name and password and click Log in. The ChemBioOffice Enterprise Manager home page appears. On this page, you will see the DocManager Enterprise menu, shown below::

DocManager basics

DocManager lets you :
Submit documents over the Web to a central repository. Documents may be in either .doc, .ppt, .txt, .xls, or .pdf format.
Perform Boolean searches over the Web for chemical structures and text throughout documents stored in the repository.
Note: DocManager does not search for structures in PDF documents.
Remove documents that are no longer shared.
