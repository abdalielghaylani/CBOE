Field Precision for some fields in the 7.2 REGDB schema are unnecessarily restrictive. In particular custom fields in the temporary_structure,batches and compound_molecule table that begin with "REAL". These were defined as NUMBER(10,8).

If you find you need to store data in the fields that are greater then XX.XXXXXX you can run the following statments.


--Custom Real fields for Compound_Molecule, Batches, and Temporary_Structures

ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_1 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_2 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_3 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_4 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_5 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_BATCH_FIELD_6 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_CMPD_FIELD_1 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_CMPD_FIELD_2 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_CMPD_FIELD_3 NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES ADD (REAL_CMPD_FIELD_4 NUMBER);


ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_1 NUMBER);
ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_2 NUMBER);
ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_3 NUMBER);
ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_4 NUMBER);
ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_5 NUMBER);
ALTER TABLE REGDB.BATCHES ADD (REAL_BATCH_FIELD_6 NUMBER);


ALTER TABLE REGDB.COMPOUND_MOLECULE ADD (REAL_CMPD_FIELD_1 NUMBER);
ALTER TABLE REGDB.COMPOUND_MOLECULE ADD (REAL_CMPD_FIELD_2 NUMBER);
ALTER TABLE REGDB.COMPOUND_MOLECULE ADD (REAL_CMPD_FIELD_3 NUMBER);
ALTER TABLE REGDB.COMPOUND_MOLECULE ADD (REAL_CMPD_FIELD_4 NUMBER);



Along the same line, fields storing MW and other fields defined with NUMBER(10,5) can be changed using the following statements.

--adjust MW and other (NUMBER (10,5)) definitions to be true floats (NUMBER)

ALTER TABLE REGDB.TEMPORARY_STRUCTURES MODIFY (SALT_MW NUMBER);
ALTER TABLE REGDB.TEMPORARY_STRUCTURES MODIFY (SALT_EQUIVALENTS NUMBER);
