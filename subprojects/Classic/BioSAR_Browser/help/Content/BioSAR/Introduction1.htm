DataView Manager

Use DataView Manager to create dataviews for ChemBioViz. After you create a dataview and assign permissions to it, the dataview is available to ChemBioViz users to run queries. You must have the appropriate permissions to use DataView Manager.

The Master DataView

Before you create any dataview for ChemBioViz, you must first create the master dataview. The master dataview is the superset of all database tables from which all dataviews listed in ChemBioViz (think of them as subsets) are made. There is only on Master dataview. Although the dataviews you create from the Master dataview are available in ChemBioViz, the Master dataview itself is not.

Creating Data Views

Only users with administrator privileges can create a dataview. After you create a dataview and assign privileges to it, users can use the dataview in ChemBioViz.

Using the Master Dataview

The first dataview must be made from the master dataview. Additional dataviews may be made either from the master dataview or from copying other dataviews.
To create a dataview from the master dataview:
1. Log into ChemBioOffice Manager as an administrative user.
2. In the DataView Manger menu, click Create from master. The DataView Manager window opens.
3. In the Name and Description text boxes, enter a name for the new data field and a brief description.
4. In the dropdown list, select a schema.
5. Select base table from the table list and click OK. The Select Tables window opens.
6. In the Select Tables window, select all the tables that you want the new dataview to contain. The base table is already selected in the list.
After you select the tables, you can finish creating the master dataview by clicking Submit. However, if you want to modify the tables or the dataview, you can use the options that are available on the Select tables window. These options let you modify, add or remove a tables in the dataview. These options are described below. Be sure to click Submit when you are finished modifying the master dataview.

Table Options

After you select the tables for your dataview, you can modify each table as needed, depending on how you want it to be used.
1. In the list of tables, click the name of a selected table that you want to edit. The Table Summary pane for that table appears.
2. Click Edit Table. The Edit Table and Fields pane appears.
The options are listed below.
Editing the table Alias
In the table Alias field, enter an alias name.
Creating a field alias
Select the field and click Create Field Alias. A new field appears in the field list with the _Alias extension.
Choosing lookup field
a. In the LookUp column, click Choose for the field you want to assign as the lookup. The Define a lookup pane appears.
b. In the left column, expand the tree and select the table to use.
c. In the right column, select the field. and a Sort Direction from the dropdown list.
d. Click Submit.
Assigning a Parent Column
a. Under Parent Column, click Choose for the field you want to assign as the parent. The Define Relationships pane appears.
b. In the Current Data View column, expand the tree and select the table to use.
c. In the Join Type dropdown list, select an option.
d. Click Submit.
Assigning an index type
In the Index Type column, select an option from the dropdown list.
Assigning a content type
In the Content Type column, select an option from the dropdown list.
Remove fields
To remove a field from the table, click Remove in the Action column.
3. After you finish setting the option, click OK in the Edit Table and Fields pane.

Security settings

1. In the Select Tables pane, click Security Settings.
2. Select the appropriate radio button for the users you want to have access to the dataview.
3. If you want to choose the users for the dataview, select the users in the list and click Add User.
4. If you want to choose the roles for the dataview, select the roles in the list and click Add Role.
5. Click Next.The Security Settings pane closes.
Review the Validation Summary
When you finish selecting the dataview options, review the schema summary. When you are finished, the DataView Manager will validate your settings.
1. After you review your dataview settings, click Submit. The Validation Summary pane opens.
2. After you review the validation summary, click Submit. The Validation Summary pane closes.
3. Click Home to return to the ChemBioOffice Manager.
