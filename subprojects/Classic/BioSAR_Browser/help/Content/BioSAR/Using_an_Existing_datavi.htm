Using an Existing dataview

To create a data view from an existing dataview
1. Log into ChemBioOffice Manager as an administrator.
2. In the DataView Manager menu, click Create from existing. The DataView Manager window opens.
3. Expand the list in the DataView Manager window.
4. Select an existing dataview from which to create the new dataview. The dataview property box appears to the right of the list.
5. Click Next.
6. In the Name and Description text boxes, enter a name for the new data field and a brief description.
7. In the dropdown list, select a schema.
8. Select base table from the table list and click OK.
9. Select all the tables that you want the new dataview to contain. The base table is already selected in the list.

Specifying security settings

1. In the Select Tables pane, click Security Settings.
2. Select the appropriate radio button for the users you want to have access to the dataview.
3. If you want to choose the users for the dataview, select the users in the list and click Add User.
4. If you want to choose the roles for the dataview, select the roles in the list and click Add Role.
5. Click Next.The Security Settings pane closes.

Selecting a base table

1. In the Select Tables pane, click Select Base Table. The Select Base Table pane appears.
2. Select a base table from the list of tables and click OK. The Select Base Table pane closes.
3. In the Select Tables pane, click Submit. The Select Tables pane closes and the Validation Summary pane appears.
4. After you review the validation summary, click Submit. The Validation Summary pane closes.
5. Click Home to return to the ChemBioOffice Manager.
