Creating a form

You can create a new form by either duplicating an existing form or using a blank template. After you create your form, you can edit it. See See "Editing forms".

Duplicating a form

To duplicate an existing form:
1. Open any private, public, or user form.
Tip: To save time, consider starting with a form that already resembles the one you want to create.
2. Right-click the Form tab and go to Add Tab>Duplicate Form. A new tab appears, displaying a duplicate of the form you opened.
3. (Optional) If you want to change the name of the form, right-click the form tab and select Rename Tab.

Using a blank template

To create a new form from a blank template, right-click the Form tab and go to Add Tab>Blank Form. A new tab appears, displaying a blank form. The blank form is completely empty; therefore, you will need to add fields to it before using it to perform a query or view data. See See "Editing forms".

Saving a form

Forms that you save are listed in the navigation pane and available for you to use and edit.
