Editing a Description/Password

To edit the description or password for a schema:
1. Expand the schema whose description/password you want to edit.
2. Click the Edit Schema Description/Password link. The following page appears:

 

NOTE:The display name for the schema is the name that will be visible to users. The default display name is the name of the schema. It is recommended that you enter a more descriptive name for the schema so that it can be easily identified.

The password for the schema is assigned when the schema is created. For information about schema passwords for CambridgeSoft schemas, contact esupport@cambridgesoft.com.

NOTE:For non-CambridgeSoft databases, you might need to contact the system administrator.
3. Change the display name and password of the schema according to your requirements.
4. Click the Save Changes button to save the changes.
