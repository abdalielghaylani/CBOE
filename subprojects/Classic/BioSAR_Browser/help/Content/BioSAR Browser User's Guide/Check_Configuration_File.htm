Check Configuration File Settings

Configuration files are used by BioSAR Enterprise to define parameters that allow the BioSAR Enterprise application to find necessary information and how to react in certain situations. If the settings in these files are not correctly recorded, the application may not be able to operate.

The configuration files for BioSAR Enterprise are found in:

<webroot>/ChemOffice/biosar_browser/config

If you are having difficulties, check to make sure the following settings are correct in the cfserver.ini file:
• DBA_PWD
• DBA_USERNAME
• ORA_SERVICENAME
