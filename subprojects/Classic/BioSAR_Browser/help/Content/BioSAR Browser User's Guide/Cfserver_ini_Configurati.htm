Cfserver.ini Configuration File

The CFServer.ini file is found in the following location: <webroot>/ChemOffice/biosar_browser/config

It defines parameters that allow the BioSAR Enterprise application to find necessary information and how to react in certain situations. When editing fields in this file, it is important to be sure what is going to change because many of these settings are necessary for the application to function properly.

The Cfserver.ini file has two sections:
• GLOBALS
• CS_SECURITY
NOTE:It is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application. New versions of the ChemOffice Enterprise use default configurations.
