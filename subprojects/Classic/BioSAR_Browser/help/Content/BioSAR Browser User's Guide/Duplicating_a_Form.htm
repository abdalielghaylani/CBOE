Duplicating a Form

Duplicating a form is useful when you need to include many properties of an existing form in a new form. For example, if you are creating a new form with the same base table and field properties of an existing form, duplicating the existing form can save time in defining the new form.

To duplicate a form:
1. Click the Manage your forms link within the BioSAR Enterprise section in the ChemOffice Enterprise home page to display the Form Management interface.
2. Select the form to be duplicated from the drop down list in the Form Management interface.
3. Click the Duplicate link. The form is duplicated and appears in the forms drop down list. The name of the duplicate form is the name of the old form preceded by "Copy of", as shown in the following figure:
