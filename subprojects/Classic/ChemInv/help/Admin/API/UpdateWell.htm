UpdateWell

Updates the attributes of a well.

Syntax

/cheminv/api/UpdateWell.asp?PlateIDs=adVarChar&WellIDs=adVarChar&ValuePairs=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
WellIDs adVarChar Specifies a comma delimited list of the IDs of the wells to be updated.   Y INV_WELLS WELL_ID
PlateIDs adVarChar Specifies a comma delimited list of the IDs of the plates, the wells associated with which are to be updated.   Y INV_WELLS PLATE_ID_FK
ValuePairs adVarChar Specifies a "::" delimited list of the name and value pairs of the well fields to be updated. For example: colnam1%3Dval1::colnam2%3Dval2.   Y N/A N/A

Output

None.

Example

The following code updates the qty_remaining and quantity_unit_fk attribute of the A-1 and A-2 wells in the P111 plate:

/cheminv/api/UpdateWell.asp?PlateIDs=P111&WellIDs=A-1,A-2&ValuePairs=qty_remaining=10::quantity_unit_fk=2
