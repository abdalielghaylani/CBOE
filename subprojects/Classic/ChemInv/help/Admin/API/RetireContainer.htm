RetireContainer

Retires a container and sets the status and location of the container to the specified values. When a container is retired its contents are not disposed, if specified explicitly.

Syntax

/cheminv/api/RetireContainer.asp?ContainerID=adNumeric&LocationID=adNumeric&ContainerStatusID=adNumeric&QtyRemaining=adDouble

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerID adNumeric Specifies the ID of the container to be retired.   Y INV_CONTAINERS CONTAINER_ID
LocationID adNumeric Specifies the ID of the location where the retired container needs to be moved. By default, the retired container is moved to the Disposed location.   Y INV_CONTAINERS LOCATION_ID_FK
ContainerStatusID adNumeric Specifies the ID of the status to be used for the retired container. By default, the retired container is set to the Disposed status.   Y INV_CONTAINERS CONTAINER_STATUS_ID_FK
QtyRemaining adDouble Specifies the quantity to be left in the container when it is retired. By default, the QtyRemaining parameter is set to zero.   N INV_CONTAINERS QTY_REMAINING

Output

None.

Example

The following code retires a container with ID 12345, and set both, its location and status to Disposed. The ID of the Disposed location is 2 and the ID of the Disposed status is 6:

/cheminv/api/RetireContainer.asp?ContainerID=12345&LocationID=2&ContainerStatusID=6
