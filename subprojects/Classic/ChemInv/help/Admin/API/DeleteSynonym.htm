DeleteSynonym

Deletes a synonym of a substance.

Syntax

/cheminv/api/DeleteSynonym.asp?SynonymID=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
SynonymID adNumeric Specifies the ID of the synonym to be deleted.   Y INV_SYNONYMS SYNONYM_ID

Output

The ID of the deleted synonym, which is stored in the SynonymID parameter.

Example

The following code deletes a synonym with ID 45:

/cheminv/api/DeleteSynonym.asp?SynonymID=45
