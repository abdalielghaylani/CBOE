DeletePlateType

Deletes the specified plate type.   

Syntax

/cheminv/api/DeletePlateType.asp?PlateTypeID=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
PlateTypeID adNumeric Specifies the ID of the plate type to be deleted.   Y INV_PLATE_TYPES PLATE_TYPE_ID

Output

The ID of the deleted plate type, which is stored in the PlateTypeID parameter.

Example

The following code deletes a plate type with ID 1: 

/cheminv/api/DeletePlateFormatID.asp?PlateTypeID=1
