ReformatPlate

Reformats plates to create a single plate from the compounds of the multiple plates.

Syntax

/cheminv/api/ReformatPlate.asp?SourcePLateIDList=adVarChar&ReformatMapID=adNumeric&BarcodeList=adVarChar&LocationID=adNumeric&PlateTypeID=adNumeric&Amt=Int&AmtUnitID=adNumeric&SolventID=adNumeric&SolventVolume=Int&SolventVolumeUnitID=adNumeric[&NumTargetPlates=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
SourcePlateIDList adVarChar Specifies a comma delimited list of the IDs of the plates to be reformatted.   Y INV_PLATES PLATE_ID
ReformatMapID adNumeric Specifies the ID of the reformat map to be used for reformatting of the plates.   Y N/A N/A
BarcodeList adVarChar Specifies a list of the barcodes of the target plates.   Y INV_PLATES PLATE_BARCODE
LocationID adNumeric Specifies the ID of the location where the target plates are to be stored.   Y INV_PLATES LOCATION_ID_FK
PlateTypeID adNumeric Specifies the ID associated with the type of the target plates.   Y INV_PLATES PLATE_TYPE_ID_FK
Amt adNumeric Specifies the amount to be taken from the source plates.   Y N/A N/A
AmtUnitID adNumeric Specifies the ID of the unit in which the amount taken from the source plates is to be measured.   Y N/A N/A
SolventID adNumeric Specifies the ID of the solvents with which the source plates are to be solvated.   Y INV_PLATES SOLVENT_ID_FK
SolventVolume adNumeric Specifies the volume of the solvents.   Y INV_PLATES SOLVENT_VOLUME
SolventVolumeUnitID adNumeric Specifies the ID of the unit in which the solvent volume is to be measured.   Y INV_PLATES SOLVENT_VOLUME_UNIT_ID_FK
NumTargetPlates adInteger Specifies the number of the target plates to be created.   N N/A N/A

Output

A Pipe (|) delimited list of the IDs of the newly created plates. For example: 100|101. This list of plate IDs is stored in the PlateIDs parameter, the data type of which is adVarChar.

Example

The following code reformats plate with ID P1001 to create 2 target plates: 

/cheminv/api/ReformatPlate.asp?SourcePlateIDList=P1001&ReformatMapID=3&BarcodeList=R1002&PlateTypeID=2&Amt=10&AmtUnitID=2&SolventID=4&SolventVolume=1&SolventVolumeUnitID=2&LocationID=101[&NumTargetPlates=2]
