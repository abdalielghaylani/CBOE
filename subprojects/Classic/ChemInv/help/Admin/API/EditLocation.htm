EditLocation

Updates attributes of a location.

Syntax

/cheminv/api/EditLocation.asp?LocationID=adNumeric&LocationName=adVarChar[&Barcode=adVarChar&LocationDesc=adVarChar&LocationTypeID=adNumeric&GridFormatID=adNumeric&PlateTypeList=adVarChar&LocationOwnerID=adVarChar&PropagateAddress=adNumeric]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the location to be updated.   Y INV_LOCATIONS LOCATION_ID
LocationName adVarChar Specifies the name of the location.   Y INV_LOCATIONS LOCATION_NAME
Barcode adVarChar Specifies the barcode for the location.   NINV_LOCATIONS LOCATION_BARCODE
LocationDesc adVarChar Specifies a description for the location.   N INV_LOCATIONS LOCATION_DESCRIPTION
LocationTypeID adNumeric Specifies the ID associated with the type of the location.   N INV_LOCATIONS LOCATION_TYPE_ID_FK
GridFormatID adNumeric Specifies the ID of the grid format associated with the location.   N INV_GRID_FORMAT GRID_FORMAT_ID
PlateTypeList adVarChar Specifies the list of the plate types allowed to be stored at the location.   N N/A N/A
LocationOwnerID adVarChar Specifies the ID for the owner of the location.   N INV_LOCATIONS OWNER_ID_FK
PropagateAddress adNumeric Indicates whether to propagate the address of the parent location to its child locations, if no address is specified for the children locations explicitly. The address is propagated if the value of this parameter is 1. 1|0 N N/A N/A

Outputs

The ID of the updated location, which is stored in the LocationID parameter. The data type of the LocationID parameter is adVarChar.

Example

The following code edits a location with ID L1022 and name TestLocation:

/cheminv/api/EditLocation.asp?LocationID=L1022&LocationName=TestLocation
