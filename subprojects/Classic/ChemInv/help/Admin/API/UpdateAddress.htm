UpdateAddress

Updates the address of the specified supplier or location.

Syntax

/cheminv/api/UpdateAddress.asp?TableName=adVarChar&TablePKID=adVarChar[&AddressID=adVarChar&ContactName=adVarChar&Address1=adVarChar&Address2=adVarChar&Address3=adVarChar&Address4=adVarChar&City=adVarChar&StateIDFK=adNumeric&CountryIDFK=adNumeric&ZIP=adVarChar&FAX=adVarChar&Phone=adVarChar&Email=adVarChar]

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
TableName adVarChar Specifies the table the address associated with which is to be updated. INV_SUPPLIERS| INV_LOCATIONS Y N/A N/A
TablePKID adVarChar Specifies the primary key ID. This ID specifies the location or supplier, whose address is to be updated.   Y N/A N/A
AddressID adVarChar Specifies the ID of the address to be updated.   N INV_ADDRESS ADDRESS_ID
ContactName adVarChar Specifies the contact name.   N INV_ADDRESS CONTACT_NAME
Address1 adVarChar Specifies the first line of the address.   N INV_ADDRESS ADDRESS1
Address2 adVarChar Specifies the second line of the address.   N INV_ADDRESS ADDRESS2
Address3 adVarChar Specifies the third line of the address.   N INV_ADDRESS ADDRESS3
Address4 adVarChar Specifies the fourth line of the address.   N INV_ADDRESS ADDRESS4
City adVarChar Specifies the name of the city.   N INV_ADDRESS CITY
StateIDFK adNumeric Specifies the ID of the state.   N INV_ADDRESS STATE_ID_FK
CountryIDFK adNumeric Specifies the ID of the country.   N INV_ADDRESS COUNTRY_ID_FK
ZIP adVarChar Specifies the zip code.   N INV_ADDRESS ZIP
FAX adVarChar Specifies the fax number.   N INV_ADDRESS FAX
Phone adVarChar Specifies the phone number.   N INV_ADDRESS PHONE
Email adVarChar Specifies the email address.   N INV_ADDRESS EMAIL

Output

The ID of the updated address, which is stored in the AddressID parameter.

Example

The following code updates the address of a supplier:

/cheminv/api/UpdateAddress.asp?TableName=INV_SUPPLIERS&TablePKID=12
