ReconcileContainer

Rectifies the contents of a location by performing the following tasks:

  • Move the misplaced containers to the location being rectified. Misplaced containers are those that are not present at the location being rectified, but should be there.
  • Move the missing containers to the Missing location. Missing containers are those that are present at the location being rectified, but should not be there.
  • Mark the containers as Verified if they are verified to be present at the location being rectified.

Syntax

/cheminv/api/ReconcileContainer.asp?LocationID=adNumeric&VerifiedContainerIDList=adVarChar[&AwolContainerIDList=adVarChar&MissingContainerIDList=adVarChar]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the location, the containers stored in which are to be reconciled.  

Y

 INV_LOCATIONS LOCATION_ID
VerifiedContainerIDList adVarChar Specifies a comma delimited list of the IDs of containers verified as being present at the specified location.   Y -  if AwolContainerIDList and Missing ContainerIDList are empty N/A N/A
AwolContainerIDList adVarChar Specifies a comma delimited list of the IDs of the misplaced containers.   Y - if VerifiedContainerIDList and Missing ContainerIDList are empty N/A N/A
CurrentUserID adVarChar Specifies the ID of the current user in the container.   Y - if VerifiedContainerIDList and Missing ContainerIDList are empty INV_CONTAINERS CURRENT_USER_ID_FK
MakeDefault adNumeric Specifies the default location.     N/A N/A
MissingContainerIDList adVarChar Specifies a comma delimited list of the IDs of the missing containers.   Y - if AwolContainerIDList and VerifiedContainerIDList are empty.   N/A N/A

Output

None.

Example

The following code reconciles the contents of the location with ID 101:

 
/cheminv/api/ReconcileContainer.asp?LocationID=101&VerifiedContainerIDList=12345,12245[&AwolContainerIDList=12225]
