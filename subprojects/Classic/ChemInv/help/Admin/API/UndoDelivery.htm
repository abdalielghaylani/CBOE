UndoDelivery

Undoes the delivery of a request and moves the container associated with the request back to its original location.

Syntax

/cheminv/api/UndoDelivery.asp?RequestID=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
RequestID adNumeric Specifies the ID of the request, whose delivery is to be undone.   Y INV_REQUESTS REQUEST_ID

Output

None.

Example

The following code undoes the delivery of a request with ID 12323:

/cheminv/api/UndoDelivery.asp?RequestID=12323
