UpdateLink

Updates the attributes of a link, such as URL, link text, or image source.

Syntax

/cheminv/api/UpdateLink.asp?FK_Value=adVarChar&FK_Name=adVarChar&Table_name=adVarChar[&URLID=Int&URLHref=adVarChar&ImageSource=adVarChar&LinkText=adVarChar]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
URLID adInteger Specifies the ID of the link to be updated.   N INV_URL URL_ID
FK_Name adVarChar Specifies the field that uniquely identifies the row with which the link is to be associated.   Y INV_URL FK_NAME
FK_Value adVarChar Specifies the value of the field specified in the FK_Name parameter.   Y INV_URL FK_VALUE
Table_name adVarChar Specifies the table name that contains the row with which the link is to be associated.   Y INV_URL TABLE_NAME
URLHref adVarChar Specifies the URL of the link.   N INV_URL URL
URLType adVarChar Specifies the type of the URL.   N INV_URL URL_TYPE
LinkText adVarChar Specifies the text for the link. If no text is specified, the link text is set to the URL of the link.    N INV_URL LINK_TXT
ImageSource adVarChar Specifies the path of the GIF or JPEG image to be used for the link.    N INV_URL IMAGE_SRC

Output

The ID of the updated link. This ID is stored in the URLID parameter, the data type of which is adNumeric.

Example

The following code updates the LinkText parameter of an already created link:

/cheminv/api/UpdateLink.asp?FK_Value=1500&FK_Name=CONTAINER_ID&Table_name=INV_CONTAINERS[&LinkText="Search Engine"]
