CreatePlateMap

Creates a plate map and returns the ID of the newly created plate map.

Syntax

/cheminv/api/CreatePlateMap.asp?LocationID=adNumeric&PlateID=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the Plate Map location where the plate map is to be stored.   Y INV_LOCATIONS LOCATION_ID
PlateID adNumeric Specifies the ID of the plate for which the plate map is to be created.   Y INV_PLATES PLATE_ID

Output

 

Example

The following code creates a plate map from a plate with ID A02 and stores the plate map at the location with ID 1032:

/cheminv/api/CreatePlateMap.asp?LocationID=1032&PlateID=A02
