GetKeyContainerAttributes

Allows you to determine the key attributes of a particular container.

Syntax

/cheminv/api/GetKeyContainerAttributes.asp?ContainerID=adVNumeric&ContainerBarcode=adVarChar

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerID adNumeric Specifies the ID of the container, whose key attributes are to be determined.    Y, if ContainerBarcode is not provided. INV_CONTAINERS CONTAINER_ID
ContainerBarcode adVarChar Specifies the barcode of the container, whose key attributes are to be determined.   Y, if ContainerID is not provided. INV_CONTAINERS BARCODE

Outputs

Parameter Name Data Type Description
ContainerID adNumeric Specifies the unique ID of the container.
ContainerBarcode adVarChar Specifies the unique barcode of the container.
CurrentUser adVarChar Specifies the current user of the container.
UOM adVarChar Specifies the unit of measurement for the contents of the container.
QtyRemaining adNumeric Specifies the quantity of the container contents.
Path adVarChar Specifies the location of the container.

Example

The following code allows you to view the key attributes of a container with ID C111:

/cheminv/api/GetKeyContainerAttributes.asp?ContainerID=C111

The following code allows you to view the key attributes of a container with barcode C8BGT17:

/cheminv/api/GetKeyContainerAttributes.asp?ContainerBarcode=C8BGT17
