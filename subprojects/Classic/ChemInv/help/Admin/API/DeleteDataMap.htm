DeleteDataMap

Deletes the specified data map.

Syntax

/cheminv/api/DeleteDataMap.asp?DataMapID=adNumeric

Inputs       

Parameter Name Data Type Description Options Required? Table Name Database Field Name
DataMapID adNumeric Specifies the ID of the data map to be deleted.   Y INV_DATA_MAPS DATA_MAP_ID

Output

None.

Example

The following code deletes a data map with ID 1006:

/cheminv/api/DeleteDataMap.asp?DataMapID=1006
