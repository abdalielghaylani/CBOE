DeletePlate

Deletes the specified plate.   

Syntax

/cheminv/api/DeletePlate.asp?PlateID=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
PlateID adNumeric Specifies the ID of the plate to be deleted.   Y INV_PLATES PLATE_ID

Output

The ID of the deleted plate, which is stored in the plate_id parameter. The data type of the plate_id parameter is adNumeric.

Example

The following code deletes a plate with ID 1111: 

/cheminv/api/DeletePlate.asp?PlateID=1111
