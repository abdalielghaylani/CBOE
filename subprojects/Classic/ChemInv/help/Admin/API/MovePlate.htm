MovePlate

Moves plates from their current location to the specified location.

Syntax

/cheminv/api/MovePlate.asp?LocationID=adNumeric&PlateID=adVarChar&DoFillGrid=adVarChar[&Preview=adVarChar]

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the location to which the plate is to be moved.   Y INV_LOCATIONS LOCATION_ID_FK
PlateID adVarChar Specifies the ID of the plate to be moved.   Y INV_LOCATIONS PLATE_ID
DoFillGrid adVarChar     Y N/A N/A
Preview adVarChar     N N/A N/A

Output

The ID of the moved plate. This ID is stored in the plate_id parameter, the data type of which is adNumeric.

Example

The following code moves a plate with ID P1234 to a location with ID 111: 

/cheminv/api/MovePlate.asp?LocationID=111&PlateID=P1234
