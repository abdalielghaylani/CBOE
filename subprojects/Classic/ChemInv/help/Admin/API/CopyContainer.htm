CopyContainer

Creates multiple copies of a container. The attributes of the copied containers are exactly similar to the original container.  

Syntax

/cheminv/api/CopyContainer.asp?ContainerID=adNumeric&NumContainers=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerID adNumeric Specifies the ID of the container to be copied.   Y INV_CONTAINERS CONTAINER_ID
NumContainers adNumeric Specifies the number of the container copies to be created.   Y N/A N/A

Output

A Pipe (|) delimited list of the IDs of the new containers. For example: 100|101. These IDs are stored in the NewContainerID parameter, the data type of which is adVarChar.

Example

The following code allows you to create 4 copies of the container with ID 111:

/cheminv/api/CopyContainer.asp?ContainerID=111&NumContainers=4
