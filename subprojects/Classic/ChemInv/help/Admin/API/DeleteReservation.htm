DeleteReservation

Deletes a reservation.

Syntax

/cheminv/api/DeleteReservation.asp?ReservationID=adNumeric&ContainerID=adNumeric

Inputs

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ReservationID adNumeric Specifies the ID of the reservation to be deleted.   Y INV_RESERVATIONS RESERVATION_ID
ContainerID adNumeric Specifies the ID of the container associated with the reservation.   Y INV_RESERVATIONS CONTAINER_ID_FK

Output

The ID of the deleted reservation, which is stored in the ReservationID parameter.

Example

The following code deletes a reservation with ID 5 that is associated with a container with ID 23745:

/cheminv/api/DeleteReservation.asp?ReservationID=5&ContainerID=23745
