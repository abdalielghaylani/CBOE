DeleteContainer

Deletes a container from Inventory Enterprise.

Syntax

/cheminv/api/DeleteContainer.asp?ContainerID=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
ContainerID adNumeric Specifies the ID of the container to be deleted.   Y INV_CONTAINERS CONTAINER_ID

Output

None.

Example

The following code allows you to delete a container with ID 12737:

/cheminv/api/DeleteContainer.asp?ContainerID=12737
