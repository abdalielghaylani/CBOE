UpdateAllPlateFields

Updates the plate fields passed as parameters.

Syntax

/cheminv/api/UpdateAllPlateFields.asp?PlateID=adVarChar&LocationID=adNumeric&PlateName=adVarChar&PlateTypeID=adNumeric&PLateFormatID=adNumeric&GroupID=adNumeric&QtyUnit=adVarChar&QtyRemaining=adNumeric[&QtyInitial=adNumeric&ContainerStatusID=adNumeric&SupplierShipmentCode=adVarChar&SupplierShipmentNumber=adVarChar&SupplierShipmentDate=Date&LibraryID=adNumeric&Solvent=adVarChar&FTCycles=adNumeric&WeightUnit=adVarChar&Weight=adNumeric&ConcUnit=adVarChar&Concentration=adNumeric&Field_1=adVarChar&Field_2=adVarChar&Field_3=adVarChar&Field_4=adVarChar&Field_5=adVarChar&Date_1= Date&Date_2= Date]

Inputs

PlateID 

The ID for the plate being updated.

LocationID

The ID for the location of the plate.

PlateName

The name of the plate.

PlateTypeID

The ID of the plate type.

PlateFormatID

The ID for the plate format applied to this plate.

GroupID

The ID for the group this plate is a part of.

QtyUnit

The units of measure that the quantities in the plate are assumed to be.

QtyRemaining

The remaining quantity in the plates in the units of QtyUnit.

[QtyInitial]

The initial quantity in the plate.

[ContainerStatusID]

The ID for the status of the plate.

[SupplierShipmentCode]

A code, somtimes provided by the supplier, to identify the shipment.

[SupplierShipmentNumber]
A number, somtimes provided by the supplier, to identify the shipment.
[SupplierShipmentDate]

The shipping date, provided by the supplier as date-time literal (ie. 01-27-2001).

[LibraryID]

The ID associated with the library the plate is a part of.

[Solvent]

THe solvent used in the plate.

[FTCycles]

The number of freeze thaw cycles allowed for this plate.

[WeightUnit]

The units of weight used to measure the weight of the plate.

[Weight]

The weight of the plate in the units specified under WeightUnit.

[ConcUnit]

The units of concentration used to measure the concentration of the contents of the plate.

[Concentration]

The concentration of thecontents of the plate in the units specified under ConcUnit.

[Field_1]

The value of the configurable field, Field_1.

[Field_2]

The value of the configurable field, Field_2.

[Field_3]

The value of the configurable field, Field_3.

[Field_4]

The value of the configurable field, Field_4.

[Field_5]

The value of the configurable field, Field_5.

[Date_1]

The date in the configurable date field, Date_1 as date-time literal (ie. 01-27-2001).

[Date_2]       

The date in the configurable date field, Date_2 as date-time literal (ie. 01-27-2001).

Outputs

None

Example

A user wants to update some fields for the plate with ID P1234.  It now resides in Location with ID 111, it is an assay plate (ID=4) with the plate format associated with ID 10.  It is in Group 1 with a quantity remaining of 10 mL.  Its new status is In Use (ID=5).  

/cheminv/api/UpdateAllPlateFields.asp?PlateID=P1234&LocationID=111&PlateName=Plate1&PlateTypeID=4&PLateFormatID=10&GroupID=1&QtyUnit=mL&QtyRemaining=1&ContainerStatusID=5
