GetLocationFromID

Allows you to determine the barcode and name of a location on specifying its ID.

Syntax

/cheminv/api/GetLocationFromID.asp?LocationID=adNumeric

Input

Parameter Name Data Type Description Options Required? Table Name Database Field Name
LocationID adNumeric Specifies the ID of the location, whose details are to be determined.   Y INV_LOCATIONS LOCATION_ID

Outputs

Parameter Name Data Type Description
LocationID adNumeric Specifies the ID of the location.
LocationBarcode adVarChar Specifies the barcode of the location.
LocationName adVarChar Specifies the name of the location.

Example

The following code determines the barcode and name of a location with ID 123:

/cheminv/api/GetLocationFromID.asp?LocationID=123
