UpdateSubstance

Updates the attributes of an already existing substance.

Syntax

/cheminv/api/UpdateSubstance.asp?CompoundID=adNumeric&SubstanceName=adVarChar[&Structure=CLOB&CAS=adVarChar&ACX_ID=adVarChar&ALT_ID_1=adVarChar&ALT_ID_2=adVarChar&ALT_ID_3=adVarChar&ALT_ID_4=adVarChar&ALT_ID_5=adVarChar]

Inputs 

Parameter Name Data Type Description Options Required? Table Name Database Field Name
CompoundID adNumeric Specifies the ID of the compound to be updated.   Y INV_COMPOUNDS COMPOUND_ID
SubstanceName adVarChar Specifies the name of the compound.   Y INV_COMPOUNDS SUBSTANCE_NAME
Structure CLOB Specifies the chemical structure of the compound. The Structure parameter stores the chemical structure of the substance as the base64 encoded cdx data. Due to the length of cdx data, it is recommended to pass this data as the form data instead of passing it through the query string.   N INV_COMPOUNDS BASE64_CDX
CAS adVarChar Specifies the CAS number of the substance.   N INV_COMPOUNDS CAS
ACX_ID adVarChar Specifies the ACX ID of the substance.   N INV_COMPOUNDS ACX_ID
ALT_ID_1 adVarChar Specifies the value of the configurable field ALT_ID_1.   N INV_COMPOUNDS ALT_ID_1
ALT_ID_2 adVarChar Specifies the value of the configurable field ALT_ID_2.   N INV_COMPOUNDS ALT_ID_2
ALT_ID_3 adVarChar Specifies the value of the configurable field ALT_ID_3.   N INV_COMPOUNDS ALT_ID_3
ALT_ID_4 adVarChar Specifies the value of the configurable field ALT_ID_4.   N INV_COMPOUNDS ALT_ID_4
ALT_ID_5 adVarChar Specifies the value of the configurable field ALT_ID_5.   N INV_COMPOUNDS ALT_ID_5

Output

The ID of the updated compound. This ID is stored in the compound_id parameter, the data type of which is adNumeric.

Example

The following code updates a substance with ID 111:  

cheminv/api/UpdateSubstance.asp?EditCompoundID=111&SubstanceName=Benzene[&CAS=71-43-2]
