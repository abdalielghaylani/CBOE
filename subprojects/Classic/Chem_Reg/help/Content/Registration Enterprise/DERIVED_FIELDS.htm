DERIVED_FIELDS
Example: DERIVED_FIELDS = FORMULA,MW,FORMULA2,MW2,ENTRY_PERSON,ENTRY_DATE, FORMULA_WEIGHT,PERCENT_ACTIVE,CHIRAL,CLOGP,H_BOND_ACCEPTORS,H_BOND_DONORS
Description: Determinies which fields in the add_compound and add_batch form should should be hidden from user input because it if populated by the business tier.
Default: FORMULA,MW,FORMULA2,MW2,ENTRY_PERSON,ENTRY_DATE,FORMULA_WEIGHT,PERCENT_ACTIVE, CHIRAL,CLOGP,H_BOND_ACCEPTORS,H_BOND_DONORS,CHEM_NAME_AUTOGEN
Options: Comma delimited list of fields
NOTE:   It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application. New versions of the ChemOffice Enterprise use default configurations.
