Adding a New Solvate Record
To add a new solvate record:
1. Click Solvates in the Manage Tables area. The Solvates table appears:

 
2. Click new. The following page appears:

 
3. Enter the required information in the Solvate Name and Solvate MW fields.
4. Select True from the Active list box if you want the new solvate to appear in the Solvate Name drop-down list.
5. Click Add Record. The new solvate is added to the Solvates table.
