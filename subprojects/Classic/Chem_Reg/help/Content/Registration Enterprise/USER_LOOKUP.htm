USER_LOOKUP
Example: USER_LOOKUP = 0
Description: Indicates if the user lookup feature should be used. The user lookup feature will provide a drop-down with user codes (from the peoples table). Choosing one will then populate the chemist field with the actual person.
Default: 0
Options: 0 | 1
NOTE:   It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application. New versions of the ChemOffice Enterprise use default configurations.
