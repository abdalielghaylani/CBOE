Adding a Batch or Lot

In Registration Enterprise, you can add batch or lot information for a compound. This allows you to maintain different records for the compounds that are similar but have slightly different properties. For example, you can create batch records for the compounds that are similar but contain different impurities, behave differently in tests, or are procured from the different suppliers.
You can add batch information for a compound only if the compound is registered in Registration Enterprise and has a Registration number associated with it. When you associate a batch record with a registered compound, a unique batch number is assigned to the batch record and the batch record is added to the temporary table. When the batch record is registered, the batch number is suffixed to the Registry number of the compound, with which the batch is associated. For more information about registering batch records and temporary compounds, see "Registering a Temporary Record"
To associate a batch with a registered compound:
1. Click Add Batch in the Add to Temporary section in the Registration Enterprise home page. The following page appears:

 

NOTE:   You can also access this page by clicking the Batch link within the Registration Enterprise section in the home page of ChemOffice Enterprise.
2. Enter the registration number of the compound with which the batch is to be associated in the Registry Number text box.
3. Click OK. The New Batch Submission Form containing the information of the compound appears:

 
4. Specify the required information in the New Batch Submission Form.
NOTE:   For more information about fields in the New Batch Submission form, see "Form Fields"
5. Click Add Record. A dialog box confirming the addition of batch to the temporary table appears.
6. Click OK. The New Batch Submission Form reappears.
