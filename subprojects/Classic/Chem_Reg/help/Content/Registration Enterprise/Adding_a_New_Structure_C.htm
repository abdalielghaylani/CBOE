Adding a New Structure Comment
To add a new structure comment to the Structure Comments table:
1. Click Structure Comments in the Manage Tables area. The Structure Comments table appears:

 
2. Click new. The following page appears:

 
3. Enter the comment in the Structure Comment text box.
4. Select True from the Active list box if you want the new structure comment to appear in the Structure/Stereochemistry Comment drop-down list.
5. Click Add Record. The new structure comment is added to the Structure Comment table.
NOTE:   The "no comment" structure comment allows you to add a compound to Registration Enterprise without specifying the structure of the compound, even if the structure is a required field.
