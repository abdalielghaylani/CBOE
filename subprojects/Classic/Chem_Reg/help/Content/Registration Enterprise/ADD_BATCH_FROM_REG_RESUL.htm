ADD_BATCH_FROM_REG_RESULTS
If the ADD_BATCH_FROM_REG_RESULTS Cfserver.ini setting is as follows: ADD_BATCH_FROM_REG_RESULTS=1 a button is enabled in detail view called Add Batch/Lot. This button performs the same function as the button on the homepage with the same name performs, but the user does not have to enter a registration number. After adding a batch the user is returned to the record last viewed.
