How it is implemented
RLS is implemented through the use of Oracle’s Fine Grain Access Control. Tables are assigned policies. When a table is accessed, the policy is enforced. A policy is an additional piece of SQL that enforces a rule so a user can only select records that are linked to a project for which he has rights. Since policies are in Oracle, if there is an attempt to access data from any application (e.g. SQL Worksheet), the rule is enforced.
