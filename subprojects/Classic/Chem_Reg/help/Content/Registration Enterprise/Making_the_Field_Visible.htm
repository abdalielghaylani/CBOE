Making the Field Visible

To make a previously hidden field visible:
1. Open the cfserver.ini file and scroll down to the section title "hide fields from GUI and ChemLoader".
2. Find the EXACT FIELD NAME in the GUI_FIELDS_TO_HIDE entries, and delete the name.
3. Save your changes.
NOTE:   To see your changes in a Registration Enterprise Field, it is necessary to reset IIS.
For more information about the Cfserver.ini file, please see "CFServer.ini Configuration File".
