PRIMARY_STRWHERE
If the PRIMARY_STRWHERE Cfserver.ini file setting is as follows: PRIMARY_STRWHERE=COMPOUND_PROJECT Registration Enterprise inspects both the compound and the project when deciding if a duplicate exists. If the compound being added is a duplicate in the entire system, but not a duplicate within a project an end user would see no duplicate window appear and the compound is registered as usual. However since the same compound is a system duplicate the record is placed in the Duplicates Table. If a compound is a duplicate within a project, the Duplicate Window appears.
