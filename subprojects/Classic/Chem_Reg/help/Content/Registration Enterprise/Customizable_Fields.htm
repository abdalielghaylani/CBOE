Customizable Fields

Registration Enterprise contains some fields that can be customized according to the requirements. You can customize the name, data type, and display type for these fields. The customizable fields for the compound records are:
• 4 text fields
• 4 integer fields
• 4 real number fields
• 4 date fields
The customizable fields for the batch records are:
• 6 integer fields
• 6 real number fields
• 6 date fields
By default, the customizable fields are hidden. For information about making the customizable fields visible to the users and customizing them, see “Customizable Fields” in the ChemBioOffice 2008 Administration and Configuration Guide.
