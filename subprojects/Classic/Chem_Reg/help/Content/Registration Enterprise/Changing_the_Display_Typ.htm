Changing the Display Type of a field
To change the display type of a field:
1. Open the CFserver.ini file and scroll down to the section title DISPLAY_TYPES.
2. Find the appropriate field name in the list. The listing are in the following format:
• DISPLAY_TYPES#=FieldName ; Input Field : Option [; Display Field : Option] Be sure not to change the text to the left of the = or the FieldName.
For more information about Input Field options, please see "CFServer.ini Configuration File".
3. Save your changes.
