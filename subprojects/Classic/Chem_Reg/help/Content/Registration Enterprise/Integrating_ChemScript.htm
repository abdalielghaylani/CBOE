Integrating ChemScript

Here we describe how to integrate ChemScript with Registration Enterprise. ChemScript is a chemical programming environment that is integrated with the ChemOffice Enterprise applications. Using ChemScript, you can create your own chemistry business rules and execute those rules on scientific data in batch mode. ChemScript is based on the non-proprietary scripting language Python, which provides clear syntax, object-oriented programming, dynamic data typing, and high performance across a broad range of operating systems including Windows, UNIX, and Linux.
