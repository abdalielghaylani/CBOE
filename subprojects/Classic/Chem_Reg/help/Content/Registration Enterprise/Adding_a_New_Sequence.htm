Adding a New Sequence
To add a new prefix and starting number to the Sequence table:
1. Click Sequence in the Manage Tables area. The Sequence table appears:

 
2. Click new. The following page appears:

 
3. Enter the required information in the Prefix, Next In Sequence, and Root Number Length fields.
4. Select True from the Active list box if you want the new record to appear in the Prefix drop-down list.
5. Click Add Record. The new record is added to the Sequence table.
