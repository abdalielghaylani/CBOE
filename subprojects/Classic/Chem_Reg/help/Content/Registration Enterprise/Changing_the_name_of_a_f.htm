Changing the name of a field
To change the name of a field:
1. Open the CFserver.ini file and scroll down to the section title Field_Labels format.
2. Find the appropriate field name in the list. The listing are in the following format:
• FIELD_LABELS#=EXACT FIELD NAME : FIELD LABEL DISPLAY< BR> The only part of this listing that should be changed is the FIELD LABEL DISPLAY.
3. Change the text after the : to the name you would like displayed in the Registration Form for this field.
4. Save your changes.
