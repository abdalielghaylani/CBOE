Adding a field to the required field list
The only information necessary to add a field to the required field list is the table the field is found in, the name of the field, and the field's datatype.
For example, to make the CAS number a required field, we need to know the following values:

Syntax Variable

Value

<TableName>

Temporary_Structures

<FieldName>

CAS_Number

<DataType>

0

<Displayed Name>

CAS Number
To make this field required, add the following to the end of the required fields entry: ,Temporary_Structures.CAS_Number;0
