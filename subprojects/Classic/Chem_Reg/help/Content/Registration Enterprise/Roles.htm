Roles

Different users need to perform different tasks using Registration Enterprise. Therefore, while setting up new user accounts, system administrator assigns different roles to different users. Each Registration role is assigned specific privileges, which specify the task that can be performed using the role. For example, BROWSER only has the privilege to view the Registration Enterprise records, however, CHEMICAL_ADMINISTRATOR is allowed to add, edit, and delete the records, as well.
