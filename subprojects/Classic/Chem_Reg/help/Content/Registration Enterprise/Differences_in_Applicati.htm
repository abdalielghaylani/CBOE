Differences in Applications that can Register Compounds in Registration Enterprise
Applications with the ability to register compounds in Registration Enterprise do not necessarily allow the user to specify a project. With RLS, if a post to Registration Enterprise is performed and a project_id is not linked to the user found - the record is dumped to the Temporary Table and the project is assigned to Unspecified.
