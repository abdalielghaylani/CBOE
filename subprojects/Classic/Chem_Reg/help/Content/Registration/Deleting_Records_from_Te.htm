Deleting Records from Temporary Table

If you have the appropriate privileges, you can delete a record from the temporary table, if required.

NOTE:Records cannot be recovered after you delete them.

To delete a record from the temporary table:

1. Search the temporary table for the record to delete.
2. Mark the record to delete.
3. Click Delete Marked. The Delete Records page appears.
4. Add a description for the records to be deleted.
5. Click Delete and then Done to delete the records or click Cancel if you do not want to delete the record.
6. Click Home to go to the registration Enterprise home page.
