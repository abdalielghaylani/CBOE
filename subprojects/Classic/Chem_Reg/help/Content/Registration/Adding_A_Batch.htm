Adding A Batch

A batch is created when you register a record to the permanent table. You can add more than one batches to a registered record. You can view any batch information by selecting the specific batch in the tree structure on the left hand side of the page. Refer figure below:

 

To add a batch:

1. Click View Record link corresponding to the record. The Registry Record page appears.
2. Click Add Batch. A new batch appears in the tree structure on the left hand side of the page.
3. Fill in the Batch Information by selecting a value in the Project field. If you need to add more than one project, click Add Project.
4. Fill in the Fragments by component information by selecting a value in Fragment Types field and specifying values in code and Equivalent fields. To add more than one fragment type, click New Fragment.
5. Click Submit. The Registry Record page appears.

You can edit, move, or delete a batch.
