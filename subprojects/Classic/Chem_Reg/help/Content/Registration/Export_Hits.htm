Export Hits

You can save and export a hitlist. The structure and molecular weight are automatically exported. You can also include any other fields used to generate the hitlist. You can save the file in the following file type:

• SDF Flat Correlated-
• SDF Flat Uncorrelated
• SDF Nested
• Excel

An example of SDF Flat Uncorrelated(A), SDF Flat Correlated(B), and SDF Nested(C) is shown in the following image:

 

SDF Flat Correlated, SDF Flat Uncorrelated, and SDF Nested formats differ in the manner in which the records in the subtables are handled. In SDF Flat uncorrelated, each row in the subtable appears as a new record and hence the adjacent records contain duplicate values for the main data fields. In SDF Flat correlated, all related rows of the subtable are included in a single record and hence main data fields are not duplicated. In SDF Nested, each subtable is condensed to a single block of text and included in a single column of the record.
