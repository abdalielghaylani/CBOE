Saving Queries

You can save a query and use it later when required. Saving a query is useful when you expect the same search to be performed more than once.

To save a query:

1. After performing a search, click Save under the Query option.
2. Enter a name and description for the query.
3. Select Is Public if you want the query to be available to others.
4. Click Save.
