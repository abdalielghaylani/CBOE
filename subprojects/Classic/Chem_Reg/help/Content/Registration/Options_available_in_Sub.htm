Options available in Submit Record Form

Once you fill in the Submit Record form, you can not only submit the record to the temporary table but also save the form and use it later. You can also register the record directly to the permanent table if you have adequate privileges.

Options available in the Submit Record form include:

• Save
• Load
• Submit
• Register
• Cancel

 

Save

You can save the record as a draft for later use. For example, assume you are awaiting analytical information to determine the purity of a component you want to submit and have filled in all other fields in the form. You can save the form and add the information when it is available.

To save a record:

1. Click Save. The Save Registry form appears.
2. Enter a name for the record in the Registry Name field and a description in Registry Description field.
3. Select Public to make the record available to other users.
4. Click Save.

Load

You can open saved records using this option.

To load a record:

1. Click Load. The Load Registry form appears.
2. Click on the Registry Name of the saved record you want to open. The record information is loaded.

Submit

Use this option to submit records to the temporary table. For more information on submitting records, see Creating Registry Records.

 

Cancel

Use this option to exit from the Submit Record form and go to the main menu of Registration Enterprise.
