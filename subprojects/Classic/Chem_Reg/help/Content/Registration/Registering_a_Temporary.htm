Registering a Temporary Record

When you register a record, Registration assigns a unique number to it, and when you register a batch record, the batch number is added to the Registration number previously assigned to the compound. In addition to registering each record individually, Registration Enterprise also allows you to register all the temporary table records, at once.

NOTE:The records stored in the permanent table can only be accessed if Row Level Security is not enabled.

Registering a Compound Record

You can register a record by either selecting the Review Register option in the Registration home page or by searching for records in the temporary table using Search Temp option in the Registration home page. On selecting Review Register option the Search Temporary Record form displaying all records in temporary table appears.

You can register one record at a time or perform bulk registration.

To register a single record:

1. Click Review Record link corresponding to that record. The temporary record page which displays the record information appears.
2. Click Edit to edit the record information and click Submit to save any changes made.
3. Click Register to register the record. The record is registered and the Registry Number appears. Refer Figure below:

 

4. Click Done.

To perform bulk registration:

1. Mark the records to register.
2. Click Register Marked. The Bulk register marked page appears.
3. Choose the duplicate action to perform.
4. Click Register Marked.
