Searching Records in Temporary Table

You can either retrieve all the records in the temporary table or search for specific records in the temporary table. You can search by structure, temporary ID or by any other parameter for records in the temporary table.

To search the temporary table for a record:

1. Click Search Temp in the Registration Enterprise home page. The Search Temporary Records page appears.

 

2. Click Retrieve All to retrieve all records in the temporary table or enter a specific search criteria under Component information or Batch Information and click Search. A list of records matching the search criteria you specified appears. For more information on specifying search criteria, See Search Criteria.

To create a hitlist of all the records displayed, click Mark All. The count of the marked hits appears on the right hand side of the page.

To deselect all the selected records, click Un- Mark All. For more information on marking records, See Marking Records.
