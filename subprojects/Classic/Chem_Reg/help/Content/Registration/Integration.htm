Integration

Registration can be integrated with other CambridgeSoft applications such as Inventory Enterprise, ChemACX database, BioSAR Enterprise, BioAssay Enterprise, and E-Notebook. When integrated, it provides a solution to the problems associated with organizing, storing, and sharing chemical information. Each product shares information through common databases. Each product also provides interfaces to operations performed in different application. For example, you can register a compound in Registration Enterprise through E-Notebook or BioAssay HTS. Sharing a common database makes all information consistent with a change in any one application.This means that every user in the research and development process has access to the up-to-date data. It saves time and reduces the number of errors in re-interpreting and re-entering data in multiple locations.

 

You can link Registration Enterprise to any of these ChemBioOffice Enterprise applications, if they are installed:

• Inventory Enterprise
• BioSAR Enterprise
• BioAssay HTS
• E-Notebook
• ChemACX Database

On integrating Registration Enterprise with:

• Inventory Enterprise- you can search compounds in Registration Enterprise using Inventory Enterprise. This lets you include the compounds in Registration Enterprise in inventory containers without re-entering data. You can also send Registration Enterprise search results to Inventory Enterprise. Any changes made to the record in the Registration Enterprise is immediately reflected in Inventory Enterprise. Inventory Enterprise linked with Registration Enterprise and ChemACX, provides solution for complete management of organization’s inventory.
• ChemACX- you can register compounds from a catalog listing. This populates the registration record with all the data in the ChemACX record.
• BioSAR Enterprise- you can search the registration data and biological data and create a report containing only the information you want.
• BioAssay HTS- you can register a compound in Registration Enterprise through BioAssay HTS or E-Notebook.
• E-Notebook- you can register a compound in Registration Enterprise through E-Notebook.
