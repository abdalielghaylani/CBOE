Submitting New Records

You can submit a registry record containing a single component, multiple components, or batch information. You can submit a record to the temporary table in the database or, if you have adequate privileges, register the record directly to the permanent table.

In Registration Enterprise, you can register pure compounds, mixtures, and batches.
