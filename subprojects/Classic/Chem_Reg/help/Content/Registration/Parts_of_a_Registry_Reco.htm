Parts of a Registry Record

A Registry Record represents a substance and all its associated registry data. It includes:

• Components- represents a distinct pure substance within a registered record
• Batch- represents a particular preparation of a Registry Record.
• Batch Component- information about the components can differ for each preparation so it is tracked for each batch separately.

A registry record data contains four types of information:

• Registry Information
• Component Information
• Batch Information
• Batch Component Information
