Batch Information

Batch information consists of the following fields:

• Batch ID- Unique batch number assigned to the batch record.
• Date Created- Date on which the batch was created.
• Last Modification Date- Date on which the batch was last modified.
• Projects- Projects associated with the batch. Chemist can associate different batches to different Projects.
• Batch Custom Properties:
• Chemist- Scientist who synthesized the batch.
• Synthesis Date- Date on which the batch was synthesized
• Notebook Reference- The reference to the chemist notebook that contains the batch synthesis information
• Amount- Substance quantity obtained for this batch
• Units- Units of the amount obtained.
• Appearance- Visual appearance of the substance.
• Purity
• Purity Comments
• Analytical Reference
• Solubility
• Batch Comments
• Storage Requirements and
• Warnings
