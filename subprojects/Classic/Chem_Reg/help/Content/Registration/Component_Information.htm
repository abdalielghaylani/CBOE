Component Information

Component information consists of the following fields:

• Component ID- Each component is defined by a unique ID. The format of Component ID is defined by the Administrators. The default format is: Prefix: Sequence Number.
• Identifiers- The component identifiers describe the pure substance. The default component Identifiers are:
• Chemical Name
• Chemical Name Autogenerated
• CAS
• Synonym
• Structure- Each component may have an associated structure.
• Fragments- When a component is a chemical compound, only the base structure is drawn on the Structure box and the rest of the structure compound information is filled in the Fragment list. Fragment Information differs from one batch to another. Salts and Solvate are the most typical example of Fragment types
• Component Custom Properties:
• Component Comments- Comments associated with each component.
• Stereochemistry Comments- Any stereochemistry remark.
NOTE:Identifiers and Component ID format, and fragment types are defined by the administrator.
