Container

The term, container, refers to a physical entity that can store the chemical substances. You can create different types of containers in Inventory Enterprise 11.0, such as tube, bottle, box, vial, and cylinder. Unlike physical containers, you can associate only single primary chemical substances with an Inventory container. You can associate secondary chemical substances, such as solvent, with the container using the additional container fields. For a complete list of container fields, see "Inventory Enterprise 11.0 Form Fields".

You can also associate Registration Enterprise or ChemACX substances with the Inventory containers, if Inventory Enterprise 11.0 is integrated with Registration Enterprise or ChemACX applications.
