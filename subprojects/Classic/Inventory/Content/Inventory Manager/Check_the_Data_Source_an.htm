Check the Data Source and UDL

The ODBC Data Source and UDL give the application information about the Oracle Database.

If you are having difficulties, be sure to check the following:

ODBC Data Source

1. Open the ODBC Control Panel by going to Start >Programs>Administrative Tools>Data Sources
2. Select System DSN tab
3. If there is a listing for Chem_Inv, highlight it and click Configure. Skip to step 7.
4. If there is no listing for Chem_Inv, click Add.
5. Highlight Oracle ODBC Driver driver.
6. Click Finish.
7. Fill in the following information:
• Data Source Name: Chem_Inv
• Description: Chem_Inv
• Service Name: <Oracle Service Name> for your target database.
NOTE:For information about your Oracle Service Name, see your Oracle DBA or consultant.
• User ID: cheminvdb2
8. Click Ok.

Inventory Enterprise UDLs

1. Browse to <webroot>/ChemOffice /cheminv/config/.
2. Open cheminv.udl by double-clicking on it.
3. Change the Data Source to match your Oracle service name.
4. Set the password to oracle.
5. Click the Test Connection button.
6. If the connection succeeds, click Ok.
