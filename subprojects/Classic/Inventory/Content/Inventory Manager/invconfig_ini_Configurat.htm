invconfig.ini Configuration File

The Invconfig.ini file is found in the following location: <webroot>/ChemOffice/cheminv/config. This file defines many parameters that allow Inventory Enterprise to function properly, including the format of custom fields. Most of this file should not be edited since editing this file in the wrong place could cause major problems when running the Inventory Enterprise application.

NOTE:   It is the responsibility of a site's administrator to propagate any changes made to ChemBioOffice Enterprise application files to future versions of the application. New versions of the ChemBioOffice Enterprise use default configurations.

Description: This setting is equal to the server name where Document Manager is installed. This is necessary for successful integration with Document Manger. Set this value to NULL if DOcument Manager integration is not desired.

Example: CUSTOM_FIELDS = FIELD_1:Custom Field 1;0,FIELD_2:Custom Field 2;0

Description: A comma delimited list of the optional customizable fields that are in use. If the field appears in this list, it will appear on all relevant forms. Names of field are in the following format:

<Field_Name>:<Field_Label>;<0|1>

where Field_Name is the name of the configurable field (as shown in the INI file), Field Label is how the field should be labeled in the interface, and the last parameter indicates if the field is required (1) or not (0). NOTE: Putting spaces in field names may make the field display in a less then optimal way.

Default: ""

Options: A comma delimited list in the above format.
