Zebra Label Printing

In order to utilize the Zebra label printing feature, the LPR definition and Zebra label printer itself must be created within the application. This is done using Tasks>Manage Tables.

Under the [MANAGE_TABLES] section in the invconfig.ini file, the Editable_Tables entry has been modified to include the following values:

• cheminvdb2.inv_label_printers:Label Printers
• cheminvdb2.inv_lpr_definition:LPR Definitions

These values correspond to the new tables created to support Zebra label printing. In addition, the Editable_Tables entry has been expanded to cover multiple lines as the entry has now exceeded the allowable limit.

 

Setting up a new Zebra label printer involves two steps:

• Defining the appearance of the actual label/barcode
• Grouping the definition with a label type and assigning it to a printer
