Searching Special Locations

Special Locations are locations, defined by an administrator, as possible locations that should be excluded from a search. For example, a user may want to exclude the Disposed location from searches because the container is probably no longer available.

Please see your system administrator if you would like to add a particular location to the Special Locations list.

To select the special locations to exclude:

  1. Click on the Special Link (next to the Exclude Special Locations checkbox in a search form).
    The Special Inventory Locations dialog box appears.

  2. Select the checkbox next to any location name that should be recognized as a special location.
  3. Click  OK.
