Summary Tab

The Summary Tab of the View Container frame provides a summary of important details about the container and its contents.  For more information regarding the fields found under the Summary Tab, click on the appropriate field in the image below.

The system administrator also has the opportunity to add customized links to the right side of the Summary Tab above the ACX Lookup link. These links can link to other data sources such as MSDS or Analytical Data. Please see your system administrator for more details.
