Manage Roles

The Manage Roles interface of Inventory Enterprise allows you to add, update, and delete information related to roles in various ChemOffice Enterprise applications, such as BioAssay Enterprise and BioSAR Enterprise. The Manage Roles interface also allows you to add or remove users corresponding to a particular role. In addition, the Manage Roles interface allows you to combine the privileges assigned to roles of different applications. Add Role

To add a new role to a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Roles from drop down list in the Manage Roles page.
  2. Click the Refresh link. The list of roles of the selected application is displayed in the Select a Role list box.
  3. Click the Add Role button to display the Add new Role page.
  4. Type the name of the new role in the Role Name text box.
  5. Select the privileges to be assigned to the new role.
  6. Click the OK button. The new role is added to the list of roles of the selected application.

Update Role

To update the information about an existing role of a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Roles from drop down list in the Manage Roles page.
  2. Click the Refresh link. The list of roles of the selected application is displayed in the Select a Role list box.
  3. Select the role whose information you need to update, in the Select a Role list box.
  4. Click the Update Role button to display the Update Role page.
  5. Modify the privileges of the role according to your requirements.
  6. Click the OK button. The information about the selected role is updated.

Manage Grantees

To manage grantees:

  1. Select the appropriate application in the View Roles from drop down list in the Manage Roles page.
  2. Click the Refresh link. The list of roles of the selected application is displayed in the Select a Role list box.
  3. Select the role whose grantees you need to manage.
  4. Click the Users button to display the Manage Grantees for this Role page.
  5. Add or remove users from the selected role according to your requirements.
  6. Click the OK button.

Combine Roles

To combine roles of different applications:

  1. Select the appropriate application in the View Roles from drop down list in the Manage Roles page.
  2. Click the Refresh link. The list of roles of the selected application is displayed in the Select a Role list box.
  3. Select the role with which you need to combine roles of other applications.
  4. Click the Roles button to display the Manage Roles granted to this Role page.
  5. Add or remove roles of other applications from the selected role according to your requirements.
  6. Click the OK button.

Delete Roles

To delete an existing role of a ChemOffice Enterprise application:

  1. Select the appropriate application in the View Roles from drop down list in the Manage Roles page.
  2. Click the Refresh link. The list of roles of the selected application is displayed in the Select a Role list box.
  3. Select the role that you need to delete, in the Select a Role list box.
  4. Click the Delete Role button to display a page, which confirms whether you really want to delete the selected role.
  5. Click the OK button. The selected role is deleted.
