View History

The view history area of Inventory Enterprise is available as part of substance management. To access this section, please take the following steps:

  1. From the Substance Management area, access the search form.
  2. Perform a search for the appropriate substance. 
  3. While in list view, click the  Details button for the substance.

  4. Click on the View History link.

The View History window opens.

 

This area allows you to view an audit report of substances. A user can filter the details according to Filter Criteria.

 

A user can view Audit details in two ways, Standard & Aggregate. The application displays the Standard report by default:

To view an Aggregate report:

  1. Click Aggregate.

User can view the audit details group by fields. User can also select radio button to view group by date.

  1. Enter details in appropriate fields.
  2. Click Filter.
