Create an Order

The Create Order link is accessed from the Filled Sample Requests screen or from the Manage Orders screen.

To Create an Order:

  1. Click Create Order.

    The Create Order dialog appears.


     
  2. Scan in the containers in the order.
  3. Enter the Delivery Location.
  4. Enter the Shipping Conditions.
  5. Click Create Order.
