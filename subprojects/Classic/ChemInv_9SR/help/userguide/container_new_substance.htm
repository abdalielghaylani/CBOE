Substance Tab

The Substance Tab of the New Container window holds information about the substance in the container. When the tab is first opened, the tab is empty except for two links, New Substance and Select Substance.

In order to insert a substance into the record, click on New Substance, if you would like to insert a substance that is not registered yet, or Select Substance, if you would like to insert a substance that is already in the database.

If New Substance  was selected, a new window opens that allows you to input information about the substance.

If Select Substance was selected, a new window opens that allows you to search for a substance. Do the following to search:

  1. Enter the desired information.
  2. Click Search.
  3. A list of substances is returned. Click on the details button for the appropriate substance.
  4. Click OK to insert the substance.
