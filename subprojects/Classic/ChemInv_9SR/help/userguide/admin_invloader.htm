Inventory Loader

Inventory Loader is an application that allows you to import compounds from a ChemFinder database and load them into the Inventory Enterprise or Registration Enterprise applications. Inventory Loader can run only on that computer, which can access the ChemOffice Enterprise application server.

Most often, the compounds that are to be loaded into the Inventory Enterprise or Registration Enterprise applications are stored in a SDFile. In such situation, you can load the compounds by converting the SDFile into a ChemFinder database. You can convert the SDFile into a ChemFinder database using the ChemFinder application.

The Inventory Loader application can import compounds either directly into the Inventory Enterprise or into a series of new plates. For information about loading compounds into a series of new plates or loading the compounds directly, please see Loading Compound into Plates and Loading Compounds Directly.
