Loading Compounds Directly

To import the compounds directly into the Inventory Enterprise and Registration Enterprise databases:

  1. Start the Inventory Loader application. The Introduction screen of the Import ChemFinder Database wizard appears:
  2. Click the Next button to begin the wizard. The Login to the Plate Inventory Database page appears.
  3. Enter the name of the ChemOffice server and your user name and password in the Server, User ID, and Password text boxes, respectively.

Note: Ensure that the user name that you have entered in the preceding page has sufficient permissions to create plates and load compounds in it.

  1. Click the Next button. The Select ChemFinder Database page appears.

Note: If an error message appears, stating that no plate locations or formats are configured in Inventory Enterprise, then you need to create a location that can hold plates and/or create a valid plate format, in Inventory Enterprise.

  1. Click the Browse button to locate and specify the ChemFinder database from which the compounds are to be imported.
  2. Select the Load Structures Only radio button.
  3. Click the Next button. The Select Fields page appears:
  4. Set the fields displayed in the Value column. You can set a field either by mapping it with a field of ChemFinder database table, in the Field Name column or by specifying a default value for the field in the Default column.
  5. Click the Next button. The Log Plate XML page appears.
  6. Select the Log data to the following location check box and specify the location at which you want to log the data in the text box displayed below the check box, if you want to log the plate xmls for the future reference.
  7. Click the Next button. The Ready to Import page appears.
  8. Click the Finish button to begin to import the compounds. You can view the progress of the import process in the Ready to Import page.
  9. Click the Finish button when the import process finishes.
