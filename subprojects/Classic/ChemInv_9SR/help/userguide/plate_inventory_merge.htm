Merge Plates

To see if you have the appropriate privileges to merge plates, see roles and privileges .

The Merge Plate function allows a user to merge the contents of two plates together.  Plates can be merged if they were created from the same parent plate.  The parent ID of a plate is listed under the Summary Tab in the Details Frame.  If there is no parent listed, the plate cannot be merged with any other plates.

To merge two plates:

  1. Search for or browse to the first plate you wish to merge.
  2. Open the plate in the Details Frame.
  3. Click Merge Plates.
    The Merge Plate dialog appears.

     
  4. Select the plate to merge with the current plate..
  5. Click  OK.
