Multi Scan

The Multi Scan feature allows a user to scan more than one container in the same list. A user can perform few operations such as “Check In”, Check Out”, “Move Container”, “Retire Container”, Delete Container” and “Clear List”, on the scanned container(s).

In order to scan new container:

  1. From the Current Location Frame, click Multi Scan.

The Multi Scan window opens.

  1. Scan in a container using a barcode scanner or enter a Container ID in the field and press Tab.


    2. Note: A user can scan multiple Container IDs. Note: A user can also remove container by clicking Remove link.
