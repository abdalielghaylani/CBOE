Global Search - Registration Enterprise Substances

Inventory result records for Registration Enterprise substances use a blue color scheme to identify themselves. These substances are linked directly to the Registration Enterprise to allow the user to retrieve Registration data from Inventory Enterprise.

  • Click the linked Reg# to view the record for this substance in the Registration Enterprise. Click  Names to view a list of alternate IDs for the substance.
  • Click  Details to view containers with this substance.
  • Click  Mark to mark the record. For more information about marked records, please see Marking Records.
