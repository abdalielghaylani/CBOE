Text File Template Options

Text File imported into Inventory Enterprise through the Create Plates from Text File interface must be in one of the following supported formats:

Template Name # of Header Rows File Delimiter # of Columns
CS Default 0 comma 30
Source/Target Plate-Tab Delimited 1 tab 2
Source/Target Well-Tab Delimited 0 tab 4
Source/Target Unified-Comma Delimited 7 comma 14
Source/Target Z Pattern-Comma Delimited 0 comma 7

CS Default

The CS Default format is delimited by commas and contains 0 header lines. This import template is not plate format specific. The following columns make up this type of file:

Column Contains ...
A Source Plate Barcode
B Source Well Number
C Target Plate Barcode
D Target Well Number
E Initial Quantity
F Units of Measure ID
G Concentration
H Units of Concentration ID
I Solvent ID
J Field 1
K Field 2
L Field 3
M Field 4
N Field 5
O Field 6
P Field 7
Q Well Field 1
R Well Field 2
S Well Field 3
T Well Field 4
U Well Field 5
V Well Date 1
W Well Date 2

Source/Target Plate-Tab Delimited

The Source/Target Plate-Tab Delimited format is delimited by tabs and contains 1 header line. This import template is not plate format specific. The following columns make up this type of file:

Column Contains ...
A Target Plate Barcode
B Source Plate Barcode

'

Source/Target Well-Tab Delimited

The Source/Target Well-Tab Delimited Format is delimited by tabs and contains 0 header lines. This import template is not plate format specific. The following columns make up this type of file:

Column Contains ...
A Target Plate Barcode
B Target Well Coordinate
C Source Plate Barcode
D Source Well Coordinate

Source/Target Unified-Comma Delimited

The Source/Target Unified-Comma Delimited format is delimited by commas and contains 7 header lines.  This import template is not plate format specific. The following columns make up this type of file:

Column Contains ...
A N/A
B N/A
C N/A
D N/A
E Source Plate Barcode
F Source Well Number
G N/A
H N/A
I N/A
J Target Plate Barcode
K Target Well Number
L N/A
M N/A
N N/A

Source/Target Z Pattern-Comma Delimited

The Source/Target Z Pattern-Comma Delimited format is delimited by commas and contains 0 header lines. When imported, this file uses a z pattern interleaving process to layout the new plate. This format is usually used to create 1 384 well plate from 4 96 well plates.  The following columns make up this type of file:

Column Contains ...
A N/A
B N/A
C Target Plate Barcode
D Target Quadrant
E Source Plate Barcode
F N/A
G N/A
