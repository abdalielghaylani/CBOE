Creating Dithered Reformat Map

In case of the dithered reformat map, if P1, P2, P3, and P4 plates are reformatted, the well a1 of the new plate will contain compound of P1, well a2 will contain compound of P2, well b1 will contain compound of P3, and well b2 will contain compound of P4.

Note: The order in which the compounds of the different parent plate are distributed to the different wells of the reformatted plate can vary. You can specify the required plate fill order while creating the reformat map.

To create a dithered reformat map:

  1. Click the Tasks link within the Inventory Enterprise section in the home page of ChemOffice Enterprise. The Administrative Menu page appears.
  2. Click the Plate Settings link in the Plate Management section.
  3. Click the New link next to the Reformatting Maps list box. The Manage Reformat Maps dialog box appears:

  1. Perform the following tasks in the Manage Reformat Maps dialog box:
  • Specify the name of the reformat map in the Reformat Map Name text box.
  • Ensure that Dithered is selected in the Reformat Map Type list box and set the plate fill order.
  • Select format of the source plate from the Source Plate Format list box.
  • Select format of the target plate from the Target Plate Format list box.
  1. Click the OK button. A screen, informing you about the successful creation of the reformat map, is displayed.
  2. Click the OK button.
