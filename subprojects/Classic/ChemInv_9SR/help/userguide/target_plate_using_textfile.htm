Creating a Target Plate

To create a target plate from a text file:

  1. Click the Tasks link within the Inventory Enterprise section in the home page of ChemOffice Enterprise. The Administrative Menu page appears.
  2. Click the Create Plates from Text File link. The Create Plates From Text File dialog box appears:

  1. Perform the following tasks in the Create Plates From Text File dialog box:
  • Select appropriate Plate Import template from the Import Template list box.
  • Click the Browse button next to the Text File text box and locate the text file from which the plate is to be created.
  • Select plate type from the Plate Type list box.
  • Select plate format from the Plate Format list box.
  • Select plate status from the Plate Status list box.
  1. Click the OK button.
