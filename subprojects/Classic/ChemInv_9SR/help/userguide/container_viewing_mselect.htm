Multi Select

The Multi Select Link changes the container list frame to details view and places a checkbox next to each Container ID. This allows users to select more than one containers and performs one action on all of those containers at once.

For example, if you would like to move 3 containers to a new location:

  1. Click Multi Select.

    The containers are listed in Details view with checkboxes next to each container ID.

  2. Select the checkbox next to any container, on which you would like to perform any operation.

The following operations can perform on selected containers:

Check Out

Check In

Move Containers

Retire Containers

Delete Containers Update Container
