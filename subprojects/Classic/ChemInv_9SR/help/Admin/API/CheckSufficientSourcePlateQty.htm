CheckSufficientSourcePlateQty

Checks to see if the parent plate contains enough substance to be alloted as requested. This functions assumes that the amount necessary = NumPlates*WellAmt.  

Syntax

/cheminv/api/CheckSufficientSourcePlateQty.asp?PlateID=adVarChar&WellAmt=Int&NumPLates= Int

Inputs

PlateID

The PlateID for the source (or parent) plate. 

WellAmt

The amount of substance to be placed in each new plate.

NumPlates

The number of plates that will be created with the parent plate's substance. 

Outputs

0 indicates a sufficient quantity is available,  1 indicates insufficient quantity

Example

A plate (ID = P12345) will be sampled, taking 1 microliter for each of 4 plates

/cheminv/api/CheckSufficientSourcePlateQty.asp?PlateID=P12345&WellAmt=1&NumPLates= 4
