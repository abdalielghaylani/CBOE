GetLocationFromID

Returns the location ID for a specified Location Barcode.

Syntax

 /cheminv/api/GetLocationFromID.asp?LocationID= adVarChar

Inputs

LocationID

The Location ID for the container you are requesting information for.           

Outputs

LocationBarcode
LocationID
Location Name

Example

A user requests the Locaiton Barcode for the location with ID 123.

 /cheminv/api/GetLocationFromID.asp?LocationID= 123
