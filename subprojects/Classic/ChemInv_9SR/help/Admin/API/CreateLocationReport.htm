CreateLocationReport

Causes the RPT report writer to produce a printable Location report.  Loation reports contain a list of all containers within an inventory location.  Additionally containers to appear in the report may be filtered by Compound_ID.  The function returns the file name of the report as saved to the server's reports directory.  Typically the URL to the reports directory is /cfserverasp/RPT/Reports/.  Check the invconfig.ini file for the location of the reports directory on your server. 

Syntax

/cheminv/api/CreateLocationReport.asp?LocationID=adNumeric&ReportName=adVarChar&CompoundID=adNumeric&[ReportFormat=adVarChar]

Inputs

LocationID

Location_ID value of the loaction for which the report is to be produced.

ReportName

The name of the report template as defined in the MS Access reports database.

CompoundID

Filters the list of containers in the Location to match the substance associated with Compound_ID.

ContainerID

Filters the list of containers in the Location to be limited to containers with equivalent ContainerIDs.

[ReportFormat]

The format of the printable report.  Allowed values include SNP (Access SnapShot format), PDF (Adobe Portable Document Format), RTF (Rich Text Format) and XLS (Microsoft Excel document).  Defaults to SNP.

Outputs

None

Example

A user wants to create a location report for Location ID 121 named LReport121 including the following containers: 111,123,142 in the SNP format.

/cheminv/api/CreateLocationReport.asp?LocationID=121&ReportName=LReport121&CompoundID=111,123,142
