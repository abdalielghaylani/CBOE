SplitPlate

Removes a specified amount of substance from the source plate and creates new plates with that substance, according to the parameters specified.

Syntax

/cheminv/api/SplitPlate.asp?PlateID=adVarChar&WellAmt=Int&PlateTypeID=Int&SolventAmt=Int&Solvent=adVarChar&NumPlates= Int

Inputs

PlateID

The ID for the plate being split.

WellAmt

The amount to be removed from the source plate into the new plate.

PlateTypeID

The ID for the Plate Type of the new plate.

SolventAmt

The amount of solvent added to the plate.      

Solvent

The name of the solvent added to the plate.  

NumPlates

The number of plates that are created. Note: The amount in QtyAmt is removed for each of the new plates.       

Outputs

PlateID

The new plate IDs.

Example

A user splits a plate (ID=P777) into 3 new Assay Plates (ID=2), each conatiner 10 microliters from the source plate.  100 microliters of water is added to the new plates.  

/cheminv/api/SplitPlate.asp?PlateID=P777&WellAmt=10&PlateTypeID=2&SolventAmt=100&Solvent=H2O&NumPlates=3
