UpdateAllContainerFields

Updates ALL container attributes. Caution: Ommiting a given optional parameter from the function call will set the ommited attribute to NULL.

Syntax

/cheminv/api/UpdateAllContainerFields.asp?ContainerID=adNumeric&LocationID=adNumeric&UOMID=adNumeric&QtyMax=adDouble&ContainerTypeID=adNumeric[&ContainerStatusID=adNumeric&CompoundID=adNumeric&ExpDate=adDBTimeStamp&QtyRemaining=adDouble&MinStockQty=adDouble&MaxStockQty=adDouble&ContainerName=adVarChar&ContainerDesc=adVarChar&RegBatchID=adVarChar&Barcode=adVarChar&TareWeight=adDouble&UOWID=adNumeric&Purity=AdDouble&UOPID=adNumeric&Concentration=adDouble&UOCID=adNumeric&Grade=adVarChar&Solvent=adVarChar&Comments=adVarChar&SupplierID=adNumeric&SupplierCatNum=adVarChar&LotNum=adVarChar&DateProduced=adDBTimeStamp&DateOrdered=adDBTimeStamp&DateReceived=adDBTimeStamp&ContainerCost=adNumeric&OwnerID=adVarChar&CurrentUserID=adVarChar&PONumber=adVarChar&ReqNumber=adVarChar]

Inputs

ConatinerID

ContainerID of container to be moved.

LocationID

The Location_ID value of the location where the container is stored.

UOMID

The unit of measure to be used for this container as a UNIT_OF_MEAS_ID value. Use ShowContainerUnits to see valid UNIT_OF_MEAS_ID values

QtyMax

Maximum capacity of the container in the units defined by UOMID.

ContainerTypeID

Type of container as a Type_ID value. Use ShowContainerTypes to see valid Type_ID values.

[ContainerStatusID]

Status of container as a Status_ID value. 

[QtyRemaining]

Quantity of material remaining in the container at the present time in the units defined by UOMID.

[MinStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue a re-stock warning.

[MaxStockQty]

Quantity of material in the units defined by UOMID upon which the system should issue an over-stock warning.

[CompoundID] 

Compound_ID value associating the substance in the container with a preexisting inventory compund.

[ContainerDesc]

A description of the container.

[ContainerName]

The name of the container.  If empty, defaults to the system's internal ContainerID.

[RegID] 

A registry number from the Registration System.

[BatchNumber] 

A registry batch number from the Registration System.

[Barcode]

A unique alphanumerical identifier used to print a container barcode.

[TareWeight]

The weight of the physical container in the units of UOWID.

[NetWeight]

in the units of UOWID.

[FinalWeight]

in units of UOWID.

[UOWID]

Unit of weight value as a Unit_ID value from the Units table.

[Concenctration]

The concentration of the substance designated by CompoundID in units of UOCID.

[Density]

The densityof the substance designated by CompoundID in units of UODID.

[UOCID]

Unit of concentration value as a Unit_ID value from the Units table.

[UODID]

Unit of density value as a Unit_ID value from the Units table.

[Purity]

The purity of the substance designated by CompoundID in units of UOPID.

[UOPID]

Unit of purity value as a Unit_ID value from the Units table.

[Grade]

The quality or purity designation associated with the substance designated by CompoundID.

[Solvent]

The name of the solvent.

[Comments]

Comments or instructions associated with this container. Maximum length is 4000 characters. It may be passed as form value to avoid truncation by querystring size limit.

[SupplierID]

Supplier_ID value associating the Supplier with a preexisting Supplier Name.

[SupplierCatNum]

The catalog number for the compound assigned by the supplier.

[LotNum]

Supplier's Lot Number

[ContainerCost]

The cost of the container.

[OwnerID]

User_ID value of the user who owns responsibility of the container.

[CurrentUserID]

User_ID value of the user currently responsible for the container.

[PONumber]

The PONumer under which the container was ordered.

[POLineNumber]
[ReqNumber]

The internal requisition number under which the container was ordered.

[Field_1]

The value in the optional field, Field_1.

[Field_2]

The value in the optional field, Field_2.

[Field_3]

The value in the optional field, Field_3.

[Field_4]

The value in the optional field, Field_4.

[Field_5]

The value in the optional field, Field_5.

[Field_6]

The value in the optional field, Field_6.

[Field_7]

The value in the optional field, Field_7.

[Field_8]

The value in the optional field, Field_8.

[Field_9]

The value in the optional field, Field_9.

[Field_10]

The value in the optional field, Field_10.

[ExpDate]

The date the compound will expire as date-time literal (ie. 01-27-2001).

[DateProduced]

The date the compound was produced as date-time literal (ie. 01-27-2001).

[DateOrdered]

The date the compound was ordered as date-time literal (ie. 01-27-2001).

[DateReceived]

The date the compound was received as date-time literal (ie. 01-27-2001).

[Date_1]

The date used in the option field Date_1 as date-time literal (ie. 01-27-2001).

[Date_2]

The date used in the option field Date_2 as date-time literal (ie. 01-27-2001).

[Date_3]

The date used in the option field Date_3 as date-time literal (ie. 01-27-2001).

[Date_4]

The date used in the option field Date_4 as date-time literal (ie. 01-27-2001).

[Date_5]

The date used in the option field Date_5 as date-time literal (ie. 01-27-2001).
