UpdateContainer2

Updates the container fields passed as parameters.

Syntax

 /cheminv/api/UpdateContainer2.asp?

Inputs

ContainerIDs 

List of container IDs to be updated.  IDs are separated by a comma.

ValuePairs

The Value Pairs that will update the containers appropriately.

Outputs

0 if successful
