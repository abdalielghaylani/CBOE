CalcMolarConc

Calculates the molar concentration from the inputs.   

Syntax

/cheminv/api/CalcMolarConc.asp?MolarAmt=Int&SolventVolume=Int&SolventVolumeUnitID= Int

Inputs

MolarAmt

The number of moles. 

SolventVolume

The volume of solvent.

SolventVolumeUnitID

ID associated with the units used when calculating the volume.

Outputs

MolarConcentration 

Example

A user wants to calculate the molar concentration for a mixture which includes 2 moles of the substance and 100 microliters of the solvent.  micoliters has an ID of 5.  

/cheminv/api/CalcMolarConc.asp?MolarAmt=2&SolventVolume=100&SolventVolumeUnitID= 5
