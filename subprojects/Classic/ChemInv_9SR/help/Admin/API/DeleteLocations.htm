DeleteLocations

Deletes the locations specified in the LocationList. Returns the number of locations deleted. For any locations to be deleted, all locations given in the LocationList must be empty. Use DeleteContainers to ensure every location to be deleted is empty.

Syntax

/cheminv/api/DeleteLocations.asp?LocationList=List

Inputs

LocationList

A list of Location_ID values for the locations to be deleted. Use a comma to separate multiple Location_ID values. Use a dash to express a range of Location_Id values.

Outputs

None

Example

A user wants to delete the locations 111 and 112.

/cheminv/api/DeleteLocations.asp?LocationList=111,112
