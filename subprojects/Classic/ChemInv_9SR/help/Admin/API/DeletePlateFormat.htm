DeletePlateFormat

Deletes the specified plate format.   

Syntax

 /cheminv/api/DeletePlateFormatID.asp?PlateFormatID= adNumeric

Inputs

PlateFormatID

The ID of the plate format to be deleted.      

Outputs

None

Example

A user deletes the plate format with ID 1. 

 /cheminv/api/DeletePlateFormatID.asp?PlateFormatID= 1
