UpdateSubstance

Updates a substance using the entered parameters.

Syntax

/cheminv/api/UpdateSubstance.asp?EditCompoundID=adNumeric&SubstanceName=adVarChar[&Structure=Structure&CAS=adVarChar&ACX_ID=adVarChar&ALT_ID_1=adVarChar&ALT_ID_2=adVarChar&ALT_ID_3=adVarChar&ALT_ID_4=adVarChar&ALT_ID_5= adVarChar]

Inputs 

EditCompoundID

The ID for the compound being updated.

SubstanceName

The name of the substance.

[Structure]

The chemical structure of the substance.

[CAS]

The CAS number of the substance.

[ACX_ID]

The ACX ID for the substance.

[ALT_ID_1]

The value of the configurable field ALT_ID_1.

[ALT_ID_2]

The value of the configurable field ALT_ID_2.

[ALT_ID_3]

The value of the configurable field ALT_ID_3.

[ALT_ID_4]

The value of the configurable field ALT_ID_4.

[ALT_ID_5]

The value of the configurable field ALT_ID_5.

Outputs

None

Example

A user wants to update a substance with ID 111.  It is Benzene with a CAS number of  71-43-2.  

cheminv/api/UpdateSubstance.asp?EditCompoundID=111&SubstanceName=Benzene[&CAS=71-43-2]
