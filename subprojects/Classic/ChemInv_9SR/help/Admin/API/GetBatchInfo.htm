GetBatchInfo

Lists information for a specified batch or a registered compound.

Syntax

 /cheminv/api/GetBatchInfo.asp?RegID=adVarChar&RegNumber= adVarChar[&BatchNumber= adNumeric]

Inputs

RegID

The Reg ID (from Registration System) for the batch you are requesting information for.  If this is not provided, RegNumber is required.      

RegNumber

The Reg Number (from Registration System) for the batch you are requesting information for.  If this is not provided, RegID is required.      

[BatchNumber]

The batch number (from Registration System) for the batch you are requesting information for.  If this is not specified, and Reg ID is specified, the batch is assumed to be 1.      

Outputs

For each batch:

RegNumber
Notebook Page
UserID
Amount
Amount Units
RegID
Batch Number

Example

 A user requests to view information for the batch represented with the following RegNumber: AB12345-3

  /cheminv/api/GetBatchInfo.asp?RegNumber= AB12345-3

OR, a user requests to view information for the third batch of the compound with RegID: AB12345

 /cheminv/api/GetBatchInfo.asp?RegID=AB12345[&BatchNumber= 3]
