UpdateRequest

Updates a request using the entered parameters.

Syntax

/cheminv/api/UpdateRequest.asp?RequestID=adNumeric&USerID=adVarChar&QtyRequired=Int&LocationID=adNUmeric[DateRequested=Date&RequestComments= adVarChar]

Inputs 

RequestID

The ID for the request being updated.

UserID

The ID of the user who is updating the request.

QtyRequired

The required quantity being requested.

Location ID

The ID for the location where the request should be delivered.

[DateRequested]

The date on which the request should be delivered .

[RequestComments]

Comments entered about the request.

Outputs

None

Example

A user wants to update a request (ID=121) to deliver the substance to location ID 1123.  

/cheminv/api/UpdateRequest.asp?RequestID=121&USerID=mjs&QtyRequired=10&LocationID=1123
