CopyContainer

Makes the specified number of copies of the entered container (creating new containers for each copy).  

Syntax

/cheminv/api/CopyContainer.asp?ContainerID=adNumeric&NumContainers=Int&Action= adVarChar

Inputs

ContainerID

The ID of the conatiner (the parent container) to be split or sampled. 

NumContainers

The number of  conatiners the parent container is split or sampled into.

Action = split|sample

This is the action taking place to necessitate a Copy Container.  The only two options are split and sample. When a container is split, the container contents if dispersed among new containers, the original conatiner being removed.  On the other hand, when a container is sampled some quantity of the entire quantity of the container is moved into new containers, the original container left in tact, with less contents. 

Outputs

Pipe (|) delimited list of new container IDs. For example: 100|101

Example

A conatiner (ID=111) is sampled (therefore creating multiple copies of itself), into 4 new containers.

/cheminv/api/CopyContainer.asp?ContainerID=111&NumContainers=4&Action= sample
