GetKeyContainerAttributes

Lists Key container attributes for the given containerID or Container Barcode.

Syntax

 /cheminv/api/GetKeyContainerAttributes.asp?ContainerID= adVarChar&ContainerBarcode= adVarChar

Inputs

ContainerID

The Container ID for the container you are requesting information for.  If this is not provided, ContainerBarcode is required.      

ContainerBarcode

The Container Barcode for the container you are requesting information for.  If this is not provided, ContainerID is required.         

Outputs

ContainerID
Barcode
CurrentUser
UOM
QtyRemaining
Path

Example

 A user requests to view information for the container with the following ID: C111

 /cheminv/api/GetKeyContainerAttributes.asp?ContainerID= C111

OR, a user requests to view information for the container with the following Barcode: C8BGT17

 /cheminv/api/GetKeyContainerAttributes.asp?ContainerBarcode= C8BGT17
