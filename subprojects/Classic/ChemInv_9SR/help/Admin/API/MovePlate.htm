MovePlate

Moves a plate to a new location.

Syntax

 /cheminv/api/MovePlate.asp?LocationID= adNumeric&PlateID= adVarChar

Inputs

LocationID

Location ID for the destination location.

PlateID

PlateID for the plate that is being moved.         

Outputs

Destination Location ID

Example

A user wants to move a plate with ID P1234 to the location with ID 111. 

 /cheminv/api/MovePlate.asp?LocationID= 111&PlateID= aP1234
