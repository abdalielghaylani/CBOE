UpdatePlateType

Updates a plate type using the entered parameters.

Syntax

/cheminv/api/UpdatePlateType.asp?PlateTypeID=adNumeric&PlateTypeName=adVarChar&MaxFreezeThaw= adNumeric

Inputs 

PlateTypeID

The ID for the plate type being updated.

PlateTypeName

The Name of the plate type being updated.

MaxFreezeThaw

The maximum number of freeze thaw cycles allowed for this plate type.

Outputs

None

Example

A user wants to update the plate type (ID=4) to use the allow 5 freeze thaw cycles instead of 3.  

/cheminv/api/UpdatePlateType.asp?PlateTypeID=4&PlateTypeName=NewPlateType&MaxFreezeThaw=5
