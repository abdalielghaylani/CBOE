UpdatePlateFormat

Updates a plate format using the entered parameters.

Syntax

/cheminv/api/UpdatePlateFormat.asp?PlateFormatID=adNumeric&PlateFormatName=adVarChar&PhysPlateID= adNumeric

Inputs 

PlateFormatID

The ID for the plate format being updated.

PlateFormatName

The Name of the plate format being updated.

PhysPlateID

The ID for the physical plate type used to define the plate format.

Outputs

None

Example

A user wants to update the plate format (ID=21) to use the physical plate ID 5 instead of 3.  

/cheminv/api/UpdatePlateFormat.asp?PlateFormatID=21&PlateFormatName=NewPLateFormat&PhysPlateID=5
