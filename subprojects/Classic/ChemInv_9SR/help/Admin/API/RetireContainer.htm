RetireContainer

Moves a container to a specified location, changes its status to the specified value and sets its amount remaining to zero.   Typically used to move containers to the default retired location, set them to the default retired status and zero their amount remaining.

Syntax

/cheminv/api/RetireContainer.asp?ContainerID=adNumeric&LocationID=adDouble&ContainerStatusID=adNumeric&QtyRemaining=adDouble

Inputs

ConatinerID

ContainerID of container to be moved.

LocationID

The Location_ID value of the destination location where the container should be moved to.

ContainerStatusID

Status of container as a Status_ID value.

[QtyRemaining]

The amount of substance remaining in the container to be retired. If absent, the remaining amount will be set to zero.

Outputs

None

Example

A user wants to retire a container (ID=12345)  to the trash bin (ID=10) and give it a retired status (ID=7).  The containe ris currently empty.

/cheminv/api/RetireContainer.asp?ContainerID=12345&LocationID=10&ContainerStatusID=7
