UpdateSynonym

Assigns a synonym which is already created to a given substance

Syntax

/cheminv/api/UpdateSynonym.asp?SynonymID=adNumeric&SubstanceName=adVarChar(255)

Inputs

SynonymID

Synonym_ID value assigned during the creating of a synonym .

SubstanceName

The name of the substance.
