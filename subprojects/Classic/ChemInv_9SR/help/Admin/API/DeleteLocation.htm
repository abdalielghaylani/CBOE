DeleteLocation

Deletes a single location. Returns the Location_ID of the parent of the deleted location. The location must be empty before it can be deleted. Use DeleteContainers to ensure the location to be deleted is empty.

Syntax

/cheminv/api/DeleteLocation.asp?LocationID=adNumeric&Recursively=Int

Inputs

LocationID

Location_ID value for the location to be deleted.

Recursively = 0|1

Indicates if you would like to perform the action recursively (1) or not (0)..

Outputs

None

Example

A user wants to delete a location with ID=101 and all of the locations it contains.

/cheminv/api/DeleteLocation.asp?LocationID=101&Recursively=1
