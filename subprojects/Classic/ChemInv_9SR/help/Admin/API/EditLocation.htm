EditLocation

[Deprecated function. Do Not Use!]

Makes changes to the attributes of a location specified by the given LocationID.  Returns the LocationID.  Parameters not entered will default to NULL.

Syntax

/cheminv/api/EditLocation.asp?LocationID=adNumeric&LocationName=adVarChar[&Barcode=adVarChar&LocationDesc=adVarChar&LocationTypeID=adNumeric]

Inputs

LocationID

The Location_ID value of the location to be edited.

LocationName

The name of the location.

[Barcode]

The barcode associated with the location to be edited.

[LocationDesc]

A description of the location to be edited.

[LocationTypeID]

A Type_ID value describing the type of location to be edited.

Outputs
