UpdateGridFormat

Updates the grid format (also known as storage format) with the passed parameters.

Syntax

 /cheminv/api/UpdateGridFormat.asp?grid_format_id=adNumeric&Name=adVarChar&row_count=adNumeric&col-count=adNumeric[&row_prefix=adVarChar&col_prefix=adVarChar&row_use_letters=adNumeric&col_use_letters=adNumeric&name_separator=adVarChar&number_start_corner=adNUmeric&number_direction=adNUmeric&description=adVarChar&grid_format_type= adNUmeric]

Inputs

grid_format_id 

The ID for the grid format being updated.

Name

The name of the grid format.

Row_Count

The number of rows in the grid.

Col_Count

The number of columns in the grid.

[row_prefix]

A prefix added to each row for identification purposes.

[col_prefix]

A prefix added to each column for identification purposes.

[row_use_letters]

Indicte if consecutive letters should be used to identify rows instead of numbers.

[col_use_letters]

Indicte if consecutive letters should be used to identify columns instead of numbers.

[name_separator]

.

[number_start_corner]

.

[number_direction]

.

[description]

.

[grid_format_type]

.

Outputs

None

Example

A user wants to update the grid format with name OldGrid (ID=14) to have 8 columns and 12 rows and place a C prefix on all columns.  

/cheminv/api/UpdateGridFormat.asp?grid_format_id=14&Name=OldGrid_Invert&row_count=12&col_count=8[&col_prefix=C]
