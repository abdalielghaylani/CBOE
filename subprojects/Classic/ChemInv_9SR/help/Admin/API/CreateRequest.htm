CreateRequest

Creates a new request.  

Syntax

/cheminv/api/CreateRequest.asp?ContainerID=adNumeric&UserID=adVarChar&QtyRequired=Int&LocationID=adNumeric&DateRequired=Date[&RequestComments= adVarChar]

Inputs

ContainerID

The ID for the container which is being requested. 

UserID

The ID for the user who is making the request.   

QtyRequired

The amount og substance needed by the user.  

LocationID

The ID for the location where the substance should be delivered.  

DateRequired

The date for when the substance is required at the delivery location as date-time literal (ie. 01-27-2001).    

RequestComments

Comments entered for the request.  

Outputs

None

Example

A user (ID=mjs) enters a request for 10 mL of the substance in container ID 111 to be delivered on or before November 13 2003 at location ID 12345. 

/cheminv/api/CreateRequest.asp?ContainerID=111&UserID=mjs&QtyRequired=10&LocationID=12345&DateRequired= 11-13-2003
