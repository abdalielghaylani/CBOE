CreateSynonym

Creates a new synonym

Syntax

/cheminv/api/CreateSynonym.asp?CompoundID=adNumeric&SubstanceName=adVarChar(255)

Inputs

CompoundID

Compound_ID value associating the substance in the container with a preexisting inventory compund .

SubstanceName

The name of the substance.

Outputs

None

Example

A user creates a synonym for a compound with ID 23424.  The synonym is Carbolic Acid.

/cheminv/api/CreateSynonym.asp?CompoundID=23424&SubstanceName="Carbolic Acid"
