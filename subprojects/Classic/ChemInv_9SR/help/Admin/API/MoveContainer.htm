MoveContainer

Moves a single container to a new location.

Syntax

/cheminv/api/MoveContainer.asp?LocationID=adNumeric&ContainerID=adNumeric

Inputs

LocationID

The Location_ID value of the destination location where the container should be moved to.

ConatinerID

ContainerID of container to be moved.

Outputs

LocationID

The new Location ID.

Example

A user wants to move a container with ID 13423 to the location with ID 111.

/cheminv/api/MoveContainer.asp?LocationID=111&ContainerID=13432
