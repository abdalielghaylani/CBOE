DeleteSynonym

Deletes a given synonym.

Syntax

/cheminv/api/DeleteSynonym.asp?SynonymID=adNumeric

Inputs

SynonymID

Synonym_ID value for the synonym to be deleted .

Outputs

None

Example

A user wants to delete the synonym with ID 45.

/cheminv/api/DeleteSynonym.asp?SynonymID=45
