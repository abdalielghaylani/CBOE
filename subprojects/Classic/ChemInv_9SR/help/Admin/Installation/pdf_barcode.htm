Enable PDF and Barcode Support

PDF Support

The RPT report writer installed with the Inventory Enterprise is capable of producing reports in four different formats: SNP (MS Access Report Viewer), RTF (Rich Text Format), XLS (Excel Workbook), PDF. Support for the PDF format is optional and requires a PDF printer driver to be installed on the server. Following are the optional instructions required to enable PDF support:

  1. Browse to <webroot>\ChemOffice\ ChemInv\Installation\win2pdf .
  2. Install the win2pdf driver by double-clicking on w2pdf151.exe.
  3. Go to START>Settings> Control Panel>Printers and check that the win2pdf printer is the default Windows Printer.
    1. Right-click on the printer.
    2. Select Set As Default Printer.
    3. Purchase a license and registration code from: http://www.daneprairie.com/
    4. Right click on the win2pdf printer (in START> Control Panel>Printers )and select Properties.
    5. Enter the regcode you purchased.
  4. Browse to <webroot>/ChemOffice/WebServer_source/cfserverasp/RPT
  5. Stop the Reports Service (RPTService).
  6. Open the ChemInv_reports.mdb file (found in your ChemOffice_data/Cheminv folder).
  7. Open the ReportFormats table and check the Available checkbox for the PDF format.
  8. Close Microsoft Access.
  9. Restart the Reports Service.

Barcode Support

  1. Browse to <webroot>\ChemOffice\ ChemInv\Installation\Tbarcode .
  2. Run the Tbarcode.ocx installer.
  3. Perform an IISRESET after finishing the Inventory Enterprise installation.

Step 7: Test the Installation
