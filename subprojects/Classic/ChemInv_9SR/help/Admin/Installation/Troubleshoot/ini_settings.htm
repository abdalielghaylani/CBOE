Check Configuration File Settings

Configuration files are used by Inventory Enterprise to define parameters that allow the Inventory Enterprise application to find necessary information and how to react in certain situations.  If the settings in these files are not correctly recorded, the application may not be able to operate.

The configuration files for Inventory Enterprise are found in:
<webroot>/ChemOffice/cheminv/config

The installation instructions for Inventory Enterprise indicate the need to set a number of ini settings to the correct value.  See Required Configuration for more information.

If you are having difficulties, check to make sure the following settings are correct:

  • cfserver.ini
    • DBA_PWD
    • DBA_USERNAME
    • ORA_SERVICENAME
