Setting up PDF Support

The RPT report writer installed with the Chemical Inventory is capable of producing reports in four different formats: SNP (MS Access Report Viewer) , RTF (Rich Text Format), XLS (Excel Workbook), PDF (Adobe Protable Document).  Support for the PDF format is optional and requires a PDF printer dirver to be installed on the server.  Following are the optional instructions required to enable PDF support:

  1. Browse to ChemInv\Installation\win2pdf.
  2. Install the win2pdf driver.
  3. Go to the Control Panel>Printers and check that the win2pdf printer is the default Windows Printer.
  4. Purchase a license and registration code from:  http://www.daneprairie.com/
  5. Right click on the win2pdf printer and select properties. Enter the regcode.
  6. If the report writer is running double click on its icon in the taskbar tray and click stop.
  7. Open the ChemInv_reports.mdb file
  8. Open the ReportFormats table and check the Available checkbox for the PDF format.
  9. Close Access
  10. Restart the report writer
