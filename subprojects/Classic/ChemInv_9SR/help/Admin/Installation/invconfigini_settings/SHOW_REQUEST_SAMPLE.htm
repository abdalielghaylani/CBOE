SHOW_REQUEST_SAMPLE

Example: SHOW_REQUEST_SAMPLE = 1

Description: Indicates if the Request Sample feature should be turned on. The Request Sample feature allows you to request part of a container or batch of containers as opposed to the Request Container feature in which you request the whole container.

In addition to the Request Sample feature, this setting also enables the Manage Sample Requests and Manage Orders features, if set to 1.

Default:

Options:0 | 1
