ALT_IDS

Example: ALT_IDS = inv_compounds.ALT_ID_1:Substance ID 1;1-U,inv_compounds.ALT_ID_2:Substance ID 2;0

Description: A comma delimted list of the alternate compound IDs. Alt IDs are in the following format:
<Field_Name>:<Field_Label>;<0|1>
where Field_Name is the name of the alt ID field (as shown in the INI file), Field Label is how the alt id should show up in the interface, and the last parameter indicates if the field is required (1) or not (0). Add a -U to the end of an alt ID's definition to make the ID a unique identifier.

Default: inv_compounds.ALT_ID_1:Substance ID 1;1-U,inv_compounds.ALT_ID_2:Substance ID 2;0,inv_compounds.ALT_ID_3:Substance ID 3;0,inv_compounds.ALT_ID_4:Substance ID 4;0,inv_compounds.ALT_ID_5:Substance ID 5;0

Options: A comma delimited list in the above format.

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
