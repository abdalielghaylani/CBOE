USE_CUSTOM_BROWSEBANNER_FRAME_LINKS

Example: USE_CUSTOM_BROWSEBANNER_FRAME_LINKS = "False"

Description: Indicates if the application should look for a custom file to define the links found in the Browse Banner Frame (top frame in browse mode). True indicates a custom file is used, False indicates the standard should be used.

Default: False

Options: True | False

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice Enterprise application files to future versions of the application.  New versions of the ChemOffice Enterprise use default configurations.
