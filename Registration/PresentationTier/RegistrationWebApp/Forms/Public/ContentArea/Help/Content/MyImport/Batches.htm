Batches

A batch is a set of quantities and properties of the same substance. In Registration Enterprise 11.0 a batch is automatically created for any record registered. You can further add batches and manage them as desired.

Adding a batch

A batch is created when you register a record to the permanent table. You can add more than one batches to a registered record. You can view any batch information by selecting the specific batch in the tree structure on the left hand side of the page. Refer figure below:

 

To add a batch:
1. Click View Record link corresponding to the record. The Registry Record page appears.
2. Click Add Batch. A new batch appears in the tree structure on the left hand side of the page.
3. Fill in the Batch Information by selecting a value in the Project field. If you need to add more than one project, click Add Project.
4. Fill in the Fragments by component information by selecting a value in Fragment Types field and specifying values in code and Equivalent fields. To add more than one fragment type, click New Fragment.
5. Click Submit. The Registry Record page appears.
You can edit, move, or delete a batch.

Editing a batch

You can modify batch information and save the changes. You can select a batch by selecting the corresponding batch in the tree structure.
To edit a batch:
1. Click Edit. The Registry Record page appears.
2. Make the required changes in the Batch information section and Fragments by component section.
3. Click Submit.

Deleting a batch

You can not only edit a batch but also delete a batch if it is not required.
To delete a batch:
1. Click Delete Batch. The Move/Delete Batch page appears.
2. Select Delete in the Select Action option.
3. Specify the Batch ID.
4. Click Delete Batch.

Moving a batch

You can move a batch from one registry record to another using the Move Batch option.
To move a batch:
1. Click Move batch. The Move/Delete Batch page appears.
2. Select Move in the Select Action option.
3. Specify the Batch ID and the Registry Number of the registered record to which the batch is to be moved.
4. Click Move Batch.
