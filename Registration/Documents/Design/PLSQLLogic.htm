Perform a duplicate checking before insert

            If no duplicate found

                    If insert fails

                            Insert into temporary table

            If single duplicate found

                    Based on the duplicate action parameter, do following

                    If "add_batch"

                            Add a batch to the registered compound

                    If "override"

                            Insert into permanent tables anyway based on the Insert Logic

                    If insert fails

                                    Insert into temporary table

                    If "user_input"

                             Insert into temporary table

                             Mark record as duplicate in temp table

            If multiple duplicate found

                     Insert into temporary table

    end if
