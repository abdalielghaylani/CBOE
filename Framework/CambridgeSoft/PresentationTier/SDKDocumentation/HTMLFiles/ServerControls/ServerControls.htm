COEFramework Server Controls

 

The COEFramework provides a rich set of server controls that can be used in your application. The various controls provided by COEFramework are:

 

  • ChemDrawEmbeded
  • DataMapper
  • DataViewManager
  • COEFormGenerator
  • FormManager
  • TableEditorControl
  • COEDatabasePublishManager

 

To use the COEFramework server controls in your application:

 

1.             Create a Visual Studio Web Application

2.             Add a reference, CambridgeSoft.COE.Framework.dll, to the application.

3.             Right-click the Toolbox to display a shortcut menu. Select the Choose Items option from the shortcut menu to display the Choose Toolbox Items dialog box as shown below:

 

 

 

1.             Specify the location of the CambridgeSoft.COE.Framework.dll file using the Browse button and click the OK button to add the COEFramework server controls to the Toolbox.
