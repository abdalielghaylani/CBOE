COEDatabasePublishingService

 

The primary role of the COEDatabasePublishingService is to design the schema of the databases in the predefined XML format (DataView), which is used by the various services, such as Search service. It also enables you to generate the alternative DataView by editing the default DataView generated using the COEDatabasePublishingService. The following tasks can be performed using the COEDatabasePublishingService:    

 

§         GetDatabase: Gets the list of databases used by a service of COE Framework.

§         Get UserList: Gets the list of users of the specified database.

§         PublishDatabase: Publishes the specified database into the predefined XML format and save it into the COEDataView table. It also assigns permissions for accessing the database to the specified list of users.
