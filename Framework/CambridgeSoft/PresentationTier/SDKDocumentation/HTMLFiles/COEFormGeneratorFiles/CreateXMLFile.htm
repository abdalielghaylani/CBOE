Creating XML File

 

In the XML file, you need to define the two tags, FormDisplay and FormElements tags. The display information of the form is provided under the FormDisplay tag and the list of controls is provided under the FormElements tag. The FormDisplay tag contains the tags, such as height, width, and style, to define the various attributes like height, width, and style, of the form. The FormElements tag contains the FormElement tags to define the different controls of the form. The FormElement tag contains the following sub-tags to define the various properties of the form element or the control:

 

  • defaultvalue: Contains the default value of the control.
  • bindingExpression: Contains the binding expression for the control.
  • configInfo: Contains the configuration information of the control.
  • DisplayInfo: Contains the display information of the control. The  displayInfo tag further contains the following tags:

·               height: Contains the height of the control.

·               width: Contains the width of the control.

·               top: Contains the Y axis location of the control.

·               left: Contains the X axis location of the control.

·               assembly: Contains the name of the assembly that implements the control.

·               type: Contains the name of the class that implements the control.
