EXTRA_FRAGS_OK

Example: EXTRA_FRAGS_OK= "0"

Description: Internal setting determining if duplicate checking should take permit extraneous fragments into account. 
        Note: This is not a preferences setting.  This ini settings determines how duplicate checking works.  

Default: 0   

Options: 0 | 1

It is important to note that it is the responsibility of a site's administrator to propagate any changes made to ChemOffice WebServer application files to future versions of the application.  New versions of the ChemOffice WebServer use default configurations.
