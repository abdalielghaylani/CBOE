Accessing MSDS Data

You can view the Material Safety Data (MSDS) for a chemical by clicking on its corresponding MSDS link. The MSDS link is available in the List view and Details view, as shown below:

Note: If MSDS data is not available for a chemical record, the record's MSDS link is disabled.

To view the MSDS data for a chemical:

  1. Click MSDS in the chemical record in either List or Details view. A page listing all the data sheets available for the chemical appears:

Note: If only one data sheet is available for the chemical, the data sheet appears directly and the list of the data sheet is not displayed.

  1. Click on "View MSDS" button corresponding to the required data sheet. The data sheet appears as shown below:
