Registering a Temporary Record

Registration assigns a unique number to each record that is registered. When a batch is registered, the batch number is added to the registration number previously assigned to the compound. In addition to registering each record individually, Registration Enterprise also lets users register all the temporary table records at once.

Records in the permanent table can be accessed only if Row Level Security is not enabled.

Registering a Compound Record

Records may be registered by either selecting Review Register in the Registration home page or by searching for records in the temporary table using Search Temp in the Registration home page. On selecting Review Register, the Search Temporary Record form displaying all records in the temporary table appears.

You can register one record at a time or several records using bulk registration.

To register a single record:

  1. Log in to ChemBioOffice Enterprise with Registrar privileges.
  2. Click on Review Register icon in the Registration home page, the list of records in the temporary database displays
  3. Click the Review Record link corresponding to the record. The temporary record page that displays the record information appears.

  1. To Register the record click on Register button, the record is registered and the Registry Number appears as shown below:

  1. Click Done.

To perform a Bulk Registration:

  1. Mark the records to register
  2. Click Register Marked. The Bulk register marked page appears.

  1. Choose the duplicate action to perform
  • Duplicate: to register as a new compound any duplicate record it encounters.
  • Temporary: to register all records except duplicates. The duplicate records remain in the temporary table.
  • Batch: to add new batches for the duplicate compounds.
  • None: to leave the record in temporary table.

  1. Click Register Marked button.
