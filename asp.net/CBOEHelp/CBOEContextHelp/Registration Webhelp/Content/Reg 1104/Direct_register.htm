Direct register

If a user has registrar privileges, he/she can use this option to bypass the temporary table and submit a record directly to the permanent table.

To register directly to the permanent table:

  1. Fill in the required/optional fields in the Submit New Record form.
  2. Click on Register button. All unique records are registered and Registry number is generated. If duplicate records exists, the duplicate checking page appears. For information about duplicate resolution please refer to the "Managing Duplicates" topic.
  3. The record is registered and the Registry number is generated.
  4. Click Done.

Note: If ChemDraw Chemical Warning Checker is enabled and ChemDraw chemical warning is present, you will not be allowed to Register the component until the underlying error is resolved. For more information on ChemDraw chemical warning, See "ChemDraw Chemical Warning".
