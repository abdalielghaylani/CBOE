Creating an Export Template

To create a template:

  1. Go to File>Export Hits. The Export Hits window appears.
  2. In the drop-down list, select Create New Template. A new window appears.
  3. In the new window, enter the Template Name and a Template Description.
  4. To enable others to view the query, select Public.
  5. Select the fields to export when using the template.
  6. Click OK.
