Options in Submit Component Form

Once the Submit Component form is filled in, submitter can either submit the record to the temporary table or save the form for later use.

If the user has adequate privileges he/she can also approve and register the record directly to the permanent table.

The options available in the Submit Component form include:

  • Save Template
  • Load Template
  • Submit
  • Cancel
  • Clear

Submit

Use this option to submit records to the temporary table. For more information on submitting records, see "Creating a registry record".

Cancel

Use this option to exit from the Submit Component form and go to the main menu of the Registration Enterprise.

Clear

Use this option to delete the content of the Submit Component form and re-enter new values.
