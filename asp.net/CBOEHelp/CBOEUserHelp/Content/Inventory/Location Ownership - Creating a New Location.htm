Location Ownership - Creating a New Location

To create a new location and specify the ownership:

  1. Log in to COE Manager as INVADMIN or any other user
  2. Click on Browse link to go to Inventory application

  1. In the left pane go to Location>>New

  1. Fill in the required and Optional fields (if desired) and then in the Location admin field select an appropriate user/group based on the radio button by user/by group you select

  1. When completed, click on Ok button the new location is created

Note: It is not possible to edit a location or to add a sub-location to a location when you are not an owner. The Field will appear in read only mode as shown in the screen shot below.
