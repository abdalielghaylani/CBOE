Retrieve Data from Inventory System

The user is allowed to pull data from Inventory system and to use the structures in the Reaction section.

To retrieve data from Inventory:

  1. Log in to E-Notebook
  2. Create an experiment and then add Reaction section
  3. In the Reaction toolbar, click on the Add button, the following dialog appears

  1. Select an appropriate datasource: “Inventory Search” from the ‘Search in’ drop down list
  2. The search form consists of a ChemDraw control to render structures for exact and substructure searching

  1. A set of text boxes are available for entering textual search data. The textboxes are as follows:
  • Substance Name
  • CAS Registry Number
  • ACX ID
  • Container Barcode
  • Molecular Formula
  • MolWeight Range
  1. Check the Exclude Special Locations check box, to avoid fetching compounds from Inventory special locations . Special locations are configured by your system administrator in the Cows Web.config file.
  2. Enter Search criteria and click on the search button to execute the search

  1. A clear button is available for clearing the form
