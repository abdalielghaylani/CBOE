Query Management tab

The query management tab has four options for better query management:
Current
Saved
History
Advanced

Current

This option lists the last generated query. On clicking the list, the records matching the query criteria is retrieved and displayed.

Saved

This option lists all saved queries. Click on the list to retrieve matching records.

History

This option list all queries generated including the saved queries.

Advanced

You can use this option to select a hitlist and edit or restore it.
Editing a hitlist
To edit a hitlist:
1. Select the required hitlist and click Edit.
2. Specify a name and a description.
3. Select ‘Is Public’ if you want the information to be available to others also.
4. Click Edit.
Restoring a hitlist
You can integrate the results of a saved query with the results of the current query. Thus you can perform boolean operations on two different searches.
To restore:
1. Select the required hitlist and click Restore
2. Select any one of the restore type described below and click Restore:
Replace current list-discards the current hit list and displays the records from the hit list you chose to open.
Intersect with current list-displays only those records that appear in both the current hit list and in the saved hit list. This is a Boolean AND operation.
Subtract from current list-displays only those records that are in the saved hit list but are not in the current hit list.
Union with current list-displays all records in either the current list or the saved hit list. This is a Boolean OR operation.
