Dataviews

Dataviews are defined via the Dataview Manager within the ChemBioOffice Enterprise framework. A dataview is a defined set of database tables that are made available for Datalytix users for searching purpose. Dataviews define relationships between tables and fields that are visible to the user. Once a dataview is configured it can be used over and over again without intervention, even as new data is added to the tables being referenced.
From the Datalytix interface, you can access a single dataview with multiple tables. Advanced searches can also be performed on the dataviews using Table Names or Tag Names.
