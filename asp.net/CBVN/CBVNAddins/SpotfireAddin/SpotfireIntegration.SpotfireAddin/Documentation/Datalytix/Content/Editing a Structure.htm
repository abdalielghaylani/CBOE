Editing a Structure

To edit a structure:
 
1. Click on the Edit button in the toolbar to open the Structure Editor. If applicable, select the editor to edit the current structure. Alternatively, you can also double click in the Structure pane to open the editor for the current structure.
 
2. Edit the query structure in the Structure Editor window using appropriate tools.
 
3. Click OK to replace the current structure in the Structure pane with the new structure.
