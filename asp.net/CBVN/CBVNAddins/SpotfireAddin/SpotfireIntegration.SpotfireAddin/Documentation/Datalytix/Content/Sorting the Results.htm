Sorting the Results

The query results can be sorted by any field in an ascending or descending order. Sorting can be performed on a single column or more than one column. Sorting is configured as part of the Result Definition and is reflected when then the tables and fields are imported and displayed in Spotfire tables.
 
To sort results:
1. Right mouse click on a field on the Result Definition form.
 

 
2. Select the Sort Order sub-menu item. Select to sort on this field in ascending or descending order. By default, None is selected.
 

 
A resulting sort arrow is displayed in the Sort column for the field. In the following example, the Formula field will be sorted in descending order.
 

 
Alternatively, you can toggle the Sort order between None, Ascending and Descending by clicking directly on the icon in the Sort column for the selected field.
