Get Data

 

The GetData() method of the COESearch class of COESearchService retrieves a page of data either from a previously saved existing hitlist or from the base table of the current dataview.   The data is returned as a Dataset object containing tables as defined by the input ResultsCritria. 

 

 

The GetData() method contains the following parameters:

 

  • resultsCriteria: Is the object of the SearchCriteria class.
  • pagingInfo: Is the Object of the PagingInfo class.
  • dataView: Is the object for the COEDataView class.

 

You require the Results Criteria, Paging Info, and Data View XML files that need to be converted to the corresponding objects. For example, the XML file for the Results Criteria needs to be converted to the object of the ResultsCriteria class.

 

The presence or absence of a hitlistID value in the PagingInfo XML file determines the behavior of the GetData process. The sample PagingInfo xml file is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<PagingInfo xmlns="COE.PagingInfo">

  <PagingInfoID>0</PagingInfoID>

  <HitListID>0</HitListID>

  <KeepAlive>NONE</KeepAlive>

  <RecordCount>10</RecordCount>

  <Start>0</Start>

  <End>0</End>

</PagingInfo>

 

You can specify the a desired value to the HitListID in the PagingInfo xml file to get the data from an existing Hitlist having the specified HitlistID. If you want to browse the base table of the dataView, you need to set the value,0, to the HitListID in the PagingInfo xml file.

 

The following code snippet shows how to use the GetData() method of the COESearchService:

 

class GetDataTest

{

        COESearch _coeSearch = new COESearch();

        COEDataView dataView = new COEDataView();

        ResultsCriteria resultsCriteria = new ResultsCriteria();

        PagingInfo pagingInfo = new PagingInfo();

 

        // Temp Path to xml file

        string tempPath = Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData) + @"\" + Application.ProductName + @"\";

 

        public void TestGetData ()

        {

           

 

            // Build DataView

            dataView = new COEDataView();

            BuildDataViewFromXML(ref dataView);

 

            // Build ResultsCriteria

            ResultsCriteria resultsCriteria = new ResultsCriteria();

            BuildResultsCriteriaFromXML(ref resultsCriteria);

 

            //PagingInfo

            pagingInfo = new PagingInfo();

            BuildPagingInfoFromXML(ref pagingInfo);

 

            try

            {

                DataSet dataset = null;

                dataset = _coeSearch.GetData(resultsCriteria, pagingInfo, dataView);

            }

            catch (Exception ex)

            {

                HandleException(ex);

            } 

        }

 

 

  private void BuildDataViewFromXML(ref COEDataView dataView)

        {

            XmlDocument doc = new XmlDocument();

            doc.Load(tempPath + “DataView.xml”);

            dataView = new COEDataView(doc);

           

        }

 

 

private void BuildResultsCriteriaFromXML(ref ResultsCriteria resultsCriteria)

        {

            XmlDocument doc = new XmlDocument();

            doc.Load(tempPath + “ResultsCriteria.xml”);

            resultsCriteria = new ResultsCriteria(doc);

        }

 

 

        private void BuildPagingInfoFromXML(ref PagingInfo pagingInfo)

        {

            XmlDocument doc = new XmlDocument();

            doc.Load(tempPath + “PagingInfo.xml”);

            pagingInfo = new PagingInfo(doc);

        }

}

 

The above code snippet defines GetHitList class that contains the following methods to use the GetData() method of the Search Service:

 

  • TestGetData: This method calls the GetData() method of the COESearch class of the COESearchService, which returns a dataset containing the desired data.
  • BuildDataViewFromXML: This method converts the DataView XML into the DataVeiw object.
  • BuildResultsCriteriaFromXML: This method converts the SearchCriteria XML into the ResultsCriteria object.
  • BuildPagingInfoFromXML: This method converts the PagingInfo XML into the PagingInfo object.
